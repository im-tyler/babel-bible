## Intuition [Beginner]

A canonical transformation is a change of phase-space coordinates that preserves Hamilton's equations. But how do you actually *specify* one? You could write down the transformation equations $(Q, P) = \Phi(q, p)$ directly, but then you have to check that the result is canonical — a messy condition involving partial derivatives.

There is a better way. Every canonical transformation can be encoded in a single scalar function, called a **generating function**. Instead of specifying the $2n$ transformation equations directly, you specify one function of $2n$ variables (half old coordinates, half new coordinates), and the transformation equations follow by taking partial derivatives. The generating function is a "potential" for the canonical transformation, in the same way that the electric potential encodes the electric field: differentiate in the right directions and the transformation pops out.

There are four standard choices for which half of the old and new variables serve as the independent arguments of the generating function. These give the four types, conventionally labelled $F_1, F_2, F_3, F_4$. The most commonly used is $F_2(q, P, t)$ — old positions and new momenta — because it connects directly to the Hamilton-Jacobi equation. But all four are related by Legendre transforms of one another, and the choice of which to use is a matter of convenience.

The key insight is that a generating function does not just *describe* a canonical transformation — it *generates* one. Different choices of generating function produce different canonical transformations. This is what makes them powerful: instead of searching for good coordinates by trial and error, you search for a good generating function, which is a scalar optimisation problem.

## Visual [Beginner]

Imagine the graph of a canonical transformation in a combined $(q, p, Q, P)$ space. The graph is a $2n$-dimensional surface sitting inside a $4n$-dimensional space. The generating function is a way of describing this surface as the graph of a function, by choosing which $2n$ variables are "inputs" and which $2n$ are "outputs."

For the type-2 generating function $F_2(q, P)$, the inputs are $(q, P)$ — old positions and new momenta — and the outputs are $(p, Q)$ — old momenta and new positions. The transformation equations $p_i = d F_2/d q^i$ and $Q^i = d F_2/d P_i$ say: the old momentum is the slope of $F_2$ in the old-position direction, and the new position is the slope of $F_2$ in the new-momentum direction.

![Schematic of the four generating-function types as different "slices" of the canonical-transformation graph. Each type picks a different pair of old/new variables as independent inputs, with the remaining pair derived by differentiation. The type-2 box (q, P) -> (p, Q) is highlighted as the most common choice.](/img/09.05.03-generating-functions-four-types-placeholder.svg)

The four types correspond to the four ways of choosing $2n$ independent variables from the set $\{q, p, Q, P\}$ such that one comes from the old set and one from the new set for each degree of freedom. A transformation that is regular with respect to one choice may be singular with respect to another, so having all four types ensures you can always find one that works.

## Worked example [Beginner]

The simplest interesting canonical transformation is the **exchange transformation** that swaps position and momentum (up to signs): $Q = p$, $P = -q$. This is canonical because it preserves the Poisson brackets: $\{Q, P\} = \{p, -q\} = 1$.

To construct it as a generating function, use the type-1 form $F_1(q, Q)$ with the relations $p = d F_1/d q$ and $P = -d F_1/d Q$. The exchange transformation requires $p = d F_1/d q = Q$ (since $Q = p$) and $P = -d F_1/d Q = -q$. The first equation gives $F_1 = qQ + f(Q)$ for some function $f$. The second gives $d F_1/d Q = q + f'(Q) = q$, so $f'(Q) = 0$, i.e., $f$ is constant. Dropping the constant, $F_1(q, Q) = qQ$.

Now try the type-2 form. The exchange transformation has $P = -q$, so $q = -P$, but $F_2$ requires $q$ and $P$ as independent variables. Since $P = -q$, these are not independent — the type-2 generating function does not exist for the exchange transformation. This is why we need all four types: the exchange transformation is singular in the type-2 parameterisation but regular in type-1.

The second example: the **identity transformation** $Q = q$, $P = p$ is generated by $F_2(q, P) = q \cdot P$. Check: $p_i = d F_2/d q^i = P_i$ and $Q^i = d F_2/d P_i = q^i$. So $F_2 = q \cdot P$ generates the identity. A generating function close to the identity, $F_2 = q \cdot P + \epsilon S(q, P)$, generates a canonical transformation close to the identity — a perturbation. This is the starting point for canonical perturbation theory.

## Formal definition [Intermediate+]

### The symplectic condition

A transformation $(q, p) \mapsto (Q(q, p), P(q, p))$ on phase space is **canonical** if and only if the Jacobian matrix $M$ with entries $M_{ij} = \partial(Q^1, \ldots, Q^n, P_1, \ldots, P_n)/\partial(q^1, \ldots, q^n, p_1, \ldots, p_n)$ satisfies the **symplectic condition**:

$$
M^T J M = J, \qquad J = \begin{pmatrix} 0 & I_n \\ -I_n & 0 \end{pmatrix}.
$$

This is equivalent to the preservation of the Poisson brackets: $\{Q^i, Q^j\} = 0$, $\{P_i, P_j\} = 0$, $\{Q^i, P_j\} = \delta^i_j$.

### The generating-function principle

The generating-function approach starts from the modified Hamilton principle. A curve $(q(t), p(t))$ satisfies Hamilton's equations if and only if the action integral $\int(p_i\,dq^i - H\,dt)$ is stationary. After a canonical transformation to $(Q, P)$, the action must be $\int(P_i\,dQ^i - K\,dt)$, also stationary. The two integrals differ at most by a total differential $dF$ of some function $F$, since adding $dF$ to the integrand does not affect stationarity:

$$
p_i\,dq^i - H\,dt = P_i\,dQ^i - K\,dt + dF.
$$

The function $F$ is the generating function. Its natural arguments depend on which variables are chosen as independent.

### Type 1: $F_1(q, Q, t)$

Choose $(q, Q)$ as independent variables. Expanding $dF_1 = (\partial F_1/\partial q^i)\,dq^i + (\partial F_1/\partial Q^i)\,dQ^i + (\partial F_1/\partial t)\,dt$ and substituting:

$$
p_i = \frac{\partial F_1}{\partial q^i}, \qquad P_i = -\frac{\partial F_1}{\partial Q^i}, \qquad K = H + \frac{\partial F_1}{\partial t}.
$$

The first two equations define the canonical transformation; the third relates the old and new Hamiltonians. For a time-independent transformation, $K = H$. The condition for $F_1$ to define a valid transformation is that the mixed Hessian $\det(\partial^2 F_1/\partial q^i \partial Q^j) \neq 0$, ensuring the equations can be solved for $(q, p)$ in terms of $(Q, P)$ or vice versa.

### Type 2: $F_2(q, P, t)$

Perform a Legendre transform of $F_1$ in the $Q$-variables: $F_2 = F_1 + P_i Q^i$. Then $dF_2 = dF_1 + P_i\,dQ^i + Q^i\,dP_i = p_i\,dq^i + Q^i\,dP_i + (K - H)\,dt$, giving:

$$
p_i = \frac{\partial F_2}{\partial q^i}, \qquad Q^i = \frac{\partial F_2}{\partial P_i}, \qquad K = H + \frac{\partial F_2}{\partial t}.
$$

Type 2 is the most widely used because $F_2$ depends on old positions and new momenta — the natural variables for the Hamilton-Jacobi equation. The type-2 generating function $S(q, P, t)$ that satisfies $H(q, \partial S/\partial q, t) + \partial S/\partial t = 0$ is the Hamilton-Jacobi function [09.05.02].

### Type 3: $F_3(p, Q, t)$

Legendre transform of $F_1$ in the $q$-variables: $F_3 = F_1 - p_i q^i$. Then:

$$
q^i = -\frac{\partial F_3}{\partial p_i}, \qquad P_i = -\frac{\partial F_3}{\partial Q^i}, \qquad K = H + \frac{\partial F_3}{\partial t}.
$$

Type 3 is useful when the old momenta $p$ (rather than positions $q$) are the natural independent variables, for example in the momentum representation of quantum mechanics.

### Type 4: $F_4(p, P, t)$

Legendre transform of $F_1$ in both $q$ and $Q$: $F_4 = F_1 - p_i q^i + P_i Q^i$. Then:

$$
q^i = -\frac{\partial F_4}{\partial p_i}, \qquad Q^i = \frac{\partial F_4}{\partial P_i}, \qquad K = H + \frac{\partial F_4}{\partial t}.
$$

Type 4 is the least commonly used but is essential for transformations where both positions are dependent variables — for example, the exchange transformation $(Q, P) = (p, -q)$ is naturally expressed in type 4 as $F_4(p, P) = -p \cdot P$, giving $q = P$ and $Q = -P$... wait, let me recompute: $q^i = -\partial(-p_j P_j)/\partial p_i = P_i$ and $Q^i = \partial(-p_j P_j)/\partial P_i = -p_i$. So $Q = -p$ and $q = P$, which differs from the exchange by a sign. The sign convention depends on which form of the exchange is desired.

### Regularity condition

A generating function of a given type exists if and only if the canonical transformation is **regular** with respect to the chosen independent variables. The condition for type 1 is $\det(\partial Q/\partial p) \neq 0$; for type 2, $\det(\partial P/\partial p) \neq 0$; for type 3, $\det(\partial q/\partial Q) \neq 0$; for type 4, $\det(\partial Q/\partial q) \neq 0$. Any canonical transformation that is sufficiently close to the identity satisfies the type-2 regularity condition, which is why type 2 works for perturbation theory.

## Key derivation [Intermediate+]

**Key derivation: The generating function guarantees symplecticity.**

We prove that the transformation defined by a type-2 generating function $F_2(q, P)$ is automatically canonical, i.e., it satisfies the symplectic condition. The argument for the other three types is analogous.

Define the transformation by $p_i = \partial F_2/\partial q^i = f_i(q, P)$ and $Q^i = \partial F_2/\partial P_i = g^i(q, P)$. We must show $\{Q^i, Q^j\} = 0$, $\{P_i, P_j\} = 0$, and $\{Q^i, P_j\} = \delta^i_j$.

The first is automatic: $P_i$ and $P_j$ are the new momenta, which are independent variables in the $(q, P)$ parameterisation, so $\{P_i, P_j\} = 0$.

For $\{Q^i, P_j\}$: using the chain rule in the $(q, P)$ coordinate system,

$$
\{Q^i, P_j\} = \frac{\partial Q^i}{\partial q^k}\frac{\partial P_j}{\partial P_k} - \frac{\partial Q^i}{\partial P_k}\frac{\partial P_j}{\partial q^k} = \frac{\partial g^i}{\partial q^k}\delta^k_j - 0 = \frac{\partial^2 F_2}{\partial P_j \partial q^i}.
$$

By equality of mixed partials, this equals $\partial^2 F_2/\partial q^i \partial P_j = \partial p_i/\partial P_j$... wait, that is not right. Let me redo: $\{Q^i, P_j\} = \partial Q^i/\partial q^k \cdot \delta^k_j = \partial Q^i/\partial q^j = \partial^2 F_2/(\partial q^j \partial P_i)$... actually $\{Q^i, P_j\}$ in the old $(q,p)$ Poisson bracket is

$$
\{Q^i, P_j\}_{q,p} = \frac{\partial Q^i}{\partial q^k}\frac{\partial P_j}{\partial p_k} - \frac{\partial Q^i}{\partial p_k}\frac{\partial P_j}{\partial q^k}.
$$

This requires inverting the map $(q, p) \to (q, P)$, i.e., expressing everything in old coordinates. The cleaner proof uses the differential-form identity directly.

The generating function guarantees $p_i\,dq^i + Q^i\,dP_i = dF_2$ (from the definition of $F_2$). The canonical 1-form $\theta = p_i\,dq^i$ satisfies $d\theta = dp_i \wedge dq^i = \omega_{\rm old}$. Under the transformation, the new canonical form is $\Theta = P_i\,dQ^i$ with $d\Theta = dP_i \wedge dQ^i = \omega_{\rm new}$. From $p_i\,dq^i = dF_2 - Q^i\,dP_i$, taking $d$ of both sides:

$$
dp_i \wedge dq^i = -dQ^i \wedge dP_i = dP_i \wedge dQ^i.
$$

So $\omega_{\rm old} = \omega_{\rm new}$, and the transformation is symplectic. This proof is clean and coordinate-free: the generating function ensures that the symplectic 2-form is preserved because the two canonical 1-forms differ by an exact form ($dF_2$), whose exterior derivative vanishes.

**Bridge.** This derivation builds toward the Hamilton-Jacobi equation [09.05.02], which is the PDE that a type-2 generating function must satisfy in order to trivialise the Hamiltonian; the foundational reason it works is exactly the symplecticity guaranteed here. The generating-function construction appears again in the normal-form method of perturbation theory [09.04.06], where the near-identity transformation that eliminates fast oscillations is specified by its type-2 generating function. Putting these together, the generating function is the central tool for constructing canonical transformations: it reduces the problem of finding a symplectomorphism (a nonlinear map satisfying a matrix condition) to the problem of choosing a scalar function and differentiating it.

## Exercises [Intermediate+]

<aside class="exercise" data-difficulty="easy" data-type="symbolic" markdown="1">

**Exercise 1 (easy, symbolic).**

Verify that $F_2(q, P) = q \cdot P$ generates the identity transformation by computing $p_i = \partial F_2/\partial q^i$ and $Q^i = \partial F_2/\partial P_i$.

<details class="exercise__hint"><summary>Hint</summary>

Differentiate a dot product.

</details>

<details class="exercise__answer"><summary>Answer</summary>

$p_i = \partial(q^k P_k)/\partial q^i = P_i$ and $Q^i = \partial(q^k P_k)/\partial P_i = q^i$. So $Q = q$ and $p = P$: the identity. Adding any function of $q$ alone to $F_2$ changes the transformation by a shift in $P$; adding any function of $P$ alone shifts $Q$.

</details>

</aside>

<aside class="exercise" data-difficulty="medium" data-type="symbolic" markdown="1">

**Exercise 2 (medium, symbolic).**

Find a type-2 generating function for the scaling transformation $Q^i = \lambda q^i$, $P_i = p_i/\lambda$ for constant $\lambda > 0$. Verify that this transformation is canonical by computing the Poisson bracket $\{Q^i, P_j\}$.

<details class="exercise__hint"><summary>Hint</summary>

You need $Q^i = \partial F_2/\partial P_i = \lambda q^i$ and $p_i = \partial F_2/\partial q^i = P_i/\lambda$.

</details>

<details class="exercise__answer"><summary>Answer</summary>

From $p_i = \partial F_2/\partial q^i = P_i/\lambda$, we get $F_2 = q^i P_i/\lambda + f(P)$. From $Q^i = \partial F_2/\partial P_i = q^i/\lambda + f'(P_i)$, we need $q^i/\lambda + f'(P_i) = \lambda q^i$, so $f'(P_i) = (\lambda - 1/\lambda)q^i$... this does not work because $f$ must be a function of $P$ alone. The resolution: use $F_2 = \lambda q \cdot P$. Then $p_i = \lambda P_i$ (so $P_i = p_i/\lambda$) and $Q^i = \lambda q^i$. So the transformation is $Q = \lambda q$, $P = p/\lambda$. Poisson bracket: $\{Q^i, P_j\} = \lambda \delta^i_k \cdot (1/\lambda)\delta^k_j = \delta^i_j$. Canonical.

</details>

</aside>

<aside class="exercise" data-difficulty="medium" data-type="symbolic" markdown="1">

**Exercise 3 (medium, symbolic).**

The harmonic oscillator Hamiltonian is $H = \tfrac{1}{2}(p^2 + \omega^2 q^2)$. Find a type-2 generating function $F_2(q, P)$ that transforms to coordinates $(Q, P)$ in which the Hamiltonian is $K = \omega P$ (i.e., $Q$ is cyclic). Identify the new variables physically.

<details class="exercise__hint"><summary>Hint</summary>

You need $K = H(q, p)$ with $p = \partial F_2/\partial q$. Set $K = \omega P$ and solve the PDE for $F_2$.

</details>

<details class="exercise__answer"><summary>Answer</summary>

The relation $K = H$ requires $\omega P = \tfrac{1}{2}[(\partial F_2/\partial q)^2 + \omega^2 q^2]$. This is a nonlinear first-order ODE for $F_2(q, P)$ at each fixed $P$. The solution is $F_2(q, P) = \tfrac{1}{2}\omega q^2\cot\phi$ where $\phi$ is determined by $P$. More precisely, the action-angle generating function gives $F_2(q, I) = \tfrac{1}{2}\omega q^2\cot\theta$ where $I = P$ is the action and $\theta$ is related to $q$ by $q = \sqrt{2I/\omega}\sin\theta$.

In the new variables: $P = I = E/\omega$ is the action (energy divided by frequency), and $Q = \theta$ is the angle variable (phase of the oscillation). The Hamiltonian $K = \omega I$ is linear in the action, $\dot{I} = -\partial K/\partial Q = 0$ (action conserved), and $\dot{Q} = \partial K/\partial I = \omega$ (angle advances at constant frequency). This is the action-angle transformation for the harmonic oscillator.

</details>

</aside>

<aside class="exercise" data-difficulty="hard" data-type="symbolic" markdown="1">

**Exercise 4 (hard, symbolic).**

Show that the composition of two canonical transformations generated by type-2 functions $F_2^{(1)}(q, P)$ and $F_2^{(2)}(Q, R)$ is itself canonical. Express the generating function for the composition in terms of the two component generating functions.

<details class="exercise__hint"><summary>Hint</summary>

The first transformation maps $(q, p) \to (Q, P)$ and the second maps $(Q, P) \to (R, S)$. The composition maps $(q, p) \to (R, S)$ directly. Use the generating-function identity $p_i\,dq^i + Q^i\,dP_i = dF_2^{(1)}$ and $P_i\,dQ^i + R^i\,dS_i = dF_2^{(2)}$.

</details>

<details class="exercise__answer"><summary>Answer</summary>

The first transformation gives $p_i\,dq^i + Q^i\,dP_i = dF_2^{(1)}(q, P)$. The second gives $P_i\,dQ^i + R^i\,dS_i = dF_2^{(2)}(Q, S)$. Adding: $p_i\,dq^i + (Q^i\,dP_i + P_i\,dQ^i) + R^i\,dS_i = dF_2^{(1)} + dF_2^{(2)}$. The middle term is $d(Q^i P_i)$. So $p_i\,dq^i + R^i\,dS_i = d(F_2^{(1)} + F_2^{(2)} - Q \cdot P)$. The composition is generated by $F_2^{(3)}(q, S) = F_2^{(1)}(q, P) + F_2^{(2)}(Q, S) - Q^iP_i$ with the intermediate variables $(Q, P)$ eliminated using the two transformation equations. This is a Baker-Campbell-Hausdorff-type formula for canonical transformations: the composition generating function is the sum of the two component functions minus the pairing term $Q \cdot P$.

</details>

</aside>

Show that $p = \partial F_1/\partial q$ and $P = -\partial F_1/\partial Q$ are satisfied.

<details class="exercise__hint"><summary>Hint</summary>

Express $p$ and $P$ in terms of $q$ and $Q$. Write $p = \omega q/\tan Q$ and $P = \tfrac{1}{2}\omega q^2(1 + 1/\tan^2 Q) = \omega q^2/(2\sin^2 Q)$.

</details>

<details class="exercise__answer"><summary>Answer</summary>

From $Q = \arctan(\omega q/p)$, we have $p = \omega q/\tan Q$. And $P = \tfrac{1}{2}(p^2/\omega + \omega q^2) = \tfrac{1}{2}(\omega q^2/\tan^2 Q + \omega q^2) = \omega q^2/(2\sin^2 Q)$.

We need $F_1(q, Q)$ with $p = \partial F_1/\partial q = \omega q/\tan Q$, so $F_1 = \omega q^2/(2\tan Q) + f(Q)$. And $P = -\partial F_1/\partial Q = \omega q^2/(2\sin^2 Q) - f'(Q)$. Comparing with $P = \omega q^2/(2\sin^2 Q)$, we get $f'(Q) = 0$, so $f$ is constant. Thus $F_1(q, Q) = \omega q^2/(2\tan Q)$.

The generating function is singular at $Q = 0$ and $Q = \pi$ (where $\tan Q = 0$), corresponding to the turning points where $q$ passes through zero. This is a generic feature of the type-1 parameterisation of action-angle variables: the angle variable passes through values where the mixed Hessian $\partial^2 F_1/\partial q \partial Q$ vanishes, and the generating function is not globally smooth. The type-2 parameterisation avoids this problem.

</details>

</aside>

**Exercise 6 (hard, symbolic).**

A generating function $F_2(q, P) = q^1 P_1 + q^2 P_2 + \epsilon(q^1 P_2 - q^2 P_1)$ generates a canonical transformation that is the identity plus a small perturbation. Compute the transformation to first order in $\epsilon$ and interpret the result geometrically.

<details class="exercise__hint"><summary>Hint</summary>

$p_i = \partial F_2/\partial q^i$ and $Q^i = \partial F_2/\partial P_i$. Expand to $O(\epsilon)$.

</details>

<details class="exercise__answer"><summary>Answer</summary>

$p_1 = P_1 + \epsilon P_2$, $p_2 = P_2 - \epsilon P_1$. And $Q^1 = q^1 + \epsilon q^2$, $Q^2 = q^2 - \epsilon q^1$.

To $O(\epsilon)$: invert $P_i \approx p_i - \epsilon(p_2, -p_1)_i$... more directly, the position-space transformation is $\mathbf{Q} = \mathbf{q} + \epsilon\,\hat{z}\times\mathbf{q}$, which is an infinitesimal rotation by angle $\epsilon$ about the $z$-axis. The momentum transformation is $\mathbf{p} = \mathbf{P} + \epsilon\,\hat{z}\times\mathbf{P}$, the same rotation. The generator $G = q^1 p_2 - q^2 p_1 = L_z$ is the $z$-component of angular momentum. The infinitesimal canonical transformation generated by $L_z$ is a rotation about the $z$-axis — this is the connection between symmetries and generating functions.

</details>

</aside>

**Exercise 7 (medium, symbolic).**

Find the type-2 generating function for the canonical transformation that corresponds to a point transformation $Q^i = f^i(q)$ on configuration space, i.e., the canonical lift of a configuration-space diffeomorphism to the cotangent bundle.

<details class="exercise__hint"><summary>Hint</summary>

The canonical lift has $P_i = p_j\,\partial q^j/\partial Q^i$. Write this as a generating-function condition.

</details>

<details class="exercise__answer"><summary>Answer</summary>

For a point transformation $Q^i = f^i(q)$, the canonical lift is $P_i = p_j\,\partial q^j/\partial Q^i$. The type-2 generating function is $F_2(q, P) = P_i f^i(q)$. Check: $p_j = \partial F_2/\partial q^j = P_i\,\partial f^i/\partial q^j$, which gives $P_i = p_j(\partial f^i/\partial q^j)^{-1}$... actually, writing $J^i_j = \partial f^i/\partial q^j$, we have $p_j = P_i J^i_j$, i.e., $P = p \cdot J^{-1}$, or equivalently $P_i = p_j (J^{-1})^j_i = p_j\,\partial q^j/\partial Q^i$. And $Q^i = \partial F_2/\partial P_i = f^i(q)$. So $F_2(q, P) = P_i f^i(q)$ generates the canonical lift. Point transformations [09.05.04] are a special case of canonical transformations with this particularly simple generating function.

</details>

</aside>

**Exercise 8 (hard, symbolic).**

Prove that a canonical transformation is symplectic (preserves $\omega = dp_i \wedge dq^i$) if and only if it is generated by one of the four types of generating function (assuming the appropriate regularity condition).

<details class="exercise__hint"><summary>Hint</summary>

The "if" direction was done in the Key derivation: $dF$ is exact, so $d^2F = 0$ gives symplecticity. For the "only if" direction, use the fact that the graph of a symplectomorphism is a Lagrangian submanifold of the product symplectic manifold $(T^*Q \times T^*Q, \pi_1^*\omega - \pi_2^*\omega)$, and a Lagrangian submanifold is locally the graph of an exact 1-form by the Poincare lemma.

</details>

<details class="exercise__answer"><summary>Answer</summary>

The "if" direction: a generating function $F$ ensures $p_i\,dq^i - P_i\,dQ^i = dF$ (up to signs depending on type). Taking $d$: $dp_i \wedge dq^i - dP_i \wedge dQ^i = d^2F = 0$. So $\omega_{\rm old} = \omega_{\rm new}$: symplectic.

The "only if" direction: consider the product symplectic manifold $M = T^*Q \times T^*Q$ with symplectic form $\Omega = \pi_1^*\omega - \pi_2^*\omega$. The graph $\Gamma_\Phi = \{(x, \Phi(x)) : x \in T^*Q\}$ of a symplectomorphism $\Phi$ satisfies $\Omega|_\Gamma = 0$: for vectors tangent to $\Gamma$, $\Omega((v_1, D\Phi\cdot v_1), (v_2, D\Phi\cdot v_2)) = \omega(v_1, v_2) - \omega(D\Phi\cdot v_1, D\Phi\cdot v_2) = 0$ since $\Phi^*\omega = \omega$. So $\Gamma_\Phi$ is an $n$-dimensional submanifold on which the $2n$-form $\Omega$ restricts to zero: $\Gamma_\Phi$ is a **Lagrangian submanifold** of $(M, \Omega)$. By the Poincare lemma, the restriction of $\theta = p_i\,dq^i - P_i\,dQ^i$ to $\Gamma_\Phi$ is locally exact: $\theta|_\Gamma = dF$ for some function $F$ on $\Gamma$. Choosing a local parameterisation of $\Gamma$ by $(q, Q)$, $(q, P)$, $(p, Q)$, or $(p, P)$ gives the four types of generating function. The regularity condition ensures the chosen parameterisation is valid (the Lagrangian submanifold is transverse to the corresponding foliation).

</details>

</aside>

## Advanced results [Master]

### The Lagrangian-submanifold interpretation

The deepest way to understand generating functions is through the theory of **Lagrangian submanifolds**. A Lagrangian submanifold $L$ of a $2n$-dimensional symplectic manifold $(M, \omega)$ is an $n$-dimensional submanifold on which $\omega$ vanishes: $\omega|_L = 0$. Lagrangian submanifolds are the "maximally isotropic" submanifolds — they have the largest possible dimension consistent with $\omega = 0$.

The graph of a canonical transformation $\Phi : T^*Q \to T^*Q$ is a Lagrangian submanifold of $(T^*Q \times T^*Q, \pi_1^*\omega - \pi_2^*\omega)$. The four types of generating function correspond to the four ways of projecting this Lagrangian submanifold onto coordinate $n$-planes in the $4n$-dimensional product space. A projection is regular (the Lagrangian submanifold is transverse to the fibres of the projection) precisely when the corresponding generating function exists.

This geometric picture generalises to arbitrary symplectic manifolds, not just cotangent bundles. On a general symplectic manifold, there is no canonical splitting into "position" and "momentum," and the generating function depends on the choice of a polarisation (a Lagrangian foliation). The four types correspond to four choices of polarisation on the product manifold.

### Generating functions and the time evolution

The time-$t$ evolution map $\phi_t : T^*Q \to T^*Q$ of a Hamiltonian system is a canonical transformation (a symplectomorphism). Its generating function is related to **Hamilton's principal function** $S(q, Q, t)$, defined as the action evaluated along the classical trajectory from $q$ at time $0$ to $Q$ at time $t$:

$$
S(q, Q, t) = \int_0^t L(q(s), \dot{q}(s))\,ds
$$

where $q(s)$ is the solution of Hamilton's equations with $q(0) = q$ and $q(t) = Q$. Hamilton's principal function is the type-1 generating function of the time-evolution map: $p(0) = \partial S/\partial q$ and $p(t) = -\partial S/\partial Q$. This is the deep connection between the action principle and canonical transformations: the classical action *is* the generating function of the classical flow.

### Singular generating functions and the Maslov class

Not every canonical transformation admits a global generating function of a single type. The exchange transformation $Q = p$, $P = -q$ is singular in the type-2 parameterisation because $\det(\partial P/\partial p) = 0$. More generally, for action-angle variables on a torus, the type-1 generating function $F_1(q, Q)$ is multivalued (it increases by $2\pi I$ each time $Q$ wraps around the torus).

The **Maslov index** and the **Maslov class** quantify the obstruction to finding a globally smooth generating function of a single type. A canonical transformation on a compact phase space (like a torus) requires at least two charts, each with its own generating function, related by transition functions on the overlaps. The cocycle condition on the transition functions defines a cohomology class — the Maslov class — which measures the topological obstruction to a global generating function. For the harmonic oscillator on the circle, the Maslov index is 2 (the generating function has two caustics per period), which is the origin of the Maslov correction $\mu = 2$ in the WKB quantisation condition $\oint p\,dq = 2\pi\hbar(n + \mu/4)$.

**Synthesis.** The generating-function framework unifies the algebraic, geometric, and topological aspects of canonical transformations: the four types arise from the choice of Lagrangian splitting on the product manifold, the symplecticity of the transformation follows from the exactness of the generating-function differential, and the topological obstructions to a global generating function are measured by the Maslov class. The central insight is that a canonical transformation is completely determined by a single scalar function — the generating function — and all the transformation equations follow by differentiation. Putting these together, the generating function is the bridge between the abstract symplectic geometry of canonical transformations and the concrete computational tools (Hamilton-Jacobi equation, action-angle variables, perturbation theory) that make Hamiltonian mechanics the most powerful framework for classical and quantum physics.

## Connections [Master]

- Canonical transformations [09.05.01] are the symplectomorphisms whose generating functions are developed here.
- The Hamilton-Jacobi equation [09.05.02] is the PDE satisfied by the type-2 generating function that trivialises the Hamiltonian.
- Hamilton's equations [09.04.02] are preserved by every canonical transformation generated by the functions in this unit.
- Action-angle variables [09.06.01] are constructed using a type-2 generating function that separates the conserved actions from the cyclic angles.
- Hamiltonian perturbation theory [09.04.06] uses near-identity type-2 generating functions to construct the normal-form transformation.

## Historical & philosophical context [Master]

The concept of generating functions for canonical transformations was introduced by Carl Gustav Jacob Jacobi in his *Vorlesungen uber Dynamik* (lectures delivered 1842-1843, published posthumously 1866). Jacobi's insight was that the Hamilton-Jacobi equation — the PDE satisfied by the generating function of the trivialising transformation — reduces the problem of solving Hamilton's equations to the problem of solving a single first-order nonlinear PDE. This "Jacobi's theorem" established the equivalence between the Hamilton-Jacobi PDE and the ODE system of Hamilton's equations, and it became the dominant approach to solving mechanical problems in the 19th century.

The four-type classification was systematised by various authors in the early 20th century. Goldstein's *Classical Mechanics* (1950, now in its 3rd edition) presents the standard textbook treatment. The Lagrangian-submanifold interpretation is due to Weinstein (1971) and is developed in Arnold's *Mathematical Methods* (1974, English translation 1978). The Maslov class was introduced by Maslov in *Theorie des perturbations et methodes asymptotiques* (1972) and developed by Leray, Hörmander, and others into the modern theory of Fourier integral operators.

The philosophical significance of generating functions lies in the reduction of a high-dimensional geometric problem (finding a symplectomorphism) to a scalar function. This is a recurring pattern in mathematical physics: potentials encode vector fields, actions encode trajectories, and generating functions encode coordinate changes. In each case, the scalar object is easier to manipulate and optimise than the vector or tensor object it generates, and the physical content is preserved by the differentiation procedure that recovers the original object.
