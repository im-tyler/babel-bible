## Intuition [Beginner]

A canonical transformation is a relabelling of phase space -- new names for the same physical states -- that preserves Hamilton's equations. Unit [09.05.01] established what these transformations are and why they matter. This unit answers the practical question: how do you actually *construct* one?

You could write down the $2n$ transformation equations $(Q, P) = \Phi(q, p)$ directly and then verify that the Poisson brackets are preserved -- a tedious calculation involving cross-derivatives. There is a better way. Every canonical transformation can be encoded in a single scalar function called a **generating function**. You specify one function of $2n$ variables -- half old, half new -- and the transformation equations emerge by taking derivatives. The generating function is a "potential" for the canonical transformation, just as an electrostatic potential encodes the electric field: differentiate in the right directions and the transformation pops out.

Why four types? The generating function takes $2n$ of the $4n$ phase-space variables as its independent arguments. There are four natural choices, corresponding to which half of the old set and which half of the new set you pick:

| Type | Independent variables | Derived variables |
|------|-----------------------|-------------------|
| $F_1$ | $q, Q$ | $p, P$ |
| $F_2$ | $q, P$ | $p, Q$ |
| $F_3$ | $p, Q$ | $q, P$ |
| $F_4$ | $p, P$ | $q, Q$ |

Each type picks one member from the old pair $(q, p)$ and one from the new pair $(Q, P)$ for each degree of freedom. The four types are related by Legendre transforms -- changing which variables you treat as "independent" in the same way that switching from $U(V, S)$ to $H(S, P)$ in thermodynamics is a Legendre transform.

Not every canonical transformation can be written in every type. The identity transformation $Q = q, P = p$ is singular in the $F_1$ parameterisation (because $q$ and $Q$ are not independent near the identity) but perfectly regular in the $F_2$ parameterisation. The exchange transformation $Q = p, P = -q$ is regular in $F_1$ but singular in $F_2$. Having all four types ensures you can always find one that works for the transformation you need.

The strategic payoff: instead of searching for good coordinates by trial and error, you search for a good generating function -- a scalar optimisation problem. The Hamilton-Jacobi equation, perturbation theory, and action-angle variables all reduce to choosing the right generating function.

## Visual [Beginner]

Imagine the graph of a canonical transformation plotted in a combined $(q, p, Q, P)$ space. The graph is a $2n$-dimensional surface inside a $4n$-dimensional space. A generating function describes this surface by choosing which $2n$ variables are "inputs" and which $2n$ are "outputs," then writing the surface as the graph of a scalar potential.

For $F_2(q, P)$, the inputs are old positions $q$ and new momenta $P$; the outputs are old momenta $p$ and new positions $Q$. The transformation equations say: the old momentum is the slope of $F_2$ along the $q$-direction, and the new position is the slope of $F_2$ along the $P$-direction.

![Schematic of the four generating-function types as different "slices" of the canonical-transformation graph. Each type picks a different pair of old/new variables as independent inputs, with the remaining pair derived by differentiation. The type-2 box (q, P) to (p, Q) is highlighted as the most common choice.](/img/09.05.03-generating-functions-four-types-placeholder.svg)

The four types correspond to the four projections of the graph onto coordinate $n$-planes. A projection is regular (one-to-one) precisely when the corresponding generating function exists. Where one projection folds over and becomes singular, another projection remains regular -- this is why we need all four.

## Worked example [Beginner]

### The identity transformation

The simplest canonical transformation is doing nothing: $Q = q, P = p$. The type-2 generating function is $F_2(q, P) = q \cdot P$. Applying the type-2 rules: $p_i = dF_2/dq^i = P_i$ and $Q^i = dF_2/dP_i = q^i$. Confirmed: the identity.

This might seem like a tautology, but it has deep consequences. A generating function close to the identity, $F_2 = q \cdot P + \epsilon S(q, P)$, generates a canonical transformation close to the identity -- a small perturbation. This is the starting point for canonical perturbation theory and for the Hamilton-Jacobi method.

### The exchange transformation

Consider swapping position and momentum: $Q = p, P = -q$. This is canonical because $\{Q, P\} = \{p, -q\} = 1$, preserving the fundamental bracket.

Using the type-1 form $F_1(q, Q)$ with the rules $p = dF_1/dq$ and $P = -dF_1/dQ$: the exchange requires $dF_1/dq = Q$ and $-dF_1/dQ = -q$, i.e., $dF_1/dQ = q$. The first equation gives $F_1 = qQ + f(Q)$. The second gives $q + f'(Q) = q$, so $f$ is constant. Therefore $F_1(q, Q) = qQ$.

Now try type 2. The exchange has $P = -q$, but $F_2$ requires $q$ and $P$ as independent variables. Since $P = -q$, these are dependent -- the type-2 generating function does not exist. This is exactly why we need all four types: the exchange is singular in type 2 but regular in type 1.

### Rotation in phase space

The transformation $Q = q\cos\alpha - p\sin\alpha$, $P = q\sin\alpha + p\cos\alpha$ is a rotation by angle $\alpha$ in the $(q, p)$ plane. The generating function $F_1(q, Q) = \frac{1}{2}(q^2 + Q^2)\cot\alpha - qQ\csc\alpha$ produces exactly this rotation. Taking derivatives:

$dF_1/dq = q\cot\alpha - Q\csc\alpha = q\cos\alpha/\sin\alpha - Q/\sin\alpha$. Setting this equal to $p$ and solving: $Q = q\cos\alpha - p\sin\alpha$.

$-dF_1/dQ = -(Q\cot\alpha - q\csc\alpha) = q/\sin\alpha - Q\cos\alpha/\sin\alpha = P$. Substituting $Q$: $P = q\sin\alpha + p\cos\alpha$.

The Poisson bracket is $\{Q, P\} = (dQ/dq)(dP/dp) - (dQ/dp)(dP/dq) = \cos^2\alpha + \sin^2\alpha = 1$. Canonical.

## Formal definition [Intermediate+]

### The generating-function principle

A curve $(q(t), p(t))$ satisfies Hamilton's equations if and only if the action integral $I = \int (p_i\,dq^i - H\,dt)$ is stationary [09.04.02]. After a canonical transformation to $(Q, P)$ with new Hamiltonian $K$, the action must be $I' = \int (P_i\,dQ^i - K\,dt)$, also stationary. Two integrals with the same stationary points differ at most by an exact differential $dF$:

$$
p_i\,dq^i - H\,dt = P_i\,dQ^i - K\,dt + dF.
$$

The function $F$ is the generating function. Its natural arguments depend on which variables are chosen as independent.

### Type 1: $F_1(q, Q, t)$

Choose $(q, Q)$ as independent variables. Expanding $dF_1$ and matching coefficients:

$$
\boxed{p_i = \frac{\partial F_1}{\partial q^i}, \qquad P_i = -\frac{\partial F_1}{\partial Q^i}, \qquad K = H + \frac{\partial F_1}{\partial t}.}
$$

The first two equations define the canonical transformation implicitly. The third relates the old and new Hamiltonians. For time-independent transformations, $K = H$. The regularity condition is that the $n \times n$ mixed Hessian $\det(\partial^2 F_1/\partial q^i \partial Q^j) \neq 0$, ensuring the equations can be solved for $(q, p)$ in terms of $(Q, P)$.

### Type 2: $F_2(q, P, t)$

Legendre-transform $F_1$ in the $Q$-variables: $F_2 = F_1 + P_iQ^i$. Then $dF_2 = p_i\,dq^i + Q^i\,dP_i + (K - H)\,dt$, giving:

$$
\boxed{p_i = \frac{\partial F_2}{\partial q^i}, \qquad Q^i = \frac{\partial F_2}{\partial P_i}, \qquad K = H + \frac{\partial F_2}{\partial t}.}
$$

Type 2 is the most widely used because $F_2$ depends on old positions and new momenta -- the natural variables for the Hamilton-Jacobi equation [09.05.02]. The type-2 generating function $S(q, P, t)$ satisfying $H(q, \partial S/\partial q, t) + \partial S/\partial t = 0$ is the Hamilton-Jacobi function.

### Type 3: $F_3(p, Q, t)$

Legendre-transform $F_1$ in the $q$-variables: $F_3 = F_1 - p_iq^i$. Then:

$$
\boxed{q^i = -\frac{\partial F_3}{\partial p_i}, \qquad P_i = -\frac{\partial F_3}{\partial Q^i}, \qquad K = H + \frac{\partial F_3}{\partial t}.}
$$

Type 3 is useful when old momenta $p$ are the natural independent variables, for instance in the momentum representation.

### Type 4: $F_4(p, P, t)$

Legendre-transform $F_1$ in both $q$ and $Q$: $F_4 = F_1 - p_iq^i + P_iQ^i$. Then:

$$
\boxed{q^i = -\frac{\partial F_4}{\partial p_i}, \qquad Q^i = \frac{\partial F_4}{\partial P_i}, \qquad K = H + \frac{\partial F_4}{\partial t}.}
$$

Type 4 is the least commonly used but is essential when both positions are dependent. The exchange transformation $(Q, P) = (p, -q)$ is naturally expressed in type 4 as $F_4(p, P) = -p \cdot P$: $q^i = -\partial(-p_jP_j)/\partial p_i = P_i$ and $Q^i = \partial(-p_jP_j)/\partial P_i = -p_i$, giving $Q = -p, q = P$.

### Summary of regularity conditions

Each type requires an invertibility condition on the transformation:

| Type | Independent vars | Regularity condition |
|------|-----------------|---------------------|
| $F_1$ | $q, Q$ | $\det(\partial Q/\partial p) \neq 0$ |
| $F_2$ | $q, P$ | $\det(\partial P/\partial p) \neq 0$ |
| $F_3$ | $p, Q$ | $\det(\partial q/\partial Q) \neq 0$ |
| $F_4$ | $p, P$ | $\det(\partial Q/\partial q) \neq 0$ |

Any canonical transformation sufficiently close to the identity satisfies the type-2 condition, which is why type 2 works for perturbation theory.

### The symplectic condition

A transformation $(q, p) \mapsto (Q, P)$ is canonical if and only if the Jacobian matrix $M$ satisfies the **symplectic condition**:

$$
M^T J M = J, \qquad J = \begin{pmatrix} 0 & I_n \\ -I_n & 0 \end{pmatrix}.
$$

This is equivalent to the preservation of the fundamental Poisson brackets [09.04.03]: $\{Q^i, Q^j\} = 0$, $\{P_i, P_j\} = 0$, $\{Q^i, P_j\} = \delta^i_j$.

## Key derivation [Intermediate+]

**Theorem.** *The transformation defined by a type-2 generating function $F_2(q, P)$ is automatically symplectic.*

**Proof.** The generating function guarantees $p_i\,dq^i + Q^i\,dP_i = dF_2$. The old canonical 1-form is $\theta = p_i\,dq^i$ with $d\theta = dp_i \wedge dq^i = \omega_{\mathrm{old}}$. The new canonical 1-form is $\Theta = P_i\,dQ^i$ with $d\Theta = dP_i \wedge dQ^i = \omega_{\mathrm{new}}$.

From $p_i\,dq^i = dF_2 - Q^i\,dP_i$, take the exterior derivative of both sides:

$$
dp_i \wedge dq^i = -dQ^i \wedge dP_i = dP_i \wedge dQ^i.
$$

So $\omega_{\mathrm{old}} = \omega_{\mathrm{new}}$. The transformation is symplectic. The key fact: the two canonical 1-forms differ by an exact form $dF_2$, whose exterior derivative vanishes ($d^2F_2 = 0$). The argument for types 1, 3, and 4 is identical up to sign conventions.

**Bridge.** This derivation is the foundation for the Hamilton-Jacobi equation [09.05.02], which is the PDE that a type-2 generating function must satisfy to trivialise the Hamiltonian ($K = 0$). The symplecticity guaranteed here is the reason the Hamilton-Jacobi method works: the generating function automatically produces a canonical transformation, so the simplified Hamiltonian $K$ still governs a genuine Hamiltonian system. The same construction appears in canonical perturbation theory, where the near-identity transformation that eliminates fast oscillations is specified by its type-2 generating function.

<aside class="exercise" data-difficulty="medium" data-type="symbolic" markdown="1">

**Exercise 3 (medium, symbolic).**

Find a type-2 generating function for the scaling transformation $Q^i = \lambda q^i$, $P_i = p_i/\lambda$ for constant $\lambda \neq 0$. Verify canonicity by computing $\{Q^i, P_j\}$.

<details class="exercise__hint"><summary>Hint</summary>

You need $Q^i = \partial F_2/\partial P_i = \lambda q^i$ and $p_i = \partial F_2/\partial q^i = P_i/\lambda$.

</details>

<details class="exercise__answer"><summary>Answer</summary>

From $p_i = \partial F_2/\partial q^i = \lambda P_i$, we get $F_2 = \lambda q^i P_i + f(P)$. From $Q^i = \partial F_2/\partial P_i = \lambda q^i + \partial f/\partial P_i$, we need $\partial f/\partial P_i = 0$, so $f$ is constant. Thus $F_2 = \lambda q \cdot P$. Check: $p_i = \lambda P_i$ so $P_i = p_i/\lambda$, and $Q^i = \lambda q^i$. The Poisson bracket $\{Q^i, P_j\} = \lambda \delta^i_k \cdot (1/\lambda)\delta^k_j = \delta^i_j$. Canonical.

</details>

</aside>

<aside class="exercise" data-difficulty="medium" data-type="symbolic" markdown="1">

**Exercise 4 (medium, symbolic).**

The harmonic oscillator has $H = \frac{1}{2}(p^2 + \omega^2 q^2)$. Find a type-2 generating function $F_2(q, P)$ that transforms to coordinates in which the Hamiltonian is $K = \omega P$ (i.e., $Q$ is cyclic). Identify the new variables physically.

<details class="exercise__hint"><summary>Hint</summary>

Set $K = \omega P$ and use $K = H(q, p)$ with $p = \partial F_2/\partial q$. Solve the resulting PDE for $F_2$.

</details>

<details class="exercise__answer"><summary>Answer</summary>

The relation $\omega P = \frac{1}{2}[(\partial F_2/\partial q)^2 + \omega^2 q^2]$ is a first-order ODE for $F_2$ at each fixed $P$. The action-angle solution is $F_2(q, I) = \frac{1}{2}\omega q^2/\tan\theta$ where $I = P$ and $q = \sqrt{2I/\omega}\sin\theta$.

In the new variables: $P = I = E/\omega$ is the action (energy divided by frequency), and $Q = \theta$ is the angle variable (phase of oscillation). The Hamiltonian $K = \omega I$ is linear in the action: $\dot{I} = -\partial K/\partial Q = 0$ (action conserved) and $\dot{Q} = \partial K/\partial I = \omega$ (angle advances at constant frequency). This is the action-angle transformation for the harmonic oscillator, the simplest example of the Liouville-Arnold construction [09.06.01].

</details>

</aside>

<aside class="exercise" data-difficulty="medium" data-type="symbolic" markdown="1">

**Exercise 5 (medium, symbolic).**

A generating function $F_2(q, P) = q^1P_1 + q^2P_2 + \epsilon(q^1P_2 - q^2P_1)$ generates a near-identity canonical transformation. Compute the transformation to first order in $\epsilon$ and interpret geometrically.

<details class="exercise__hint"><summary>Hint</summary>

Compute $p_i = \partial F_2/\partial q^i$ and $Q^i = \partial F_2/\partial P_i$, then expand to $O(\epsilon)$.

</details>

<details class="exercise__answer"><summary>Answer</summary>

$p_1 = P_1 + \epsilon P_2$, $p_2 = P_2 - \epsilon P_1$. $Q^1 = q^1 + \epsilon q^2$, $Q^2 = q^2 - \epsilon q^1$. The position-space transformation is $\mathbf{Q} = \mathbf{q} + \epsilon\,\hat{z} \times \mathbf{q}$: an infinitesimal rotation by angle $\epsilon$ about the $z$-axis. The momentum transformation is the same rotation. The perturbation term $G = q^1P_2 - q^2P_1 = L_z$ is the $z$-component of angular momentum -- the generator of rotations. This is the connection between symmetries and generating functions: the conserved quantity $L_z$ generates the rotation symmetry via its Hamiltonian flow.

</details>

</aside>

<aside class="exercise" data-difficulty="medium" data-type="symbolic" markdown="1">

**Exercise 6 (medium, symbolic).**

Find the type-2 generating function for the canonical lift of the point transformation $Q^i = f^i(q)$. That is, find $F_2(q, P)$ such that $Q^i = f^i(q)$ and the momenta transform as cotangent vectors.

<details class="exercise__hint"><summary>Hint</summary>

The canonical lift has $P_i = p_j\,\partial q^j/\partial Q^i$. Write $F_2$ so that $p_j = \partial F_2/\partial q^j$ produces this relation.

</details>

<details class="exercise__answer"><summary>Answer</summary>

Set $F_2(q, P) = P_i f^i(q)$. Then $p_j = \partial F_2/\partial q^j = P_i\,\partial f^i/\partial q^j$, which gives $P = p \cdot J^{-1}$ where $J^i_j = \partial f^i/\partial q^j$. And $Q^i = \partial F_2/\partial P_i = f^i(q)$. Point transformations [09.05.04] are canonical transformations with this particularly simple generating function -- a linear function of the new momenta.

</details>

</aside>

**Exercise 7 (hard, symbolic).**

The first maps $(q, p) \to (Q, P)$ and the second maps $(Q, P) \to (R, S)$. Add the differential identities: $p_i\,dq^i + Q^i\,dP_i = dF_2^{(1)}$ and $P_i\,dQ^i + R^i\,dS_i = dF_2^{(2)}$.

</details>

<details class="exercise__answer"><summary>Answer</summary>

The first gives $p_i\,dq^i + Q^i\,dP_i = dF_2^{(1)}(q, P)$. The second gives $P_i\,dQ^i + R^i\,dS_i = dF_2^{(2)}(Q, S)$. Adding: $p_i\,dq^i + (Q^i\,dP_i + P_i\,dQ^i) + R^i\,dS_i = d(F_2^{(1)} + F_2^{(2)})$. The middle term is $d(Q^iP_i)$. So $p_i\,dq^i + R^i\,dS_i = d(F_2^{(1)} + F_2^{(2)} - Q \cdot P)$.

The composition is generated by $F_2^{(3)}(q, S) = F_2^{(1)}(q, P) + F_2^{(2)}(Q, S) - Q^iP_i$ with intermediate variables $(Q, P)$ eliminated using the two transformation equations. This is the group law for canonical transformations expressed in generating-function form.

**Exercise 8 (hard, symbolic).**

Prove that a canonical transformation is symplectic (preserves $\omega = dp_i \wedge dq^i$) if and only if it is locally generated by one of the four types of generating function (assuming the appropriate regularity condition).

<details class="exercise__hint"><summary>Hint</summary>

The "if" direction: $dF$ is exact, so $d^2F = 0$ gives symplecticity. For the "only if" direction, use the fact that the graph of a symplectomorphism is a Lagrangian submanifold of the product manifold with symplectic form $\pi_1^*\omega - \pi_2^*\omega$.

</details>

<details class="exercise__answer"><summary>Answer</summary>

*If direction*: a generating function ensures $p_i\,dq^i - P_i\,dQ^i = dF$ (up to signs). Taking $d$: $dp_i \wedge dq^i - dP_i \wedge dQ^i = d^2F = 0$, so $\omega_{\mathrm{old}} = \omega_{\mathrm{new}}$.

*Only-if direction*: the graph $\Gamma_\Phi$ of a symplectomorphism $\Phi$ is a Lagrangian submanifold of $(T^*Q \times T^*Q, \pi_1^*\omega - \pi_2^*\omega)$, since for tangent vectors $(v, D\Phi \cdot v)$: $(\pi_1^*\omega - \pi_2^*\omega)((v_1, D\Phi v_1), (v_2, D\Phi v_2)) = \omega(v_1,v_2) - \omega(D\Phi v_1, D\Phi v_2) = 0$. By the Poincare lemma, the restriction of the 1-form $\theta = p_i\,dq^i - P_i\,dQ^i$ to $\Gamma_\Phi$ is locally exact: $\theta|_\Gamma = dF$. Choosing a parameterisation of $\Gamma$ by $(q, Q)$, $(q, P)$, $(p, Q)$, or $(p, P)$ gives the four types. The regularity condition ensures the chosen parameterisation is valid (the Lagrangian submanifold is transverse to the corresponding foliation).

## Lean formalization [Intermediate+]

The generating-function formalism is a central missing piece in Mathlib's Hamiltonian mechanics. The specific ingredients needed are:

1. The definition of a nonlinear canonical transformation as a symplectomorphism $\Phi : T^*Q \to T^*Q$ satisfying $\Phi^*\omega = \omega$.
2. Generating functions of each type as formal objects: a smooth function of $2n$ mixed variables with nonsingular mixed Hessian.
3. The transformation equations (e.g., $p_i = \partial F_2/\partial q^i$, $Q^i = \partial F_2/\partial P_i$) and the proof that the resulting map is symplectic.
4. The Legendre-transform relations between the four types.
5. The regularity conditions under which each type exists.
6. The Lagrangian-submanifold interpretation: the graph of a symplectomorphism is Lagrangian in the product, and a Lagrangian submanifold transverse to a fibration is locally the graph of an exact 1-form.

Items (1)--(3) are within reach given existing Mathlib infrastructure (smooth maps, exterior derivatives, the symplectic group). Item (6) requires the theory of Lagrangian submanifolds, which is absent. No `lean_module` ships with this unit.

## Advanced results [Master]

### Generating functions as symplectomorphism classification

The four types of generating function are not merely a calculational device -- they classify canonical transformations by the topology of their graph in the product manifold. Let $(M_1 \times M_2, \Omega)$ be the product symplectic manifold with $\Omega = \pi_1^*\omega - \pi_2^*\omega$. The graph of a symplectomorphism is a Lagrangian submanifold $L \subset (M_1 \times M_2, \Omega)$.

The four types correspond to four Lagrangian fibrations of $M_1 \times M_2$ (projections onto coordinate $n$-planes). A generating function of type $k$ exists when $L$ is transverse to the $k$-th fibration. The Maslov index quantifies the obstruction to a single generating function being globally smooth: as you move around the Lagrangian submanifold, it may become tangent to a given fibration, causing the generating function to blow up. The transition between charts (different types) is governed by the Maslov class in $H^1(L, \mathbb{Z})$.

For the harmonic oscillator on the circle, the Maslov index is 2 (two caustics per period), which produces the correction $\mu = 2$ in the WKB quantisation condition.

### Time-dependent generating functions

A canonical transformation can depend explicitly on time: $\Phi(q, p, t) = (Q(q,p,t), P(q,p,t))$. The extended phase space $T^*Q \times \mathbb{R}$ carries the contact form $\alpha = p_i\,dq^i - H\,dt$ and the extended symplectic form $\Omega = dp_i \wedge dq^i - dH \wedge dt$. A transformation is canonical in the extended sense if it preserves $\Omega$.

**Proposition (Time-dependent generating functions).** *A time-dependent canonical transformation is locally generated by $F_2(q, P, t)$ via $p_i = \partial F_2/\partial q^i$, $Q^i = \partial F_2/\partial P_i$, with the new Hamiltonian $K = H + \partial F_2/\partial t$.*

*Proof.* The closed 1-form $\alpha - \beta$ on the extended phase space, with $\alpha = p_i\,dq^i - H\,dt$ and $\beta = Q^i\,dP_i + K\,dt$, satisfies $d(\alpha - \beta) = \Omega - \Omega = 0$ when $\Phi$ preserves $\Omega$. Locally $\alpha - \beta = dF_2$ for some $F_2(q, P, t)$, giving the three relations. $\square$

Time-dependent generating functions arise naturally when changing to a rotating or accelerating frame. Setting $K = 0$ recovers the Hamilton-Jacobi equation.

### Hamilton-Jacobi as the type-2 generating function

The deepest application of generating functions is the Hamilton-Jacobi equation. Choose a type-2 generating function $S(q, P, t)$ such that the new Hamiltonian vanishes: $K = 0$. The relation $K = H + \partial S/\partial t$ gives:

$$
H\!\left(q, \frac{\partial S}{\partial q}, t\right) + \frac{\partial S}{\partial t} = 0.
$$

This is the **Hamilton-Jacobi equation** [09.05.02]. Its solution $S(q, \alpha, t)$ (where $\alpha = P$ are $n$ integration constants) generates a canonical transformation that entirely trivialises the dynamics: $K = 0$ implies $\dot{P}_i = 0$ and $\dot{Q}^i = 0$, so the new variables are constants. The system is solved by quadrature from the generating function.

Jacobi's theorem establishes the equivalence: solving the $2n$ Hamilton's ODEs is equivalent to solving the single first-order PDE for $S$. The PDE is generally harder to solve, but when separation of variables applies (as it does for all Liouville-integrable systems), it reduces to $n$ independent ODEs -- a dramatic simplification.

### Time evolution as a generating function

The time-$t$ evolution map $\phi_t : T^*Q \to T^*Q$ is a canonical transformation. Its type-1 generating function is **Hamilton's principal function** $S(q, Q, t)$, defined as the action evaluated along the classical trajectory from $q$ at time $0$ to $Q$ at time $t$:

$$
S(q, Q, t) = \int_0^t L(q(s), \dot{q}(s))\,ds.
$$

Hamilton's principal function satisfies $p(0) = \partial S/\partial q$ and $p(t) = -\partial S/\partial Q$. The classical action *is* the generating function of the classical flow. This is the deep connection between the action principle and canonical transformations: the variational principle selects trajectories that are integral curves of the generating function of the time-evolution symplectomorphism.

**Synthesis.** The generating-function framework unifies the algebraic, geometric, and topological aspects of canonical transformations. The four types arise from the choice of Lagrangian splitting on the product manifold. The symplecticity of the transformation follows from the exactness of the generating-function differential ($d^2 = 0$). The topological obstructions to a global generating function are measured by the Maslov class. The central insight is that a canonical transformation is completely determined by a single scalar function, and all transformation equations follow by differentiation. The Hamilton-Jacobi equation exploits this by choosing the generating function that trivialises the dynamics, reducing mechanics to the problem of solving one PDE.

## Full proof set [Master]

**Proposition 1 (Generating function guarantees symplecticity).** *The transformation defined by any of the four types of generating function is symplectic.*

*Proof.* For type 2: $p_i\,dq^i + Q^i\,dP_i = dF_2$ implies $dp_i \wedge dq^i + dQ^i \wedge dP_i = d^2F_2 = 0$, so $\omega_{\mathrm{old}} = dp_i \wedge dq^i = dP_i \wedge dQ^i = \omega_{\mathrm{new}}$. For type 1: $p_i\,dq^i - P_i\,dQ^i = dF_1$ implies $dp_i \wedge dq^i - dP_i \wedge dQ^i = 0$, giving the same result. Types 3 and 4 differ only by signs in the 1-form identity, which cancel upon taking $d$. $\square$

**Proposition 2 (Existence of a generating function for any symplectomorphism).** *Let $\Phi : (T^*Q, \omega) \to (T^*Q, \omega)$ be a symplectomorphism. If $\det(\partial Q/\partial p) \neq 0$ at a point, then locally there exists $F_1(q, Q)$ generating $\Phi$.*

*Proof.* The graph $\Gamma_\Phi = \{(z, \Phi(z))\}$ satisfies $\Omega|_\Gamma = 0$ where $\Omega = \pi_1^*\omega - \pi_2^*\omega$. So $\Gamma_\Phi$ is Lagrangian in $(T^*Q \times T^*Q, \Omega)$. The 1-form $\theta = p_i\,dq^i - P_i\,dQ^i$ restricts to a closed form on $\Gamma_\Phi$ (since $d\theta|_\Gamma = \Omega|_\Gamma = 0$). By the Poincare lemma, $\theta|_\Gamma = dF_1$ locally. The condition $\det(\partial Q/\partial p) \neq 0$ ensures $(q, Q)$ serve as local coordinates on $\Gamma_\Phi$. $\square$

**Proposition 3 (Legendre relations between the four types).** *$F_2 = F_1 + P_iQ^i$, $F_3 = F_1 - p_iq^i$, $F_4 = F_1 - p_iq^i + P_iQ^i$, where the dependent variables in each expression are eliminated using the type-1 transformation equations.*

*Proof.* Start from $p_i\,dq^i - P_i\,dQ^i = dF_1$. Add $d(P_iQ^i) = P_i\,dQ^i + Q^i\,dP_i$ to both sides: $p_i\,dq^i + Q^i\,dP_i = d(F_1 + P_iQ^i)$. The left side is $dF_2$, so $F_2 = F_1 + P_iQ^i$. Subtract $d(p_iq^i) = p_i\,dq^i + q^i\,dp_i$: $-q^i\,dp_i - P_i\,dQ^i = d(F_1 - p_iq^i)$, so $F_3 = F_1 - p_iq^i$. Both transforms together give $F_4 = F_1 - p_iq^i + P_iQ^i$. $\square$

**Proposition 4 (Infinitesimal canonical transformations).** *The generating function $F_2(q, P) = q^iP_i + \epsilon G(q, P)$ generates, to first order in $\epsilon$, the infinitesimal canonical transformation $Q^i = q^i + \epsilon\,\partial G/\partial P_i$, $p_i = P_i + \epsilon\,\partial G/\partial q^i$. The function $G$ is the generator of the transformation.*

*Proof.* $p_i = \partial F_2/\partial q^i = P_i + \epsilon\,\partial G/\partial q^i$. $Q^i = \partial F_2/\partial P_i = q^i + \epsilon\,\partial G/\partial P_i$. The transformation differs from the identity by $O(\epsilon)$. Expressing $P_i \approx p_i - \epsilon\,\partial G/\partial q^i$ and substituting: $Q^i \approx q^i + \epsilon\,\partial G(q, p)/\partial p_i$, giving $\delta q^i = \epsilon\,\partial G/\partial p_i$ and $\delta p_i = -\epsilon\,\partial G/\partial q^i$. This is the Hamiltonian flow generated by $G$, evaluated at parameter $s = \epsilon$. $\square$

## Historical & philosophical context [Master]

The concept of generating functions for canonical transformations was introduced by Carl Gustav Jacob Jacobi in his *Vorlesungen uber Dynamik* (lectures delivered 1842--1843, published posthumously 1866). Jacobi recognised that the Hamilton-Jacobi equation -- the PDE satisfied by the generating function of the trivialising transformation -- reduces the problem of solving Hamilton's equations to the problem of solving a single first-order nonlinear PDE. This "Jacobi's theorem" established the equivalence between the Hamilton-Jacobi PDE and the ODE system of Hamilton's equations, and it became the dominant approach to solving mechanical problems in the 19th century.

William Rowan Hamilton had earlier (1833--1834) introduced what is now called Hamilton's principal function $S(q, Q, t)$ -- the type-1 generating function of the time-evolution map -- as the solution of a variational problem. Hamilton's insight was that the action functional, evaluated on classical trajectories, satisfies a first-order PDE (the Hamilton-Jacobi equation in its time-dependent form). Jacobi sharpened this by showing that *any* complete integral of the HJ equation (a solution depending on $n$ arbitrary constants) generates a canonical transformation that solves the mechanical problem, whether or not the function was obtained from an action integral.

The four-type classification was systematised in the early 20th century. The standard textbook presentation is due to Goldstein (*Classical Mechanics*, 1950, now in its 3rd edition). Landau and Lifshitz (*Mechanics*, 1976) develop primarily the type-2 generating function, which they denote $S(q, P)$, because it connects most directly to the Hamilton-Jacobi equation.

The Lagrangian-submanifold interpretation is due to Weinstein (1971) and is developed in Arnold's *Mathematical Methods* (1974, English translation 1978). This geometric viewpoint reveals that the four types are not arbitrary conventions but reflect the four natural Lagrangian fibrations of the product symplectic manifold $(T^*Q \times T^*Q, \pi_1^*\omega - \pi_2^*\omega)$. The Maslov class, introduced by Maslov (1972) and developed by Leray, Hormander, and others, quantifies the topological obstruction to a globally smooth generating function of a single type.

The philosophical significance of generating functions is the reduction of a high-dimensional geometric problem (finding a symplectomorphism) to a scalar function. This is a recurring pattern in mathematical physics: potentials encode vector fields, actions encode trajectories, and generating functions encode coordinate changes. In each case the scalar object is easier to manipulate and optimise than the vector or tensor object it generates. The transition from quantum to classical mechanics (the WKB approximation, path integrals) relies on the generating function as the bridge between the quantum amplitude and the classical trajectory.
