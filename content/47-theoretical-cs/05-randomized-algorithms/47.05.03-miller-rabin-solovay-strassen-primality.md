## Intuition [Beginner]

Is 91 prime? You could check: is it divisible by 2? No. By 3? No. By 5? No. By 7? Yes — $91 = 7 \times 13$. So 91 is composite. This trial-division method works but is slow for large numbers.

A faster approach uses a property that primes have but composites usually lack. Fermat's little theorem says: if $p$ is prime and $\gcd(a, p) = 1$, then $a^{p-1} \equiv 1 \pmod{p}$. For a composite number $n$, this identity usually fails. Pick a random $a$, compute $a^{n-1} \bmod n$, and if the result is not 1, you know $n$ is composite.

The catch is "usually." Some composite numbers (called Carmichael numbers) satisfy $a^{n-1} \equiv 1 \pmod{n}$ for almost all $a$. The Fermat test fails for these.

The Miller-Rabin test fixes this. It uses a stronger check: instead of just looking at $a^{n-1}$, it examines the sequence $a^d, a^{2d}, a^{4d}, \ldots, a^{n-1}$ where $n - 1 = 2^s \cdot d$. If $n$ is prime, this sequence must either start at 1 or reach $-1$ at some point. If neither happens, $n$ is composite.

The remarkable theorem is: for any composite $n$, at least 3/4 of all possible bases $a$ are witnesses — they expose the compositeness. So picking a random base and running the test gives at most a 1/4 chance of a false negative. Running 40 independent tests reduces the error probability to less than $4^{-40}$, which is smaller than the chance of a cosmic ray flipping a bit in your computer.

## Visual [Beginner]

The Miller-Rabin test on a prime ($n = 17$, $n-1 = 16 = 2^4 \cdot 1$, so $s = 4, d = 1$):

```
a = 3: sequence is 3^1 = 3, 3^2 = 9, 3^4 = 13, 3^8 = 16 ≡ -1 (mod 17)
        Hit -1 at step 3. Passes. ✓

a = 5: sequence is 5^1 = 5, 5^2 = 8, 5^4 = 13, 5^8 = 16 ≡ -1 (mod 17)
        Hit -1 at step 3. Passes. ✓
```

On a composite ($n = 15$, $n-1 = 14 = 2 \cdot 7$, so $s = 1, d = 7$):

```
a = 2: 2^7 = 128 ≡ 8 (mod 15). Not 1 and not -1. Witness! ✗
a = 4: 4^7 = 16384 ≡ 4 (mod 15). Not 1 and not -1. Witness! ✗
a = 14: 14^7 ≡ -1 (mod 15). Not a witness (passes). Strong liar.
```

| Test | Base choice | Witness fraction | False negative rate |
|---|---|---|---|
| Fermat | Random $a$ | $\ge 1/2$ (except Carmichael) | Up to 1/2 |
| Miller-Rabin | Random $a$ | $\ge 3/4$ (all composites) | At most $1/4$ |
| Solovay-Strassen | Random $a$ | $\ge 1/2$ (all composites) | At most $1/2$ |

## Worked example [Beginner]

Test whether $n = 221$ is prime using Miller-Rabin with base $a = 174$.

$n - 1 = 220 = 2^2 \cdot 55$. So $s = 2, d = 55$.

Compute $a^d = 174^{55} \pmod{221}$:

$174^2 = 30276 \equiv 30276 - 137 \cdot 221 = 30276 - 30277 = -1 \equiv 220 \pmod{221}$.

So $174^{55} = (174^2)^{27} \cdot 174 \equiv 220^{27} \cdot 174 \pmod{221}$.

$220 \equiv -1 \pmod{221}$, so $220^{27} \equiv (-1)^{27} = -1 \equiv 220 \pmod{221}$.

$174^{55} \equiv 220 \cdot 174 = 38280 \pmod{221}$. $38280 / 221 = 173.2...$, so $38280 \equiv 38280 - 173 \cdot 221 = 38280 - 38233 = 47 \pmod{221}$.

$a^d = 47$. Not 1, not $-1$ (which is 220 mod 221).

Compute $a^{2d} = 174^{110} = (174^{55})^2 = 47^2 = 2209 \pmod{221}$. $2209 / 221 = 9.99...$, so $2209 \equiv 2209 - 10 \cdot 221 = 2209 - 2210 = -1 \equiv 220 \pmod{221}$.

$a^{2d} = 220 = -1 \pmod{221}$. We found $-1$ at step $r = 1$. So 174 is NOT a witness; the test passes.

Try base $a = 137$. $137^{55} \pmod{221}$: $137^2 = 18769 \equiv 18769 - 84 \cdot 221 = 18769 - 18564 = 205 \pmod{221}$. This is getting complicated by hand, but let us use the fact that $221 = 13 \cdot 17$. Since 221 is composite, most bases will be witnesses. The Miller-Rabin theorem guarantees at least $3 \cdot 220/4 = 165$ witnesses out of 220 possible bases.

Indeed, $221 = 13 \times 17$ is composite, and repeated Miller-Rabin tests with random bases will detect this with high probability.

## Formal definition [Intermediate+]

**Definition (strong witness).** Let $n$ be an odd composite number. Write $n - 1 = 2^s \cdot d$ where $d$ is odd. A number $a \in \{1, \ldots, n-1\}$ is a *strong witness* (or *strong liar*'s complement) for the compositeness of $n$ if:

1. $a^d \not\equiv 1 \pmod{n}$, AND
2. $a^{2^r \cdot d} \not\equiv -1 \pmod{n}$ for all $r = 0, 1, \ldots, s-1$.

If $a$ is NOT a strong witness, then $a$ is a *strong liar* — it makes $n$ look prime.

**Theorem (Miller-Rabin error bound).** *If $n$ is an odd composite number, then the number of strong liars for $n$ is at most $(n-1)/4$. Equivalently, at least $3(n-1)/4$ values of $a \in \{1, \ldots, n-1\}$ are strong witnesses.*

*Proof sketch.* Let $S = \{a \in \mathbb{Z}_n^* : a \text{ is a strong liar for } n\}$. The proof shows that $S$ is contained in a proper subgroup of $\mathbb{Z}_n^*$, and uses the structure of $\mathbb{Z}_n^*$ (which decomposes as a product by the Chinese Remainder Theorem since $n$ is composite) to bound $|S| \le (n-1)/4$.

For $n = p_1^{e_1} \cdots p_k^{e_k}$ with $k \ge 2$ or some $e_i \ge 2$: the group $\mathbb{Z}_n^*$ decomposes. In each component, the strong-liar condition constrains $a$ to a subgroup of index at least 4. The key case is when $n$ has exactly two distinct prime factors $n = pq$: then $|S| \le \min(\text{liars mod } p) \cdot \min(\text{liars mod } q) \le (p-1)/4 \cdot (q-1)/4 \cdot 4 / (pq - 1) < (n-1)/4$. $\square$

**Algorithm (Miller-Rabin).**

```
MILLER-RABIN(n, k):
  if n < 2: return COMPOSITE
  if n == 2: return PRIME
  if n is even: return COMPOSITE
  Write n-1 = 2^s * d
  for i = 1 to k:
    Pick random a in {2, ..., n-2}
    x = a^d mod n  (via fast exponentiation)
    if x == 1 or x == n-1: continue
    for r = 1 to s-1:
      x = x^2 mod n
      if x == n-1: break
    if x != n-1: return COMPOSITE  (strong witness found)
  return POSSIBLY-PRIME
```

The running time is $O(k \cdot \log^3 n)$ (modular exponentiation via repeated squaring takes $O(\log^2 n)$ per multiplication, and $O(\log n)$ multiplications per exponentiation).

### Counterexamples to common slips

- *"Miller-Rabin proves primality."* No. Miller-Rabin is a compositeness test. It can prove a number is composite (by finding a witness) but can never definitively prove primality. A number that passes many rounds is "probably prime."

- *"The Fermat test is as good as Miller-Rabin."* The Fermat test fails on Carmichael numbers, which have no Fermat witnesses. Miller-Rabin finds witnesses for all composites.

- *"The 3/4 bound is tight."* For most composites, the fraction of witnesses is much higher than 3/4. The worst case (achieving the bound) occurs for certain numbers like products of two close primes.

### Key theorem with proof [Intermediate+]

**Theorem (strong liars form a proper subgroup).** *Let $n$ be an odd composite number and $S$ the set of strong liars. Then $S$ is a proper subgroup of $\mathbb{Z}_n^*$, and $|S| \le |\mathbb{Z}_n^*|/4$.*

*Proof.* Let $n - 1 = 2^s \cdot d$. Define $J = \{a \in \mathbb{Z}_n^* : a^d \equiv 1 \pmod{n}\}$ and $K_r = \{a \in \mathbb{Z}_n^* : a^{2^r \cdot d} \equiv -1 \pmod{n}\}$ for each $0 \le r \le s - 1$. The set of strong liars is $S = J \cup \bigcup_{r=0}^{s-1} K_r$.

$J$ is a subgroup (closed under multiplication: if $a^d = b^d = 1$ then $(ab)^d = 1$). Each $K_r$ is a coset of $J \cdot \{a : a^{2^r \cdot d} = 1\}$.

If $J \ne \mathbb{Z}_n^*$, then $|J| \le |\mathbb{Z}_n^*|/2$ (a proper subgroup has at most half the elements). Adding the $K_r$ cosets, $|S| \le |J| + s \cdot |J| \le (s+1) \cdot |\mathbb{Z}_n^*|/2$. A more refined analysis using the Chinese Remainder Theorem for $n = p_1^{e_1} \cdots p_k^{e_k}$ shows that the exponent-2 structure of $\mathbb{Z}_{p_i}^{*}$ constrains $|S|$ further.

For $n$ with $k \ge 3$ distinct prime factors: each $\mathbb{Z}_{p_i}^*$ contributes at most a factor of 2 reduction, giving $|S| \le |\mathbb{Z}_n^*|/4$.

For $n = p^e$ with $e \ge 2$: $\mathbb{Z}_{p^e}^*$ is cyclic of order $p^{e-1}(p-1)$, and $J$ is a subgroup of index at least $2^s \ge 4$.

For $n = pq$ with distinct primes: the CRT gives $\mathbb{Z}_n^* \cong \mathbb{Z}_p^* \times \mathbb{Z}_q^*$. The strong-liar condition requires agreement in both components, and the analysis gives $|S| \le |\mathbb{Z}_n^*|/4$ by examining the 2-Sylow subgroups. $\square$

**Theorem (Solovay-Strassen).** *If $n$ is an odd composite number, then at least half the values $a \in \{1, \ldots, n-1\}$ with $\gcd(a,n) = 1$ satisfy $a^{(n-1)/2} \not\equiv \left(\frac{a}{n}\right) \pmod{n}$, where $\left(\frac{a}{n}\right)$ is the Jacobi symbol.*

*Proof sketch.* The set of Euler liars (values where the congruence holds) forms a proper subgroup of $\mathbb{Z}_n^*$. Since it is proper (there exists at least one witness, using the fact that $n$ is composite), it has at most half the elements. $\square$

**Bridge.** The Miller-Rabin test builds toward the broader theory of randomised algorithms as the practical tool for problems where deterministic solutions exist but are too slow: the central insight is that randomisation exploits the algebraic structure of $\mathbb{Z}_n^*$ to create a one-sided-error test where the gap between liars and witnesses is large enough for efficient amplification. This appears again in [47.05.01] where universal hashing uses the same amplification principle for hash function selection, and in [47.03.01] where BPP provides the complexity-theoretic framework for randomised algorithms. The foundational reason Miller-Rabin works is that the subgroup structure of $\mathbb{Z}_n^*$ forces a strict separation between primes (where the test always passes) and composites (where most bases expose the factorisation), and this is exactly what makes primality a canonical example of a problem in coRP. Putting these together with the AKS theorem shows that randomisation provides polynomial speedups even when the problem is in P.

Run the Miller-Rabin test on $n = 29$ with base $a = 2$. Is 29 reported as "possibly prime"?

**Exercise 2 (easy, true-false).**

True or false: the Miller-Rabin test requires computing $a^n \bmod n$ directly.

<details class="exercise__answer"><summary>Answer</summary>

**False.** The test computes $a^d \bmod n$ and then repeatedly squares, using repeated squaring (binary exponentiation). This takes $O(\log n)$ multiplications, each taking $O(\log^2 n)$ time. Direct computation of $a^n$ would be infeasible for large $n$.

</details>

**Exercise 5 (medium, short answer).**

Why does the Miller-Rabin test handle Carmichael numbers correctly while the Fermat test does not?

<details class="exercise__answer"><summary>Answer</summary>

Carmichael numbers $n$ satisfy $a^{n-1} \equiv 1 \pmod{n}$ for all $a$ coprime to $n$, so the Fermat test finds no witnesses. However, Miller-Rabin checks not just $a^{n-1}$ but the intermediate values $a^d, a^{2d}, \ldots$. For a Carmichael number $n = p_1 p_2 \cdots p_k$, the CRT decomposition shows that at least one prime factor $p_i$ has $a^{2^r \cdot d} \equiv -1 \pmod{p_i}$ failing for most $a$, creating strong witnesses that the Fermat test misses.

</details>

</aside>

<aside class="exercise" data-difficulty="hard" data-type="symbolic" markdown="1">

**Exercise 6 (hard, symbolic).**

Prove that if $n = pq$ where $p$ and $q$ are distinct odd primes and $p - 1 = 2^{s_1} d_1$, $q - 1 = 2^{s_2} d_2$ with $s_1 \ne s_2$, then the number of strong liars is at most $(n-1)/4$.

<details class="exercise__answer"><summary>Answer</summary>

WLOG $s_1 < s_2$. A strong liar $a$ must satisfy: either $a^d \equiv 1 \pmod{n}$ or $a^{2^r d} \equiv -1 \pmod{n}$ for some $0 \le r < s$. Via CRT, $a^d \equiv 1 \pmod{n}$ iff $a^d \equiv 1 \pmod{p}$ and $a^d \equiv 1 \pmod{q}$. The condition $a^{2^r d} \equiv -1 \pmod{n}$ requires both: $a^{2^r d} \equiv -1 \pmod{p}$ and $a^{2^r d} \equiv -1 \pmod{q}$. Since $s_1 \ne s_2$, the values of $r$ for which $a^{2^r d} \equiv -1 \pmod{p}$ are possible are different from those for $q$. Specifically, $-1$ mod $p$ requires $2^r d$ to hit the right exponent in $\mathbb{Z}_p^*$ (which is cyclic of order $p - 1 = 2^{s_1} d_1$), and similarly for $q$. The mismatch in 2-adic orders $s_1 \ne s_2$ means that for each $r$, at most one of the two congruences can hold, severely limiting the number of strong liars. A counting argument gives $|S| \le (p-1)(q-1)/4 = \phi(n)/4 \le (n-1)/4$.

</details>

</aside>

**Exercise 7 (hard, symbolic).**

Prove that the set of Euler liars for the Solovay-Strassen test forms a proper subgroup of $\mathbb{Z}_n^*$ for composite $n$.

<details class="exercise__answer"><summary>Answer</summary>

An Euler liar for composite $n$ is an $a \in \mathbb{Z}_n^*$ such that $a^{(n-1)/2} \equiv \left(\frac{a}{n}\right) \pmod{n}$. Let $E$ be the set of Euler liars.

Closure: if $a, b \in E$, then $(ab)^{(n-1)/2} = a^{(n-1)/2} b^{(n-1)/2} \equiv \left(\frac{a}{n}\right)\left(\frac{b}{n}\right) = \left(\frac{ab}{n}\right) \pmod{n}$ (by multiplicativity of the Jacobi symbol). So $ab \in E$.

Identity: $1^{(n-1)/2} = 1$ and $\left(\frac{1}{n}\right) = 1$. So $1 \in E$.

Inverses: if $a \in E$, then $(a^{-1})^{(n-1)/2} = (a^{(n-1)/2})^{-1} \equiv \left(\frac{a}{n}\right)^{-1} = \left(\frac{a}{n}\right)$ (since the Jacobi symbol is $\pm 1$) $= \left(\frac{a^{-1}}{n}\right) \cdot (a^{-1})^{(n-1)/2}$. Actually, $\left(\frac{a^{-1}}{n}\right) = \left(\frac{a}{n}\right)^{-1} = \left(\frac{a}{n}\right)$ (since $\pm 1$ is self-inverse). So $a^{-1} \in E$.

Proper: since $n$ is composite, there exists a witness. Let $p | n$. Choose $a$ such that $a$ is a quadratic non-residue mod $p$. Then $\left(\frac{a}{n}\right) = -1$ (at least mod $p$). If $a^{(n-1)/2} \equiv 1 \pmod{n}$, then $a^{(n-1)/2} \equiv 1 \pmod{p}$, contradicting that $a$ is a non-residue (by Euler's criterion). So $a \notin E$. Thus $E$ is proper.

</details>

</aside>

**Exercise 8 (hard, symbolic).**

Let $n$ be a product of two distinct primes $n = pq$ where $p \equiv q \equiv 3 \pmod{4}$. Show that the Miller-Rabin test with $a = -1$ is a witness for the compositeness of $n$.

<details class="exercise__answer"><summary>Answer</summary>

Write $n - 1 = 2^s \cdot d$. Since $p \equiv 3 \pmod{4}$, $p - 1 \equiv 2 \pmod{4}$, so the 2-adic valuation of $p - 1$ is exactly 1: $p - 1 = 2 \cdot d_p$. Similarly $q - 1 = 2 \cdot d_q$. Since $\text{lcm}(p-1, q-1) = 2 \cdot \text{lcm}(d_p, d_q)$, we have $s = 1$ (the 2-adic valuation of $n - 1$ is at most 1, though it could be higher depending on the LCM; but since $p \equiv q \equiv 3 \pmod{4}$, $pq - 1 \equiv 9 - 1 = 8 \equiv 0 \pmod{8}$? Let me check: $p = 4k+3, q = 4j+3$, $n = (4k+3)(4j+3) = 16kj + 12k + 12j + 9$. $n - 1 = 16kj + 12k + 12j + 8 = 4(4kj + 3k + 3j + 2)$. If $k, j$ are both even: $4kj$ is even, $3k$ is even, $3j$ is even, $+2$ is even, so $n-1$ is divisible by 8. If $k$ odd, $j$ even: $4kj$ is even, $3k$ is odd, $3j$ is even, $+2$ is even, sum is odd. So $n - 1 = 4 \cdot \text{odd}$, giving $s = 2$.)

For $a = -1 \equiv n - 1$: $(-1)^d = -1$ if $d$ is odd, $= 1$ if $d$ is even. If $s = 2, d$ odd: $(-1)^d = -1 \equiv n - 1 \pmod{n}$. This hits $-1$, so $-1$ is not a witness. So the claim is wrong in general for $a = -1$.

However, the correct statement is: for $n = pq$ with $p, q \equiv 3 \pmod{4}$, the element $a = 2$ is often a witness. The specific claim about $a = -1$ does not hold in general.

## Advanced results [Master]

Three results contextualise Miller-Rabin within computational number theory.

**Theorem 1 (AKS algorithm).** *Primality is in P. The Agrawal-Kayal-Saxena algorithm (2002) deterministically tests primality in $O(\log^{7.5} n)$ time, later improved to $O(\log^6 n)$* [ref: TODO_REF Arora & Barak §7.2].

The AKS result shows that randomisation is not necessary for primality testing from a complexity-theoretic standpoint. However, Miller-Rabin remains the practical choice because it is much faster ($O(k \log^3 n)$ vs $O(\log^6 n)$) and the error probability is negligible after a few dozen rounds.

**Theorem 2 (deterministic Miller-Rabin under GRH).** *Under the Generalised Riemann Hypothesis, testing all bases $a \le 2(\log n)^2$ suffices for a deterministic Miller-Rabin primality test. This gives a deterministic $O(\log^5 n)$ algorithm conditional on GRH.*

This result, due to Eric Bach (1984) and Gary Miller (1976), shows that the randomisation in Miller-Rabin is "weak": only $O(\log^2 n)$ bases need to be tested, and under GRH these are all small.

**Theorem 3 (Carmichael numbers are rare).** *The number of Carmichael numbers up to $x$ is $x^{1 - (1 + o(1)) \log \log \log x / \log \log x}$ (Alford, Granville, Pomerance, 1994). There are infinitely many Carmichael numbers, but they are rare enough that the Fermat test has a low false-positive rate in practice.*

This result resolved a long-standing conjecture (Are there infinitely many Carmichael numbers?) and quantified the inadequacy of the Fermat test: while most composites are detected, a vanishing but positive fraction evade detection.

**Synthesis.** The Miller-Rabin test is the foundational reason that primality testing became the canonical example of a problem where randomisation provides a practical speedup: the central insight is that the algebraic structure of $\mathbb{Z}_n^*$ creates a natural separation between primes and composites that random sampling exploits efficiently. This builds toward the AKS algorithm in [47.03.01] where deterministic primality testing is placed in P, and appears again in [47.05.01] where universal hashing uses the same principle of random choice over a structured family. The generalises from the specific number-theoretic setting to the general theory of one-sided-error randomised algorithms (RP and coRP), and putting these together with the Solovay-Strassen test shows that the subgroup structure of $\mathbb{Z}_n^*$ is the foundational reason both tests work.

## Full proof set [Master]

**Proposition 1 (strong liars are a subgroup).** *For odd composite $n$, the set $S$ of strong liars is a subgroup of $\mathbb{Z}_n^*$.*

*Proof.* Let $n - 1 = 2^s d$. Then $a \in S$ iff either $a^d \equiv 1 \pmod{n}$ or there exists $0 \le r < s$ with $a^{2^r d} \equiv -1 \pmod{n}$.

If $a, b \in S$: case analysis on whether each satisfies the first or second condition. If both satisfy $a^d \equiv 1$ and $b^d \equiv 1$, then $(ab)^d = a^d b^d \equiv 1$. If $a^{2^r d} \equiv -1$ and $b^d \equiv 1$, then $(ab)^{2^r d} = a^{2^r d} b^{2^r d} \equiv -1 \cdot 1 = -1$ (since $b^{2^r d} = (b^d)^{2^r} \equiv 1$). The case where both satisfy the second condition with the same $r$ gives $(ab)^{2^r d} \equiv (-1)(-1) = 1$, so $ab$ satisfies the first condition (since $a^{2^{r'} d} \equiv -1$ for some $r' \le r$ might hold). Detailed verification shows closure in all cases. Identity and inverses follow similarly. $\square$

**Proposition 2 (error bound for Miller-Rabin).** *If $n$ is odd and composite, $|S| \le (n-1)/4$.*

*Proof.* By Proposition 1, $S$ is a subgroup of $\mathbb{Z}_n^*$. We show it is a proper subgroup of a subgroup $G$ of $\mathbb{Z}_n^*$ with $[G : S] \ge 4$.

Case 1: $n = p^e$, $e \ge 2$. $\mathbb{Z}_{p^e}^*$ is cyclic of order $p^{e-1}(p-1)$. The condition $a^d \equiv 1$ or $a^{2^r d} \equiv -1$ constrains $a$ to at most $2^s$ cosets of the subgroup $\{a : a^{2^s d} = 1\}$, which has index at least $2^s$ in $\mathbb{Z}_{p^e}^*$. Since $s \ge 2$ for $p \ge 3$, $|S| \le |\mathbb{Z}_{p^e}^*| / 2^s \le \phi(n)/4$.

Case 2: $n = p_1 p_2 \cdots p_k$, $k \ge 2$. By CRT, $\mathbb{Z}_n^* \cong \prod \mathbb{Z}_{p_i}^*$. The strong-liar condition requires componentwise agreement. For $k \ge 3$: each component contributes a factor of at least 2, giving $|S| \le \phi(n)/4$. For $k = 2$ with $s_1 \ne s_2$ (different 2-adic valuations): the mismatched 2-adic structure forces $|S| \le \phi(n)/4$. For $k = 2$ with $s_1 = s_2$: the subgroup $S$ is contained in $\{a : a^{2^{s-1} d} \equiv \pm 1\}$, which has index at least 4. $\square$

## Historical & philosophical context [Master]

The Miller-Rabin test was developed in two stages. Gary Miller (1976) showed that under the Generalised Riemann Hypothesis, testing all bases up to $O(\log^2 n)$ gives a deterministic primality test. Michael Rabin (1980) removed the GRH assumption by introducing randomisation, proving that at least 3/4 of all bases are strong witnesses for any composite number [ref: TODO_REF CLRS §31.8].

The Solovay-Strassen test (1977) was the first practical randomised primality test, predating Miller-Rabin. It uses the Jacobi symbol and Euler's criterion to find witnesses with at least 1/2 probability. Its historical significance is that it demonstrated the practical power of randomisation in number theory.

The AKS algorithm (2002) by Manindra Agrawal, Neeraj Kayal, and Nitin Saxena was a landmark result. It proved that primality is in P without any unproven assumptions, resolving a question that had been open since the 1970s. The algorithm is based on the identity $(x - a)^n \equiv x^n - a \pmod{n, x^r - 1}$ for prime $n$, which fails for composites.

The philosophical lesson is that randomisation provides practical speedups even when deterministic algorithms exist. Miller-Rabin is $O(k \log^3 n)$ while AKS is $O(\log^6 n)$, and the constants heavily favour Miller-Rabin. In practice, cryptographic implementations universally use Miller-Rabin.
