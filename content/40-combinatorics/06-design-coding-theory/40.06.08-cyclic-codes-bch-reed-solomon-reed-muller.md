## Intuition [Beginner]

A code is a chosen collection of allowed strings, spread far apart so that a few flipped symbols cannot carry one allowed string all the way to another. Most codes are just lists with no further pattern. A **cyclic code** adds one extra rule of remarkable power: if a string is allowed, then so is the string you get by sliding every symbol one step to the right and wrapping the last symbol around to the front. Rotate a codeword and you land on another codeword. This single symmetry, repeated, ties the whole code together and makes it cheap to build and to decode in real hardware.

The trick that unlocks the symmetry is to stop thinking of a string of length $n$ as a flat list and instead read it as the coefficients of a polynomial. The string $a_0 a_1 a_2$ becomes $a_0 + a_1 x + a_2 x^2$. Sliding the symbols one step and wrapping around turns out to be the same as multiplying that polynomial by $x$ and folding $x^n$ back to $1$. So a cyclic code is a family of polynomials closed under multiplication by $x$, and one special polynomial — the **generator** — divides every codeword and so describes the whole code at a glance.

Why care? Because choosing the generator's roots lets you order up a code to spec. Want to correct three errors? Plant a run of related roots and a clean count guarantees the codewords stay far apart. This is how the codes on a scratched compact disc, in a QR square, and on a probe past Pluto are made: Reed-Solomon and BCH codes, both cyclic, both built by choosing roots.

## Visual [Beginner]

Picture a codeword as beads on a ring rather than on a straight wire. A cyclic shift just rotates the ring by one bead; the cyclic-code rule says every rotation of an allowed pattern is again allowed. So codewords come in whole rotation families, and the code is built from these spinning necklaces rather than from isolated strings.

The table shows the engine: shifting a string is multiplying its polynomial by $x$, and when a power reaches $x^7$ it folds back to $1$. A cyclic code is exactly a set of polynomials you can multiply by $x$ and stay inside.

## Worked example [Beginner]

Build a small cyclic code of length $7$ over the two-symbol alphabet $\{0,1\}$ by hand. Start from the generator polynomial $g(x) = 1 + x + x^3$, a divisor of $x^7 - 1$. Codewords are the multiples of $g$ that still have length $7$, namely $g(x)$ times a message polynomial of degree at most $3$.

Step 1. Read off the generator as a string. The polynomial $1 + x + x^3$ has coefficient $1$ at positions $0$, $1$, and $3$, so $g$ is the string $1101000$.

Step 2. Make a codeword by multiplying. Take the message $m(x) = 1 + x$, the string $1100000$. Multiply: $(1 + x)(1 + x + x^3) = 1 + x + x^3 + x + x^2 + x^4 = 1 + x^2 + x^3 + x^4$, since $x + x = 0$ in this alphabet. That is the string $1011100$.

Step 3. Check the cyclic-shift rule. Slide $1011100$ right by one with wraparound to get $0101110$. As a polynomial this is $x + x^3 + x^4 + x^5$, which equals $x \cdot (1 + x^2 + x^3 + x^4)$ folded — and it is again a multiple of $g$, so it is another codeword.

Step 4. Count the codewords. The message has $4$ free coefficients, so there are $2^4 = 16$ codewords. This is the $[7,4]$ Hamming code in cyclic dress.

What this tells us: one short generator polynomial, by multiplication alone, produces an entire code that is automatically closed under rotation. Picking a different generator with cleverly chosen roots is how engineers dial in exactly how many errors the code will fix.

## Formal definition [Intermediate+]

Throughout, $\mathbb{F}_q$ is the finite field with $q$ elements (its structure is developed in [21.02.01]; here it is applied), $q$ a prime power, and $\gcd(n, q) = 1$ so that $x^n - 1$ has $n$ distinct roots in a splitting field. Words of length $n$ are identified with polynomials of degree below $n$ via $a = (a_0, \dots, a_{n-1}) \leftrightarrow a(x) = \sum_{i=0}^{n-1} a_i x^i$, inside the quotient ring $R_n = \mathbb{F}_q[x]/(x^n - 1)$. The Hamming metric and the parameters $[n,k,d]_q$ are those of [40.06.06].

**Definition (cyclic code).** A linear code $C \subseteq \mathbb{F}_q^n$ is **cyclic** if it is closed under the cyclic shift $(a_0, \dots, a_{n-1}) \mapsto (a_{n-1}, a_0, \dots, a_{n-2})$. Under the identification above, this shift is multiplication by $x$ in $R_n$, so $C$ is cyclic exactly when $C$ is an **ideal** of $R_n$.

**Definition (generator and check polynomials).** Since $R_n$ is a principal ideal ring, every cyclic code $C$ is generated by a unique monic divisor $g(x) \mid x^n - 1$ of least degree, the **generator polynomial**; then $C = \langle g(x) \rangle$, $\dim C = n - \deg g$, and a generator matrix has rows the shifts $g(x), x g(x), \dots, x^{k-1} g(x)$ with $k = n - \deg g$. The **check polynomial** is $h(x) = (x^n - 1)/g(x)$; a word $c(x)$ lies in $C$ iff $c(x) h(x) \equiv 0 \pmod{x^n - 1}$, and $C^\perp$ is the cyclic code with generator the reciprocal of $h$.

**Definition (defining set).** Fix a primitive $n$-th root of unity $\alpha$ in an extension $\mathbb{F}_{q^m}$ (so $m$ is the multiplicative order of $q$ modulo $n$). The **defining set** of $C$ is
$$
T = \{ i \in \mathbb{Z}_n : g(\alpha^i) = 0 \},
$$
the exponents of the roots of $g$ among the powers of $\alpha$. Because $g$ has coefficients in $\mathbb{F}_q$, the set $T$ is a union of **$q$-cyclotomic cosets** $C_s = \{ s, sq, sq^2, \dots \} \pmod n$, and $c(x) \in C$ iff $c(\alpha^i) = 0$ for all $i \in T$.

**Definition (BCH and Reed-Solomon codes).** The **BCH code** of length $n$ and designed distance $\delta$ over $\mathbb{F}_q$ is the cyclic code whose defining set contains a run of $\delta - 1$ consecutive exponents $b, b+1, \dots, b + \delta - 2$; equivalently $g$ is the least-degree polynomial over $\mathbb{F}_q$ vanishing at $\alpha^b, \dots, \alpha^{b+\delta-2}$, namely the least common multiple of their minimal polynomials. It is **narrow-sense** when $b = 1$ and **primitive** when $n = q^m - 1$. A **Reed-Solomon code** is the special case $n = q - 1$, $\alpha$ a primitive element of $\mathbb{F}_q$ itself ($m = 1$): then each minimal polynomial is linear, $g(x) = \prod_{i=1}^{\delta - 1}(x - \alpha^i)$ has degree $\delta - 1$, and the code is $[\,q-1,\ q-\delta,\ \delta\,]_q$.

**Definition (Reed-Muller code).** For $0 \le r \le m$ the **Reed-Muller code** $\mathrm{RM}(r,m)$ is the binary code of length $2^m$ obtained by evaluating every Boolean polynomial in $m$ variables of degree at most $r$ at all $2^m$ points of $\mathbb{F}_2^m$:
$$
\mathrm{RM}(r,m) = \{ (f(v))_{v \in \mathbb{F}_2^m} : f \in \mathbb{F}_2[x_1,\dots,x_m],\ \deg f \le r \},
$$
with dimension $\sum_{i=0}^{r} \binom{m}{i}$ and minimum distance $2^{m-r}$.

### Counterexamples to common slips [Intermediate+]

- *The generator is not any divisor — it is the monic divisor of least degree generating the ideal.* Two different divisors of $x^n - 1$ generate different codes; the dimension is $n - \deg g$, so a larger $g$ means a smaller code.

- *The BCH bound uses a run of consecutive exponents, not just $\delta - 1$ roots.* Any $\delta - 1$ roots scattered arbitrarily give no distance guarantee; the consecutive-run hypothesis is what powers the Vandermonde argument. Conversely the **true** minimum distance can exceed the designed distance $\delta$.

- *Reed-Solomon needs $n \le q$.* Because the evaluation points are field elements (and the roots of the generator are powers of a primitive element of $\mathbb{F}_q$ itself), $n = q - 1$ for the cyclic form and $n \le q$ for the extended/evaluation form; you cannot have a length-$255$ Reed-Solomon code over $\mathbb{F}_2$ — it lives over $\mathbb{F}_{256}$.

- *$\mathrm{RM}(r,m)$ is graded by degree, not by a single monomial.* The codewords come from all polynomials up to degree $r$, so $\mathrm{RM}(0,m)$ is the repetition code, $\mathrm{RM}(m,m)$ is the whole space, and $\mathrm{RM}(m-1,m)$ is the even-weight (parity-check) code.

## Key theorem with proof [Intermediate+]

The structural engine of designed-distance codes is the BCH bound: a run of consecutive zeros of the generator forces a lower bound on minimum distance, proved by a Vandermonde determinant. From it the Reed-Solomon MDS optimality follows at once.

**Theorem (BCH bound).** *Let $C$ be a cyclic code of length $n$ over $\mathbb{F}_q$ with defining set $T$, and suppose $T$ contains $\delta - 1$ consecutive exponents $b, b+1, \dots, b+\delta-2$. Then the minimum distance satisfies $d(C) \ge \delta$* [ref: TODO_REF Bose-Ray-Chaudhuri 1960].

*Proof.* Let $\alpha$ be the fixed primitive $n$-th root of unity, so $\alpha^b, \alpha^{b+1}, \dots, \alpha^{b+\delta-2}$ are roots of every codeword polynomial. Let $c(x) = \sum_{j} c_j x^j$ be a nonzero codeword and write its support $\mathrm{supp}(c) = \{ j_1, \dots, j_w \}$ with $w = \mathrm{wt}(c)$; the nonzero coefficients are $c_{j_1}, \dots, c_{j_w}$. The condition $c(\alpha^{b+\ell}) = 0$ for $\ell = 0, \dots, \delta - 2$ reads
$$
\sum_{s=1}^{w} c_{j_s}\, (\alpha^{j_s})^{\,b+\ell} = 0, \qquad \ell = 0, 1, \dots, \delta - 2.
$$
Set $\beta_s = \alpha^{j_s}$ and $d_s = c_{j_s} \beta_s^{\,b}$; the $\beta_s$ are distinct powers of $\alpha$, and each $d_s \ne 0$. The equations become $\sum_{s=1}^{w} d_s\, \beta_s^{\,\ell} = 0$ for $\ell = 0, \dots, \delta - 2$. Suppose, for contradiction, that $w \le \delta - 1$. Take the first $w$ of these equations ($\ell = 0, \dots, w-1$); in matrix form they are $V \mathbf{d} = 0$, where $V = (\beta_s^{\,\ell})_{\ell, s}$ is a $w \times w$ Vandermonde matrix on the distinct nodes $\beta_1, \dots, \beta_w$. Its determinant is $\prod_{s < t}(\beta_t - \beta_s) \ne 0$, so $V$ is invertible and $\mathbf{d} = 0$, contradicting $d_s \ne 0$. Hence $w \ge \delta$, and since $C$ is linear $d(C) = \min_{c \ne 0} \mathrm{wt}(c) \ge \delta$. $\square$

**Corollary (Reed-Solomon codes are MDS).** *The Reed-Solomon code with $n = q - 1$ and generator $g(x) = \prod_{i=1}^{\delta-1}(x - \alpha^i)$ is an $[\,q-1,\ q-\delta,\ \delta\,]_q$ code meeting the Singleton bound* [ref: TODO_REF Reed-Solomon 1960]. Here $\deg g = \delta - 1$, so $k = n - (\delta - 1) = q - \delta$, and the defining set is the consecutive run $\{1, \dots, \delta - 1\}$, giving $d \ge \delta$ by the BCH bound. The Singleton bound of [40.06.06] gives $d \le n - k + 1 = (q-1) - (q - \delta) + 1 = \delta$. The two inequalities force $d = \delta = n - k + 1$, so the code is maximum distance separable.

**Bridge.** The BCH bound builds toward every designed-distance construction by converting "plant $\delta - 1$ consecutive roots" into the certificate $d \ge \delta$, and it appears again in the master tier as the spectral skeleton of Reed-Solomon decoding, where the same Vandermonde structure becomes the key equation of Berlekamp-Massey. The foundational reason a consecutive run works — and scattered roots do not — is that consecutiveness is exactly the hypothesis making the coefficient matrix a Vandermonde on distinct nodes, so non-vanishing of $\prod_{s<t}(\beta_t - \beta_s)$ is the whole proof. This is dual to the Singleton bound of [40.06.06]: Singleton caps $d$ from above by the rank of a parity-check matrix, the BCH bound props $d$ up from below by the rank of a Vandermonde, and putting these together pins Reed-Solomon to the MDS boundary where the two meet. The central insight, that a code's distance is read off the *zeros* of its generator rather than its coefficients, generalises into the design-theoretic Assmus-Mattson theorem, where the placement of dual zeros below the redundancy forces the fixed-weight codewords into $t$-designs.

**Exercise 4 (medium, symbolic).**

Prove the $(u, u+v)$ construction for Reed-Muller codes: $\mathrm{RM}(r, m) = \{ (u,\ u+v) : u \in \mathrm{RM}(r, m-1),\ v \in \mathrm{RM}(r-1, m-1) \}$, and use it to derive the dimension recursion $\dim \mathrm{RM}(r,m) = \dim \mathrm{RM}(r,m-1) + \dim \mathrm{RM}(r-1,m-1)$.

<details class="exercise__hint"><summary>Hint</summary>

Split the variables as $x_1, \dots, x_{m-1}$ and $x_m$. Write any degree-$\le r$ polynomial as $f(x_1,\dots,x_{m-1}) + x_m\, h(x_1,\dots,x_{m-1})$ and read off the degrees of $f$ and $h$. Evaluate on the two halves $x_m = 0$ and $x_m = 1$.

</details>

<details class="exercise__answer"><summary>Answer</summary>

Index the $2^m$ points of $\mathbb{F}_2^m$ by $(\mathbf{w}, x_m)$ with $\mathbf{w} \in \mathbb{F}_2^{m-1}$, listing the $x_m = 0$ half first. A Boolean polynomial of degree $\le r$ is uniquely $f(\mathbf{x}') + x_m\, h(\mathbf{x}')$ with $\mathbf{x}' = (x_1,\dots,x_{m-1})$, where $\deg f \le r$ and $\deg h \le r - 1$ (the factor $x_m$ already contributes degree $1$). Evaluating: on the half $x_m = 0$ the codeword is $\mathrm{ev}(f) =: u \in \mathrm{RM}(r,m-1)$; on the half $x_m = 1$ it is $\mathrm{ev}(f) + \mathrm{ev}(h) = u + v$ with $v = \mathrm{ev}(h) \in \mathrm{RM}(r-1,m-1)$. Thus every codeword has the form $(u, u+v)$, and conversely each such pair arises from a unique $(f,h)$, giving the stated equality. Since $u$ and $v$ range independently over the two codes, $\dim \mathrm{RM}(r,m) = \dim \mathrm{RM}(r,m-1) + \dim \mathrm{RM}(r-1,m-1)$, which unrolls to $\sum_{i=0}^{r}\binom{m}{i}$ by Pascal's identity. Rubric: full credit for the unique split $f + x_m h$ with correct degree bounds, the two-half evaluation giving $(u, u+v)$, and the independent-range dimension count.

</details>

</aside>

**Exercise 5 (medium, numeric).**

A Reed-Solomon code over $\mathbb{F}_{256}$ is used on a compact disc with $n = 255$, $k = 251$. How many symbol errors can it correct, and how many symbol erasures can it recover?

<details class="exercise__hint"><summary>Hint</summary>

It is MDS, so $d = n - k + 1$. Errors corrected is $\lfloor (d-1)/2\rfloor$; erasures recovered is $d - 1$ (each erasure costs one unit of distance, not two).

</details>

<details class="exercise__answer"><summary>Answer</summary>

$d = n - k + 1 = 255 - 251 + 1 = 5$. Errors corrected: $\lfloor (5-1)/2 \rfloor = 2$ symbol errors. Erasures recovered: $d - 1 = 4$ erasures (known-location losses). Because each symbol is a byte over $\mathbb{F}_{256}$, a single symbol error absorbs a whole burst of up to $8$ corrupted bits, which is why Reed-Solomon excels against scratch-induced bursts on CDs. Rubric: full credit for $d = 5$, $2$ errors, $4$ erasures.

</details>

</aside>

**Exercise 6 (medium, multiple choice).**

Which statement about the Reed-Muller duality $\mathrm{RM}(r,m)^\perp = \mathrm{RM}(m-r-1, m)$ is correct?

A. $\mathrm{RM}(1,m)$ is self-dual for all $m$  
B. $\mathrm{RM}(m-1, m)$, the even-weight code, is dual to $\mathrm{RM}(0,m)$, the repetition code  
C. The duality fails when $m$ is even  
D. $\mathrm{RM}(r,m)$ is always self-dual

<details class="exercise__hint"><summary>Hint</summary>

Set $r = m-1$ in the duality formula: $m - r - 1 = m - (m-1) - 1 = 0$. The repetition code $\mathrm{RM}(0,m)$ and the even-weight code $\mathrm{RM}(m-1,m)$ are classic duals.

</details>

<details class="exercise__answer"><summary>Answer</summary>

**B.** Setting $r = m - 1$ gives $\mathrm{RM}(m-1,m)^\perp = \mathrm{RM}(0,m)$: the even-weight (single parity-check) code is dual to the repetition code, the standard dual pair. Feedback-wrong: $\mathrm{RM}(r,m)$ is self-dual only when $2r + 1 = m$, i.e. $m = 2r+1$ (option D is false in general, A holds only for $m=3$), and the duality holds for every $m$ (option C false).

</details>

**Exercise 7 (hard, symbolic).**

Prove that $\mathrm{RM}(r,m)$ has minimum distance exactly $2^{m-r}$. Show the lower bound by induction on $m$ via the $(u, u+v)$ construction, and exhibit a weight-$2^{m-r}$ codeword.

<details class="exercise__hint"><summary>Hint</summary>

For the weight bound use $\mathrm{wt}(u, u+v) \ge \min(2\,\mathrm{wt}(u), \mathrm{wt}(v))$ when $v \ne 0$, plus the inductive distances $2^{(m-1)-r}$ and $2^{(m-1)-(r-1)}$. For the achieving word evaluate the monomial $x_1 x_2 \cdots x_r$.

</details>

<details class="exercise__answer"><summary>Answer</summary>

Induct on $m$. Base $m = r$: $\mathrm{RM}(m,m)$ is the whole space, distance $1 = 2^0$; and $\mathrm{RM}(0,m)$ is the repetition code, distance $2^m$. Inductive step: a nonzero codeword of $\mathrm{RM}(r,m)$ is $(u, u+v)$ with $u \in \mathrm{RM}(r,m-1)$, $v \in \mathrm{RM}(r-1,m-1)$. If $v = 0$ then the word is $(u,u)$ with $u \ne 0$, of weight $2\,\mathrm{wt}(u) \ge 2 \cdot 2^{(m-1)-r} = 2^{m-r}$. If $v \ne 0$, then since $\mathrm{wt}(u + v) \ge \mathrm{wt}(v) - \mathrm{wt}(u)$ and $\mathrm{wt}(u) + \mathrm{wt}(u+v) \ge \mathrm{wt}(v)$,
$$
\mathrm{wt}(u, u+v) = \mathrm{wt}(u) + \mathrm{wt}(u+v) \ge \mathrm{wt}(v) \ge 2^{(m-1)-(r-1)} = 2^{m-r}.
$$
Either way the weight is at least $2^{m-r}$, so $d \ge 2^{m-r}$. For equality, the monomial $f = x_1 x_2 \cdots x_r$ vanishes unless $x_1 = \dots = x_r = 1$, leaving the $x_{r+1}, \dots, x_m$ free: exactly $2^{m-r}$ points evaluate to $1$. So $\mathrm{ev}(x_1\cdots x_r)$ has weight $2^{m-r}$, and $d = 2^{m-r}$. Rubric: full credit for both cases of the weight bound, the inductive distances, and the achieving monomial.

</details>

</aside>

<aside class="exercise" data-difficulty="hard" data-type="symbolic" markdown="1">

**Exercise 8 (hard, symbolic).**

Show that a primitive narrow-sense binary BCH code of length $n = 2^m - 1$ with designed distance $\delta = 2t+1$ has dimension $k \ge n - mt$, and explain why the bound can be loose.

<details class="exercise__hint"><summary>Hint</summary>

The generator is the lcm of the minimal polynomials of $\alpha, \alpha^2, \dots, \alpha^{2t}$. Even powers repeat earlier cosets, so only $\alpha, \alpha^3, \dots, \alpha^{2t-1}$ contribute new minimal polynomials, each of degree $\le m$.

</details>

<details class="exercise__answer"><summary>Answer</summary>

The defining set must contain $\{1, 2, \dots, 2t\}$. Over $\mathbb{F}_2$, each even exponent $2j$ lies in the same cyclotomic coset as $j$ (since squaring is the Frobenius and $2j \equiv 2 \cdot j$), so the distinct cosets needed are those of the $t$ odd exponents $1, 3, 5, \dots, 2t-1$. Each minimal polynomial $m_i(x)$ has degree equal to the coset size, at most $m$. Hence $\deg g = \deg \mathrm{lcm}(m_1, m_3, \dots, m_{2t-1}) \le mt$, giving $k = n - \deg g \ge n - mt$. The bound is loose because (i) some cosets are smaller than $m$ (when the order of $2$ modulo the relevant divisor is less than $m$), shrinking $\deg g$; and (ii) distinct odd exponents can share a coset, so fewer than $t$ minimal polynomials may actually occur. The true dimension is read off the exact cyclotomic-coset partition, and the true minimum distance often exceeds $\delta$. Rubric: full credit for the even-exponent coset collapse, the $t$ odd contributors of degree $\le m$, the bound $k \ge n - mt$, and at least one reason for looseness.

</details>

</aside>

## Advanced results [Master]

The cyclic framework unifies the named families and culminates in a design-theoretic statement. The results below trace the BCH-as-subfield-subcode-of-Reed-Solomon picture, the affine-geometric face of Reed-Muller, majority-logic decoding, and the Assmus-Mattson theorem that turns weight classes into $t$-designs.

**Theorem 1 (BCH is a subfield subcode of Reed-Solomon).** Let $\mathrm{RS}_{q^m}$ be the Reed-Solomon code over $\mathbb{F}_{q^m}$ with the same length $n \mid q^m - 1$ and the same consecutive defining set $\{b, \dots, b + \delta - 2\}$. The BCH code over $\mathbb{F}_q$ with that designed distance is exactly the **subfield subcode**
$$
\mathrm{BCH}_q = \mathrm{RS}_{q^m} \cap \mathbb{F}_q^n,
$$
the codewords of the Reed-Solomon code all of whose coordinates already lie in $\mathbb{F}_q$ [ref: TODO_REF MacWilliams-Sloane Ch. 9]. The minimum distance of $\mathrm{BCH}_q$ is at least that of $\mathrm{RS}_{q^m}$, namely $\ge \delta$, recovering the BCH bound; the dimension drops because the subfield constraint is $m$ linear conditions per Reed-Solomon check, encoded by the cyclotomic cosets. This is the conceptual home of the BCH bound: a BCH code inherits its distance from an ambient MDS code and pays for the smaller alphabet in dimension.

**Theorem 2 (Reed-Muller as affine geometry and as a punctured-cyclic code).** The codewords of $\mathrm{RM}(r,m)$ of minimum weight $2^{m-r}$ are exactly the indicator vectors of the $(m-r)$-dimensional **affine subspaces** of $\mathbb{F}_2^m$, the flats of the affine geometry $\mathrm{AG}(m,2)$ [ref: TODO_REF MacWilliams-Sloane Ch. 13]. Punctured at one coordinate, $\mathrm{RM}(r,m)$ minus the all-zero point becomes a cyclic code of length $2^m - 1$ whose defining set is described by the $2$-adic weights of exponents (the Kasami-Lin-Peterson characterisation): $\alpha^j$ is a zero iff the binary digit sum of $j$ is less than $m - r$. So Reed-Muller codes are cyclic up to puncturing, and the affine-geometry codes of points-versus-flats incidence realise them combinatorially.

**Theorem 3 (Reed's majority-logic decoding).** $\mathrm{RM}(r,m)$ decodes by **majority logic** in $r+1$ rounds [ref: TODO_REF Reed 1954]. Each degree-$r$ monomial coefficient is recovered by a set of $2^{m-r}$ disjoint parity-check sums (one per coset of the complementary subspace), each an unbiased estimate of that coefficient; a majority vote over the $2^{m-r}$ sums corrects up to $2^{m-r-1} - 1$ errors, half the minimum distance. Subtracting the recovered top-degree part reduces to $\mathrm{RM}(r-1,m)$, and the rounds cascade down to the constant term. The decoder is fully parallel and was used on the Mariner deep-space probes (the $[32,6,16]$ code $\mathrm{RM}(1,5)$).

**Theorem 4 (generalized Reed-Solomon and the evaluation form).** Fix distinct $a_1, \dots, a_n \in \mathbb{F}_q$ and nonzero multipliers $v_1, \dots, v_n$. The **generalized Reed-Solomon code** is $\mathrm{GRS}_k = \{ (v_1 f(a_1), \dots, v_n f(a_n)) : \deg f < k \}$, an $[n, k, n-k+1]_q$ MDS code for any $n \le q$ [ref: TODO_REF Reed-Solomon 1960]. Its dual is again a GRS code (with multipliers from the Lagrange/derivative weights of the $a_i$), so the GRS family is closed under duality, and the narrow-sense cyclic Reed-Solomon code is the case $a_i = \alpha^{i}$, $v_i = 1$. The evaluation form makes the erasure-recovery property of [40.06.06] manifest: any $k$ coordinates determine $f$ by Lagrange interpolation.

**Theorem 5 (Assmus-Mattson theorem).** Let $C$ be an $[n,k,d]_q$ code with dual $C^\perp$ of minimum distance $d^\perp$. Fix $t < d$, and let $s$ be the number of nonzero weights of $C^\perp$ in the range $\{1, 2, \dots, n - t\}$. If $s \le d - t$, then for every weight $w$ with $d \le w \le n$, the supports of the weight-$w$ codewords of $C$ form a $t$-design (a $t$-$(n, w, \lambda_w)$ design for a suitable $\lambda_w$), and the same holds for $C^\perp$ in the analogous range [ref: TODO_REF Assmus-Mattson 1969]. Applied to the extended binary Golay code $\mathcal{G}_{24}$, whose dual is itself and whose weights are $0, 8, 12, 16, 24$: taking $t = 5$, the dual weights in $\{1, \dots, 19\}$ are $8, 12, 16$, so $s = 3 = d - t = 8 - 5$, and the weight-$8$ codewords (the $759$ octads) form a $5$-$(24, 8, 1)$ design — the Steiner system $S(5,8,24)$ of [40.06.04]. The ternary $\mathcal{G}_{12}$ yields $S(5,6,12)$ the same way.

**Synthesis.** Putting these together, the algebra of zeros, the geometry of flats, and the combinatorics of designs are three readings of one object pinned by the generator polynomial. The foundational reason Reed-Solomon sits exactly on the MDS boundary is that its zeros form a maximal consecutive run, so the BCH lower bound and the Singleton upper bound of [40.06.06] meet — this is exactly the rank-meets-Vandermonde coincidence of the Key theorem, and every BCH code is the shadow of such an MDS code over a larger field, inheriting distance and paying in dimension. This is dual, in the precise MacWilliams sense of [40.06.06], to the weight-enumerator picture: the Assmus-Mattson hypothesis is a statement about how few nonzero *dual* weights fall below the redundancy, and the central insight is that this scarcity forces the Pless power moments to be independent of which $t$-subset one fixes, which is precisely the defining balance of a $t$-design. The bridge from codes to designs runs both ways — the Golay codes *produce* the Steiner systems $S(5,8,24)$ and $S(5,6,12)$ of [40.06.04], and those designs reconstruct the codes — so the perfect-code arithmetic of [40.06.07], the design-graph dictionary of [40.06.09], and the Hadamard codes of [40.06.05] are corners of a single design-code-graph triangle, and Reed-Muller's first-order code $\mathrm{RM}(1,m)$ is exactly the Hadamard code that sits on the triangle's coding vertex.

## Full proof set [Master]

**Proposition 1 (structure of cyclic codes).** *The ideals of $R_n = \mathbb{F}_q[x]/(x^n-1)$ are exactly the principal ideals $\langle g(x) \rangle$ with $g$ a monic divisor of $x^n - 1$; the corresponding code has dimension $n - \deg g$, and $\dim C + \dim C^\perp = n$ with $C^\perp$ cyclic.*

*Proof.* $R_n$ is a quotient of the principal ideal domain $\mathbb{F}_q[x]$, hence every ideal is the image of an ideal $\langle f \rangle \supseteq \langle x^n - 1 \rangle$ of $\mathbb{F}_q[x]$, i.e. $\langle g \rangle$ with $g \mid x^n - 1$ monic of least degree in the ideal. The set $\{ g, xg, \dots, x^{k-1}g \}$ with $k = n - \deg g$ is $\mathbb{F}_q$-linearly independent (distinct leading degrees below $n$) and spans the ideal modulo $x^n - 1$, so $\dim C = k$. For the dual, with $h = (x^n-1)/g$ the map $c \mapsto c \cdot h \bmod (x^n - 1)$ has kernel $C$ and image of dimension $\deg g$, and a coordinate computation identifies $C^\perp$ with $\langle \tilde h \rangle$, where $\tilde h(x) = x^{\deg h} h(1/x)$ is the reciprocal of $h$; thus $C^\perp$ is cyclic of dimension $\deg g = n - k$, and $\dim C + \dim C^\perp = n$ as in the rank-nullity statement of [40.06.06]. $\square$

**Proposition 2 (BCH bound via Vandermonde).** *A cyclic code whose defining set contains $\delta - 1$ consecutive exponents has $d \ge \delta$.*

*Proof.* As in the Key theorem, a nonzero codeword $c$ of weight $w$ with support $\{j_1, \dots, j_w\}$ satisfies $\sum_{s=1}^w c_{j_s} (\alpha^{j_s})^{b+\ell} = 0$ for $\ell = 0, \dots, \delta - 2$, where $\alpha^b, \dots, \alpha^{b + \delta - 2}$ are the consecutive zeros. Writing $\beta_s = \alpha^{j_s}$ (distinct) and $d_s = c_{j_s}\beta_s^b$ (nonzero), the first $w$ equations form a $w \times w$ Vandermonde system $V\mathbf{d}=0$ with $\det V = \prod_{s<t}(\beta_t - \beta_s) \ne 0$. If $w \le \delta - 1$ the system has enough equations to force $\mathbf{d} = 0$, contradicting $d_s \ne 0$; hence $w \ge \delta$. $\square$

**Proposition 3 (Reed-Solomon MDS and dual).** *The Reed-Solomon code $[\,n, k, n-k+1\,]_q$ ($n \le q$) is MDS, and its dual is a generalized Reed-Solomon code, hence also MDS.*

*Proof.* In the evaluation form $C = \{ (f(a_1), \dots, f(a_n)) : \deg f < k \}$ a nonzero $f$ of degree $< k$ has at most $k - 1$ roots among the distinct $a_i$, so at least $n - (k-1) = n - k + 1$ coordinates are nonzero; thus $d \ge n - k + 1$, and Singleton of [40.06.06] gives $d \le n - k + 1$, so $d = n - k + 1$ and $C$ is MDS. For the dual, the bilinear form pairs $C$ with the evaluations of degree-$<n-k$ polynomials against the Lagrange-derivative weights $u_i = \prod_{j \ne i}(a_i - a_j)^{-1}$: one checks $\sum_i u_i f(a_i) g(a_i) = 0$ whenever $\deg f < k$ and $\deg g < n-k$, because $\sum_i u_i a_i^\ell = 0$ for $0 \le \ell \le n - 2$ (a Lagrange-interpolation identity for the coefficient of the top power). Hence $C^\perp = \{ (u_i g(a_i))_i : \deg g < n - k \}$, a GRS code with multipliers $u_i$, MDS of distance $k + 1$. $\square$

**Proposition 4 (Reed-Muller dimension, distance, and duality).** *$\dim \mathrm{RM}(r,m) = \sum_{i=0}^r \binom{m}{i}$, $d = 2^{m-r}$, and $\mathrm{RM}(r,m)^\perp = \mathrm{RM}(m-r-1,m)$.*

*Proof.* The evaluation map sends the $\sum_{i \le r}\binom{m}{i}$ squarefree monomials of degree $\le r$ to linearly independent vectors (a Boolean function on $\mathbb{F}_2^m$ has a unique multilinear representative, since $x^2 = x$), giving the dimension. The distance is $2^{m-r}$ by the $(u,u+v)$ induction of Exercise 7. For duality, take $f$ of degree $\le r$ and $g$ of degree $\le m - r - 1$; the product $fg$ has degree $\le m - 1$, so the sum of $fg$ over all of $\mathbb{F}_2^m$ vanishes (a Boolean polynomial of degree $< m$ has even weight, as the top monomial $x_1\cdots x_m$ is the only odd-weight one). Hence $\langle \mathrm{ev}(f), \mathrm{ev}(g)\rangle = \sum_v f(v)g(v) = 0$, so $\mathrm{RM}(m-r-1,m) \subseteq \mathrm{RM}(r,m)^\perp$. The dimensions match: $\sum_{i \le m-r-1}\binom{m}{i} = 2^m - \sum_{i \le r}\binom{m}{i}$ by the symmetry $\binom{m}{i} = \binom{m}{m-i}$, equal to $2^m - \dim \mathrm{RM}(r,m)$. Equality of dimensions forces $\mathrm{RM}(r,m)^\perp = \mathrm{RM}(m-r-1,m)$. $\square$

**Proposition 5 (Assmus-Mattson theorem).** *With $C$, $t < d$, and $s = |\{ \text{nonzero weights of } C^\perp \text{ in } [1, n-t] \}| \le d - t$, the weight-$w$ supports of $C$ form $t$-designs for all $d \le w \le n$.*

*Proof.* Fix a $t$-subset $T$ of coordinates and let $A_w(T)$ count weight-$w$ codewords whose support contains $T$. The Pless power-moment identities express, for each $\nu \le t$, the symmetric sums $\sum_w \binom{n - \nu}{w - \nu} A_w$ in terms of the dual weight distribution $A_j^\perp$ for $j \le \nu \le t$. The shortened code $C_T$ (codewords zero on $T$) and the punctured dual relate the local counts $A_w(T)$ to the global dual spectrum through the MacWilliams transform of [40.06.06]. The hypothesis $s \le d - t$ means the dual has at most $d - t$ nonzero weights below $n - t + 1$; this is exactly the number of free parameters in the linear system whose unknowns are the $A_w(T)$ and whose equations are the power moments for $\nu = 0, \dots, t$. The system then has a unique solution *independent of $T$* — the count $A_w(T)$ depends only on $|T| = t$, not on which $t$-set $T$ is chosen. A weight-$w$ class in which every $t$-subset lies in the same number $\lambda_w = A_w(T)$ of supports is, by definition, a $t$-$(n, w, \lambda_w)$ design. The dual statement follows by symmetry of the hypothesis. $\square$

## Connections [Master]

- The co-produced unit [40.06.07] on perfect and Golay codes constructs both Golay codes as cyclic (quadratic-residue) codes — the binary $\mathcal{G}_{23}$ from a degree-$11$ factor of $x^{23}-1$ over $\mathbb{F}_2$, the ternary $\mathcal{G}_{11}$ from a factor of $x^{11}-1$ over $\mathbb{F}_3$ — exactly the generator-polynomial mechanism of this unit. The Assmus-Mattson theorem proved here is what turns the Golay weight enumerators computed there into the $5$-designs; the two units share the same codes seen from the cyclic-structure and the perfect-packing sides.

- The Steiner systems of [40.06.04] are the output of the Assmus-Mattson theorem applied to the Golay codes: the $759$ octads form $S(5,8,24)$ and the $132$ hexads form $S(5,6,12)$. The foundational reason a code can carry a $5$-design is the dual-weight scarcity hypothesis — few nonzero dual weights below the redundancy — and this is exactly the condition that forces the fixed-weight supports into the balanced incidence structure that [40.06.04] studies axiomatically.

- The arithmetic of the alphabet is the finite-field theory of [21.02.01]: the defining set of a cyclic code is a union of $q$-cyclotomic cosets because the generator has coefficients in $\mathbb{F}_q$ while its roots live in $\mathbb{F}_{q^m}$, and Reed-Solomon codes evaluate polynomials at the primitive element of $\mathbb{F}_q$ itself, so the length is bounded by $q - 1$. This is dual to the additive-character structure used in [40.06.06] for the MacWilliams identity, on which the Assmus-Mattson proof rests.

- The first-order Reed-Muller code $\mathrm{RM}(1,m)$ is the Hadamard / biorthogonal code of [40.06.05]: its $2^{m+1}$ codewords are the rows of a $2^m \times 2^m$ Hadamard matrix and their complements, with all nonzero weights $2^{m-1}$, so the Paley-Hadamard constructions there and the affine-geometry codes here describe the same equidistant object. Together with the strongly regular graphs of [40.06.09], this closes the design-code-graph triangle that organises the whole chapter.

## Historical & philosophical context [Master]

Cyclic codes crystallized in the late 1950s once Eugene Prange recast codes as ideals in $\mathbb{F}_q[x]/(x^n-1)$, making the generator polynomial the central object. The multiple-error-correcting family was found independently by Alexis Hocquenghem in 1959 [ref: TODO_REF MacWilliams-Sloane Ch. 9] and by Raj Chandra Bose and Dwijendra Kumar Ray-Chaudhuri in 1960 [ref: TODO_REF Bose-Ray-Chaudhuri 1960]; their initials name the BCH codes, and the BCH bound — a consecutive run of zeros forces a distance — is the construction's engine.

Irving Reed and Gustave Solomon published their polynomial codes in 1960 [ref: TODO_REF Reed-Solomon 1960], a four-page paper in the *Journal of the SIAM* defining codes by evaluating low-degree polynomials over a finite field. The maximum-distance-separable optimality made them the workhorse of digital storage and transmission: compact discs, DVDs, QR codes, RAID arrays, and the Voyager probes all carry Reed-Solomon codes. The earlier Reed-Muller codes came from David Muller's 1954 Boolean-function construction [ref: TODO_REF Muller 1954] and Irving Reed's companion 1954 majority-logic decoder [ref: TODO_REF Reed 1954]; the $[32,6,16]$ first-order code flew on Mariner 9 to return the first close images of Mars.

The design-theoretic capstone is the Assmus-Mattson theorem of Edward Assmus and Harold Mattson in 1969 [ref: TODO_REF Assmus-Mattson 1969], which gave a purely coding-theoretic sufficient condition for the fixed-weight codewords to form $t$-designs, recovering the Mathieu $5$-designs from the Golay codes without group theory. The theorem tied error-correcting codes to the design theory of [40.06.04] and to the sporadic simple groups, and remains a primary source of $t$-designs with large $t$.
