## Intuition [Beginner]

Finding the eigenvalues of a full square matrix is hard work, and the machine that does it, the QR algorithm, has to grind through many rounds. Each round costs roughly the cube of the matrix size, so for a dense matrix the whole job is expensive. There is a cheaper way to set things up first.

The idea is to comb the matrix into a tidier shape before the eigenvalue machine ever runs. We squeeze almost all of its entries down to zero, leaving only a narrow band near the diagonal. A general matrix gets combed into a shape where everything below the first slot under the diagonal is zero; this is called Hessenberg form. A symmetric matrix combs down even further, to a thin three-stripe band called tridiagonal form.

The combing has to be done carefully, because we are not allowed to change the eigenvalues. We use a special kind of rotation, a reflection, applied on both sides of the matrix at once. Applying the same reflection on the left and on the right is a change of viewpoint that leaves every eigenvalue exactly where it was. So we get a matrix with the same eigenvalues but a far simpler shape, and the eigenvalue machine then races through it.

## Visual [Beginner]

The picture shows a five-by-five matrix before and after combing. On the left every cell can be nonzero, a full block of numbers. On the right almost the whole lower triangle has been zeroed out, leaving a staircase: the diagonal, the one stripe just below it, and everything above. That staircase is the Hessenberg shape.

```
   Full matrix            Upper-Hessenberg form
  x x x x x                x x x x x
  x x x x x                x x x x x
  x x x x x      ===>      0 x x x x
  x x x x x                0 0 x x x
  x x x x x                0 0 0 x x

   Symmetric matrix        Tridiagonal form
  x x x x x                x x 0 0 0
  x x x x x                x x x 0 0
  x x x x x      ===>      0 x x x 0
  x x x x x                0 0 x x x
  x x x x x                0 0 0 x x
```

Two things are worth seeing. First, the combed matrix keeps the same eigenvalues as the full one, because each combing step is a balanced two-sided reflection. Second, the symmetric case lands on the thinnest band of all, a single stripe on each side of the diagonal, because zeroing a column of a symmetric matrix zeroes the matching row for free.

## Worked example [Beginner]

Take the symmetric three-by-three matrix

$$
A = \begin{pmatrix} 4 & 1 & 2 \\ 1 & 3 & 0 \\ 2 & 0 & 5 \end{pmatrix}.
$$

We want to clear the entry $2$ in the bottom-left corner, the one that sits below the first subdiagonal in column one. We will use a reflection acting on the last two coordinates to fold the vector $(1, 2)$ formed by the two below-diagonal entries of column one onto a single axis.

Step 1. Look at the part of column one below the diagonal: the numbers $1$ and $2$. Their combined length is $\sqrt{1^2 + 2^2} = \sqrt{5} \approx 2.236$. A reflection on these two coordinates can rotate $(1, 2)$ onto $(2.236, 0)$, putting all the weight into the first of the two slots and clearing the second.

Step 2. Build the reflection $H$ that does this, and form the new matrix $H A H$. Applying $H$ on the left clears the bottom-left entry; applying the same $H$ on the right keeps the eigenvalues unchanged and, because $A$ is symmetric, clears the top-right entry to match.

Step 3. The result is the tridiagonal matrix

$$
\begin{pmatrix} 4 & 2.236 & 0 \\ 2.236 & 4.6 & 1.2 \\ 0 & 1.2 & 3.4 \end{pmatrix},
$$

where the corner entries are now zero and only the diagonal and its two neighbouring stripes survive.

What this tells us: one reflection turned a full symmetric matrix into a three-stripe band, and the eigenvalues are unchanged because we reflected on both sides. A larger matrix needs a handful more reflections, one per column, but the principle is the same.

## Formal definition [Intermediate+]

Let $A \in M_n(\mathbb{F})$ with $\mathbb{F} \in \{\mathbb{R}, \mathbb{C}\}$. A matrix $H = (h_{ij})$ is **upper-Hessenberg** when $h_{ij} = 0$ for all $i > j + 1$; that is, the only possibly-nonzero entries below the main diagonal lie on the first subdiagonal. A matrix $T$ is **tridiagonal** when $t_{ij} = 0$ for $|i - j| > 1$, so it is upper-Hessenberg and lower-Hessenberg at once. A Hessenberg matrix is **unreduced** when every subdiagonal entry is nonzero, $h_{i+1, i} \neq 0$ for $1 \leq i \leq n-1$.

A **Householder reflector** is a unitary matrix of the form $H = I - 2 v v^* / (v^* v)$ for a nonzero vector $v$, in the sense of [01.01.09]; it is Hermitian and unitary, $H = H^* = H^{-1}$, and it reflects across the hyperplane $v^\perp$. Given any vector $x$, a Householder reflector can be chosen so that $Hx = \alpha e_1$ with $|\alpha| = \|x\|$, collapsing all but the first coordinate of $x$ onto a single axis.

Two matrices $A$ and $B$ are **similar** when $B = S^{-1} A S$ for an invertible $S$; they share the same characteristic polynomial and hence the same eigenvalues with the same algebraic multiplicities. When $S = Q$ is unitary, $B = Q^* A Q$ is a **unitary** (orthogonal, in the real case) **similarity**, and it preserves not only the spectrum but the Hermitian structure: if $A = A^*$ then $(Q^* A Q)^* = Q^* A Q$.

**Definition (Hessenberg reduction).** The Hessenberg reduction of $A$ is a unitary $Q$ with $Q^* A Q = H$ upper-Hessenberg. It is built as a product $Q = Q_1 Q_2 \cdots Q_{n-2}$ of Householder reflectors: $Q_k$ acts as the identity on the first $k$ coordinates and, applied as the two-sided similarity $A \mapsto Q_k^* A Q_k$, zeroes the entries of column $k$ below row $k+1$. When $A = A^*$ the same construction yields $Q^* A Q = T$ **tridiagonal**, because each reflector preserves symmetry and the cleared subcolumn forces a cleared superrow.

Notation: $e_1$ is the first standard basis vector; $\mathrm{fl}(\cdot)$ denotes the computed (floating-point) result of an operation; $\|\cdot\|$ without subscript is the Euclidean / spectral norm; $\epsilon_{\mathrm{mach}}$ is the unit roundoff. The reflector $Q_k$ is identified with the embedded $(n-k)\times(n-k)$ reflection $\mathrm{diag}(I_k, \tilde H_k)$ acting on coordinates $k+1, \ldots, n$.

### Counterexamples to common slips

- The reflector must act as a **two-sided** similarity $Q_k^* A Q_k$, not a one-sided multiplication. A one-sided $Q_k^* A$ would zero the subcolumn but change the eigenvalues; the right multiplication by $Q_k$ restores the spectrum and is why the cleared region is below the *first* subdiagonal, not below the diagonal — the right multiply refills the entries in row $k+1$ that a triangularising left-only sweep would have cleared.
- One cannot zero the entry $h_{k+1, k}$ on the first subdiagonal: the reflector $Q_k$ is built to act on coordinates $k+1$ through $n$, so it can collapse the subcolumn $(a_{k+1,k}, \ldots, a_{n,k})$ onto its first slot $a_{k+1,k}$ but cannot remove that slot. This is the structural reason finite Householder reduction stops at Hessenberg, not at triangular form.
- Tridiagonal form requires $A = A^*$. For a non-symmetric $A$, clearing the subcolumn does not clear the matching superrow, so the upper triangle stays full and the result is only Hessenberg.

## Key theorem with proof [Intermediate+]

**Theorem (orthogonal reduction to Hessenberg/tridiagonal form; Trefethen-Bau Lecture 26 [ref: TODO_REF]; Golub-Van Loan §7.4.3 [ref: TODO_REF]).** *Every $A \in M_n(\mathbb{F})$ admits a unitary $Q$, a product of at most $n-2$ Householder reflectors, with $Q^* A Q = H$ upper-Hessenberg. If $A = A^*$ then $Q^* A Q = T$ is tridiagonal. The reduction uses $O(n^3)$ arithmetic operations — $\tfrac{10}{3}n^3$ for the general case and $\tfrac{4}{3}n^3$ for the symmetric case — and $A$, $H$, $T$ all share the same spectrum.*

**Proof.** Construct $Q$ one column at a time. Suppose after $k-1$ steps the matrix $A_{k-1} = Q_{k-1}^* \cdots Q_1^* A Q_1 \cdots Q_{k-1}$ is upper-Hessenberg in its first $k-1$ columns. Write the $k$-th column below the diagonal as $x = (a_{k+1,k}, \ldots, a_{n,k})^\top \in \mathbb{F}^{n-k}$. Choose a Householder reflector $\tilde H_k$ on $\mathbb{F}^{n-k}$ with $\tilde H_k x = \alpha e_1$, $|\alpha| = \|x\|$, and set $Q_k = \mathrm{diag}(I_k, \tilde H_k)$, the identity on the first $k$ coordinates.

Left-multiplication $Q_k^* A_{k-1}$ replaces the subcolumn $x$ by $\alpha e_1$, zeroing rows $k+2, \ldots, n$ of column $k$ while leaving the diagonal and subdiagonal entries in rows $1, \ldots, k+1$ intact. Right-multiplication by $Q_k$ acts only on columns $k+1, \ldots, n$ — because $Q_k$ fixes the first $k$ coordinates — so it does not disturb columns $1, \ldots, k$ and in particular preserves the zeros just created in column $k$. Thus $A_k = Q_k^* A_{k-1} Q_k$ is upper-Hessenberg in its first $k$ columns. After $k = n-2$ steps every column below the first subdiagonal is zero, so $A_{n-2} = H$ is upper-Hessenberg, with $Q = Q_1 \cdots Q_{n-2}$.

When $A = A^*$, each $A_k = Q_k^* A_{k-1} Q_k$ is again Hermitian, since the $Q_k$ are unitary and Hermitian conjugation commutes with the similarity. A Hermitian upper-Hessenberg matrix is tridiagonal: $h_{ij} = 0$ for $i > j+1$ forces $h_{ji} = \overline{h_{ij}} = 0$ for $j > i+1$, so both bands vanish beyond the first off-diagonals. Hence $H = T$ is tridiagonal.

The operation count: step $k$ applies $\tilde H_k$ to an $(n-k) \times (n-k)$ trailing block from both sides, costing $O((n-k)^2)$ per side; summing $\sum_{k=1}^{n-2} 4(n-k)^2 = \tfrac{4}{3}n^3 + O(n^2)$ in the symmetric case (only the lower band is touched) and $\tfrac{10}{3}n^3$ in the general case where both the trailing rows and columns are updated. Each $A_k$ is unitarily similar to $A$, so the characteristic polynomials agree and the spectra coincide. $\square$

**Bridge.** This construction builds toward the QR algorithm of [43.06.03], where the Hessenberg form produced here is the standing input that makes each QR sweep cost $O(n^2)$ instead of $O(n^3)$; the band structure appears again in the Arnoldi iteration of [43.07.02], whose orthonormal Krylov basis produces exactly a Hessenberg matrix $H_m = Q_m^* A Q_m$, the same object reached here by direct reflectors rather than by iteration. The foundational reason the reduction stops at Hessenberg and not at triangular form is the Abel/Galois obstruction: a finite rational-plus-root algorithm cannot in general expose the eigenvalues, so the best a finite orthogonal procedure can do is the cheapest non-triangular band, and this is exactly the division of labour between a finite direct phase and an infinite iterative phase. The two-sided similarity that preserves the spectrum here generalises the change-of-basis invariance of eigenvalues, and putting these together, the reduction is the bridge from the static spectral theory of [01.01.08] to the production eigensolvers: it converts a dense problem into a banded one at one-time $O(n^3)$ cost, after which the iteration is nearly free.

**Exercise 3 (medium, symbolic).**

Show that if $A = A^*$ and $Q$ is unitary, then $Q^* A Q$ is Hermitian. Deduce that the Hessenberg reduction of a Hermitian matrix is tridiagonal.

<details class="exercise__hint"><summary>Hint</summary>

Take the conjugate transpose of $Q^* A Q$, and use $A = A^*$ and $(Q^*)^* = Q$. Then combine "Hermitian" with "upper-Hessenberg".

</details>

<details class="exercise__answer"><summary>Answer</summary>

Compute $(Q^* A Q)^* = Q^* A^* (Q^*)^* = Q^* A^* Q = Q^* A Q$, using $A^* = A$ and $(Q^*)^* = Q$, so $Q^* A Q$ is Hermitian. Now suppose $H = Q^* A Q$ is also upper-Hessenberg, so $h_{ij} = 0$ whenever $i > j + 1$. Hermitian symmetry gives $h_{ji} = \overline{h_{ij}}$, so for $j > i + 1$ we have $h_{ji} = \overline{h_{ij}} = \overline{0} = 0$: the superdiagonal band beyond the first stripe also vanishes. Hence $h_{ij} = 0$ for all $|i - j| > 1$, which is tridiagonality. Rubric: full credit for the Hermitian computation and the both-bands-vanish deduction.

</details>

<aside class="exercise" data-difficulty="medium" data-type="symbolic" markdown="1">

**Exercise 4 (medium, symbolic).**

Explain precisely why the Householder reflector $Q_k$ used at step $k$ must act as the identity on the first $k$ coordinates. What goes wrong if it acts on coordinate $k$ as well?

<details class="exercise__hint"><summary>Hint</summary>

The right-multiplication by $Q_k$ touches exactly the columns on which $Q_k$ acts nontrivially. Which previously-cleared zeros would be disturbed if $Q_k$ acted on column $k$?

</details>

<details class="exercise__answer"><summary>Answer</summary>

Right-multiplication $A_{k-1} Q_k$ forms linear combinations of exactly those columns on which $Q_k$ acts nontrivially. If $Q_k$ acted on coordinate $k$, the right-multiply would mix column $k$ into the others, refilling the subdiagonal zeros already placed in columns $1, \ldots, k-1$, and the reduction would not progress. By fixing the first $k$ coordinates, $Q_k$ leaves columns $1, \ldots, k$ untouched on the right while its left action clears rows $k+2, \ldots, n$ of column $k$. The price of this restriction is that $Q_k$ can only collapse the subcolumn onto row $k+1$, not row $k$, so the entry $h_{k+1,k}$ survives — this is why the reduction halts at Hessenberg rather than triangular form. Rubric: full credit for identifying the refill problem and connecting the coordinate restriction to the Hessenberg stopping point.

</details>

</aside>

<aside class="exercise" data-difficulty="hard" data-type="symbolic" markdown="1">

**Exercise 7 (hard, short-answer).**

Prove that the upper-Hessenberg structure is preserved by one step of the QR algorithm: if $H$ is upper-Hessenberg with QR factorisation $H = QR$, then $\hat H = RQ$ is again upper-Hessenberg.

<details class="exercise__hint"><summary>Hint</summary>

$Q = H R^{-1}$, and $R^{-1}$ is upper-triangular. An upper-Hessenberg matrix times an upper-triangular matrix is upper-Hessenberg; so is upper-triangular times upper-Hessenberg.

</details>

<details class="exercise__answer"><summary>Answer</summary>

From $H = QR$ with $R$ invertible upper-triangular, $Q = H R^{-1}$. The product of an upper-Hessenberg matrix $H$ with an upper-triangular matrix $R^{-1}$ is upper-Hessenberg: entry $(i,j)$ of $H R^{-1}$ is $\sum_m h_{im} (R^{-1})_{mj}$, and $(R^{-1})_{mj} = 0$ for $m > j$ while $h_{im} = 0$ for $i > m + 1$, so a nonzero term needs $i \leq m + 1 \leq j + 1$, giving zero whenever $i > j + 1$. Hence $Q$ is upper-Hessenberg. Then $\hat H = R Q$ is upper-triangular times upper-Hessenberg: entry $(i,j)$ is $\sum_m r_{im} q_{mj}$ with $r_{im} = 0$ for $m < i$ and $q_{mj} = 0$ for $m > j + 1$, so nonzero terms need $i \leq m \leq j + 1$, again forcing the entry to vanish for $i > j + 1$. Therefore $\hat H$ is upper-Hessenberg, and the band structure is invariant across the iteration. Rubric: full credit for both products and the index bookkeeping.

</details>

</aside>

<aside class="exercise" data-difficulty="hard" data-type="symbolic" markdown="1">

**Exercise 8 (hard, symbolic).**

Show that an unreduced Hessenberg matrix $H$ (every subdiagonal entry nonzero) has geometric multiplicity exactly $1$ for each eigenvalue — that is, $H$ is non-derogatory.

<details class="exercise__hint"><summary>Hint</summary>

For any scalar $\lambda$, examine the rank of $H - \lambda I$ by looking at its first $n-1$ columns and the nonzero subdiagonal entries.

</details>

<details class="exercise__answer"><summary>Answer</summary>

Fix any $\lambda$ and consider $M = H - \lambda I$, still upper-Hessenberg with the same (nonzero) subdiagonal entries $h_{i+1,i}$. Delete the first row and the last column of $M$ to obtain an $(n-1) \times (n-1)$ matrix whose main diagonal consists of the subdiagonal entries $h_{2,1}, h_{3,2}, \ldots, h_{n,n-1}$, and which is upper-triangular (every entry strictly below this diagonal came from below the first subdiagonal of $M$, hence is zero). Its determinant is $\prod_{i=1}^{n-1} h_{i+1,i} \neq 0$, so this submatrix is invertible and $\mathrm{rank}(M) \geq n - 1$. Therefore $\dim \ker(H - \lambda I) = n - \mathrm{rank}(M) \leq 1$ for every $\lambda$, so each eigenvalue has geometric multiplicity exactly $1$. Rubric: full credit for the deleted-submatrix triangularity argument and the rank conclusion.

</details>

## Advanced results [Master]

**Theorem (backward stability of the orthogonal reduction; Wilkinson Ch. 6 [ref: TODO_REF]).** *Let $A \in M_n(\mathbb{F})$ and let $\hat H$ be the upper-Hessenberg matrix computed by Householder reduction in floating-point arithmetic with unit roundoff $\epsilon_{\mathrm{mach}}$. There is a unitary $\hat Q$, and a perturbation $E$, such that*

$$
\hat Q^* (A + E) \hat Q = \hat H, \qquad \|E\| = O\!\left( n^2 \epsilon_{\mathrm{mach}} \|A\| \right).
$$

The computed reduction is the exact reduction of a nearby matrix. The mechanism is that each Householder reflector, applied in floating point, equals an exactly-unitary reflector applied to a slightly perturbed operand, with the perturbation bounded by a low-degree polynomial in $n$ times $\epsilon_{\mathrm{mach}}\|A\|$; composing $n-2$ such steps multiplies the bound by the number of steps and keeps the accumulated $\hat Q$ exactly unitary to working precision. Because the backward error $E$ is of the order of the unavoidable rounding of the input, the reduction adds no instability of its own: the eigenvalues of $\hat H$ are the eigenvalues of a matrix within $O(n^2 \epsilon_{\mathrm{mach}}\|A\|)$ of $A$, and by the perturbation theory of [43.01.02] they are accurate to the conditioning of the eigenproblem. This is the precise sense in which orthogonal similarities are the right tool: a non-orthogonal reduction with a large condition number $\kappa(S)$ would magnify $E$ by $\kappa(S)$, whereas a unitary $\hat Q$ has $\kappa(\hat Q) = 1$.

**Theorem (the implicit-Q / uniqueness theorem; Golub-Van Loan §7.4.5 [ref: TODO_REF]).** *Let $A \in M_n(\mathbb{F})$ and let $Q$, $V$ be unitary with $Q^* A Q = H$ and $V^* A V = G$ both unreduced upper-Hessenberg. If the first columns agree, $Q e_1 = V e_1$, then $Q$ and $V$ agree up to a diagonal unitary sign matrix, and correspondingly $H$ and $G$ agree up to that diagonal similarity.* The unreduced Hessenberg form is therefore essentially determined by its first column. This is the structural fact the implicit (bulge-chasing) QR algorithm rests on: a shifted QR sweep need not form $H - \mu I$ explicitly; it suffices to apply the correct first transformation and then restore Hessenberg form by chasing the resulting bulge down the band, because the uniqueness theorem guarantees the result equals the explicit sweep.

**Theorem (band reduction and the operation-count hierarchy).** *For a banded Hermitian $A$ with bandwidth $b$, the Householder tridiagonalisation can be organised to cost $O(n^2 b)$ rather than $O(n^3)$ by chasing fill-in within the band; for a full Hermitian $A$ the cost is $\tfrac{4}{3}n^3$, half the general $\tfrac{10}{3}n^3$ because only one triangle is updated.* The symmetric case is cheaper at every level because the two-sided similarity acts on a self-conjugate object, so the work and the storage halve; the band-chasing organisation is the lever that turns an already-sparse input into a tridiagonal output at sub-cubic cost, and it is the same fill-control idea that governs sparse-direct factorisation.

**Synthesis.** The reduction to Hessenberg/tridiagonal form is the hinge of the two-phase eigenvalue computation, and the foundational reason it exists is the Abel/Galois obstruction: no finite rational-and-radical procedure exposes the eigenvalues of a general matrix, so the computation must split into a finite direct phase and an infinite iterative phase. This is exactly the division this unit constructs — Householder reflectors collapse the matrix onto the cheapest band a finite orthogonal procedure can reach, and the band is preserved by every QR sweep, so the iterative phase inherits a permanent $O(n^2)$-per-step (or $O(n)$ tridiagonal) cost. The two-sided unitary similarity is the central insight that makes the scheme stable: it preserves the spectrum exactly, preserves Hermitian structure, and has unit condition number, so the reduction's backward-error bound generalises the $\kappa(S)$-free stability of the Householder QR of [01.01.09]. Putting these together, the Hessenberg form is at once the output of a finite direct algorithm and the standing input of the iterative one, and the implicit-Q theorem shows the form is rigid enough — fixed by its first column — that the iteration can be driven implicitly; the bridge to the Krylov methods of chapter $43.07$ is that the same Hessenberg object arises there as $Q_m^* A Q_m$ from the Arnoldi process, so the direct reduction and the iterative projection are two routes to one structure.

## Full proof set [Master]

**Proposition (existence and finiteness of the Hessenberg reduction).** *For every $A \in M_n(\mathbb{F})$ there is a unitary $Q$, a product of at most $n-2$ Householder reflectors, with $Q^* A Q$ upper-Hessenberg.*

**Proof.** Induct on the column index $k$, with the invariant that $A_k = Q_k^* \cdots Q_1^* A Q_1 \cdots Q_k$ is upper-Hessenberg in columns $1, \ldots, k$. The base case $k = 0$ is vacuous. Assume $A_{k-1}$ is Hessenberg in its first $k-1$ columns. Let $x = (a_{k+1,k}, \ldots, a_{n,k})^\top$ be the part of column $k$ below the subdiagonal. If $x = 0$ take $Q_k = I$; otherwise choose a Householder reflector $\tilde H_k$ on $\mathbb{F}^{n-k}$ with $\tilde H_k x = -\,\mathrm{sgn}(x_1)\|x\| e_1$ — the sign chosen to avoid cancellation — and embed $Q_k = \mathrm{diag}(I_k, \tilde H_k)$.

Left-multiplication by $Q_k^*$ acts only on rows $k+1, \ldots, n$, replacing $x$ by a multiple of $e_1$ and so zeroing rows $k+2, \ldots, n$ of column $k$; rows and columns $1, \ldots, k$ are fixed, preserving the earlier Hessenberg columns. Right-multiplication by $Q_k$ acts only on columns $k+1, \ldots, n$, hence does not touch columns $1, \ldots, k$ and preserves the new zeros in column $k$. So $A_k$ is Hessenberg in columns $1, \ldots, k$. At $k = n-2$ the matrix is fully upper-Hessenberg, and $Q = Q_1 \cdots Q_{n-2}$ is unitary as a product of unitaries. $\square$

**Proposition (preservation of Hermitian structure; tridiagonal output).** *If $A = A^*$ then the reduction above yields $Q^* A Q$ tridiagonal.*

**Proof.** Each $A_k = Q_k^* A_{k-1} Q_k$ is Hermitian whenever $A_{k-1}$ is, since $(Q_k^* A_{k-1} Q_k)^* = Q_k^* A_{k-1}^* Q_k = Q_k^* A_{k-1} Q_k$, and $A_0 = A$ is Hermitian by hypothesis; by induction $H = A_{n-2}$ is Hermitian. By the previous proposition $H$ is upper-Hessenberg, so $h_{ij} = 0$ for $i > j+1$. Hermitian symmetry gives $h_{ji} = \overline{h_{ij}}$, so $h_{ji} = 0$ for $j > i+1$ as well. Both bands beyond the first off-diagonals vanish, so $|i - j| > 1 \Rightarrow h_{ij} = 0$, which is tridiagonality. $\square$

**Proposition (spectral invariance and the operation count).** *The reduction preserves the spectrum and uses $O(n^3)$ operations, with leading term $\tfrac{4}{3}n^3$ in the Hermitian case.*

**Proof.** Spectral invariance is similarity invariance: $Q^* A Q = Q^{-1} A Q$ for unitary $Q$, so $\det(Q^* A Q - \lambda I) = \det(Q^{-1}(A - \lambda I)Q) = \det(A - \lambda I)$, and the characteristic polynomials, hence the spectra with multiplicities, coincide. For the count, step $k$ applies $\tilde H_k = I - 2 v v^*/(v^* v)$ to a trailing block: a Householder application $B \mapsto \tilde H_k B = B - v(2 v^* B / v^* v)$ on an $(n-k) \times m$ block costs $4(n-k)m$ operations (one matrix-vector product and one rank-one update). In the Hermitian case only the lower triangle is stored and each step updates an $(n-k) \times (n-k)$ symmetric trailing block at cost $\approx 4(n-k)^2$ halved by symmetry to $\approx 2(n-k)^2$ for the two-sided update plus the symmetric rank-two correction; summing $\sum_{k=1}^{n-2}$ gives leading term $\tfrac{4}{3}n^3$. The general (non-Hermitian) case updates both the trailing rows and the trailing columns, doubling the symmetric work to leading term $\tfrac{10}{3}n^3$. $\square$

**Proposition (QR-step invariance of the Hessenberg band).** *If $H$ is upper-Hessenberg with QR factorisation $H = QR$ ($Q$ unitary, $R$ upper-triangular invertible), then $\hat H = RQ$ is upper-Hessenberg.*

**Proof.** Write $Q = H R^{-1}$. For an upper-Hessenberg $H$ and an upper-triangular $T$, the product $HT$ is upper-Hessenberg: $(HT)_{ij} = \sum_m h_{im} t_{mj}$, with $t_{mj} = 0$ for $m > j$ and $h_{im} = 0$ for $m < i - 1$, so a surviving term needs $i - 1 \leq m \leq j$, impossible when $i > j + 1$. Thus $Q = H R^{-1}$ is upper-Hessenberg. Likewise for upper-triangular $T$ and upper-Hessenberg $K$, the product $TK$ is upper-Hessenberg: $(TK)_{ij} = \sum_m t_{im} k_{mj}$ with $t_{im} = 0$ for $m < i$ and $k_{mj} = 0$ for $m > j+1$, so a surviving term needs $i \leq m \leq j+1$, impossible when $i > j+1$. Applying this with $T = R$, $K = Q$ gives $\hat H = RQ$ upper-Hessenberg. The same argument with the conjugate-symmetric band shows a symmetric tridiagonal $H$ stays tridiagonal under a QR step. $\square$

## Connections [Master]

The Hessenberg/tridiagonal reduction is the direct first phase of the QR algorithm of [43.06.03]: the QR algorithm is run not on the dense $A$ but on its Hessenberg reduction, where the band-invariance proposition proved here guarantees that every QR sweep returns another Hessenberg matrix, dropping the per-sweep cost from $O(n^3)$ to $O(n^2)$ and, in the symmetric tridiagonal case, to $O(n)$; without this reduction the QR algorithm would be impractical.

The Householder reflectors that perform the reduction are the same orthogonal triangularisation tool built and analysed in [01.01.09], reused here as two-sided similarities rather than one-sided factors; the backward-stability bound $\|E\| = O(\epsilon_{\mathrm{mach}}\|A\|)$ of the reduction is the $\kappa$-independent stability of Householder transformations transplanted from the QR factorisation to the eigenvalue setting.

The inverse-iteration step of [43.06.01] solves a linear system $(A - \mu I) y = x$ at every iteration, and reducing $A$ to Hessenberg or tridiagonal form first makes each of those solves cheap — a tridiagonal solve is $O(n)$ — which is why the practical eigenvalue pipeline reduces once and then iterates, sharing the reduction across the iteration just as the QR algorithm does.

The Arnoldi iteration of [43.07.02] produces the Hessenberg matrix $H_m = Q_m^* A Q_m$ as the projection of $A$ onto a Krylov subspace, and its Hermitian specialisation, the Lanczos iteration, produces a tridiagonal $T_m$; these are the same band structures reached here by direct reflectors, so the present unit is the finite-dimensional, full-spectrum counterpart of the Krylov-subspace projection that underlies GMRES and conjugate gradients.

## Historical & philosophical context [Master]

The reduction of a matrix to Hessenberg form by orthogonal similarities is due to Alston Householder, whose 1958 introduction of the elementary reflector $I - 2 v v^*/(v^* v)$ gave numerical linear algebra its sharpest tool for selectively zeroing entries while preserving norms, and whose subsequent treatise *The Theory of Matrices in Numerical Analysis* (1964) systematised the reduction as the standard preprocessing step [ref: TODO_REF Householder 1958]. Wilkinson, in *The Algebraic Eigenvalue Problem* (1965), supplied the rounding-error analysis showing that Householder's reduction is backward stable: the computed band matrix is the exact reduction of a matrix within $O(\epsilon_{\mathrm{mach}}\|A\|)$ of the original, which is what made the two-phase eigenvalue computation trustworthy in finite precision [ref: TODO_REF Wilkinson 1965].

The strategic role of the reduction — a finite direct phase followed by an infinite iterative phase — reflects the Abel-Ruffini theorem and Galois's resolution of solvability by radicals: because the eigenvalues of a general matrix of size $\geq 5$ are the roots of an arbitrary polynomial, no finite sequence of arithmetic and root operations can produce them, so the eigenvalue problem is necessarily iterative [ref: TODO_REF Abel 1826]. The reduction does the maximal finite work — banding the matrix as far as a finite orthogonal procedure allows — and hands the irreducibly iterative remainder to the QR algorithm of Francis and Kublanovskaya (1961). The tridiagonal specialisation for symmetric matrices, and its exploitation by the symmetric QR and divide-and-conquer eigensolvers, was developed extensively by Parlett, whose *The Symmetric Eigenvalue Problem* is the standard account [ref: TODO_REF Parlett 1998].
