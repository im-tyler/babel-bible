## Intuition [Beginner]

A holomorphic function is astonishingly rigid. Once you know its values on a tiny disc — equivalently, once you know its Taylor series at one point — you have, in principle, pinned down the function *everywhere* it can possibly live. The local power series is not a snapshot; it is a seed. By re-expanding it at a point near the edge of its disc you grow a new series on a new disc, and the two agree where they overlap. Repeat, and the function spreads across the plane along whatever path you walk.

This is analytic continuation: marching a local series from disc to disc. The catch, and the whole story, is that the answer can depend on the path you took. Walk a square-root function once around the origin and it comes back as its own negative; walk a logarithm around and it gains $2\pi i$. The seed is faithful, but the field it grows can fold over itself.

## Visual [Beginner]

![A chain of overlapping discs along a curve from a starting point, each disc carrying a power series that agrees with its neighbour on the overlap; a separate loop around a central dot returns to the start carrying a different value, illustrating path-dependence.](/img/analytic-continuation-placeholder.svg)

The picture has two halves. On the left, a curve runs from a start point to an end point, covered by a chain of overlapping discs; inside each disc sits a power series, and on every overlap the neighbouring series match exactly. Following the chain transports the original local function to a new place. On the right, a loop encircles a marked centre dot and returns to where it began — yet the series it carries home is *not* the one it left with. Same start, same finish, different path, different answer. That gap is the entire subject.

## Worked example [Beginner]

Start with the square-root function near the point $1$, choosing the branch with $\sqrt{1} = 1$. Its local series is well behaved on a disc around $1$. Now carry that series along a circle of radius $1$ centred at the origin, re-expanding as you go. Track the *angle* of the input: at angle $0$ the output has angle $0$; halfway round, the input angle is $\pi$ and the output angle is $\pi/2$; all the way round, the input angle is $2\pi$ but the output angle is only $\pi$. The output has rotated half as fast.

So when the input returns to $1$, the function value has become $-1$, not $1$. We came back to the same point carrying the *opposite* branch. Go around a second time and it flips back to $+1$. The origin is the special point — the branch point — that the loop encircled. Avoiding it, every loop returns you home unchanged; enclosing it once swaps the two square roots. The same loop applied to $\log z$ adds $2\pi i$ each time, and never returns, because the logarithm has infinitely many branches.

## Formal definition [Intermediate+]

A **function element** is a pair $(f, D)$ where $D \subset \mathbb{C}$ is an open disc and $f$ is holomorphic on $D$; equivalently it is a convergent power series with its disc of convergence. Two elements $(f_1, D_1)$ and $(f_2, D_2)$ are **direct analytic continuations of each other** if $D_1 \cap D_2 \neq \varnothing$ and $f_1 = f_2$ on $D_1 \cap D_2$. The **germ** of $f$ at a point $a$ is the equivalence class of elements defined near $a$ under agreement on some neighbourhood; we write $f_a$. [ref: TODO_REF Ahlfors §8.1]

Let $\gamma : [0,1] \to \mathbb{C}$ be a path. A **continuation of the germ $f_0$ along $\gamma$** is a family of germs $\{ f_{\gamma(t)} \}_{t \in [0,1]}$, one at each point $\gamma(t)$, that is *locally consistent*: for every $t_0$ there is a neighbourhood $J$ of $t_0$ in $[0,1]$ and a single function element $(g, D)$ with $\gamma(J) \subset D$ such that the germ of $g$ at $\gamma(t)$ equals $f_{\gamma(t)}$ for all $t \in J$. Intuitively, the path is covered by finitely many overlapping discs, each carrying one element, and adjacent elements agree on overlaps — a chain of function elements running along $\gamma$. The terminal germ $f_{\gamma(1)}$ is the result of the continuation. [ref: TODO_REF Conway Ch. IX]

A **complete analytic function** is a maximal connected set of germs closed under continuation along all paths: start from one germ and adjoin every germ reachable by some chain. Its **natural domain of existence** is the set of base points covered; a boundary point past which *no* element extends is a point of the **natural boundary**.

## Key theorem with proof [Intermediate+]

**Lemma (uniqueness of continuation along a fixed path).** Continuation of a given germ $f_0$ along a fixed path $\gamma$, when it exists, is unique: any two locally consistent families starting from $f_0$ agree at every $t$.

*Proof.* Let $S = \{ t : \text{the two families have the same germ at } \gamma(t) \}$. Then $0 \in S$. If $t_0 \in S$, both families are represented near $t_0$ by single elements $(g, D)$ and $(g', D')$ whose germs agree at $\gamma(t_0)$; by the identity theorem [06.01.01] $g = g'$ on the component of $D \cap D'$ containing $\gamma(t_0)$, so the germs agree on a whole subinterval — $S$ is open. The complement is open by the same local-element argument run from a point of disagreement. Since $[0,1]$ is connected, $S = [0,1]$. $\square$

**Theorem (Monodromy Theorem; Ahlfors §8.1, Thm 2).** *Let $\Omega \subset \mathbb{C}$ be open and let the germ $f_0$ at $a \in \Omega$ admit continuation along every path in $\Omega$. If $\gamma_0, \gamma_1 : [0,1] \to \Omega$ are paths from $a$ to $b$ that are homotopic in $\Omega$ (fixed endpoints), then the continuations along $\gamma_0$ and $\gamma_1$ yield the same terminal germ at $b$. In particular, if $\Omega$ is simply connected, the germ $f_0$ continues to a single-valued holomorphic function on all of $\Omega$.*

*Proof.* Let $H : [0,1] \times [0,1] \to \Omega$ be the homotopy, $H(\cdot, 0) = \gamma_0$, $H(\cdot, 1) = \gamma_1$, with $H(0,s) = a$, $H(1,s) = b$. For each $s$ write $\gamma_s = H(\cdot, s)$ and let $f^s$ denote the terminal germ of the continuation along $\gamma_s$ (which exists by hypothesis). The claim is that $s \mapsto f^s$ is locally constant, hence constant.

Fix $s_0$. By compactness of $[0,1] \times [0,1]$ the continuation along $\gamma_{s_0}$ is represented by finitely many elements $(g_k, D_k)$ over a partition $0 = t_0 < \cdots < t_n = 1$, with $\gamma_{s_0}([t_{k-1}, t_k]) \subset D_k$ and $g_k = g_{k+1}$ on the overlap. By uniform continuity of $H$ there is $\delta > 0$ so that for $|s - s_0| < \delta$ the path $\gamma_s$ also satisfies $\gamma_s([t_{k-1}, t_k]) \subset D_k$ for every $k$. Then the *same* elements $(g_k, D_k)$ form a valid continuation along $\gamma_s$: consecutive elements still agree on the (nonempty, by closeness) overlaps containing $\gamma_s(t_k)$, by the identity theorem. Hence the terminal germ along $\gamma_s$ is the germ of $g_n$ at $b$ — identical to that along $\gamma_{s_0}$. So $f^s = f^{s_0}$ for $|s - s_0| < \delta$: the map is locally constant. By connectedness $f^1 = f^0$.

For the simply connected case, all paths from $a$ to a given $b$ are homotopic, so the terminal germ depends only on $b$, not the path; defining $F(b)$ as that common value yields a single-valued holomorphic function on $\Omega$ extending $f_0$. $\square$

**Bridge.** This theorem builds toward the very definition of a Riemann surface [06.03.01], and it is exactly the statement that separates the plane from its branched covers. The foundational reason path-dependence exists is topological: in a multiply connected domain like $\mathbb{C} \setminus \{0\}$ a loop around the puncture cannot be contracted, so the monodromy theorem grants no agreement, and indeed $\sqrt z$ and $\log z$ disagree after the loop. This pattern appears again in covering-space theory [06.02.02], where the assignment "germ at the endpoint" is precisely the monodromy action of $\pi_1(\Omega)$ on the fibre of the étale space of germs; the theorem says that this action factors through homotopy. Putting these together, the central insight is that holomorphic continuation is a covering map over its base, single-valued exactly when the base is simply connected and multivalued exactly to the extent that $\pi_1$ fails to vanish — the analytic shadow of a purely topological fact.

## Advanced results [Master]

The germs of all function elements assemble into a topological space, the **sheaf of holomorphic germs** $\mathcal{O}$, whose points are pairs $(a, f_a)$ and whose topology is generated by the sets $\{(z, g_z) : z \in D\}$ for each element $(g, D)$. The projection $\pi : \mathcal{O} \to \mathbb{C}$, $(a, f_a) \mapsto a$, is a **local homeomorphism** — an *étale space* — and a continuation along $\gamma$ is exactly a *lift* of $\gamma$ to $\mathcal{O}$ starting at the chosen germ. In this language the uniqueness lemma is the unique-path-lifting property and the monodromy theorem is the homotopy-lifting property of a covering map. Each connected component of $\mathcal{O}$ is one **complete analytic function**, and its projection to $\mathbb{C}$, suitably completed at branch points, is the Riemann surface [06.03.01] on which that function lives single-valuedly. [ref: TODO_REF Conway Ch. IX]

For an *algebraic* function — a multivalued $w(z)$ satisfying $P(z, w) = 0$ for a polynomial $P$ — continuation never escapes to a natural boundary; instead it permutes the finitely many roots $w_1(z), \dots, w_d(z)$ as $z$ loops the discriminant locus, and the monodromy representation $\pi_1(\mathbb{C} \setminus \{\text{branch points}\}) \to S_d$ has finite image. The branch points [06.02.01] are exactly the loops at which the permutation is not the identity, and Riemann's existence theorem [06.02.03] inverts the construction: any such permutation representation arises from an actual algebraic curve. The transcendental case ($\log z$, with infinite-order monodromy, or $\sum z^{2^n}$, with no continuation) sits outside this finite picture and is where natural boundaries appear.

A delicate point sharpens the simply-connected conclusion: continuation may *develop singularities*. The theorem's hypothesis "$f_0$ continues along every path in $\Omega$" is essential — a germ may continue along some paths and hit a singularity along others. The function $\sqrt{z}$ continues along every path in $\mathbb{C} \setminus \{0\}$ (not simply connected, hence multivalued), but no germ of it continues along a path *through* $0$. The interplay between the topology of $\Omega$ and the location of singularities is precisely what the passage to a surface is designed to untangle.

**Synthesis.** Analytic continuation is the mechanism by which a single power-series germ determines a global object, and the monodromy theorem is the foundational reason that object is single-valued exactly when the base is simply connected. This is exactly the analytic face of covering-space theory: the sheaf of germs is an étale space, continuation is path-lifting, and the monodromy theorem is homotopy-lifting, so the multivaluedness of $\sqrt z$ and $\log z$ generalises into the monodromy representation of $\pi_1$ on a fibre. Putting these together, the central insight is that the rigidity of holomorphic functions — guaranteed by the identity theorem [06.01.01], which forces continuation along a fixed path to be unique — coexists with genuine path-dependence, and the only honest resolution is to change the domain: lift the multivalued function to its Riemann surface [06.03.01], where the monodromy is unwound and continuation becomes single-valued. The bridge from this unit to the rest of the strand is that branch points [06.02.01], branched coverings [06.02.02], and the uniformization theorem [06.03.03] are all answers to one question this unit poses — how to make the path-dependent answer of continuation into an honest function — and the sheaf of germs that continuation builds is the same sheaf whose cohomology drives Riemann-Roch and the analytic-versus-algebraic comparison.

## Full proof set [Master]

The monodromy theorem and the uniqueness lemma are proved in full in the Key theorem section. The remaining Master claims are recorded here.

**Proposition (the étale space of germs is a local homeomorphism, and continuation is path-lifting).** *Let $\mathcal{O}$ be the set of germs $(a, f_a)$ with the topology generated by the basic sets $N(g, D) = \{(z, g_z) : z \in D\}$ for function elements $(g, D)$. Then $\pi : \mathcal{O} \to \mathbb{C}$, $(a, f_a) \mapsto a$, is a local homeomorphism, and a continuation of $f_0$ along a path $\gamma$ is the same datum as a continuous lift $\tilde\gamma : [0,1] \to \mathcal{O}$ of $\gamma$ with $\tilde\gamma(0) = (a, f_0)$.*

*Proof.* On a basic set $N(g, D)$ the projection $\pi$ restricts to $(z, g_z) \mapsto z$, a bijection onto $D$ with continuous inverse $z \mapsto (z, g_z)$ (continuous because for $z'$ near $z$ the germ $g_{z'}$ lies in the same $N(g, D)$); so $\pi$ is a homeomorphism on each basic set, hence a local homeomorphism. Given a continuation $\{f_{\gamma(t)}\}$, define $\tilde\gamma(t) = (\gamma(t), f_{\gamma(t)})$. Local consistency says each $t_0$ has a neighbourhood $J$ and an element $(g, D)$ with $\tilde\gamma(J) \subset N(g, D)$ and $\pi \circ \tilde\gamma = \gamma$ there; since $\pi$ is a homeomorphism on $N(g, D)$ and $\gamma$ is continuous, $\tilde\gamma|_J$ is continuous. So $\tilde\gamma$ is a continuous lift. Conversely a continuous lift $\tilde\gamma$ has, near each $t_0$, image inside some basic $N(g, D)$; the second coordinate of $\tilde\gamma(t)$ is then the germ of $g$ at $\gamma(t)$, exhibiting local consistency. The two notions coincide. $\square$

**Proposition (single-valued extension on a simply connected domain).** *If $\Omega$ is simply connected and the germ $f_0$ at $a \in \Omega$ continues along every path in $\Omega$, then there is a unique holomorphic $F : \Omega \to \mathbb{C}$ with germ $f_0$ at $a$.*

*Proof.* For $b \in \Omega$ pick any path $\gamma$ from $a$ to $b$ and let $f_b$ be the terminal germ; by the monodromy theorem (all such $\gamma$ are homotopic in the simply connected $\Omega$) $f_b$ is independent of $\gamma$. Define $F(b)$ as the value at $b$ of any element representing $f_b$. Near $b$, fix an element $(g, D)$ representing the continuation; for $b'$ in $D$ the path $\gamma$ followed by a short segment to $b'$ continues to the germ $g_{b'}$, so $F = g$ on $D$. Thus $F$ is holomorphic near every point, hence on $\Omega$, with germ $f_0$ at $a$. Uniqueness is the identity theorem: two such extensions share the germ $f_0$ and agree on the connected $\Omega$. $\square$

## Connections [Master]

- **Holomorphic function and the identity theorem [06.01.01]** supply the engine of the whole unit. The identity theorem — two holomorphic functions agreeing on a set with a limit point agree on the component — is what makes continuation along a fixed path *unique* and what forces neighbouring function elements to glue. Without it, germs would not determine their continuations and the entire subject would collapse; the rigidity it expresses is the precise sense in which a power series is a faithful seed.

- **Branch point and ramification [06.02.01]** is where continuation fails to close up. A branch point is a point that *every* small loop around it forces a monodromy other than the identity: $\sqrt z$ and $\log z$ both ramify at $0$. This unit explains branch points dynamically — as the obstructions that make the monodromy representation of $\pi_1$ act non-identically — and [06.02.01] catalogues their local normal forms and ramification indices.

- **Branched coverings of Riemann surfaces [06.02.02]** is the topological home of the monodromy action. The sheaf of germs is an étale space, continuation is path-lifting, and the monodromy theorem is homotopy-lifting; [06.02.02] makes this a covering-space statement, with the deck group realising the monodromy representation $\pi_1 \to S_d$ that this unit's algebraic-function discussion introduces.

- **Riemann surface [06.03.01]** is the resolution of multivaluedness. A complete analytic function — a connected component of the sheaf of germs built by continuation — projects to $\mathbb{C}$ as a (branched) covering, and that covering, completed at branch points, *is* the Riemann surface on which the function is single-valued. This unit constructs the raw material; [06.03.01] gives it the structure of a one-dimensional complex manifold.

- **Uniformization theorem [06.03.03]** is the deepest downstream consequence. Once multivalued continuation is lifted to a Riemann surface, uniformization classifies the universal cover as the sphere, the plane, or the disc, so that any germ ultimately continues single-valuedly on a *simply connected* model — the monodromy theorem's simply-connected conclusion realised globally, with the monodromy group acting as deck transformations.

- **Riemann zeta function $\zeta(s)$ [21.03.01]** is the paradigm application in number theory. The Dirichlet series $\sum n^{-s}$ converges only for $\operatorname{Re}(s) > 1$, yet its germ continues meromorphically to all of $\mathbb{C}$ with a single simple pole at $s = 1$ (Riemann 1859). Because the punctured plane $\mathbb{C} \setminus \{1\}$ is not simply connected the continuation could in principle be multivalued, but here it is single-valued — the functional equation certifies it — making $\zeta$ the prototype of every analytic-continuation argument in analytic number theory.

## Historical & philosophical context [Master]

Two nineteenth-century viewpoints converge in this unit. Karl Weierstrass, from his 1842 Münster lectures onward, founded the theory of analytic functions on the **power-series element** (*Potenzreihenelement*): a function *is* a maximal system of elements obtained from one another by continuation — an *analytisches Gebilde* — with no appeal to geometry [ref: TODO_REF Weierstrass 1894-95]. In this account multivaluedness is built in from the start: $\log z$ is not one function but a connected web of elements, and continuation is the rule that knits the web. Bernhard Riemann, in his 1857 memoir on Abelian functions, took the opposite-facing route [ref: TODO_REF Riemann 1857]: he changed the *domain*, building a many-sheeted surface over the plane on which a multivalued algebraic function becomes single-valued, so that the path-dependence of continuation is absorbed into the topology of the surface. The two pictures are the same object seen from two sides — Weierstrass's web of germs is exactly the étale space whose connected components are Riemann's surfaces — and the monodromy theorem is the precise bridge, identifying analytic continuation with covering-space lifting.

The philosophical content is a lesson about the limits of formula and the primacy of domain. A holomorphic function is determined by local data with a rigidity that has no analogue for smooth functions, yet that same rigidity makes the global object refuse to be single-valued on the naive domain. Ahlfors's modern treatment [ref: TODO_REF Ahlfors §8.1] crystallises the resolution: one does not fight the multivaluedness with cuts and conventions but *redesigns the space* so that the function can be honest. This move — replace a recalcitrant object on the wrong space by a clean object on the right space — recurs throughout geometry, from covers to sheaves to schemes, and analytic continuation is its first and most vivid instance.
