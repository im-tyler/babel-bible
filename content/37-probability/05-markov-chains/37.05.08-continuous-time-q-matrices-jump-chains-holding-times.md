## Intuition [Beginner]

A continuous-time Markov chain models a system that hops between a list of states, like a discrete-time chain, but now the clock runs continuously: the system sits in a state for a random length of time, then jumps to a new state, then waits again, and so on. The memoryless rule still holds, but it now has two parts. While the system sits in a state, only the current state decides how long the wait tends to be and where the next jump is likely to go. Nothing about the earlier history matters, and — this is the new ingredient — not even how long the system has already been waiting matters.

That last point forces the waiting times to be of one special kind. If the remaining wait should not depend on how long you have already waited, the wait must follow the "exponential" pattern: a continuous version of repeatedly flipping a coin to decide whether to jump now. Each state $i$ has a rate $q_i$ that sets how fast its exponential clock runs. A large rate means short stays and frequent jumps; a small rate means long, patient stays.

So a continuous-time chain is built from two ingredients you can picture separately. First, an ordinary discrete chain that says, "when you do jump out of state $i$, here are the odds of where you land." This is the jump chain. Second, for each state, a single number — the rate $q_i$ — that runs the exponential waiting clock. Bundle the jump odds and the rates together and you get one table, the generator or $Q$-matrix, that packages both the where and the when.

One subtlety gives the theory its only real surprise. If the rates keep getting bigger as the system wanders into faster and faster states, the waits can shrink so quickly that infinitely many jumps pile up into a finite stretch of time. The chain then "runs off the end of the clock," an event called explosion. Most chains do not explode, and there is a clean test telling them apart.

The one-sentence takeaway: a continuous-time chain waits an exponential time on each state, set by that state's rate, then jumps according to a discrete jump chain, and the rate-and-jump data is exactly the $Q$-matrix — provided the jumps do not pile up into an explosion.

## Worked example [Beginner]

We follow a two-state chain. State $1$ has rate $q_1 = 2$ (so its average wait is $1/2$), and state $2$ has rate $q_2 = 1$ (average wait $1$). From state $1$ the only place to jump is state $2$, and from state $2$ the only place to jump is state $1$, so the jump chain just alternates $1, 2, 1, 2, \dots$.

Step 1. Start in state $1$. The wait is exponential with rate $2$. The average wait is $1 / q_1 = 1/2$ time unit. Say this particular wait comes out to $0.4$.

Step 2. At time $0.4$ the system jumps. The jump chain from state $1$ sends it to state $2$ with chance $1$, so it lands in state $2$.

Step 3. Now in state $2$, the wait is exponential with rate $1$, average $1 / q_2 = 1$. Say this wait is $0.9$. The system holds state $2$ from time $0.4$ until time $0.4 + 0.9 = 1.3$.

Step 4. At time $1.3$ it jumps back to state $1$, and the pattern repeats with a fresh exponential wait of rate $2$.

Step 5. Read off the long-run fraction of time in each state. On average the system spends $1/2$ a unit in state $1$ and $1$ unit in state $2$ per round trip, so out of every $1.5$ units it spends $1$ in state $2$. The fraction of time in state $2$ is $1 / 1.5 = 2/3$, and in state $1$ it is $1/3$.

What this tells us: the rates alone fixed the average waits, the jump chain alone fixed the order of visits, and combining them gave both a sample path and a long-run time split. The state with the slower clock (smaller rate) collects more of the time, which matches the picture of patient versus hurried states.

## Formal definition [Intermediate+]

Throughout, $I$ is a countable *state space* and time is continuous, indexed by $t \in [0, \infty)$. All random objects live on a probability space $(\Omega, \mathcal{F}, \mathbb{P})$ [37.01.01]. The discrete-time apparatus of stochastic matrices, the simple and strong Markov properties, and the finite-dimensional law from [37.05.01] is assumed.

**Definition ($Q$-matrix; generator).** A $Q$-*matrix* (or *generator*) on $I$ is a family $Q = (q_{ij})_{i, j \in I}$ such that
$$0 \leq -q_{ii} < \infty \ \text{ for all } i, \qquad q_{ij} \geq 0 \ \text{ for all } i \neq j, \qquad \sum_{j \in I} q_{ij} = 0 \ \text{ for all } i.$$
Each row therefore sums to zero. Write $q_i := -q_{ii} = \sum_{j \neq i} q_{ij}$ for the *total jump rate* (the *holding rate*) out of $i$. A $Q$-matrix with $q_i < \infty$ for every $i$ (built into the definition above) is *stable*; if moreover $\sup_i q_i < \infty$ it is *uniform* or *bounded*. The matrix is *conservative* when the row sums are exactly zero, as imposed here; allowing $\sum_j q_{ij} < 0$ corresponds to an extra killing rate $q_i + q_{ii} \geq 0$, excluded in this unit.

**Definition (jump matrix).** The *jump matrix* (or *jump chain matrix*) $\Pi = (\pi_{ij})_{i, j \in I}$ associated with $Q$ is the stochastic matrix
$$\pi_{ij} = \frac{q_{ij}}{q_i} \ \text{ for } j \neq i \text{ when } q_i > 0, \qquad \pi_{ii} = 0 \text{ when } q_i > 0, \qquad \pi_{ii} = 1, \ \pi_{ij} = 0 \ (j \neq i) \text{ when } q_i = 0.$$
A state with $q_i = 0$ is *absorbing*: once entered, it is never left, and $\Pi$ keeps the chain there. For $q_i > 0$ the row $\pi_{i \cdot}$ is the conditional distribution of the next state given a jump out of $i$, and $q_{ij} = q_i \pi_{ij}$ recovers the off-diagonal generator entries. Thus the pair $(\Pi, (q_i)_{i})$ and the matrix $Q$ carry identical information.

**Definition (minimal chain via jump chain and holding times).** Let $(Y_n)_{n \geq 0}$ be a discrete-time $\mathrm{Markov}(\lambda, \Pi)$ chain [37.05.01], the *jump chain*. Independently, let $(E_n)_{n \geq 1}$ be i.i.d. $\mathrm{Exp}(1)$ random variables, and set the *holding times* $S_n = E_n / q_{Y_{n-1}}$ (with $S_n = \infty$ if $q_{Y_{n-1}} = 0$). The *jump times* are $J_0 = 0$, $J_n = S_1 + \cdots + S_n$, and the *explosion time* is $\zeta = \sup_n J_n = \sum_{n \geq 1} S_n$. The *minimal* continuous-time chain is
$$X_t = Y_n \ \text{ for } J_n \leq t < J_{n+1}, \qquad X_t = \partial \ \text{ for } t \geq \zeta,$$
where $\partial$ is an adjoined cemetery state used only to define $X_t$ past explosion; before $\zeta$ the path is right-continuous and piecewise constant with jumps at the $J_n$. Conditional on $Y_{n-1} = i$, the holding time $S_n$ is $\mathrm{Exp}(q_i)$ and is independent of $Y_n$ and of the past, the defining structural fact of the construction.

**Definition (explosion and non-explosion).** The chain is *non-explosive* (or *honest*) from $i$ if $\mathbb{P}_i(\zeta = \infty) = 1$, and *explosive* if $\mathbb{P}_i(\zeta < \infty) > 0$ for some $i$. Non-explosion means $X_t \in I$ for all $t$ almost surely, so the cemetery $\partial$ is never reached and $X$ is a genuine $I$-valued process for all time.

### Counterexamples to common slips [Intermediate+]

- *The $Q$-matrix is not a stochastic matrix.* Its rows sum to $0$, not $1$, and its diagonal is non-positive. The associated *transition semigroup* $P(t) = e^{tQ}$ (when defined) is stochastic; $Q$ is its derivative at $t = 0$, the continuous-time analogue of the discrete generator $P - I_d$ of [37.05.01], not of $P$ itself.

- *Holding times are exponential, not the increments of the jump chain.* The randomness splits cleanly: the jump chain $(Y_n)$ decides *where*, the exponentials decide *when*, and the two are independent given the visited states. Confusing the holding rate $q_i$ with a transition probability conflates the clock with the routing.

- *$\pi_{ii} = 0$ for a non-absorbing state.* The jump chain never stays put when $q_i > 0$; self-transitions of the continuous chain are invisible because a jump from $i$ to $i$ would not change the path. Self-loops in $Q$ are therefore meaningless — only off-diagonal $q_{ij}$ carry information.

- *Explosion is genuine and rate-driven.* A finite-state $Q$-matrix, or any bounded one with $\sup_i q_i < \infty$, never explodes; explosion needs $q_i \to \infty$ along a path the jump chain can traverse fast enough. The Markov property alone does not rule it out.

## Key theorem with proof [Intermediate+]

**Theorem (jump chain and holding times $\Leftrightarrow$ $Q$-matrix; the competing-exponentials law).** Let $Q$ be a $Q$-matrix on $I$ with jump matrix $\Pi$ and rates $(q_i)$. Construct $(X_t)$ as in the minimal-chain definition. Then for a state $i$ with $q_i > 0$, started at $X_0 = i$:

(a) the holding time $S_1 = \inf\{t > 0 : X_t \neq i\}$ is $\mathrm{Exp}(q_i)$;

(b) the first jump target $X_{S_1} = Y_1$ has law $\pi_{i \cdot}$, i.e. $\mathbb{P}_i(Y_1 = j) = q_{ij}/q_i$;

(c) $S_1$ and $Y_1$ are independent.

Conversely, if a process holds $i$ for an $\mathrm{Exp}(q_i)$ time and then jumps to $j \neq i$ with probability $\pi_{ij}$, independently, its generator entries are $q_{ij} = q_i \pi_{ij}$ off-diagonal and $q_{ii} = -q_i$, recovering $Q$ [ref: TODO_REF Norris 1997 §2.1-2.2].

*Proof.* Model the dynamics out of $i$ by a family of independent competing exponential clocks. For each $j \neq i$ with $q_{ij} > 0$, let $T_j \sim \mathrm{Exp}(q_{ij})$ be independent across $j$, and let the chain jump to whichever rings first.

**Step 1 (the minimum of independent exponentials).** Let $T = \min_{j \neq i} T_j$. For $t \geq 0$, independence gives
$$\mathbb{P}_i(T > t) = \prod_{j \neq i} \mathbb{P}(T_j > t) = \prod_{j \neq i} e^{-q_{ij} t} = \exp\Big(-t \sum_{j \neq i} q_{ij}\Big) = e^{-q_i t},$$
since $\sum_{j \neq i} q_{ij} = q_i$. Hence $T \sim \mathrm{Exp}(q_i)$, proving (a) with $S_1 = T$.

**Step 2 (which clock rings first).** The probability that clock $j$ is the strict minimum is computed by conditioning on $T_j = s$ and requiring every other clock to exceed $s$:
$$\mathbb{P}_i(\text{$j$ rings first}) = \int_0^\infty q_{ij} e^{-q_{ij} s} \prod_{k \neq i, j} e^{-q_{ik} s}\, ds = \int_0^\infty q_{ij}\, e^{-q_i s}\, ds = \frac{q_{ij}}{q_i} = \pi_{ij},$$
using $q_{ij} + \sum_{k \neq i, j} q_{ik} = q_i$ in the exponent. This proves (b): the jump target has law $\pi_{i \cdot}$.

**Step 3 (independence of when and where).** Compute the joint law of the first-ring time and its identity. For the event that $j$ rings first *and* this happens after time $t$,
$$\mathbb{P}_i(\text{$j$ first},\, T > t) = \int_t^\infty q_{ij} e^{-q_i s}\, ds = \frac{q_{ij}}{q_i} e^{-q_i t} = \pi_{ij}\, \mathbb{P}_i(T > t).$$
The joint probability factors into the marginal $\pi_{ij}$ of the target and the marginal $e^{-q_i t}$ of the time, so $Y_1$ and $S_1$ are independent, proving (c). The competing-exponentials model therefore produces exactly an $\mathrm{Exp}(q_i)$ holding time and an independent $\Pi$-distributed jump, which is the minimal-chain construction; and the lack-of-memory of the exponential makes the resulting process Markov, with the same data $(Y_n, S_n)$ regenerating at each jump by the strong Markov property of $(Y_n)$ [37.05.01].

**Converse.** Given holding rate $q_i$ and jump law $\pi_{i \cdot}$, set $q_{ij} = q_i \pi_{ij}$ for $j \neq i$ and $q_{ii} = -q_i$. Then $q_{ij} \geq 0$ off-diagonal and $\sum_j q_{ij} = -q_i + q_i \sum_{j \neq i} \pi_{ij} = -q_i + q_i = 0$, so $Q$ is a conservative $Q$-matrix, and the construction run from $Q$ returns the given holding-and-jump dynamics. $\square$

**Bridge.** This theorem builds toward the entire analytic theory of continuous-time chains and appears again in the forward and backward Kolmogorov equations, where $Q$ becomes the generator of the transition semigroup $P(t) = e^{tQ}$. The foundational reason the construction is canonical is that the memoryless exponential is the unique continuous holding law compatible with the Markov property, so the only freedom left is the routing matrix $\Pi$: this is exactly the split of the dynamics into a *when* (the rate $q_i$) and a *where* (the row $\pi_{i\cdot}$), and the identity $q_{ij} = q_i \pi_{ij}$ is the dictionary between them. The discrete jump chain $(Y_n)$ is the embedded skeleton governed by the discrete theory of [37.05.01], so every recurrence, transience, and irreducibility question about $X$ reduces to one about $Y$; putting these together, the continuous-time chain is the jump chain with exponential clocks bolted on, and the $Q$-matrix is dual to the diffusion generator $\mathcal{L}$ of [02.15.03] — there the generator is a second-order differential operator on a continuum, here it is a row-zero-sum matrix on a countable set, the central insight being that both are the infinitesimal description of a Markov semigroup.

A state $i$ has generator entries $q_{ij} = 3$, $q_{ik} = 1$ to the only two other reachable states $j, k$. Compute the holding rate $q_i$ and the diagonal entry $q_{ii}$. Give $q_i$ as an integer.

<aside class="exercise" data-difficulty="medium" data-type="symbolic" markdown="1">

**Exercise 3 (medium, symbolic).**

Show that if $T_1 \sim \mathrm{Exp}(\alpha)$ and $T_2 \sim \mathrm{Exp}(\beta)$ are independent, then $\min(T_1, T_2) \sim \mathrm{Exp}(\alpha + \beta)$ and $\mathbb{P}(T_1 < T_2) = \alpha/(\alpha + \beta)$.

<details class="exercise__hint"><summary>Hint</summary>

For the minimum use $\mathbb{P}(\min > t) = \mathbb{P}(T_1 > t)\mathbb{P}(T_2 > t)$. For the comparison integrate the joint density.

</details>

<details class="exercise__answer"><summary>Answer</summary>

By independence $\mathbb{P}(\min(T_1, T_2) > t) = e^{-\alpha t} e^{-\beta t} = e^{-(\alpha+\beta)t}$, so the minimum is $\mathrm{Exp}(\alpha + \beta)$. For the comparison, conditioning on $T_1 = s$,
$$\mathbb{P}(T_1 < T_2) = \int_0^\infty \alpha e^{-\alpha s} \mathbb{P}(T_2 > s)\, ds = \int_0^\infty \alpha e^{-\alpha s} e^{-\beta s}\, ds = \frac{\alpha}{\alpha + \beta}.$$
This is the two-clock case of the competing-exponentials computation underlying the Key theorem: the merged rate is the sum, and the win probability is the rate share.

</details>

</aside>

<aside class="exercise" data-difficulty="medium" data-type="symbolic" markdown="1">

**Exercise 4 (medium, symbolic).**

Let $S \sim \mathrm{Exp}(q)$ be a holding time. Show the lack-of-memory property: for $s, t \geq 0$, $\mathbb{P}(S > s + t \mid S > s) = \mathbb{P}(S > t)$. Explain in one sentence why this is what makes the construction Markov.

<details class="exercise__hint"><summary>Hint</summary>

Use $\mathbb{P}(S > u) = e^{-qu}$ and the definition of conditional probability.

</details>

<details class="exercise__answer"><summary>Answer</summary>

$$\mathbb{P}(S > s + t \mid S > s) = \frac{\mathbb{P}(S > s + t)}{\mathbb{P}(S > s)} = \frac{e^{-q(s+t)}}{e^{-qs}} = e^{-qt} = \mathbb{P}(S > t).$$
The residual holding time, given that the state has already been occupied for $s$, has the same $\mathrm{Exp}(q)$ law as a fresh holding time. So at every instant the future of the process depends only on the present state, not on the elapsed occupancy — exactly the continuous-time Markov property. The exponential is the only continuous law with this property.

</details>

</aside>

<aside class="exercise" data-difficulty="medium" data-type="symbolic" markdown="1">

**Exercise 5 (medium, symbolic).**

A pure birth chain on $I = \{0, 1, 2, \dots\}$ has $q_{n, n+1} = \beta_n > 0$ and no other off-diagonal entries. Write down its $Q$-matrix, its jump matrix $\Pi$, and the law of the holding time in state $n$.

<details class="exercise__hint"><summary>Hint</summary>

The only jump from $n$ is to $n+1$, so the routing is deterministic; the rate $q_n = \beta_n$.

</details>

<details class="exercise__answer"><summary>Answer</summary>

The generator has $q_{n,n+1} = \beta_n$, $q_{nn} = -\beta_n$, all other entries zero; each row sums to zero. The jump matrix is deterministic, $\pi_{n, n+1} = 1$ and $\pi_{nk} = 0$ otherwise, so the jump chain is the deterministic march $0, 1, 2, \dots$. The holding time in state $n$ is $\mathrm{Exp}(\beta_n)$, and the jump time to reach state $n$ is $J_n = \sum_{k=0}^{n-1} S_{k+1}$, a sum of independent exponentials with rates $\beta_0, \dots, \beta_{n-1}$. The explosion time is $\zeta = \sum_{k \geq 0} S_{k+1}$, whose finiteness depends on $\sum_k 1/\beta_k$ (Exercise 8).

</details>

</aside>

**Exercise 7 (hard, symbolic).**

For a chain with bounded rates $\sup_i q_i =: \bar{q} < \infty$, prove non-explosion: $\mathbb{P}_i(\zeta = \infty) = 1$ for every $i$.

<details class="exercise__hint"><summary>Hint</summary>

Each holding time $S_n$ stochastically dominates an $\mathrm{Exp}(\bar q)$ variable. Compare $\zeta$ to a sum of i.i.d. $\mathrm{Exp}(\bar q)$ times.

</details>

<details class="exercise__answer"><summary>Answer</summary>

Conditional on $Y_{n-1}$, the holding time $S_n \sim \mathrm{Exp}(q_{Y_{n-1}})$ with $q_{Y_{n-1}} \leq \bar q$, so $S_n$ stochastically dominates an independent $\mathrm{Exp}(\bar q)$ variable $\tilde S_n$; coupling gives $S_n \geq \tilde S_n$ almost surely with $(\tilde S_n)$ i.i.d. $\mathrm{Exp}(\bar q)$. Then $\zeta = \sum_n S_n \geq \sum_n \tilde S_n$. By the strong law of large numbers $\frac1N \sum_{n=1}^N \tilde S_n \to 1/\bar q > 0$, so $\sum_n \tilde S_n = \infty$ almost surely, whence $\zeta = \infty$ almost surely. (Equivalently, $\mathbb{E}_i[\zeta] = \sum_n \mathbb{E}[S_n] \geq \sum_n 1/\bar q = \infty$ and a Borel-Cantelli / SLLN argument upgrades this to $\zeta = \infty$ a.s.) Bounded rates therefore forbid explosion; finite state spaces are the special case $\bar q = \max_i q_i < \infty$.

</details>

</aside>

<aside class="exercise" data-difficulty="hard" data-type="symbolic" markdown="1">

**Exercise 8 (hard, symbolic).**

For the pure birth chain of Exercise 5 with $\pi_{n,n+1} = 1$, show that the chain explodes from $0$ with probability one iff $\sum_{n} 1/\beta_n < \infty$, and is non-explosive iff $\sum_n 1/\beta_n = \infty$.

<details class="exercise__hint"><summary>Hint</summary>

$\zeta = \sum_n S_{n+1}$ with $S_{n+1} \sim \mathrm{Exp}(\beta_n)$ independent; use $\mathbb{E}[\zeta] = \sum_n 1/\beta_n$ and a zero-one argument for sums of independent nonnegative variables.

</details>

<details class="exercise__answer"><summary>Answer</summary>

The jump chain is deterministic, so $\zeta = \sum_{n \geq 0} S_{n+1}$ with $S_{n+1} \sim \mathrm{Exp}(\beta_n)$ independent and $\mathbb{E}[S_{n+1}] = 1/\beta_n$. By Tonelli, $\mathbb{E}[\zeta] = \sum_n 1/\beta_n$. If $\sum_n 1/\beta_n < \infty$ then $\mathbb{E}[\zeta] < \infty$, so $\zeta < \infty$ almost surely: the chain explodes. If $\sum_n 1/\beta_n = \infty$, consider $\mathbb{E}[e^{-\zeta}] = \prod_n \mathbb{E}[e^{-S_{n+1}}] = \prod_n \frac{\beta_n}{\beta_n + 1} = \prod_n \big(1 + \beta_n^{-1}\big)^{-1}$. This product is zero iff $\sum_n \beta_n^{-1} = \infty$ (since $-\log(1 + x) \sim -x$ forces $\sum \log(1 + \beta_n^{-1}) = \infty$). Hence $\mathbb{E}[e^{-\zeta}] = 0$, forcing $e^{-\zeta} = 0$ a.s., i.e. $\zeta = \infty$ almost surely: the chain is non-explosive. The dichotomy is exactly the convergence of $\sum_n 1/\beta_n$, the classical criterion separating, e.g., $\beta_n = n$ (non-explosive, $\sum 1/n = \infty$) from $\beta_n = n^2$ (explosive, $\sum 1/n^2 < \infty$).

</details>

</aside>

## Advanced results [Master]

Beyond the pathwise construction, the theory organizes around the equivalence of the data, the embedded jump chain as the discrete skeleton, the explosion dichotomy with Reuter's analytic criterion, and the passage from $Q$ to the transition semigroup that places the matrix as the infinitesimal generator dual to the continuous-state operator of [02.15.03].

**Theorem 1 (three equivalent descriptions).** Let $X$ be the minimal right-continuous process on $I$. The following data determine one another and determine the law of $X$: (i) the conservative stable $Q$-matrix $Q$; (ii) the pair consisting of the jump matrix $\Pi$ and the rate vector $(q_i)$, via $q_{ij} = q_i \pi_{ij}$, $q_{ii} = -q_i$; (iii) the requirement that, started at $i$ with $q_i > 0$, the first holding time is $\mathrm{Exp}(q_i)$ independent of the first jump, whose law is $\pi_{i\cdot}$, and that the process restarts afresh at each jump (strong Markov at jump times). The equivalence (i) $\Leftrightarrow$ (ii) is algebraic; (ii) $\Leftrightarrow$ (iii) is the Key theorem together with the strong Markov property of the discrete jump chain [37.05.01].

**Theorem 2 (the embedded jump chain governs the qualitative theory).** The jump chain $(Y_n)$ is a discrete-time $\mathrm{Markov}(\lambda, \Pi)$ chain, and the continuous chain's irreducibility, recurrence, and transience are read off $(Y_n)$: $X$ is irreducible iff $\Pi$ is irreducible (on the non-absorbing states), and a state $i$ is recurrent for $X$ iff it is recurrent for $(Y_n)$. The continuous and discrete notions of *visiting* a state coincide because the holding times are almost surely positive and finite (for $q_i > 0$), so the set of states visited is identical for $X$ and $Y$. The distinction between *positive* and *null* recurrence, however, involves the holding rates: positive recurrence of $X$ requires the expected return time $\sum_i (\text{visits})/q_i$ to be finite, blending the routing of $\Pi$ with the rates.

**Theorem 3 (explosion dichotomy and Reuter's criterion).** Let $Q$ be a stable conservative $Q$-matrix. The minimal chain is non-explosive (honest) from every state iff the only bounded solution $z = (z_i)_{i \in I}$ with $0 \leq z_i \leq 1$ to
$$Q z = z \qquad \text{equivalently} \qquad \sum_{j} q_{ij} z_j = z_i \ \text{ for all } i, \ \text{i.e. } (Q - I_d) z = 0,$$
is $z \equiv 0$. This is *Reuter's criterion* [ref: TODO_REF Reuter 1957]. The quantity $z_i = \mathbb{E}_i[e^{-\zeta}]$ is the maximal such solution: it satisfies $Q z = z$ by a first-jump decomposition, and explosion ($z_i > 0$ for some $i$) is equivalent to the existence of a non-zero bounded solution. Sufficient conditions follow at once: a bounded $Q$-matrix ($\sup_i q_i < \infty$), or more generally any chain for which $\sum_n 1/q_{Y_{n-1}} = \infty$ almost surely, is non-explosive; the pure-birth criterion $\sum_n 1/\beta_n = \infty$ is the one-dimensional instance.

**Theorem 4 (from $Q$ to the transition semigroup; the generator).** Define $p_{ij}(t) = \mathbb{P}_i(X_t = j)$ for the minimal chain. Then $(P(t))_{t \geq 0}$ with $P(t) = (p_{ij}(t))$ is a sub-stochastic semigroup: $P(0) = I_d$, $P(s + t) = P(s)P(t)$ (the continuous-time Chapman–Kolmogorov relation), and $\sum_j p_{ij}(t) \leq 1$ with equality iff the chain is non-explosive up to time $t$. It satisfies the *backward equation* $P'(t) = Q P(t)$ and, under non-explosion and suitable regularity, the *forward equation* $P'(t) = P(t) Q$, with $P(t) = e^{tQ}$ when $Q$ is bounded. Thus $Q = P'(0)$ is the infinitesimal generator: the matrix analogue, on a countable state space, of the second-order differential generator $\mathcal{L}$ of a diffusion [02.15.03]. The off-diagonal $q_{ij}$ is the instantaneous rate $\lim_{t \downarrow 0} p_{ij}(t)/t$, and $-q_{ii} = q_i = \lim_{t \downarrow 0} (1 - p_{ii}(t))/t$ is the instantaneous rate of leaving $i$.

**Synthesis.** The foundational reason the subject coheres is that one conservative $Q$-matrix encodes the entire law of the minimal chain, and putting these together, the jump-chain/holding-time data $(\Pi, (q_i))$ is exactly the same information presented as routing-plus-clock rather than as a single generator — this is the central insight that the *when* and the *where* decouple, with the memoryless exponential the unique clock the Markov property allows. The embedded jump chain is the discrete skeleton, so every recurrence and transience question generalises the discrete theory of [37.05.01] verbatim, while the rates re-enter only for positive recurrence and for the time-change between $Y$ and $X$. The explosion dichotomy is dual to the analytic question of whether $(Q - I_d)z = 0$ admits a non-zero bounded solution: this is exactly Reuter's criterion, and the maximal solution $z_i = \mathbb{E}_i[e^{-\zeta}]$ is the bridge between the probabilistic event of explosion and the linear algebra of the generator. The passage $Q \mapsto P(t) = e^{tQ}$ shows the $Q$-matrix is dual to the diffusion generator $\mathcal{L}$ of [02.15.03] — both are $P'(0)$ for their respective semigroups, one a row-zero-sum matrix on a countable set, the other a second-order elliptic operator on a continuum — so the $Q$-matrix, the jump-and-hold construction, the explosion criterion, and the generator of the semigroup are four presentations of the single fact that an infinitesimal description determines a continuous-time Markov flow whenever its jumps do not pile up.

## Full proof set [Master]

**Proposition 1 (the jump matrix is stochastic).** For any stable conservative $Q$-matrix $Q$, the associated $\Pi$ is a stochastic matrix.

*Proof.* For $i$ with $q_i > 0$: $\pi_{ij} = q_{ij}/q_i \geq 0$ for $j \neq i$ since $q_{ij} \geq 0$, and $\pi_{ii} = 0$; the row sum is $\sum_{j \neq i} q_{ij}/q_i = q_i/q_i = 1$ using $\sum_{j \neq i} q_{ij} = q_i$. For $i$ with $q_i = 0$: $\pi_{ii} = 1$ and $\pi_{ij} = 0$ for $j \neq i$, a degenerate but valid stochastic row. Hence every row of $\Pi$ is a probability distribution, so $\Pi$ is stochastic. $\square$

**Proposition 2 (competing exponentials; full joint law).** Let $(T_j)_{j \in A}$ be independent with $T_j \sim \mathrm{Exp}(\alpha_j)$, $\alpha := \sum_{j \in A} \alpha_j < \infty$. Let $T = \min_j T_j$ and $K = \arg\min_j T_j$. Then $T \sim \mathrm{Exp}(\alpha)$, $\mathbb{P}(K = k) = \alpha_k/\alpha$, $T$ and $K$ are independent, and the minimizer is almost surely unique.

*Proof.* For the tail, $\mathbb{P}(T > t) = \prod_j \mathbb{P}(T_j > t) = \prod_j e^{-\alpha_j t} = e^{-\alpha t}$, so $T \sim \mathrm{Exp}(\alpha)$. For the joint law of $\{K = k\} \cap \{T > t\}$, condition on $T_k$:
$$\mathbb{P}(K = k, T > t) = \int_t^\infty \alpha_k e^{-\alpha_k s} \prod_{j \neq k} \mathbb{P}(T_j > s)\, ds = \int_t^\infty \alpha_k e^{-\alpha s}\, ds = \frac{\alpha_k}{\alpha} e^{-\alpha t}.$$
Setting $t = 0$ gives $\mathbb{P}(K = k) = \alpha_k/\alpha$; combined with $\mathbb{P}(T > t) = e^{-\alpha t}$ the joint factors as $\mathbb{P}(K = k)\mathbb{P}(T > t)$, proving independence. Two continuous independent variables coincide with probability zero, so the minimizer is a.s. unique. $\square$

**Proposition 3 (memorylessness characterizes the exponential).** A nonnegative random variable $S$ with $\mathbb{P}(S > 0) = 1$ satisfies $\mathbb{P}(S > s + t \mid S > s) = \mathbb{P}(S > t)$ for all $s, t \geq 0$ iff $S \sim \mathrm{Exp}(q)$ for some $q \in [0, \infty)$ (with $q = 0$ meaning $S = \infty$ a.s.).

*Proof.* If $S \sim \mathrm{Exp}(q)$ then $G(t) := \mathbb{P}(S > t) = e^{-qt}$ gives $G(s+t)/G(s) = e^{-qt} = G(t)$. Conversely the hypothesis is $G(s + t) = G(s)G(t)$, the multiplicative Cauchy equation for the right-continuous non-increasing $G$ with $G(0) = 1$. Right-continuous solutions of $G(s+t) = G(s)G(t)$ on $[0,\infty)$ with $0 \leq G \leq 1$ are $G(t) = e^{-qt}$ for some $q \in [0, \infty]$: writing $g = -\log G$ gives Cauchy's additive equation $g(s+t) = g(s) + g(t)$, whose monotone solutions are linear $g(t) = qt$. Hence $G(t) = e^{-qt}$, i.e. $S \sim \mathrm{Exp}(q)$. $\square$

**Proposition 4 ($Q z = z$ for the explosion functional).** Let $z_i = \mathbb{E}_i[e^{-\zeta}]$ for the minimal chain, $0 \leq z_i \leq 1$. Then $(Q - I_d) z = 0$ on the non-absorbing states, and $z$ is the maximal bounded solution in $[0,1]$.

*Proof.* Condition on the first holding time $S_1 \sim \mathrm{Exp}(q_i)$ and the first jump $Y_1 \sim \pi_{i\cdot}$, independent by the Key theorem. After the first jump the residual explosion time is $\zeta' = \zeta - S_1$ with law $\mathbb{E}_{Y_1}[e^{-\zeta'}]$ by the strong Markov property at the jump time [37.05.01]. Hence, for $q_i > 0$,
$$z_i = \mathbb{E}_i[e^{-S_1}]\, \mathbb{E}_i[e^{-\zeta'}] = \frac{q_i}{q_i + 1} \sum_{j} \pi_{ij} z_j,$$
using $\mathbb{E}[e^{-S_1}] = q_i/(q_i + 1)$ for $S_1 \sim \mathrm{Exp}(q_i)$. Multiplying by $(q_i + 1)$ and rearranging with $q_{ij} = q_i \pi_{ij}$, $q_{ii} = -q_i$:
$$(q_i + 1) z_i = q_i \sum_j \pi_{ij} z_j = \sum_{j \neq i} q_{ij} z_j \ \Longrightarrow\ \sum_j q_{ij} z_j = \sum_{j \neq i} q_{ij} z_j - q_i z_i = (q_i + 1) z_i - q_i z_i - z_i\cdot 0,$$
which simplifies to $\sum_j q_{ij} z_j = z_i$, i.e. $(Q - I_d)z = 0$. Maximality: any $w \in [0,1]^I$ solving the same equation satisfies $w_i = \mathbb{E}_i[e^{-J_n}\, w_{Y_n}]$ for every $n$ by iterating the first-jump identity, and $e^{-J_n} w_{Y_n} \to e^{-\zeta}\cdot(\text{bounded})$ with $w \leq 1$, giving $w_i \leq \mathbb{E}_i[e^{-\zeta}] = z_i$ by dominated convergence. $\square$

**Proposition 5 (Reuter's criterion).** The minimal chain is non-explosive from every state iff $z \equiv 0$ is the only solution of $(Q - I_d)z = 0$ in $[0,1]^I$.

*Proof.* By Proposition 4, $z_i = \mathbb{E}_i[e^{-\zeta}]$ is the maximal bounded $[0,1]$-solution. Non-explosion from every state means $\mathbb{P}_i(\zeta = \infty) = 1$ for all $i$, equivalently $e^{-\zeta} = 0$ a.s., equivalently $z_i = 0$ for all $i$. Since $z$ is maximal, $z \equiv 0$ forces every bounded $[0,1]$-solution to vanish; conversely if some $z_i > 0$ then $z$ itself is a non-zero solution and $\mathbb{P}_i(\zeta < \infty) > 0$, i.e. explosion. Hence non-explosion $\Leftrightarrow$ the only solution is $z \equiv 0$. $\square$

**Proposition 6 (bounded $Q$ gives $P(t) = e^{tQ}$, a stochastic semigroup).** If $\sup_i q_i =: \bar q < \infty$, the series $e^{tQ} = \sum_{n \geq 0} t^n Q^n / n!$ converges in operator norm on $\ell^\infty(I)$, defines a stochastic semigroup $P(t)$ with $P(t)\mathbf{1} = \mathbf{1}$, and equals the transition matrix of the minimal (non-explosive) chain.

*Proof.* On $\ell^\infty(I)$, $\|Q\|_\infty = \sup_i \sum_j |q_{ij}| = \sup_i 2 q_i = 2\bar q < \infty$, so $\sum_n t^n \|Q\|^n/n! = e^{t\|Q\|} < \infty$ and the series converges absolutely in operator norm; $P(t) := e^{tQ}$ then satisfies $P(0) = I_d$, $P'(t) = Q e^{tQ} = e^{tQ}Q$, and $P(s+t) = P(s)P(t)$ by the functional equation of the matrix exponential. Conservativity $Q\mathbf{1} = 0$ gives $P(t)\mathbf{1} = e^{tQ}\mathbf{1} = \sum_n t^n Q^n \mathbf{1}/n! = \mathbf{1}$ since $Q^n\mathbf{1} = 0$ for $n \geq 1$; nonnegativity of $P(t)$ follows by writing $P(t) = e^{-\bar q' t}\sum_n (t)^n (Q + \bar q' I_d)^n/n!$ with $\bar q' \geq \bar q$ chosen so $Q + \bar q' I_d \geq 0$ entrywise, a convergent series of nonnegative matrices. Boundedness gives non-explosion (Exercise 7), and the forward/backward equations identify $e^{tQ}$ with the minimal chain's transition matrix. $\square$

## Connections [Master]

- The discrete-time chain [37.05.01] supplies the embedded jump chain $(Y_n)$: it is a $\mathrm{Markov}(\lambda, \Pi)$ chain in the exact sense of that unit, and its stochastic-matrix algebra, simple and strong Markov properties, and recurrence/transience dichotomy transfer verbatim to the qualitative theory of the continuous chain; the continuous-time content added here is only the exponential clock and the explosion phenomenon that the discrete skeleton cannot see.

- The probability-space and extension foundations [37.01.01] underwrite the construction of the pair $((Y_n), (E_n))$ on one probability space: the jump chain is built from a consistent family of finite-dimensional laws and the holding-time exponentials are an independent i.i.d. sequence, the joint law assembled by the Kolmogorov/Ionescu–Tulcea machinery, and the explosion time $\zeta = \sum_n S_n$ is a measurable functional of that data on path space.

- The continuous-state diffusion generator [02.15.03] is the parallel theory on a continuum: the $Q$-matrix is to the countable-state semigroup $P(t) = e^{tQ}$ what the second-order elliptic operator $\mathcal{L}$ is to the diffusion semigroup $P_t = e^{t\mathcal{L}}$; both are $P'(0)$, both run forward (Kolmogorov forward / Fokker–Planck) on measures and backward on observables, and the contrast is jumps on a discrete set versus continuous sample paths on $\mathbb{R}^n$, with explosion here mirroring the explosion of solutions to the SDE there.

## Historical & philosophical context [Master]

The analytic theory of continuous-time Markov processes was founded by Andrei N. Kolmogorov in his 1931 *Mathematische Annalen* memoir [ref: TODO_REF Kolmogorov 1931], which introduced the transition function $p_{ij}(t)$ and derived the forward and backward differential equations governing it. Kolmogorov worked from the semigroup relation $p_{ik}(s+t) = \sum_j p_{ij}(s) p_{jk}(t)$ — the continuous-time Chapman–Kolmogorov equation — and obtained $Q = P'(0)$ as the matrix of instantaneous rates, though the pathwise jump-and-hold construction was not yet his concern.

William Feller, in his 1940 *Transactions of the American Mathematical Society* paper on purely discontinuous Markov processes [ref: TODO_REF Feller 1940], analyzed the integro-differential equations of jump processes on countable state spaces and confronted the possibility that the backward equation does not determine a unique transition function — the analytic shadow of explosion and of boundary behaviour. The probabilistic resolution, separating the routing (jump matrix) from the timing (exponential holding times) and constructing the *minimal* process directly, is the modern synthesis presented here.

Gerd E. H. Reuter settled the existence and uniqueness question for denumerable chains in his 1957 *Acta Mathematica* paper [ref: TODO_REF Reuter 1957], characterizing non-explosion (honesty) of the minimal process through the vanishing of every bounded $[0,1]$-solution of $(Q - I_d)z = 0$, the criterion stated above. The contraction-semigroup framework he used connects the matrix $Q$ to the Hille–Yosida generation theory for the transition semigroup on $\ell^1$, placing continuous-time chains within the general theory of Markov semigroups whose continuous-state instance is the diffusion generator of [02.15.03].
