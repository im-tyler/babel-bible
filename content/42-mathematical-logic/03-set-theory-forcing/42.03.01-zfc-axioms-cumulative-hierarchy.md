## Intuition [Beginner]

Almost everything in mathematics can be built out of one humble ingredient: the collection. A number, a function, a triangle, a probability — each can be coded as a collection of simpler things, which are themselves collections, all the way down. Set theory is the study of these collections, called **sets**, and the surprising fact is that a short list of ground rules about them is enough to support the entire building. The single relation you need is "belongs to": one set either is or is not a member of another.

You might think the safe rule is "any property you can describe carves out a set." Picture the property "is a set that does not belong to itself." Most everyday collections satisfy it. Now ask whether the collection of all such sets belongs to itself. If it does, then by its own description it should not; if it does not, then it fits the description and so should. Either way you reach a contradiction. This is **Russell's paradox**, and it shows the carefree rule is broken. The repair is to stop assuming every description gives a set and instead lay down explicit, modest rules for building sets from sets you already have.

Those rules are the **ZFC axioms**. They are deliberately cautious: you may pair two sets, pool a family into its union, form the collection of all subsets, separate out the members of an existing set that satisfy a property, and guarantee one infinite set to get started. None of these lets you scoop up "everything at once," which is exactly what caused the trouble.

The picture that makes the whole system feel inevitable is a tower built in stages. Start with nothing. At each new stage, allow every collection whose members appeared at earlier stages. You never reach a top, but every set lives at some stage. This staged tower is the **cumulative hierarchy**, and the rest of this unit is about why it captures precisely the sets the axioms describe.

## Visual [Beginner]

The cumulative hierarchy is a tower of stages. Each stage contains every collection you can form using only the items from the stages below it. The picture shows the bottom stages and the rule that builds the next stage from the one before.

```
   stage 3   { everything whose members come from stage 2 }   ... huge
              ^
              |  collect all sub-collections of the stage below
   stage 2   { {}, {a}, {b}, {a,b}, ... }        items: 4
              ^
              |
   stage 1   { {} , {a} }                          items: 2
              ^
              |   here a = {} is the only item available
   stage 0   {  }   (nothing yet)                  items: 0

   the tower never stops; every set sits at some finite or
   infinite stage, and a set's "rank" is the first stage that
   contains it
```

Read the tower from the bottom up. Stage $0$ is empty. At each step you look at everything already present and form all possible sub-collections of it; those become the new stage. Because stage $1$ already has $2$ items, stage $2$ has $2 \times 2 = 4$ sub-collections, stage $3$ has $2^4 = 16$, and the counts explode upward.

| stage | how many items it holds |
|---|---|
| stage $0$ | $0$ |
| stage $1$ | $2$ |
| stage $2$ | $4$ |
| stage $3$ | $16$ |

The key idea: a set never appears before all of its members have appeared. The first stage where a set shows up is called its **rank**, and ranking every set is what ties the loose rules together into one orderly universe.

## Worked example [Beginner]

Let us build the first few stages by hand, starting from absolutely nothing, and find the rank of a small set. The only raw material is the empty collection, written $\varnothing$ — the collection with no members at all.

Step 1. Stage $0$ is empty: it has no items. So at stage $0$ there is nothing to collect.

Step 2. Build stage $1$. We collect every sub-collection of what stage $0$ offered. The only sub-collection of nothing is the empty collection itself. So stage $1 = \{\, \varnothing \,\}$, a tower level with exactly one item, namely $\varnothing$.

Step 3. Build stage $2$. Now we have one item available, $\varnothing$, so the sub-collections are: the empty one, and the one containing $\varnothing$. So stage $2 = \{\, \varnothing,\ \{\varnothing\} \,\}$, with two items.

Step 4. Build stage $3$. We have two items available, so there are $2 \times 2 = 4$ sub-collections to form, including the new set $\{\, \{\varnothing\} \,\}$ and the full set $\{\, \varnothing,\ \{\varnothing\} \,\}$.

Step 5. Find a rank. Consider the set $x = \{\, \{\varnothing\} \,\}$. Its one member is $\{\varnothing\}$, which first appeared at stage $2$. A set appears one stage after its members, so $x$ first appears at stage $3$. Its rank is $3$.

What this tells us: with no ingredients except the empty collection, the staged rule already manufactures a rich supply of distinct sets, and each one earns a definite rank — the first stage that holds it. This single number will organise the whole universe in the formal sections.

## Formal definition [Intermediate+]

The background language is first-order logic with equality over a single binary relation symbol $\in$ (cross-ref [42.01.03]); the only nonlogical primitive is membership, and every other notion ($\subseteq$, $\varnothing$, ordered pair, function) is defined from it. Variables range over sets; there are no urelements. A **class** is not an object of the theory but a syntactic abbreviation: for a formula $\varphi(x)$ with parameters, the expression $\{x : \varphi(x)\}$ stands for $\varphi$ used as a membership predicate, and $y \in \{x : \varphi(x)\}$ abbreviates $\varphi(y)$. A class that is equal to a set (some $z$ with $\forall x\,(x \in z \leftrightarrow \varphi(x))$) is a **set**; otherwise it is a **proper class**. The universal class $V = \{x : x = x\}$ is a proper class, as is the Russell class $R = \{x : x \notin x\}$.

The **ZFC axioms** are the following sentences and schemas [ref: TODO_REF Kunen Ch. I].

*Extensionality.* $\forall x\,\forall y\,\big(\forall z\,(z \in x \leftrightarrow z \in y) \to x = y\big)$. Sets are determined by their members.

*Foundation (Regularity).* $\forall x\,\big(x \neq \varnothing \to \exists y \in x\ (y \cap x = \varnothing)\big)$. Every nonempty set has an $\in$-minimal member.

*Pairing.* $\forall x\,\forall y\,\exists z\,\forall w\,(w \in z \leftrightarrow w = x \lor w = y)$.

*Union.* $\forall \mathcal F\,\exists u\,\forall x\,\big(x \in u \leftrightarrow \exists S \in \mathcal F\ (x \in S)\big)$.

*Power Set.* $\forall x\,\exists y\,\forall z\,(z \in y \leftrightarrow z \subseteq x)$.

*Separation (Comprehension) schema.* For each formula $\varphi(z, \vec p)$ not containing $y$ free, $\forall x\,\forall \vec p\,\exists y\,\forall z\,\big(z \in y \leftrightarrow (z \in x \land \varphi(z, \vec p))\big)$. One may separate a subset of an *existing* set by any property; one may not form $\{z : \varphi(z)\}$ outright. This is the precise repair of naive comprehension that disarms Russell's paradox: $\{z \in x : z \notin z\}$ exists for every set $x$, but is not a member of $x$, so no contradiction arises.

*Replacement schema.* For each formula $\psi(z, w, \vec p)$ functional in $w$, $\forall x\,\forall \vec p\,\Big(\big(\forall z \in x\ \exists! w\ \psi\big) \to \exists y\,\forall w\,(w \in y \leftrightarrow \exists z \in x\ \psi)\big)$. The image of a set under a definable class function is a set.

*Infinity.* $\exists x\,\big(\varnothing \in x \land \forall y \in x\ (y \cup \{y\} \in x)\big)$. There is an inductive set.

*Choice (AC).* Every set of nonempty pairwise-disjoint sets has a choice set; equivalently every set is well-orderable. Choice is stated here but its theory — the equivalents (Zorn, well-ordering) and independence — is developed separately in [42.03.05].

The theory ZF is the list without Choice; ZFC adjoins it. Ordered pairs are coded à la Kuratowski, $(x, y) := \{\{x\}, \{x, y\}\}$, which satisfies $(x, y) = (x', y') \leftrightarrow (x = x' \land y = y')$; the Cartesian product $x \times y = \{(a, b) : a \in x, b \in y\}$ is a set by Separation applied to $\mathcal P(\mathcal P(x \cup y))$. A **relation** is a set of ordered pairs, a **function** a relation that is single-valued, and an **ordering** a relation with the appropriate reflexivity/transitivity/antisymmetry. All of relation theory thus lives inside the set universe.

### Counterexamples to common slips [Intermediate+]

- *"$V = \{x : x = x\}$ is a set."* If $V$ were a set, Separation would yield the Russell set $R = \{x \in V : x \notin x\}$, which both is and is not a member of itself. So $V$ is a proper class; the same argument shows the class of all singletons, the class of all ordinals, and $R$ itself are proper classes.

- *"Foundation forbids only $x \in x$."* Foundation forbids every finite or infinite descending $\in$-chain $\cdots \in x_2 \in x_1 \in x_0$, not merely the one-step loop $x \in x$. The two-step loop $x \in y \in x$ and the infinite descent are also ruled out, because the set $\{x_0, x_1, x_2, \ldots\}$ would have no $\in$-minimal element.

- *"Replacement is just Separation in disguise."* Separation only carves subsets out of a *given* set and so cannot increase rank or cardinality; Replacement can. The set $\{\omega, \mathcal P(\omega), \mathcal P(\mathcal P(\omega)), \ldots\}$ and the ordinal $\omega + \omega$ require Replacement: the relevant collection is the image of $\omega$ under a class function but is not a subset of any set produced without it.

- *"$\{\varnothing\}$ and $\varnothing$ are the same because both are 'empty'."* $\varnothing$ has no members; $\{\varnothing\}$ has exactly one member, namely $\varnothing$. By Extensionality they differ, since $\varnothing \in \{\varnothing\}$ but $\varnothing \notin \varnothing$. The rank of $\varnothing$ is $0$ and the rank of $\{\varnothing\}$ is $1$.

## Key theorem with proof [Intermediate+]

The cumulative hierarchy is defined by transfinite recursion on the ordinals, and the signature theorem is that, granting Foundation, it exhausts the universe: every set lives at some level, so $V = \bigcup_\alpha V_\alpha$. This is what licenses proofs "by induction on rank" throughout set theory.

**Definition (cumulative hierarchy).** By recursion on ordinals,
$$
V_0 = \varnothing, \qquad V_{\alpha+1} = \mathcal P(V_\alpha), \qquad V_\lambda = \bigcup_{\alpha < \lambda} V_\alpha \ \ (\lambda \text{ limit}).
$$
Each $V_\alpha$ is a set (Power Set at successors, Replacement and Union at limits). The $V_\alpha$ are transitive and $\subseteq$-increasing. The **rank** of a set $x$ is $\operatorname{rank}(x) = $ the least $\alpha$ with $x \in V_{\alpha+1}$, equivalently $\operatorname{rank}(x) = \sup\{\operatorname{rank}(y) + 1 : y \in x\}$.

**Theorem (the hierarchy exhausts $V$).** *Assume the axioms of ZF, including Foundation. Then every set belongs to some $V_\alpha$; that is, $V = \bigcup_{\alpha \in \mathrm{Ord}} V_\alpha$, and every set has an ordinal rank.*

*Proof.* Call a set $x$ **grounded** if $x \in V_\alpha$ for some ordinal $\alpha$. First note that $x$ is grounded if and only if every member of $x$ is grounded. If $x \in V_\alpha$, then since each $V_\alpha$ is transitive every $y \in x$ lies in $V_\alpha$ as well, so members of grounded sets are grounded. Conversely, suppose every $y \in x$ is grounded, with $\operatorname{rank}(y)$ defined; by Replacement the image $\{\operatorname{rank}(y) + 1 : y \in x\}$ is a set of ordinals, so it has a supremum $\beta$. Then every $y \in x$ lies in $V_\beta$, hence $x \subseteq V_\beta$, so $x \in \mathcal P(V_\beta) = V_{\beta+1}$ and $x$ is grounded.

Now suppose toward a contradiction that some set is not grounded. By Foundation applied to a suitable transitive set, choose a set $z$ that is not grounded but all of whose members are grounded: form the transitive closure $\operatorname{trcl}(x)$ of an ungrounded $x$ (a set, built by Union and Replacement over the $\omega$-iterated unions), let $A \subseteq \operatorname{trcl}(x) \cup \{x\}$ be the nonempty set of its ungrounded members together with $x$, and apply Foundation to $A$ to get an $\in$-minimal $z \in A$. By minimality, no member of $z$ is in $A$, so every member of $z$ is grounded. But then $z$ is grounded by the equivalence above — contradicting $z \in A$. Hence no ungrounded set exists, and every set lies in some $V_\alpha$ with a well-defined rank. $\square$

**Bridge.** This theorem is the foundational reason set theory can argue by induction on rank: every set sits at a definite ordinal level, so a property holding at all lower levels and propagating upward holds everywhere, which is exactly the $\in$-induction principle proved in the next section. It generalises ordinary induction on $\omega$ to the whole universe, and it builds toward the construction of the ordinals [42.03.02] and cardinals [42.03.03], which are themselves the index set $\mathrm{Ord}$ and certain ranks inside this tower. The central insight is that Foundation and the hierarchy are two faces of one fact — the membership relation is well-founded — and this is dual to the way the ordinals well-order the levels from below; putting these together, the universe $V$ is not a completed totality but the union of an unbounded, increasing sequence of sets, a picture that appears again in the Reflection Principle, where finite fragments of the theory of $V$ are already true in some level $V_\alpha$.

<aside class="exercise" data-difficulty="medium" data-type="symbolic" markdown="1">

**Exercise 3 (medium, symbolic).**

Prove that each $V_\alpha$ is transitive (every member of a member is a member), by induction on $\alpha$.

<details class="exercise__hint"><summary>Hint</summary>

Transitivity is preserved by power set and by union. Handle $0$, successor, and limit separately.

</details>

<details class="exercise__answer"><summary>Answer</summary>

By transfinite induction. Base: $V_0 = \varnothing$ is vacuously transitive. Successor: assume $V_\alpha$ transitive and let $x \in V_{\alpha+1} = \mathcal P(V_\alpha)$, so $x \subseteq V_\alpha$. If $y \in x$ then $y \in V_\alpha$, and transitivity of $V_\alpha$ together with $V_\alpha \subseteq \mathcal P(V_\alpha) = V_{\alpha+1}$ (every member of $V_\alpha$ is a subset of $V_\alpha$, hence in $\mathcal P(V_\alpha)$) gives $y \in V_{\alpha+1}$. Limit: $V_\lambda = \bigcup_{\alpha<\lambda} V_\alpha$; if $x \in V_\lambda$ then $x \in V_\alpha$ for some $\alpha < \lambda$, and by induction every $y \in x$ lies in $V_\alpha \subseteq V_\lambda$. Rubric: full credit for the inclusion $V_\alpha \subseteq V_{\alpha+1}$ at the successor step and correct handling of all three cases.

</details>

</aside>

<aside class="exercise" data-difficulty="medium" data-type="symbolic" markdown="1">

**Exercise 4 (medium, symbolic).**

Show that the Russell class $R = \{x : x \notin x\}$ is a proper class. Then show, using Foundation, that $R = V$ — that is, no set is a member of itself.

<details class="exercise__hint"><summary>Hint</summary>

For the first part, suppose $R$ is a set and ask whether $R \in R$. For the second, apply Foundation to $\{x\}$.

</details>

<details class="exercise__answer"><summary>Answer</summary>

If $R$ were a set, then $R \in R$ holds if and only if $R \notin R$ by the defining condition, a contradiction; so $R$ is a proper class. For $R = V$: let $x$ be any set and apply Foundation to $\{x\}$, which is nonempty, to get an $\in$-minimal member. The only member is $x$, and minimality says $x \cap \{x\} = \varnothing$, i.e. $x \notin x$. Hence every set satisfies $x \notin x$, so $x \in R$ for all $x$ and $R = V$. (Thus under Foundation the Russell condition is satisfied by everything, yet collecting everything still fails — the obstruction is comprehension, not the predicate.) Rubric: full credit for the self-membership contradiction and the Foundation-on-$\{x\}$ argument.

</details>

</aside>

<aside class="exercise" data-difficulty="medium" data-type="symbolic" markdown="1">

**Exercise 5 (medium, symbolic).**

Verify the Kuratowski pairing axiom: prove $(x,y) = (x',y')$ implies $x = x'$ and $y = y'$, where $(a,b) := \{\{a\},\{a,b\}\}$.

<details class="exercise__hint"><summary>Hint</summary>

Use Extensionality. Split into the cases $x = y$ and $x \neq y$.

</details>

<details class="exercise__answer"><summary>Answer</summary>

Suppose $\{\{x\},\{x,y\}\} = \{\{x'\},\{x',y'\}\}$. Case $x = y$: then $(x,y) = \{\{x\}\}$, a singleton, so the right side is a singleton too, forcing $\{x'\} = \{x',y'\}$, hence $x' = y'$ and $\{x'\} = \{x\}$, giving $x' = x = y = y'$. Case $x \neq y$: then $\{x\} \neq \{x,y\}$, so $(x,y)$ has two distinct members. The singleton member $\{x\}$ must equal the singleton member of the right side; if $\{x\} = \{x',y'\}$ then $x' = y' = x$, making the right side a singleton, contradiction, so $\{x\} = \{x'\}$ and $x = x'$. Then $\{x,y\} = \{x',y'\} = \{x,y'\}$ with $x \neq y$ forces $y = y'$. Rubric: full credit for both cases and use of Extensionality to compare the singleton components.

</details>

</aside>

<aside class="exercise" data-difficulty="medium" data-type="symbolic" markdown="1">

**Exercise 6 (medium, symbolic).**

Prove that Replacement implies Separation, given the other axioms. (So Separation is redundant in the official list, though it is kept for clarity.)

<details class="exercise__hint"><summary>Hint</summary>

Given $x$ and $\varphi$, define a class function that is the identity on members satisfying $\varphi$ and sends the rest somewhere fixed; handle the empty case separately.

</details>

<details class="exercise__answer"><summary>Answer</summary>

Fix a set $x$ and a formula $\varphi(z)$. If no $z \in x$ satisfies $\varphi$, the desired set is $\varnothing$ (a set by Separation-free means: $\varnothing = \{z \in x : z \neq z\}$ is replaced below, or take Pairing then Replacement). Otherwise fix some $z_0 \in x$ with $\varphi(z_0)$ and define the class function $\psi(z, w) :\equiv (\varphi(z) \land w = z) \lor (\neg\varphi(z) \land w = z_0)$. This is functional: each $z \in x$ has a unique image $w$ (itself if $\varphi(z)$, else $z_0$). By Replacement the image $y = \{w : \exists z \in x\ \psi(z,w)\}$ is a set, and $y = \{z \in x : \varphi(z)\}$ since every output is some $z \in x$ with $\varphi(z)$ (including $z_0$). Hence the Separation instance for $\varphi$ holds. Rubric: full credit for the functional class definition with the fixed fallback $z_0$ and the empty-case handling.

</details>

</aside>

<aside class="exercise" data-difficulty="hard" data-type="symbolic" markdown="1">

**Exercise 7 (hard, symbolic).**

Prove the principle of $\in$-induction from Foundation: if $\varphi$ is a formula such that $\forall x\,\big((\forall y \in x\ \varphi(y)) \to \varphi(x)\big)$, then $\forall x\,\varphi(x)$.

<details class="exercise__hint"><summary>Hint</summary>

Suppose $\varphi$ fails somewhere. Pass to the transitive closure of a witness and apply Foundation to the set of failures inside it.

</details>

<details class="exercise__answer"><summary>Answer</summary>

Suppose the inductive hypothesis $\forall x\,((\forall y \in x\ \varphi(y)) \to \varphi(x))$ holds but $\varphi(a)$ fails for some $a$. Let $t = \operatorname{trcl}(\{a\})$ be the transitive closure (a set, formed by iterating Union over $\{a\}, \bigcup\{a\}, \ldots$ and taking the union of the $\omega$-sequence via Replacement). The set $F = \{z \in t : \neg\varphi(z)\}$ is nonempty since $a \in F$. By Foundation choose an $\in$-minimal $z \in F$. For each $y \in z$: since $t$ is transitive, $y \in t$, and by minimality $y \notin F$, so $\varphi(y)$. Thus $\forall y \in z\ \varphi(y)$, and the inductive hypothesis gives $\varphi(z)$, contradicting $z \in F$. Hence no failure exists and $\forall x\,\varphi(x)$. Rubric: full credit for passing to the transitive closure, applying Foundation to the failure set, and using transitivity to feed the inductive hypothesis.

</details>

</aside>

<aside class="exercise" data-difficulty="hard" data-type="symbolic" markdown="1">

**Exercise 8 (hard, symbolic).**

Let $\kappa$ be an inaccessible cardinal (regular and a strong limit; this anticipates [42.03.03]). Sketch why $V_\kappa$ satisfies Replacement, and hence is a model of all of ZFC. (You may use that $V_\kappa$ already models the other axioms.)

<details class="exercise__hint"><summary>Hint</summary>

A definable function on a set of size $< \kappa$ has image of size $< \kappa$ by regularity; bound the ranks of the image using strong-limit-ness.

</details>

<details class="exercise__answer"><summary>Answer</summary>

Work inside $V_\kappa$ and take an instance of Replacement: a set $a \in V_\kappa$ and a class function $F$ (definable with parameters in $V_\kappa$) defined on $a$. Since $a \in V_\kappa$ we have $|a| < \kappa$ (as $|V_\alpha| < \kappa$ for $\alpha < \kappa$, using that $\kappa$ is a strong limit so each power-set step stays below $\kappa$). The image $F[a]$ has $|F[a]| \le |a| < \kappa$. Each $F(z)$ lies in some $V_{\beta_z}$ with $\beta_z < \kappa$; by regularity of $\kappa$, $\sup_{z \in a} \beta_z < \kappa$, call it $\beta$. Then $F[a] \subseteq V_\beta$ and $|F[a]| < \kappa$, so $F[a] \in V_{\beta+1} \subseteq V_\kappa$. Thus the image is a set in $V_\kappa$, verifying Replacement. With the other axioms holding in $V_\kappa$ (Power Set from strong-limit-ness, Infinity since $\kappa > \omega$, Choice inherited from $V$), $V_\kappa \models \mathrm{ZFC}$. Rubric: full credit for using regularity to bound the supremum of ranks and strong-limit-ness to keep cardinalities below $\kappa$.

</details>

</aside>

## Advanced results [Master]

The hierarchy makes the universe a well-ordered union of sets, and the deeper structural facts concern how the theory of $V$ is approximated from within and why this approximation is exactly as strong as it can consistently be.

**Theorem 1 ($\in$-recursion).** Let $G$ be a class function. There is a unique class function $F$ on $V$ satisfying $F(x) = G(x, F \restriction x)$ for all $x$, where $F \restriction x = \{(y, F(y)) : y \in x\}$ [ref: TODO_REF Kunen Ch. I]. The proof builds approximations $F_\alpha$ on $V_\alpha$ by recursion on rank, shows any two approximations agree on their common domain by $\in$-induction, and takes the union. Rank itself is the instance $G(x, h) = \sup\{h(y) + 1 : y \in x\}$, and the transitive closure, the von Neumann ordinals, and the constructible hierarchy $L_\alpha$ of [42.03.06] are all defined this way. $\in$-recursion is the universe-wide form of transfinite recursion: the well-foundedness of $\in$ guaranteed by Foundation is exactly the input that makes recursive definitions over $V$ legitimate.

**Theorem 2 (the Lévy-Montague Reflection Principle).** For every formula $\varphi(\vec v)$ of the language of set theory and every ordinal $\beta$, there is a limit ordinal $\alpha > \beta$ such that $\varphi$ is **absolute between $V_\alpha$ and $V$**: for all $\vec a \in V_\alpha$, $\varphi^{V_\alpha}(\vec a) \leftrightarrow \varphi(\vec a)$ [ref: TODO_REF Kunen Ch. I]. More strongly, for any *finite* list $\varphi_1, \ldots, \varphi_n$ the class of such $\alpha$ is closed and unbounded. The proof takes the finitely many subformulas, uses Replacement to define for each existential subformula a function $\alpha \mapsto$ (least level adding a witness when one exists in $V$), and iterates $\omega$ times to a closure ordinal at which every existential subformula true in $V$ already has a witness in $V_\alpha$; a Tarski-Vaught test then yields absoluteness. Reflection is a theorem schema, one instance per finite formula list, and cannot be internalised to a single sentence "$V$ reflects every formula" without a truth predicate, which Tarski's theorem forbids.

**Theorem 3 (non-finite-axiomatisability of ZF).** ZF is not finitely axiomatisable: no finite subset of ZF (indeed no single sentence consistent with ZF) proves all of ZF [ref: TODO_REF Jech Ch. 12]. Given a finite fragment $T \subseteq \mathrm{ZF}$, Reflection produces, provably in ZF, an ordinal $\alpha$ with $V_\alpha \models T$; so ZF proves "$T$ has a set model," whence ZF proves $\operatorname{Con}(T)$. If some finite $T$ proved all of ZF, then $T$ would prove $\operatorname{Con}(T)$, contradicting Gödel's second incompleteness theorem (any consistent recursively axiomatised theory interpreting arithmetic cannot prove its own consistency). The same mechanism shows ZF proves the consistency of each of its finite fragments while — if consistent — never proving $\operatorname{Con}(\mathrm{ZF})$ itself.

**Theorem 4 (unprovability of $\operatorname{Con}(\mathrm{ZFC})$).** If ZFC is consistent, then ZFC does not prove $\operatorname{Con}(\mathrm{ZFC})$, and a fortiori does not prove the existence of a set model of ZFC, equivalently does not prove that any $V_\kappa \models \mathrm{ZFC}$ [ref: TODO_REF Jech Ch. 12]. By Theorem 3, asserting "there is an $\alpha$ with $V_\alpha \models \mathrm{ZFC}$" exceeds ZFC: it implies $\operatorname{Con}(\mathrm{ZFC})$. The least $\kappa$ with $V_\kappa \models \mathrm{ZFC}$ — when one exists — need not be inaccessible (it may be of cofinality $\omega$ by a Löwenheim-Skolem-and-collapse argument), but the existence of an *inaccessible* $\kappa$, giving $V_\kappa \models \mathrm{ZFC}$ for full second-order reasons, is a genuine large-cardinal strengthening beyond ZFC. This is the first rung of the large-cardinal hierarchy and the reason Reflection cannot be pushed to a single internal schema.

**Theorem 5 (rank arithmetic and closure of $V_\alpha$).** For sets $x, y$: $\operatorname{rank}(\{x, y\}) = \max(\operatorname{rank}(x), \operatorname{rank}(y)) + 1$, $\operatorname{rank}(\mathcal P(x)) = \operatorname{rank}(x) + 1$, $\operatorname{rank}((x,y)) = \max(\operatorname{rank}(x), \operatorname{rank}(y)) + 2$, and $\operatorname{rank}(\bigcup x) \le \operatorname{rank}(x)$. Consequently $V_\alpha$ for $\alpha$ a limit is closed under Pairing, Union, and Separation, and $V_\alpha$ for $\alpha > \omega$ a limit additionally models Infinity; $V_\omega$ is exactly the hereditarily finite sets and models ZFC minus Infinity. These rank computations are what make the levels usable as models and underlie the consistency arguments above.

**Synthesis.** The cumulative hierarchy is the central insight unifying every theme of foundational set theory: Foundation makes $\in$ well-founded, well-foundedness licenses $\in$-recursion and $\in$-induction, and these in turn construct the very ordinals that index the hierarchy, so the tower defines the apparatus that defines the tower. The foundational reason the universe is the increasing union $V = \bigcup_\alpha V_\alpha$ rather than a completed object is that each level is a set while the whole is a proper class, and this is exactly what the Reflection Principle exploits: any finite portion of the theory of the proper-class $V$ is already true in some set-level $V_\alpha$, which generalises the naive idea of "taking a large enough stage" into a precise closed-unbounded reflection schema. Putting these together with Gödel's second incompleteness theorem, the schematic — never single-sentence — character of Reflection becomes inevitable: ZFC proves the consistency of each finite fragment of itself (by reflecting it into some $V_\alpha$) yet cannot prove its own consistency, so it cannot prove that any single $V_\kappa$ models all of ZFC. The hierarchy is dual to the ordinal sequence that climbs it, and the bridge from this unit to the rest of the chapter is that ordinals [42.03.02], cardinals [42.03.03], the constructible universe $L$ [42.03.06], and forcing extensions are all constructions *inside* this tower, each one a controlled way of building or shrinking the levels $V_\alpha$ while keeping the membership relation well-founded.

## Full proof set [Master]

**Proposition 1 (rank is well-defined and ordinal-valued).** *Under ZF, the assignment $\operatorname{rank}(x) = \sup\{\operatorname{rank}(y) + 1 : y \in x\}$ defines a function $\operatorname{rank} : V \to \mathrm{Ord}$, and $x \in V_\alpha \leftrightarrow \operatorname{rank}(x) < \alpha$.*

*Proof.* Existence and uniqueness of $\operatorname{rank}$ as a class function come from $\in$-recursion (Theorem 1) with $G(x, h) = \sup(\operatorname{ran}(h \restriction x) + 1)$; the image is a set of ordinals by Replacement, so its supremum is an ordinal, and the values are ordinals by induction on rank. For the equivalence, argue by induction on $\alpha$. At $\alpha = 0$ both sides fail. At a successor $\alpha = \gamma + 1$: $x \in V_{\gamma+1} = \mathcal P(V_\gamma)$ iff $x \subseteq V_\gamma$ iff every $y \in x$ has $\operatorname{rank}(y) < \gamma$ (induction) iff $\sup\{\operatorname{rank}(y)+1 : y \in x\} \le \gamma$ iff $\operatorname{rank}(x) \le \gamma$ iff $\operatorname{rank}(x) < \gamma+1$. At a limit $\lambda$: $x \in V_\lambda$ iff $x \in V_\gamma$ for some $\gamma < \lambda$ iff $\operatorname{rank}(x) < \gamma < \lambda$ for some such $\gamma$ iff $\operatorname{rank}(x) < \lambda$. $\square$

**Proposition 2 ($V_\alpha$ is transitive and $\subseteq$-increasing).** *For all ordinals $\alpha \le \beta$, $V_\alpha$ is transitive and $V_\alpha \subseteq V_\beta$.*

*Proof.* Transitivity is Exercise 3. For monotonicity, induct on $\beta$. If $\beta = \gamma + 1$ and $V_\alpha \subseteq V_\gamma$ for $\alpha \le \gamma$, then since $V_\gamma$ is transitive each $x \in V_\gamma$ satisfies $x \subseteq V_\gamma$, so $x \in \mathcal P(V_\gamma) = V_{\gamma+1}$; thus $V_\gamma \subseteq V_{\gamma+1}$ and composing gives $V_\alpha \subseteq V_{\gamma+1}$. The case $\alpha = \beta$ is identity; at limits $V_\alpha \subseteq \bigcup_{\gamma < \lambda} V_\gamma = V_\lambda$ for $\alpha < \lambda$ directly. $\square$

**Proposition 3 (Reflection, single-formula form).** *For every formula $\varphi(\vec v)$ and every ordinal $\beta_0$ there is a limit $\alpha > \beta_0$ with $\forall \vec a \in V_\alpha\,(\varphi^{V_\alpha}(\vec a) \leftrightarrow \varphi(\vec a))$.*

*Proof.* List the subformulas $\varphi = \varphi_0, \ldots, \varphi_m$ closed under subformula. For each existential subformula $\varphi_i = \exists w\,\varphi_j(w, \vec v)$ define $H_i(\vec a)$ to be the least $\gamma$ such that $V_\gamma$ contains a witness $w$ with $\varphi_j(w,\vec a)$ if any exists in $V$, and $0$ otherwise; $H_i$ is a class function by Replacement. Define an increasing $\omega$-sequence $\beta_0 < \beta_1 < \cdots$ by letting $\beta_{n+1}$ exceed $\beta_n$ and exceed $\sup\{H_i(\vec a) : \vec a \in V_{\beta_n},\ i \le m\}$ (a set, by Replacement). Let $\alpha = \sup_n \beta_n$, a limit. Then for every $\vec a \in V_\alpha$ and every existential subformula, if a witness exists in $V$ one exists in $V_\alpha$. By induction on subformula complexity (the Tarski-Vaught condition), $\varphi_i^{V_\alpha}(\vec a) \leftrightarrow \varphi_i(\vec a)$ for all $i$ and all $\vec a \in V_\alpha$; the case $\varphi_i = \varphi$ is the claim. Atomic and Boolean steps are immediate from $V_\alpha \subseteq V$; the existential step uses witness reflection. $\square$

**Proposition 4 (reflection yields consistency of finite fragments).** *For each finite $T \subseteq \mathrm{ZF}$, $\mathrm{ZF} \vdash \exists \alpha\,(V_\alpha \models T)$, hence $\mathrm{ZF} \vdash \operatorname{Con}(T)$.*

*Proof.* Let $T = \{\sigma_1, \ldots, \sigma_k\}$ and let $\varphi = \sigma_1 \land \cdots \land \sigma_k$. By Proposition 3 there is a limit $\alpha$ with $\varphi$ absolute between $V_\alpha$ and $V$. Each $\sigma_i$ is a ZF theorem, so $\sigma_i$ holds in $V$, hence $\sigma_i^{V_\alpha}$ holds; thus $V_\alpha \models T$. Since $V_\alpha$ is a set, "$T$ has a model" is provable in ZF, and by the (arithmetised) completeness theorem the existence of a model entails $\operatorname{Con}(T)$, so $\mathrm{ZF} \vdash \operatorname{Con}(T)$. If a single finite $T$ proved all of ZF, then $T \vdash \operatorname{Con}(T)$, contradicting Gödel's second incompleteness theorem; hence ZF is not finitely axiomatisable. $\square$

## Connections [Master]

- The von Neumann ordinals are the transitive sets well-ordered by $\in$, constructed by the $\in$-recursion and rank machinery established here; they are co-produced as [42.03.02] and serve as the index set $\mathrm{Ord}$ that climbs the cumulative hierarchy. The foundational reason ordinals and the hierarchy are inseparable is that $\operatorname{rank}$ takes ordinal values and the ordinals are exactly the ranks of the well-ordered "spine" of $V$, so the construction of the levels $V_\alpha$ and the construction of the indices $\alpha$ are a single transfinite recursion.

- Cardinals and cardinal arithmetic [42.03.03] live inside this tower as distinguished ordinals (initial ordinals), and the rank-arithmetic and closure facts of Theorem 5 are what let $V_\kappa$ serve as a model when $\kappa$ is inaccessible; the large-cardinal hierarchy begins exactly where Theorem 4 stops, with the assertion "$\exists \kappa\ V_\kappa \models \mathrm{ZFC}$" that ZFC cannot prove. This unit owns the ambient hierarchy; that unit owns the cardinal invariants measured inside it.

- The Axiom of Choice, stated here as the ninth ZFC axiom, has its equivalents (Zorn's lemma, the well-ordering theorem, Tychonoff) and its independence developed in [42.03.05]; the well-ordering theorem is what lets every set be assigned a position in the ordinal spine, making the rank-and-hierarchy picture compatible with the choice-driven constructions, and Gödel's constructible universe $L$ [42.03.06] is the choice-respecting inner-model refinement of the very cumulative tower defined in this unit, replacing $\mathcal P(V_\alpha)$ by the definable power set at each successor step.

## Historical & philosophical context [Master]

The axiomatic treatment of sets began with Ernst Zermelo's 1908 axiomatisation, motivated by the paradoxes of Russell (1901) and Burali-Forti and by the need to secure his 1904 well-ordering theorem. Zermelo's Separation (Aussonderung) replaced Frege's inconsistent unrestricted comprehension, blocking Russell's paradox by allowing only the carving of subsets from existing sets. Abraham Fraenkel and Thoralf Skolem independently added the Replacement schema in the early 1920s, completing what is now ZF; Skolem also clarified that the axioms are first-order schemas in the language with $\in$.

The cumulative-hierarchy picture and the rank function are due to John von Neumann (1923-1928), who introduced the ordinals as transitive sets and defined the stratification of the universe by transfinite iteration of the power set; Dmitry Mirimanoff had anticipated the well-foundedness idea in 1917. Ernst Zermelo's 1930 paper *Über Grenzzahlen und Mengenbereiche* [ref: TODO_REF Zermelo 1930] cast the universe as an open-ended sequence of models $V_\kappa$ indexed by inaccessible "boundary numbers," giving the modern picture in which reflection and large cardinals are continuous with the basic hierarchy. The Axiom of Foundation, isolating the well-founded universe, was formulated by von Neumann and Zermelo and shown by them to make $V = \bigcup_\alpha V_\alpha$.

The Reflection Principle in its schematic form was proved by Azriel Lévy and Richard Montague around 1960, and Lévy used it to establish that ZF is not finitely axiomatisable. Its connection to Gödel's 1931 second incompleteness theorem — that ZF proves the consistency of each finite fragment of itself yet, if consistent, never proves $\operatorname{Con}(\mathrm{ZF})$ — is the precise sense in which the cumulative hierarchy is inexhaustible from within. Kunen's textbook treatment [ref: TODO_REF Kunen Ch. I] and Jech's [ref: TODO_REF Jech Ch. 6] are the standard modern developments of this material.
