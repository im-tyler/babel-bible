## Intuition [Beginner]

The previous unit showed that arithmetic can mirror any computation. This unit cashes that in for the most famous result in logic. The first move is a coding trick. Pick a fixed scheme that turns every symbol, every formula, and every written proof into a whole number — the way text is stored as numbers inside a computer. Once that is fixed, a statement about formulas becomes a statement about numbers, and a theory that talks about numbers can be made to talk about its own sentences and its own proofs.

Now comes the surprising part. Using this coding, you can build a single sentence that, read through the code, says of itself: "I have no proof in this theory." Call it $G$. Think about whether $G$ could be provable. If the theory could prove $G$, then $G$ would be a proved sentence that announces it has no proof — the theory would be proving a falsehood. A trustworthy theory does not do that. So $G$ is not provable. But that is exactly what $G$ claims. So $G$ is *true* and yet *unprovable*.

The conclusion is stark. Any reasonable theory of arithmetic — one that is consistent and whose axioms a machine can list — must leave some true statements unproved. You cannot fix this by adding more axioms: the same trick builds a new unprovable truth for the patched-up theory. There is no complete, consistent, machine-checkable bible of all arithmetic truths.

A second sentence makes it worse. The statement "this theory is consistent" can itself be coded as a sentence of arithmetic. A consistent theory cannot prove its own consistency. The theory's faith in itself is one thing it can never establish from the inside.

## Visual [Beginner]

The picture is a loop that bites its own tail. A coding scheme sends each sentence to a number; a special sentence is built so that, decoded, it talks about its own number. The diagram traces how "I am unprovable" closes the loop.

```
   STEP 1: code everything as numbers
       sentence  phi   --->  number  #phi
       a proof   P     --->  number  #P
       "p is a proof of the sentence coded x"  ---> a number-relation Prf(p, x)

   STEP 2: build a sentence G that points back at its own code

        G   says:   "there is NO number p with  Prf( p , #G )"
                                                       ^^^^
                                  G's claim is ABOUT G's own code

   STEP 3: ask if G can be proved

        if the theory proves G   -->  there IS a proof of G
                                  -->  so "no proof of G" is FALSE
                                  -->  the theory proved a falsehood   (not allowed)

        therefore   G is NOT provable
        but that is exactly what G says   -->  G is TRUE

   RESULT:  a sentence that is true but has no proof.
```

A *code* is a fixed dictionary: every sentence gets a number, no two sentences share a number, and you can mechanically translate back and forth. Once "$p$ is a proof of the sentence with number $x$" is itself a relation between numbers, arithmetic can express it, and the self-pointing sentence $G$ becomes possible.

| sentence built | what it says about itself | the outcome |
|---|---|---|
| $G$ | "I am not provable" | true but unprovable |
| Con | "this theory is consistent" | true but unprovable (if the theory is consistent) |
| (a Liar) | "I am false / I am untrue" | shows truth is not codeable at all |

The same self-pointing move yields three results: an unprovable truth, an unprovable consistency, and the impossibility of a truth-detector.

## Worked example [Beginner]

We watch the self-reference get built with a tiny made-up coding, to see that nothing magical happens — it is bookkeeping. Suppose our only formulas are short strings, and we fix a dictionary that assigns each one a number. Say the formula "$x$ is even" gets the number $40$, and the formula "$x$ has no proof" gets the number $77$. These numbers are the codes.

Step 1. Set up one operation on codes: substitution. Given the code of a formula with a blank $x$, and a number $n$, "substitute $n$ for $x$" produces the code of the new sentence. Write this as $\mathrm{sub}(\text{code}, n)$. For instance, $\mathrm{sub}(40, 6)$ is the code of "$6$ is even." This is a definite numerical operation: feed it two numbers, get one number back.

Step 2. We want a sentence that talks about its own code. The clever step is to substitute a formula's code *into that same formula*. Take the formula "the sentence coded $\mathrm{sub}(x, x)$ has no proof," and let its code be some number, say $90$.

Step 3. Now plug $90$ in for $x$ in that very formula. By definition the result is the sentence "the sentence coded $\mathrm{sub}(90, 90)$ has no proof." But $\mathrm{sub}(90, 90)$ is, by construction, the code of this very sentence we just wrote. So the sentence says "the sentence coded (my own code) has no proof" — it talks about itself.

Step 4. Read what it now claims: "I have no proof." We built a self-referential sentence using only the substitution operation on codes, no paradox and no infinite regress.

What this tells us: self-reference is not a mystical trick. It is the ordinary operation of plugging a formula's code into itself, which the substitution function makes a plain numerical computation. Because arithmetic can carry out that computation, arithmetic can host sentences that speak about themselves — and "I am unprovable" is one of them.

## Formal definition [Intermediate+]

Fix the language of arithmetic $\mathcal{L}_A = \{0, S, +, \cdot, <\}$, its standard model $\mathfrak{N}$ [42.01.04], and a theory $T$ in $\mathcal{L}_A$ that is **recursively axiomatised** (its set of axiom codes is recursive) and **extends $Q$** [42.01.08]. A **Gödel numbering** is an injective effective map $\ulcorner\cdot\urcorner$ assigning to each symbol, term, formula, and finite sequence of formulas a natural number, built from a recursive coding of finite sequences (for instance $\langle a_0, \dots, a_n\rangle = \prod_{i\le n} p_i^{a_i+1}$ with $p_i$ the $i$-th prime, or Gödel's $\beta$-function coding of [42.01.08]). For a formula $\varphi$ the closed term $\underline{\ulcorner\varphi\urcorner} = S^{\ulcorner\varphi\urcorner}0$ is its **numeral**, the $\mathcal{L}_A$-name of its code; we write $\ulcorner\varphi\urcorner$ for this numeral when the context is a formula slot [ref: TODO_REF Enderton §3.4].

The **substitution function** $\mathrm{sub}(\ulcorner\varphi(x)\urcorner, n) = \ulcorner\varphi(\underline n)\urcorner$ and the **diagonal function** $d(e) = \mathrm{sub}(e, e)$ are primitive recursive, hence representable in $Q$. For recursively axiomatised $T$ the relations "$x$ codes an $\mathcal{L}_A$-formula", "$y$ codes a $T$-axiom", and the **proof predicate**
$$
\mathrm{Prf}_T(p, x) \equiv \text{"$p$ codes a finite $T$-deduction whose last line is the formula coded by $x$"}
$$
are primitive recursive, so $\mathrm{Prf}_T$ is representable in $Q$ by a $\Delta_0$ formula (also written $\mathrm{Prf}_T$). The **provability predicate** is the $\Sigma_1$ formula
$$
\mathrm{Pr}_T(x) := \exists p\,\mathrm{Prf}_T(p, x),
$$
and for a sentence $\sigma$ one abbreviates $\Box_T\sigma := \mathrm{Pr}_T(\ulcorner\sigma\urcorner)$. The **consistency statement** is the $\Pi_1$ sentence
$$
\mathrm{Con}_T := \neg\,\mathrm{Pr}_T(\ulcorner 0 = S0\urcorner),
$$
expressing that the refutable sentence $0 = S0$ has no $T$-proof.

A theory $T$ is **$\omega$-consistent** when for no formula $\theta(x)$ does $T \vdash \exists x\,\theta(x)$ while $T \vdash \neg\theta(\underline n)$ for every $n \in \mathbb{N}$; $\omega$-consistency implies consistency but is strictly stronger. The **truth set** is $\#\mathrm{Th}(\mathfrak{N}) = \{\ulcorner\sigma\urcorner : \sigma \text{ a sentence}, \mathfrak{N} \models \sigma\}$, the set of codes of true arithmetic sentences. A set $A \subseteq \mathbb{N}$ is **arithmetically definable** when $A = \{n : \mathfrak{N} \models \alpha(\underline n)\}$ for some $\mathcal{L}_A$-formula $\alpha(x)$.

### Counterexamples to common slips [Intermediate+]

- *"The Gödel sentence is provably equivalent to a paradox, so the theory is contradictory."* The Liar "I am false" is a genuine paradox because truth is not arithmetically definable (Tarski). The Gödel sentence replaces "false" with the *definable* predicate "not provable", which is $\neg\Sigma_1$, not $\neg$"true"; self-reference through a definable predicate is consistent, and $G_T$ is simply true and unprovable.

- *"Incompleteness needs the full strength of $PA$."* The first theorem needs only that $T$ extends $Q$ and is recursively axiomatised and consistent — no induction. Induction (e.g. $T \supseteq I\Sigma_1$) is needed only for the *second* theorem, whose derivability conditions require provable $\Sigma_1$-completeness inside $T$.

- *"$T \nvdash G_T$ and $T \nvdash \neg G_T$ both follow from consistency."* Only $T \nvdash G_T$ follows from plain consistency. $T \nvdash \neg G_T$ needs $\omega$-consistency in Gödel's original argument; plain consistency suffices only after Rosser's modification of the provability predicate.

- *"The second theorem says no consistency proof of $T$ is possible."* It says $T$ cannot prove $\mathrm{Con}_T$ *in $T$ itself*. A stronger theory can prove $\mathrm{Con}_T$ (Gentzen proved $\mathrm{Con}_{PA}$ using transfinite induction up to $\varepsilon_0$); the bar is on self-certification, and it is sensitive to how consistency is formalised (a Rosser or Feferman predicate can make a $\mathrm{Con}$-variant provable).

## Key theorem with proof [Intermediate+]

The hinge of the entire development is the diagonal lemma: arithmetic admits self-reference. Everything downstream — both incompleteness theorems, Tarski's theorem, Löb's theorem — is an instance of feeding a chosen predicate into a sentence that asserts that predicate of its own code.

**Lemma (Diagonal / Fixed-Point).** *Let $T \supseteq Q$ be recursively axiomatised. For every formula $\psi(x)$ with one free variable there is a sentence $G$ with*
$$
T \vdash G \leftrightarrow \psi(\ulcorner G\urcorner).
$$
*[ref: TODO_REF Enderton §3.4].*

*Proof.* The diagonal function $d(e) = \mathrm{sub}(e, e)$ is primitive recursive, hence by the representability theorem [42.01.08] there is a formula $\mathrm{Diag}(x, y)$ representing its graph in $Q$: for all $e$, $Q \vdash \forall y\,(\mathrm{Diag}(\underline e, y) \leftrightarrow y = \underline{d(e)})$. Define the formula
$$
\theta(x) := \exists y\,\big(\mathrm{Diag}(x, y) \wedge \psi(y)\big),
$$
let $g = \ulcorner\theta(x)\urcorner$ be its code, and set $G := \theta(\underline g)$, a sentence. By definition of $d$ and $\mathrm{sub}$, $d(g) = \mathrm{sub}(g, g) = \ulcorner\theta(\underline g)\urcorner = \ulcorner G\urcorner$. Now $T$ proves $\mathrm{Diag}(\underline g, y) \leftrightarrow y = \underline{d(g)} = \ulcorner G\urcorner$, so inside $T$,
$$
G = \theta(\underline g) = \exists y\,(\mathrm{Diag}(\underline g, y) \wedge \psi(y)) \;\leftrightarrow\; \psi(\underline{d(g)}) \;=\; \psi(\ulcorner G\urcorner).
$$
The existential collapses because $\mathrm{Diag}(\underline g, y)$ pins $y$ to the single value $\underline{\ulcorner G\urcorner}$ provably in $T$. Hence $T \vdash G \leftrightarrow \psi(\ulcorner G\urcorner)$. $\square$

**Theorem (Gödel's First Incompleteness Theorem).** *Let $T \supseteq Q$ be consistent and recursively axiomatised. Let $G_T$ be a fixed point of $\psi(x) = \neg\mathrm{Pr}_T(x)$, so $T \vdash G_T \leftrightarrow \neg\mathrm{Pr}_T(\ulcorner G_T\urcorner)$. Then $T \nvdash G_T$, and if $T$ is $\omega$-consistent then $T \nvdash \neg G_T$; so $T$ is incomplete. Moreover $G_T$ is a true $\Pi_1$ sentence* [ref: TODO_REF Enderton §3.5].

*Proof.* Suppose $T \vdash G_T$. Then there is a $T$-proof of $G_T$, coded by some $p$, so $\mathfrak{N} \models \mathrm{Prf}_T(\underline p, \ulcorner G_T\urcorner)$; this is a true $\Sigma_1$ fact, so by $\Sigma_1$-completeness $T \vdash \mathrm{Prf}_T(\underline p, \ulcorner G_T\urcorner)$, whence $T \vdash \mathrm{Pr}_T(\ulcorner G_T\urcorner)$. But the fixed-point equivalence gives $T \vdash G_T \to \neg\mathrm{Pr}_T(\ulcorner G_T\urcorner)$, so from $T \vdash G_T$ we get $T \vdash \neg\mathrm{Pr}_T(\ulcorner G_T\urcorner)$, contradicting consistency. Hence $T \nvdash G_T$.

For the second half, suppose $T \vdash \neg G_T$, i.e. $T \vdash \mathrm{Pr}_T(\ulcorner G_T\urcorner)$, that is $T \vdash \exists p\,\mathrm{Prf}_T(p, \ulcorner G_T\urcorner)$. Since $T \nvdash G_T$, no number $p$ actually codes a proof of $G_T$, so for every $n$ the sentence $\mathrm{Prf}_T(\underline n, \ulcorner G_T\urcorner)$ is false, hence ($\Sigma_1$-completeness, refuting false closed $\Delta_0$ sentences) $T \vdash \neg\mathrm{Prf}_T(\underline n, \ulcorner G_T\urcorner)$ for every $n$. Together with $T \vdash \exists p\,\mathrm{Prf}_T(p, \ulcorner G_T\urcorner)$ this is exactly an $\omega$-inconsistency. So under $\omega$-consistency $T \nvdash \neg G_T$. Finally, $G_T$ is true: $T \nvdash G_T$ means no $p$ codes a proof, so $\mathfrak{N} \models \neg\mathrm{Pr}_T(\ulcorner G_T\urcorner)$, and the fixed point gives $\mathfrak{N} \models G_T$; and $G_T$ is $\Pi_1$ because $\neg\mathrm{Pr}_T$ is $\neg\Sigma_1 = \Pi_1$. $\square$

**Bridge.** The diagonal lemma is the foundational reason a theory of numbers can refer to itself, and the first incompleteness theorem is exactly this self-reference pointed at the provability predicate $\mathrm{Pr}_T$ that representability [42.01.08] makes available. It builds toward Tarski's undefinability of truth — the same diagonal aimed at a hypothetical truth predicate yields a contradiction rather than an unprovable truth, the difference being that "not provable" is definable while "true" is not — and it appears again in the second incompleteness theorem [42.01.10 context], where the metatheoretic step "if $T$ is consistent then $T \nvdash G_T$" is itself formalised inside $T$ as $T \vdash \mathrm{Con}_T \to G_T$. This is the central insight: the gap between truth and provability, measured by the $\Sigma_1/\Pi_1$ asymmetry of [42.01.08], is not an accident of a particular theory but a fixed point of the theory's own provability operator. Putting these together, completeness [42.01.06] and incompleteness do not conflict — completeness equates $\vdash$ with semantic consequence $\models$ across *all* models, while incompleteness concerns truth in the *single* standard model $\mathfrak{N}$, where $G_T$ holds but is underivable because some nonstandard model of $T$ refutes it.

## Advanced results [Master]

The diagonal lemma supports four developments: the abstract first theorem and its hypotheses, Rosser's elimination of $\omega$-consistency, the second theorem through the derivability conditions and Löb's theorem, and the landscape of concrete independent statements and the misreadings the theorems invite.

**Theorem 1 (the first theorem, abstract form, and its exact hypotheses).** For consistent recursively axiomatised $T \supseteq Q$, the Gödel sentence $G_T$ obtained by diagonalising $\neg\mathrm{Pr}_T$ is unprovable; under $\omega$-consistency it is also irrefutable; in every case it is a true $\Pi_1$ sentence [ref: TODO_REF Boolos-Burgess-Jeffrey Ch. 17]. Each hypothesis does exact work: recursive axiomatisability makes $\mathrm{Prf}_T$ primitive recursive and hence representable, so $\mathrm{Pr}_T$ exists as a $\Sigma_1$ formula; extension of $Q$ supplies $\Sigma_1$-completeness, the bridge from a real proof to its formalised witness; consistency blocks $T \vdash G_T$; $\omega$-consistency blocks $T \vdash \neg G_T$. Weakening any one breaks a step: a non-recursively-axiomatised $T$ (e.g. true arithmetic $\mathrm{Th}(\mathfrak{N})$ itself) has no representable proof predicate and is complete, dodging the theorem precisely because its axiom set is undefinable. The theorem is therefore not about the *content* of arithmetic but about the tension between effective axiomatisability and completeness.

**Theorem 2 (Rosser's theorem: incompleteness from plain consistency).** Replacing $\mathrm{Pr}_T$ by the Rosser predicate $\mathrm{Pr}_T^R(x) = \exists p\,(\mathrm{Prf}_T(p,x) \wedge \forall q \le p\,\neg\mathrm{Prf}_T(q, \mathrm{neg}(x)))$ and diagonalising $\neg\mathrm{Pr}_T^R$ yields a Rosser sentence $R$ with $T \nvdash R$ and $T \nvdash \neg R$ from plain consistency alone [ref: TODO_REF Lindström Ch. 2]. The asymmetry of Gödel's argument — provability is $\Sigma_1$ but refutability had to be controlled over all numerals — is repaired by building the comparison of proof codes into the predicate, so that both non-provability claims reduce to bounded statements that $Q$ decides. Rosser's sentence is no longer simply "true": its truth value depends on the ordering of proofs, and it is not in general equivalent to $\mathrm{Con}_T$, unlike the Gödel sentence. The cost of dropping $\omega$-consistency is the loss of the clean "true but unprovable" reading.

**Theorem 3 (the second theorem and Löb's theorem).** With $\mathrm{Pr}_T$ satisfying the Hilbert-Bernays-Löb conditions (D1) $T \vdash \sigma \Rightarrow T \vdash \Box_T\sigma$; (D2) $T \vdash \Box_T(\sigma\to\tau) \to (\Box_T\sigma \to \Box_T\tau)$; (D3) $T \vdash \Box_T\sigma \to \Box_T\Box_T\sigma$, a consistent $T \supseteq I\Sigma_1$ satisfies $T \nvdash \mathrm{Con}_T$ [ref: TODO_REF Enderton §3.5]. Löb's theorem $T \vdash \Box_T\sigma \to \sigma$ iff $T \vdash \sigma$ subsumes it at $\sigma = \bot$, and resolves Henkin's problem: the self-asserting-provability sentence $H$ is provable. The conditions are where induction is consumed — (D3) is the internalised $\Sigma_1$-completeness of [42.01.08], provable only with enough induction ($I\Sigma_1$), which is why $Q$ alone gives the first theorem but not the second. Solovay's completeness theorem identifies the schematically valid principles of $\Box_T$ with the modal logic **GL** (Gödel-Löb), axiom $\Box(\Box p \to p) \to \Box p$; the second theorem and Löb's theorem are theorems of GL, so the entire calculus of provability is a decidable modal logic.

**Theorem 4 (concrete independence and the limits of the theorems).** Beyond the metamathematical $G_T$ and $\mathrm{Con}_T$, there are genuinely combinatorial $\Pi_2$ statements independent of $PA$: the **Paris-Harrington** strengthened finite Ramsey theorem and **Goodstein's theorem** on the termination of Goodstein sequences, each true (provable in $\mathsf{ZFC}$ via transfinite induction past $\varepsilon_0$) but unprovable in $PA$, with the unprovability shown by encoding the $\varepsilon_0$-recursion that bounds $PA$'s provably total functions [ref: TODO_REF Lindström Ch. 3]; these connect to the Ramsey-theoretic combinatorics of [40.05.04]. What the theorems do *not* show: not that there are absolutely unknowable truths (each $G_T$ is provable in a stronger system, e.g. $PA + \mathrm{Con}_{PA}$); not that mathematics is inconsistent or that $\mathfrak{N}$ is ill-defined; not, against the Lucas-Penrose argument, that human minds transcend machines — that argument assumes the human can *know* its own consistency to assert its Gödel sentence, the very self-certification the second theorem forbids for any consistent recursively axiomatised system, so a consistent mechanical mind is in the same position and the alleged advantage evaporates. The theorems bound *effective axiomatic capture*, not knowledge or truth.

**Synthesis.** The diagonal lemma is the foundational reason every result here exists, and putting these together it organises all four developments as one construction aimed at four predicates: $\neg\mathrm{Pr}_T$ gives the unprovable $G_T$ of Theorem 1, $\neg\mathrm{Pr}_T^R$ gives the Rosser sentence of Theorem 2 that needs only consistency, $\mathrm{Pr}_T \to \sigma$ gives Löb's sentence and through it the second theorem of Theorem 3, and $\neg\mathrm{Tr}$ gives the Liar that proves truth undefinable — the same self-application separating "definable predicate" (provability, $\Sigma_1$) from "undefinable predicate" (truth). This is the central insight: incompleteness is the fixed point of the provability operator, and the second theorem is dual to the first across the move from "$G_T$ is true" (semantic, in $\mathfrak{N}$) to "$\mathrm{Con}_T \to G_T$ is provable" (syntactic, in $T$), a move that costs exactly the induction ($I\Sigma_1$) needed for the derivability conditions. The provability operator $\Box_T$ generalises from arithmetic to the modal logic GL, so that the whole phenomenon is the decidable calculus of a single modality, and the theorems integrate with computation [42.04.02]: the proof set is recursively enumerable and complete, its undecidability is the recursion-theoretic shadow of incompleteness, and Church's theorem on the Entscheidungsproblem [42.01.10] routes through the very representability [42.01.08] that powers $\mathrm{Pr}_T$. Completeness [42.01.06] and incompleteness are the bridge between the two readings of "$\vdash$": complete across all models, incomplete against the standard one — no tension, because $G_T$ fails in some nonstandard model of $T$.

## Full proof set [Master]

**Proposition 1 (Diagonal lemma).** *For recursively axiomatised $T \supseteq Q$ and any $\psi(x)$, there is a sentence $G$ with $T \vdash G \leftrightarrow \psi(\ulcorner G\urcorner)$.*

*Proof.* The diagonal function $d(e) = \mathrm{sub}(e,e)$ is primitive recursive (substitution of a numeral into a formula is a bounded recursion on the code), hence by [42.01.08] represented in $Q$ by a formula $\mathrm{Diag}(x,y)$ with $Q \vdash \forall y\,(\mathrm{Diag}(\underline e, y) \leftrightarrow y = \underline{d(e)})$ for each $e$. Set $\theta(x) := \exists y\,(\mathrm{Diag}(x,y) \wedge \psi(y))$, $g := \ulcorner\theta\urcorner$, $G := \theta(\underline g)$. Then $d(g) = \mathrm{sub}(g,g) = \ulcorner\theta(\underline g)\urcorner = \ulcorner G\urcorner$, and $T$ extends $Q$ so $T \vdash \forall y\,(\mathrm{Diag}(\underline g, y) \leftrightarrow y = \ulcorner G\urcorner)$. Substituting into $G = \exists y\,(\mathrm{Diag}(\underline g, y) \wedge \psi(y))$, the unique $y = \ulcorner G\urcorner$ survives, giving $T \vdash G \leftrightarrow \psi(\ulcorner G\urcorner)$. $\square$

**Proposition 2 (First incompleteness, non-provability of $G_T$).** *For consistent recursively axiomatised $T \supseteq Q$ and $G_T$ a fixed point of $\neg\mathrm{Pr}_T$, $T \nvdash G_T$.*

*Proof.* If $T \vdash G_T$ then some $p$ codes a proof, so $\mathrm{Prf}_T(\underline p, \ulcorner G_T\urcorner)$ is a true closed $\Sigma_1$ (indeed $\Delta_0$) sentence; by $\Sigma_1$-completeness of $Q$ [42.01.08], $T \vdash \mathrm{Prf}_T(\underline p, \ulcorner G_T\urcorner)$, hence $T \vdash \mathrm{Pr}_T(\ulcorner G_T\urcorner)$ by existential introduction. The fixed point gives $T \vdash G_T \to \neg\mathrm{Pr}_T(\ulcorner G_T\urcorner)$, so $T \vdash \neg\mathrm{Pr}_T(\ulcorner G_T\urcorner)$, and with $T \vdash \mathrm{Pr}_T(\ulcorner G_T\urcorner)$ this makes $T$ inconsistent. So $T \nvdash G_T$, whence $\mathfrak{N} \models \neg\mathrm{Pr}_T(\ulcorner G_T\urcorner)$ and $\mathfrak{N} \models G_T$: $G_T$ is true. $\square$

**Proposition 3 (First incompleteness, non-refutability under $\omega$-consistency).** *If moreover $T$ is $\omega$-consistent, then $T \nvdash \neg G_T$.*

*Proof.* By Proposition 2, $T \nvdash G_T$, so no $n$ codes a $T$-proof of $G_T$; each $\mathrm{Prf}_T(\underline n, \ulcorner G_T\urcorner)$ is a false closed $\Delta_0$ sentence, hence $T \vdash \neg\mathrm{Prf}_T(\underline n, \ulcorner G_T\urcorner)$ for every $n$ by $\Sigma_1$-completeness. Were $T \vdash \neg G_T$, the fixed point gives $T \vdash \mathrm{Pr}_T(\ulcorner G_T\urcorner)$, i.e. $T \vdash \exists p\,\mathrm{Prf}_T(p, \ulcorner G_T\urcorner)$; together with $T \vdash \neg\mathrm{Prf}_T(\underline n, \ulcorner G_T\urcorner)$ for all $n$, this is an $\omega$-inconsistency. So $T \nvdash \neg G_T$. $\square$

**Proposition 4 (Tarski's undefinability of truth).** *No $\mathcal{L}_A$-formula $\mathrm{Tr}(x)$ satisfies $\mathfrak{N} \models \mathrm{Tr}(\ulcorner\sigma\urcorner) \leftrightarrow \sigma$ for all sentences $\sigma$; equivalently $\#\mathrm{Th}(\mathfrak{N})$ is not arithmetically definable.*

*Proof.* Suppose $\mathrm{Tr}$ works. Diagonalise $\neg\mathrm{Tr}(x)$: there is $L$ with $\mathfrak{N} \models L \leftrightarrow \neg\mathrm{Tr}(\ulcorner L\urcorner)$ (the diagonal lemma is provable in $Q$, sound in $\mathfrak{N}$). The defining property of $\mathrm{Tr}$ gives $\mathfrak{N} \models \mathrm{Tr}(\ulcorner L\urcorner) \leftrightarrow L$, so $\mathfrak{N} \models L \leftrightarrow \neg L$, impossible. If $\#\mathrm{Th}(\mathfrak{N})$ were defined by $\alpha(x)$, then $\alpha$ would be such a $\mathrm{Tr}$, contradiction. $\square$

**Proposition 5 (Löb's theorem).** *Under (D1)-(D3), $T \vdash \mathrm{Pr}_T(\ulcorner\sigma\urcorner) \to \sigma$ implies $T \vdash \sigma$.*

*Proof.* Diagonalise to get $L$ with $T \vdash L \leftrightarrow (\mathrm{Pr}_T(\ulcorner L\urcorner) \to \sigma)$. Then $T \vdash L \to (\mathrm{Pr}_T(\ulcorner L\urcorner)\to\sigma)$; applying (D1) and (D2) twice and (D3) once yields $T \vdash \mathrm{Pr}_T(\ulcorner L\urcorner) \to \mathrm{Pr}_T(\ulcorner\sigma\urcorner)$ (the box distributes over the implication and (D3) supplies $\mathrm{Pr}_T(\ulcorner L\urcorner) \to \mathrm{Pr}_T(\ulcorner\mathrm{Pr}_T(\ulcorner L\urcorner)\urcorner)$). With the hypothesis $T \vdash \mathrm{Pr}_T(\ulcorner\sigma\urcorner) \to \sigma$, $T \vdash \mathrm{Pr}_T(\ulcorner L\urcorner) \to \sigma$, which is the right side of the fixed point, so $T \vdash L$. By (D1), $T \vdash \mathrm{Pr}_T(\ulcorner L\urcorner)$, and detaching gives $T \vdash \sigma$. $\square$

**Proposition 6 (Gödel's second incompleteness theorem).** *For consistent recursively axiomatised $T \supseteq I\Sigma_1$, $T \nvdash \mathrm{Con}_T$.*

*Proof.* Take $\sigma := (0 = S0)$ in Löb's theorem; $\mathrm{Con}_T = \neg\mathrm{Pr}_T(\ulcorner\sigma\urcorner)$ and $T \vdash \neg\sigma$. If $T \vdash \mathrm{Con}_T$, i.e. $T \vdash \neg\mathrm{Pr}_T(\ulcorner\sigma\urcorner)$, then $T \vdash \mathrm{Pr}_T(\ulcorner\sigma\urcorner) \to \sigma$ vacuously (its antecedent is refuted), so by Proposition 5 $T \vdash \sigma$, i.e. $T \vdash 0 = S0$, contradicting consistency. The hypothesis $T \supseteq I\Sigma_1$ is used to secure (D3), the formalised $\Sigma_1$-completeness inside $T$, which $Q$ alone does not provide. $\square$

## Connections [Master]

- Representability of recursive functions [42.01.08] supplies every ingredient consumed here: the proof predicate $\mathrm{Prf}_T$ is primitive recursive and so $\Delta_0$-representable in $Q$, the substitution function $\mathrm{sub}$ is representable and so the diagonal lemma holds, and the $\Sigma_1$-completeness proved there is exactly what turns a real proof into a provable formal witness and supplies derivability condition (D3) for the second theorem. That unit owns the bridge from computation to provable-numeral facts; this unit aims that bridge at the theory's own syntax to manufacture self-reference, so $42.01.08$ is the engine and $42.01.09$ is the destination of the chapter.

- The completeness theorem for first-order logic [42.01.06] is the apparent foil that turns out to be no tension: completeness equates $\vdash$ with semantic consequence over *all* structures, while the first incompleteness theorem concerns truth in the *single* standard model $\mathfrak{N}$. The Gödel sentence $G_T$ is true in $\mathfrak{N}$ yet unprovable, which by completeness means some model of $T$ refutes it — a nonstandard model. Completeness even underwrites the existence of that model, so the two theorems are complementary descriptions of the same turnstile.

- The halting problem and the recursion theorem [42.04.02] are the computational form of this unit's self-reference: Kleene's recursion theorem (a program can access its own code) is the diagonal lemma in the language of computation, and the undecidability of the halting set is the undecidability of the $\Sigma_1$-complete provability set. The Gödel sentence is the proof-theoretic image of a program that halts iff it does not, and the unsolvability of halting and the unprovability of $G_T$ are one phenomenon in two metatheories.

- Church's theorem and the Entscheidungsproblem [42.01.10], co-produced with this unit, draw the undecidability conclusion: because $\mathrm{Pr}_Q$ is representable and the provability set is recursively enumerable but not recursive (its complement is productive), first-order validity is undecidable. The same arithmetisation that yields incompleteness yields the negative solution of Hilbert's decision problem, so $42.01.09$ and $42.01.10$ are the two faces — incompleteness and undecidability — of the representability of [42.01.08].

- The strengthened finite Ramsey theorem and combinatorial independence [40.05.04] give the concrete payoff: the Paris-Harrington principle is a true $\Pi_2$ Ramsey-theoretic statement unprovable in $PA$, the first natural mathematical (rather than metamathematical) example, with unprovability shown by the $\varepsilon_0$-recursion bounding $PA$'s provably total functions. It instantiates this unit's general theorem in everyday combinatorics rather than in self-referential coding.

## Historical & philosophical context [Master]

Kurt Gödel proved both incompleteness theorems in *Über formal unentscheidbare Sätze der Principia Mathematica und verwandter Systeme I* (1931), constructing the proof predicate by arithmetising the syntax of *Principia Mathematica* via primitive recursive functions and the $\beta$-function, building the self-referential sentence by the diagonal construction, and stating the second theorem (the unprovability of consistency) with only a sketch, the full proof of the derivability conditions being deferred [ref: TODO_REF Enderton §3.4]. David Hilbert and Paul Bernays supplied the detailed derivability conditions and the formalised second theorem in *Grundlagen der Mathematik* II (1939), and Martin Hilbert Löb proved the theorem now bearing his name in "Solution of a problem of Leon Henkin" (1955), answering whether a sentence asserting its own provability is provable [ref: TODO_REF Smoryński §1].

Alfred Tarski's undefinability of truth appeared in "Der Wahrheitsbegriff in den formalisierten Sprachen" (1936; Polish 1933), the semantic counterpart isolating that the obstruction is the indefinability of truth, not of provability [ref: TODO_REF Boolos-Burgess-Jeffrey Ch. 17]. J. Barkley Rosser removed the $\omega$-consistency hypothesis in "Extensions of some theorems of Gödel and Church" (1936) with the ordered-proof predicate. The collapse of Hilbert's programme — a finitistic, internal consistency proof for a theory containing arithmetic — followed directly from the second theorem; Gerhard Gentzen's 1936 consistency proof of $PA$ by transfinite induction to $\varepsilon_0$ showed what an external proof must cost. Robert Solovay's "Provability interpretations of modal logic" (1976) identified the modal logic GL as the complete calculus of the provability operator, and Petr Hájek and Pavel Pudlák fixed the exact fragment ($I\Sigma_1$) over which the derivability conditions and the second theorem hold [ref: TODO_REF Lindström Ch. 2].
