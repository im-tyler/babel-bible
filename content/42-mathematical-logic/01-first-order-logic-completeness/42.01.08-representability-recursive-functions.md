## Intuition [Beginner]

A computer can add, multiply, and compare whole numbers. A logical theory of arithmetic is a fixed list of basic rules about those same operations, together with everything you can prove from them. These look like two unrelated worlds: one is a machine grinding through steps, the other is a stack of statements and proofs. This unit shows they are secretly the same world. Whenever a machine settles on a definite answer for definite inputs, a very weak arithmetic can *prove* the matching numerical fact.

Picture a tiny rulebook for whole numbers. It knows that adding zero changes nothing, that adding one more is the successor, that multiplying builds up from repeated adding, and a few more facts of that kind. It is almost laughably weak: it cannot even prove that addition is commutative in general. Yet for any *specific* sum or product, with specific numbers written out, it can step through the rules and reach the right total. Ask it about $2 + 3$ and it will prove the answer is $5$. Ask about $7 \times 6$ and it will prove $42$.

The deep point is that "compute a function" can be turned into "prove a statement of arithmetic." A function is *represented* by a formula when the rulebook proves the formula true on exactly the right inputs and outputs, and false on all the wrong ones. The surprising theorem is that *every* function a machine can compute can be represented this way, in that same weak rulebook. Computation can be mirrored inside arithmetic.

Why does this matter? Once arithmetic can mirror any computation, it can mirror the computation that checks proofs of arithmetic itself. A theory that can talk about its own proofs is a theory that can be made to talk about itself — and that is the doorway to the incompleteness results that come next.

## Visual [Beginner]

The picture is a bridge with two banks. On the left bank, a machine computes; on the right bank, a weak arithmetic proves. Representability is the bridge: every computation on the left has a matching proof on the right.

```
   LEFT BANK                 BRIDGE                 RIGHT BANK
   (a machine computes)      (representability)     (weak arithmetic Q proves)

   input  2, 3                                       numerals  S S 0 ,  S S S 0
        |                                                 |
        v                                                 v
   add  ->  5             ===================>        Q proves:  "S S 0 + S S S 0 = S S S S S 0"
        |                                                 |
   output 5                                          a finished proof, line by line

   the SAME fact, said two ways:
        machine:   add(2,3) = 5
        arithmetic: Q |- ( 2 + 3 = 5 )   (with numbers written as S S ... 0)
```

A *numeral* is how arithmetic writes a number: start from $0$ and apply "successor" ($S$, meaning add one) that many times. So $2$ is written $SS0$ and $3$ is written $SSS0$. The machine's "add $2$ and $3$ to get $5$" becomes the arithmetic's "prove that $SS0 + SSS0$ equals $SSSSS0$."

| machine does | arithmetic proves | matches? |
|---|---|---|
| $\text{add}(2,3) = 5$ | $SS0 + SSS0 = SSSSS0$ | yes |
| $\text{mult}(2,3) = 6$ | $SS0 \times SSS0 = SSSSSS0$ | yes |
| $\text{add}(2,3) = 9$ | $SS0 + SSS0 = SSSSS0$ (so the wrong answer is *disproved*) | the wrong answer is refuted |

The bridge runs both ways for facts: the right answer gets proved, every wrong answer gets refuted.

## Worked example [Beginner]

We watch the weak rulebook prove one specific sum, $2 + 2 = 4$, using only its rules about adding. Write numbers as numerals: $2$ is $SS0$ and $4$ is $SSSS0$. The rulebook has two facts about addition we will use. First, adding zero changes nothing: $x + 0 = x$. Second, adding a successor pushes the successor outside: $x + Sy = S(x + y)$.

Step 1. Start from $SS0 + SS0$. The second number is $SS0$, which is $S(S0)$ — a successor. Apply "$x + Sy = S(x+y)$" with $x = SS0$ and $y = S0$. This rewrites $SS0 + SS0$ as $S(SS0 + S0)$.

Step 2. Look inside at $SS0 + S0$. Again the second number $S0$ is a successor, $S(0)$. Apply the same rule with $y = 0$. This rewrites $SS0 + S0$ as $S(SS0 + 0)$. So far the whole expression is $S(S(SS0 + 0))$.

Step 3. Now the inside is $SS0 + 0$. The second number is $0$, so apply "$x + 0 = x$." This gives $SS0$. The whole expression is now $S(S(SS0))$.

Step 4. Count the successors: $S(S(SS0))$ is $SSSS0$, which is the numeral for $4$. The rulebook has proved $SS0 + SS0 = SSSS0$, that is, $2 + 2 = 4$.

What this tells us: the rulebook never needed any general theory of addition. It only peeled the second numeral down one successor at a time until it hit zero, then read off the answer. Every specific sum works this way, and the same style of step-by-step peeling handles every specific product too. That is exactly why a machine's definite answer always has a matching proof: the proof simply retraces the computation.

## Formal definition [Intermediate+]

Fix the language of arithmetic $\mathcal{L}_A = \{0, S, +, \cdot, <\}$ and its standard structure $\mathfrak{N} = (\mathbb{N}; 0, S, +, \cdot, <)$ [42.01.04]. For each $n \in \mathbb{N}$ the **numeral** $\underline{n}$ is the closed term $S^n 0$, defined by $\underline{0} = 0$ and $\underline{n+1} = S\underline{n}$; it is the canonical $\mathcal{L}_A$-name of $n$, with $\underline{n}^{\mathfrak{N}} = n$.

**Robinson arithmetic** $Q$ is the $\mathcal{L}_A$-theory axiomatised by the universal closures of [ref: TODO_REF Enderton §3.3]:

$$
\begin{aligned}
&\text{Q1: } Sx \ne 0, \qquad \text{Q2: } Sx = Sy \to x = y, \qquad \text{Q3: } x \ne 0 \to \exists y\,(x = Sy),\\
&\text{Q4: } x + 0 = x, \qquad \text{Q5: } x + Sy = S(x+y),\\
&\text{Q6: } x \cdot 0 = 0, \qquad \text{Q7: } x \cdot Sy = (x \cdot y) + x, \qquad \text{Q8: } x < y \leftrightarrow \exists z\,(S z + x = y).
\end{aligned}
$$

**Peano arithmetic** $PA$ extends $Q$ (with Q3 then derivable) by the first-order **induction schema**: for every $\mathcal{L}_A$-formula $\varphi(x, \bar z)$,
$$
\forall \bar z\,\Big[\big(\varphi(0,\bar z) \wedge \forall x\,(\varphi(x,\bar z) \to \varphi(Sx,\bar z))\big) \to \forall x\,\varphi(x,\bar z)\Big].
$$
Both are sound for $\mathfrak{N}$: $\mathfrak{N} \models Q$ and $\mathfrak{N} \models PA$, so every theorem of either is true in the standard model.

A relation $R \subseteq \mathbb{N}^k$ is **represented** in a theory $T$ by a formula $\rho(x_1, \dots, x_k)$ when for all $\bar a \in \mathbb{N}^k$,
$$
\bar a \in R \implies T \vdash \rho(\underline{a_1}, \dots, \underline{a_k}), \qquad \bar a \notin R \implies T \vdash \neg\rho(\underline{a_1}, \dots, \underline{a_k}).
$$
A function $f : \mathbb{N}^k \to \mathbb{N}$ is **represented** in $T$ by $\varphi(\bar x, y)$ when for all $\bar a$,
$$
T \vdash \forall y\,\big(\varphi(\underline{a_1}, \dots, \underline{a_k}, y) \leftrightarrow y = \underline{f(\bar a)}\big).
$$
Equivalently $T$ proves the correct instance $\varphi(\underline{\bar a}, \underline{f(\bar a)})$ and proves uniqueness $\forall y\,(\varphi(\underline{\bar a}, y) \to y = \underline{f(\bar a)})$. The notion of **recursive function** (closure of the zero, successor, and projection functions under composition, primitive recursion, and minimisation) and **recursive relation** is imported from the theory of computation [42.04.01]; the present unit takes that definition as given and builds the formula that captures each closure step inside $Q$.

A formula is **$\Delta_0$** (bounded) when every quantifier in it is bounded, of the form $\forall x < t$ or $\exists x < t$ for a term $t$ not containing $x$; it is **$\Sigma_1$** when of the form $\exists \bar y\,\theta$ with $\theta \in \Delta_0$, and **$\Pi_1$** when $\forall \bar y\,\theta$ with $\theta \in \Delta_0$. Bounded quantification is decided by finite search, so the truth value of a closed $\Delta_0$ sentence in $\mathfrak{N}$ is mechanically computable.

### Counterexamples to common slips [Intermediate+]

- *"Representable means true in $\mathfrak{N}$."* Representability is a *provability* condition, strictly stronger than truth on numerals. A formula can be true of $R$ in the standard model yet fail to be proved or refuted by $Q$ on some numerals. The theorem is that recursive relations admit a formula $Q$ actually decides at every numeral, not merely one $\mathfrak{N}$ verifies.

- *"$Q$ proves induction for specific numbers, so it is almost $PA$."* $Q$ has no induction schema at all. It proves each numerical instance of a recursive computation by finite term-rewriting, but it cannot prove $\forall x\,(x + 0 = 0 + x)$ or $\forall x\,(x < Sx)$. Representability deliberately needs only this induction-free strength.

- *"Function representability is just the graph being represented as a relation."* Representing the graph $\{(\bar a, b) : f(\bar a) = b\}$ as a relation gives the two implications on numerals; function representability additionally demands $T$ prove *uniqueness*, $\forall y\,(\varphi(\underline{\bar a}, y) \to y = \underline{f(\bar a)})$, an internal functionality that relation-representability does not supply.

- *"$\Sigma_1$-completeness means $Q$ proves every true sentence."* It proves every true $\Sigma_1$ sentence. True $\Pi_1$ sentences — for instance true but unprovable consistency statements — are exactly where completeness fails; the asymmetry between $\Sigma_1$ (existential, witness-checkable) and $\Pi_1$ (universal) is the whole content.

## Key theorem with proof [Intermediate+]

The signature theorem is that the weak, induction-free theory $Q$ already represents every recursive function. Its engine is the arithmetisation of finite sequences: primitive recursion threads a function value through a growing list of earlier values, and to define it inside arithmetic one must quantify over such lists using only numbers. Gödel's $\beta$-function does exactly this.

**Lemma ($\beta$-function).** *Define $\beta(c, d, i) = \mathrm{rem}\big(c,\, 1 + (i+1)d\big)$, the remainder of $c$ on division by $1 + (i+1)d$. For every finite sequence $a_0, \dots, a_n$ of natural numbers there exist $c, d \in \mathbb{N}$ with $\beta(c, d, i) = a_i$ for all $i \le n$. Moreover $\beta$ is representable in $Q$ by a $\Delta_0$ formula* [ref: TODO_REF Enderton §3.3].

*Proof.* Put $d = m!$ where $m = \max(n, a_0, \dots, a_n)$, and let $u_i = 1 + (i+1)d$ for $i \le n$. The $u_i$ are pairwise coprime: if a prime $p$ divided both $u_i$ and $u_j$ with $i < j \le n$, it would divide their difference $(j-i)d$; since $p \mid u_i = 1 + (i+1)d$ forces $p \nmid d$, $p$ divides $j - i \le n \le m$, so $p \le m$ and $p \mid d = m!$, contradicting $p \nmid d$. Each $a_i < 1 + (i+1)d = u_i$ because $a_i \le m \le d \le (i+1)d < u_i$. By the Chinese remainder theorem there is $c < \prod_i u_i$ with $c \equiv a_i \pmod{u_i}$ for all $i \le n$; as $a_i < u_i$, the least nonnegative residue is $a_i$ itself, so $\beta(c, d, i) = a_i$. For representability, $w = \mathrm{rem}(c, v)$ is the unique $w$ with $w < v$ and $\exists q \le c\,(c = q\cdot v + w)$, a $\Delta_0$ condition; $\beta(c,d,i) = w$ unfolds to the $\Delta_0$ formula $\exists q \le c\,\big(c = q\cdot(1 + (i+1)\cdot d) + w \wedge w < 1 + (i+1)\cdot d\big)$. $\square$

**Theorem (representability).** *Every recursive function $f : \mathbb{N}^k \to \mathbb{N}$ is representable in $Q$, and every recursive relation is representable in $Q$.*

*Proof.* Induct on the construction of $f$ as a recursive function [42.04.01]. The zero function $z(x) = 0$ is represented by $y = 0$; the successor $S$ by $y = Sx$; the projection $\pi^k_j(\bar x) = x_j$ by $y = x_j$. For each, $Q$ proves the value at any numeral and proves uniqueness because the value formula already has the shape $y = (\text{numeral})$, which is its own functionality statement.

*Composition.* If $h, g_1, \dots, g_m$ are represented by $\eta, \gamma_1, \dots, \gamma_m$ and $f(\bar x) = h(g_1(\bar x), \dots, g_m(\bar x))$, set
$$
\varphi(\bar x, y) := \exists w_1 \cdots \exists w_m\,\big(\gamma_1(\bar x, w_1) \wedge \cdots \wedge \gamma_m(\bar x, w_m) \wedge \eta(w_1, \dots, w_m, y)\big).
$$
Given $\bar a$, let $b_j = g_j(\bar a)$ and $b = h(\bar b) = f(\bar a)$. Each $\gamma_j$ forces $w_j = \underline{b_j}$ provably, and then $\eta$ forces $y = \underline{b}$; $Q$ proves $\varphi(\underline{\bar a}, \underline{b})$ and the uniqueness, so $f$ is represented.

*Primitive recursion.* Suppose $f(\bar x, 0) = g(\bar x)$ and $f(\bar x, S t) = h(\bar x, t, f(\bar x, t))$ with $g, h$ represented by $\gamma, \eta$. The value $f(\bar x, t)$ is determined by the sequence $a_0 = g(\bar x), a_1, \dots, a_t = f(\bar x, t)$ with $a_{i+1} = h(\bar x, i, a_i)$. Encode that sequence by a single pair $(c, d)$ through the $\beta$-lemma and set
$$
\varphi(\bar x, t, y) := \exists c\,\exists d\,\Big[\beta(c,d,0) = \text{the value of } g \wedge \forall i < t\,\big(\beta(c,d,Si) = h(\bar x, i, \beta(c,d,i))\big) \wedge \beta(c,d,t) = y\Big],
$$
where each occurrence of $g$ and $h$ is replaced by its representing formula and $\beta(c,d,i) = u$ by its $\Delta_0$ formula. For numerals $\bar a, n$ put $b = f(\bar a, n)$ and choose $(c,d)$ coding $a_0, \dots, a_n$ by the $\beta$-lemma; $Q$ proves each conjunct on the witnessing $(\underline c, \underline d)$ and proves $y = \underline b$, and uniqueness follows because $Q$ proves the bounded conjunction pins $\beta(c,d,t)$ to $\underline b$ regardless of the coding pair. So $f$ is represented.

*Minimisation.* If $f(\bar x) = \mu y\,[g(\bar x, y) = 0]$ with $g$ represented by $\gamma$ and total where used, set
$$
\varphi(\bar x, y) := \gamma(\bar x, y, 0) \wedge \forall z < y\,\neg\,\gamma(\bar x, z, 0).
$$
For $\bar a$ with $f(\bar a) = b$, $Q$ proves $\gamma(\underline{\bar a}, \underline b, 0)$ and, for each $j < b$, $\neg\gamma(\underline{\bar a}, \underline j, 0)$; using Q8 and the provable fact $Q \vdash \forall z\,(z < \underline b \leftrightarrow z = \underline 0 \vee \cdots \vee z = \underline{b-1})$, $Q$ collapses the bounded $\forall z < \underline b$ to that finite conjunction and proves it, giving $\varphi(\underline{\bar a}, \underline b)$ and the uniqueness from the leastness. A recursive relation is represented by representing its characteristic function and reading membership off the value $\underline 1$ versus $\underline 0$. $\square$

**Bridge.** This theorem is the foundational reason a theory of numbers can host a theory of computation: $Q$ proves, numeral by numeral, exactly what a machine computes, so the recursive functions of [42.04.01] live inside arithmetic as provably-functional formulas. It builds toward the arithmetisation of syntax and the proof predicate $\mathrm{Prf}_T(p,x)$, a primitive recursive relation that is therefore representable here, and it appears again in the first incompleteness theorem [42.01.09], where the diagonal lemma feeds a representing formula back into itself. The $\beta$-function is the central insight: by trading quantification over finite sequences for quantification over two numbers it makes primitive recursion arithmetically definable, which is exactly the step that lets unbounded computation be mirrored by a bounded coding. Putting these together, the weakness of $Q$ — no induction at all — becomes a feature: incompleteness will follow already for any consistent theory extending this minimal base, so the phenomenon cannot be blamed on the strength of $PA$.

## Advanced results [Master]

The representability theorem supports four developments: the optimality of $Q$ as a base, the exact strength of $\Sigma_1$-completeness, the equivalence of $\Sigma_1$-definability with recursive enumerability that runs through the MRDP theorem, and the arithmetisation of provability that opens onto incompleteness.

**Theorem 1 (representable $=$ recursive in $Q$; optimality of the base).** A function is representable in $Q$ iff it is recursive, and likewise for relations [ref: TODO_REF Boolos-Burgess-Jeffrey Ch. 16]. The forward direction is the Key theorem; the reverse holds because a representing formula gives an effective procedure — search for a $Q$-proof of $\varphi(\underline{\bar a}, \underline m)$ over $m$, which by representability halts at $m = f(\bar a)$ — so representability in any recursively axiomatised consistent theory implies recursiveness. The base cannot be weakened past $Q$ in the relevant sense: the eight axioms are exactly what the induction over composition, primitive recursion (via $\beta$), and minimisation (via Q8) consume, and any consistent theory in which all recursive functions are representable interprets $Q$. Representability is thus an absolute, theory-independent boundary coinciding with computability, and it holds already without induction.

**Theorem 2 ($\Sigma_1$-completeness and its sharp boundary).** Every true $\Sigma_1$ sentence is a theorem of $Q$, and the bound is exact: $Q$ correctly decides every closed $\Delta_0$ sentence, true existentials are provable by numeral witnesses, but true $\Pi_1$ sentences need not be provable [ref: TODO_REF Hájek-Pudlák Ch. I]. The proof reduces a $\Sigma_1$ sentence to a closed $\Delta_0$ instance at a numeral witness and runs the bounded-evaluation lemma, which proves by metatheoretic induction on bounded formulas that $Q$ agrees with $\mathfrak{N}$ on every closed $\Delta_0$ sentence. The $\Sigma_1/\Pi_1$ asymmetry is the engine of incompleteness: a consistency statement $\mathrm{Con}_T$ is $\Pi_1$, true (when $T$ is consistent) but unprovable, and the first incompleteness sentence is $\Pi_1$ for the same reason — its negation would be a false $\Sigma_1$ sentence, which $\Sigma_1$-completeness would force $T$ to disprove.

**Theorem 3 (the $\Sigma_1$/recursively-enumerable identification and MRDP).** A relation is $\Sigma_1$-definable in $\mathfrak{N}$ iff it is recursively enumerable iff its characteristic-of-membership function is the domain of a partial recursive function [ref: TODO_REF Hájek-Pudlák Ch. I]. The forward implication uses representability to convert a recursive enumeration into an existential over proof-or-computation codes; the converse is Kleene's normal form, $\exists \bar y$ over a $\Delta_0$ Kleene-$T$ predicate. The Matiyasevich-Robinson-Davis-Putnam theorem sharpens this dramatically: every $\Sigma_1$ (equivalently c.e.) relation is **Diophantine**, defined by $\exists \bar y\,(p(\bar x, \bar y) = q(\bar x, \bar y))$ for polynomials with natural-number coefficients, so the existential quantifier prefix can be collapsed onto a single polynomial equation (co-produced as [42.04.07]). Representability in $Q$ is the textbook-level half of this tower; MRDP is its number-theoretic apex, and together they make "computably enumerable" and "arithmetically existential" one notion.

**Theorem 4 (arithmetisation of syntax and the proof predicate).** Fix a Gödel numbering $\ulcorner \cdot \urcorner$ assigning numbers to terms, formulas, and finite sequences of formulas. The relations "$x$ codes a formula," "$y$ codes a proof in $T$ of the formula coded by $x$," and the substitution function $\mathrm{sub}(\ulcorner\varphi\urcorner, n) = \ulcorner\varphi(\underline n)\urcorner$ are all primitive recursive for any recursively axiomatised $T$ [ref: TODO_REF Enderton §3.5]. By the representability theorem each is representable in $Q$: there is a $\Delta_0$ formula $\mathrm{Prf}_T(p, x)$ and the $\Sigma_1$ provability predicate $\mathrm{Pr}_T(x) = \exists p\,\mathrm{Prf}_T(p, x)$, and a representing formula for $\mathrm{sub}$. This is the arithmetisation groundwork: the theory's own syntax becomes a recursive structure on numbers, and $Q$ proves the matching numerical facts about it. The diagonal lemma — for every $\psi(x)$ a sentence $G$ with $T \vdash G \leftrightarrow \psi(\ulcorner G\urcorner)$ — is then a one-line consequence of the representability of $\mathrm{sub}$, and feeding $\psi = \neg\mathrm{Pr}_T$ yields the Gödel sentence (developed in the co-produced [42.01.09]).

**Synthesis.** The representability theorem is the foundational reason computation can be mirrored inside arithmetic, and putting these together it controls all four developments: the $\beta$-function makes primitive recursion arithmetically definable, which is exactly what lets every recursive function become a provably-functional formula of $Q$, and this is the central insight that the whole incompleteness phenomenon rests on a theory with no induction. Theorem 1's identification of representable with recursive is dual to the computability-side fact that representability gives an effective halting search, so the boundary is the same object seen from logic and from computation [42.04.01]. The $\Sigma_1$-completeness of Theorem 2 generalises the numeral-by-numeral provability of the Key theorem from individual function values to all existential truths, and its $\Pi_1$ failure is the bridge to the unprovable consistency statement; Theorem 3 widens $\Sigma_1$-definability into the recursively enumerable sets and, through MRDP [42.04.07], onto Diophantine equations, so that "halts" (the halting problem of [42.04.02]) and "satisfies a polynomial equation" and "is provable" are one notion in three languages. Theorem 4 turns the representing apparatus on the theory's own syntax: the proof predicate is primitive recursive, hence representable, hence arithmetic can speak of its own provability — and the diagonal lemma closes the loop, so that the self-reference defeating the halting checker [42.04.02] reappears as a sentence of arithmetic asserting its own unprovability [42.01.09].

## Full proof set [Master]

**Proposition 1 ($\beta$-function lemma).** *For $\beta(c,d,i) = \mathrm{rem}(c, 1 + (i+1)d)$ and any $a_0, \dots, a_n$ there are $c, d$ with $\beta(c,d,i) = a_i$ for $i \le n$; and $\beta$ is $\Delta_0$-representable in $Q$.*

*Proof.* Let $m = \max(n, a_0, \dots, a_n)$, $d = m!$, $u_i = 1 + (i+1)d$. Coprimality: a common prime $p$ of $u_i, u_j$ ($i < j \le n$) divides $(u_j - u_i) = (j - i)d$; from $u_i \equiv 1 \pmod p$ and $u_i = 1 + (i+1)d$ one gets $p \nmid d$, so $p \mid (j - i)$, hence $p \le j - i \le n \le m$ and $p \mid m! = d$, a contradiction. Bound: $a_i \le m \le m! = d < 1 + (i+1)d = u_i$. CRT yields $c$ with $c \equiv a_i \pmod{u_i}$; since $0 \le a_i < u_i$, $\mathrm{rem}(c, u_i) = a_i$. The graph $\beta(c,d,i) = w$ is $\exists q \le c\,(c = q\cdot(1 + (i+1)\cdot d) + w \wedge w < 1 + (i+1)\cdot d)$, all quantifiers bounded by $c$, hence $\Delta_0$; $Q$ proves it on numerals because the division identity and the strict inequality are closed $\Delta_0$ facts, decided by $Q$ in accordance with $\mathfrak{N}$. $\square$

**Proposition 2 (initial functions and composition).** *Zero, successor, and projections are representable in $Q$, and the representable functions are closed under composition.*

*Proof.* $z$ by $y = 0$, $S$ by $y = Sx$, $\pi^k_j$ by $y = x_j$; in each case $Q$ proves the unique value at any numeral tuple directly from the term identities and the reflexivity of equality. For composition with $f(\bar x) = h(g_1(\bar x), \dots, g_m(\bar x))$ and representing formulas $\eta, \gamma_j$, set $\varphi(\bar x, y) = \exists \bar w\,(\bigwedge_j \gamma_j(\bar x, w_j) \wedge \eta(\bar w, y))$. At $\bar a$, write $b_j = g_j(\bar a)$, $b = f(\bar a)$: $Q \vdash \gamma_j(\underline{\bar a}, \underline{b_j})$ and the uniqueness force $w_j = \underline{b_j}$, and $Q \vdash \eta(\underline{\bar b}, \underline b)$ with its uniqueness force $y = \underline b$. Hence $Q \vdash \varphi(\underline{\bar a}, \underline b)$ and $Q \vdash \forall y\,(\varphi(\underline{\bar a}, y) \to y = \underline b)$. $\square$

**Proposition 3 (primitive recursion).** *If $g, h$ are representable then so is $f$ defined by $f(\bar x, 0) = g(\bar x)$, $f(\bar x, St) = h(\bar x, t, f(\bar x, t))$.*

*Proof.* Let $\gamma, \eta$ represent $g, h$. Using Proposition 1, set
$$
\varphi(\bar x, t, y) := \exists c\,\exists d\,\Big[\exists v\,(\beta(c,d,0) = v \wedge \gamma(\bar x, v)) \wedge \forall i < t\,\exists u \exists u'\,(\beta(c,d,i) = u \wedge \beta(c,d,Si) = u' \wedge \eta(\bar x, i, u, u')) \wedge \beta(c,d,t) = y\Big].
$$
At numerals $\bar a, n$, set $a_0 = g(\bar a)$, $a_{i+1} = h(\bar a, i, a_i)$, $b = a_n = f(\bar a, n)$, and pick $(c,d)$ coding $a_0, \dots, a_n$. $Q$ proves the base conjunct via $\gamma$, each bounded-$i$ conjunct via $\eta$ and the $\beta$-graph (the bounded $\forall i < \underline n$ collapsing to the explicit finite conjunction since $Q \vdash \forall i\,(i < \underline n \leftrightarrow \bigvee_{j<n} i = \underline j)$), and the final $\beta(c,d,\underline n) = \underline b$. So $Q \vdash \varphi(\underline{\bar a}, \underline n, \underline b)$. Uniqueness: any $y'$ with $\varphi(\underline{\bar a}, \underline n, y')$ forces, through the same bounded conjunction and the provable functionality of $\gamma, \eta$ and $\beta$, that $\beta(c', d', \underline n) = \underline b$, hence $y' = \underline b$; thus $Q \vdash \forall y\,(\varphi(\underline{\bar a}, \underline n, y) \to y = \underline b)$. $\square$

**Proposition 4 (minimisation).** *If $g$ is representable and $f(\bar x) = \mu y\,[g(\bar x, y) = 0]$ is total, then $f$ is representable in $Q$.*

*Proof.* Let $\gamma$ represent $g$ and set $\varphi(\bar x, y) := \gamma(\bar x, y, 0) \wedge \forall z < y\,\neg\gamma(\bar x, z, 0)$. At $\bar a$ with $f(\bar a) = b$: $g(\bar a, b) = 0$ gives $Q \vdash \gamma(\underline{\bar a}, \underline b, 0)$; for each $j < b$, $g(\bar a, j) \ne 0$ gives $Q \vdash \neg\gamma(\underline{\bar a}, \underline j, 0)$ by representability of $g$ as a function (its value at $\underline j$ is a numeral $\ne \underline 0$, and $Q$ proves distinct numerals unequal). The bounded $\forall z < \underline b$ collapses to $\bigwedge_{j<b}\neg\gamma(\underline{\bar a}, \underline j, 0)$ via $Q \vdash \forall z\,(z < \underline b \leftrightarrow \bigvee_{j<b} z = \underline j)$ (provable from Q8 and the numeral case analysis), which $Q$ proves. Hence $Q \vdash \varphi(\underline{\bar a}, \underline b)$, and uniqueness follows because any $y'$ satisfying $\varphi$ has $\gamma(\underline{\bar a}, y', 0)$ and is below none of the witnesses, forcing $y' = \underline b$ by the leastness and numeral trichotomy. $\square$

**Proposition 5 ($\Sigma_1$-completeness of $Q$).** *Every true $\Sigma_1$ sentence is provable in $Q$.*

*Proof.* First, the bounded-evaluation lemma: for every closed $\Delta_0$ sentence $\theta$, $\mathfrak{N} \models \theta \Rightarrow Q \vdash \theta$ and $\mathfrak{N} \not\models \theta \Rightarrow Q \vdash \neg\theta$, by metatheoretic induction on $\theta$. Atomic closed equalities and inequalities between closed terms evaluate to numerals, and $Q$ proves $\underline m = \underline n$ when $m = n$ and $\underline m \ne \underline n$ when $m \ne n$ (from Q1, Q2), and decides $\underline m < \underline n$ from Q8; Boolean connectives pass through; a bounded quantifier $\exists z < \underline k\,\psi$ is equivalent over $Q$ to the finite disjunction $\bigvee_{j<k}\psi(\underline j)$ (using $Q \vdash \forall z\,(z < \underline k \leftrightarrow \bigvee_{j<k} z = \underline j)$), and $\forall z < \underline k\,\psi$ to the finite conjunction, reducing to the inductive hypothesis on each closed instance. Now a $\Sigma_1$ sentence $\exists \bar y\,\theta(\bar y)$ true in $\mathfrak{N}$ has a numeral witness $\bar a$ with $\mathfrak{N} \models \theta(\underline{\bar a})$; by the lemma $Q \vdash \theta(\underline{\bar a})$, and existential introduction gives $Q \vdash \exists \bar y\,\theta(\bar y)$. $\square$

**Proposition 6 (the proof predicate is representable; provability is $\Sigma_1$).** *For recursively axiomatised $T$ the relation $\mathrm{Prf}_T(p, x)$ is primitive recursive, hence representable in $Q$ by a $\Delta_0$ formula, and $\mathrm{Pr}_T(x) = \exists p\,\mathrm{Prf}_T(p, x)$ is $\Sigma_1$.*

*Proof.* Gödel numbering renders "$x$ codes an $\mathcal{L}_A$-formula" and "$p$ codes a finite sequence of formulas each of which is a logical axiom, an axiom of $T$, or follows from earlier ones by a rule, the last being the formula coded by $x$" as primitive recursive predicates: each clause is a bounded check over the components of $p$, decidable by primitive recursion on the sequence-coding (itself built from the $\beta$-function of Proposition 1). Being primitive recursive, $\mathrm{Prf}_T$ is recursive, so by Propositions 2-4 it is representable in $Q$; as the checking is bounded by $p$ the representing formula is $\Delta_0$. Then $\mathrm{Pr}_T(x) = \exists p\,\mathrm{Prf}_T(p, x)$ is $\Sigma_1$ by definition. By Proposition 5, every true instance $\mathrm{Pr}_T(\ulcorner\sigma\urcorner)$ — that is, every actual theorem $\sigma$ — is $\Sigma_1$-provably so in $Q$, the formal counterpart of "if $T \vdash \sigma$ then $Q \vdash \mathrm{Pr}_T(\ulcorner\sigma\urcorner)$." $\square$

## Connections [Master]

- Structures and Tarski semantics [42.01.04] supply the standard model $\mathfrak{N}$ and the satisfaction relation against which truth of $\Delta_0$ and $\Sigma_1$ sentences is measured, and the numeral $\underline n$ is the closed term whose value $\underline n^{\mathfrak{N}} = n$ makes "true on numerals" precise. That unit owns the semantics of the arithmetic language; this unit adds the provability layer over it, the gap between $\mathfrak{N} \models \sigma$ and $Q \vdash \sigma$ that $\Sigma_1$-completeness measures and that incompleteness exploits.

- Models of computation and the recursive functions [42.04.01] define the recursive functions whose representability is the theorem of this unit; the inductive closure conditions — composition, primitive recursion, minimisation — are exactly the cases of the representability induction, and the equivalence "representable in $Q$ iff recursive" identifies that unit's notion of computability with this unit's notion of provable-numeral-functionality. The $\beta$-function is the device that imports primitive recursion, defined there, into the formula language here.

- The halting problem and the recursion theorem [42.04.02] are the computational shadow of this arithmetisation: the self-application that defeats the halting checker is the same diagonal that, once the proof predicate is representable here, produces the Gödel sentence; and the undecidability of arithmetic — $K \le_m \{\ulcorner\sigma\urcorner : Q \vdash \sigma\}$ — is immediate once provability is $\Sigma_1$ and representability lets $Q$ track computation. Church's theorem on the Entscheidungsproblem routes through exactly the representability established here.

- Gödel's incompleteness theorems [42.01.09], co-produced with this unit, are its destination: the diagonal lemma is a one-line consequence of the representability of the substitution function $\mathrm{sub}$ proved here, and the first incompleteness sentence $G \leftrightarrow \neg\mathrm{Pr}_T(\ulcorner G\urcorner)$ is built from the $\Sigma_1$ provability predicate of Proposition 6. The second incompleteness theorem additionally needs the Hilbert-Bernays-Löb derivability conditions, whose first condition is precisely the $\Sigma_1$-completeness of Proposition 5 formalised inside $T$.

## Historical & philosophical context [Master]

The representability of the recursive functions in a weak arithmetic is the technical core of Kurt Gödel's 1931 paper *Über formal unentscheidbare Sätze der Principia Mathematica und verwandter Systeme I*, where the $\beta$-function (Gödel's "$\beta$") was introduced precisely to express primitive recursion inside arithmetic without a primitive sequence type, using the Chinese remainder theorem to code finite sequences by pairs of numbers [ref: TODO_REF Enderton §3.3]. Gödel proved that the primitive recursive functions are definable in the arithmetic of *Principia*; the extension to all recursive functions and the isolation of the minimal theory followed from the convergence of the competing analyses of computability — Gödel and Herbrand's general recursive functions, Church's $\lambda$-definability, and Turing's machines — into the single class fixed by the Church-Turing thesis.

Raphael Robinson isolated the finitely axiomatised theory $Q$ in 1950 (in "An essentially undecidable axiom system"), showing that this induction-free fragment already suffices for representability and hence for essential undecidability, so that the incompleteness and undecidability phenomena attach to a theory far weaker than full Peano arithmetic [ref: TODO_REF Boolos-Burgess-Jeffrey Ch. 16]. Alfred Tarski, Andrzej Mostowski, and Robinson systematised the method of essentially undecidable theories in *Undecidable Theories* (1953), making $Q$-representability the standard route to undecidability results across mathematics. The sharpening of $\Sigma_1$-definability to Diophantine definability — that every recursively enumerable set is the solution set of a polynomial equation — was completed by Yuri Matiyasevich in 1970, building on Julia Robinson, Martin Davis, and Hilary Putnam, resolving Hilbert's tenth problem in the negative [ref: TODO_REF Hájek-Pudlák Ch. I].
