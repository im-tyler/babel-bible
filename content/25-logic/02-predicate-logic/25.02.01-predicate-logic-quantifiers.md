## Intuition [Beginner]

Propositional logic treats each statement as an indivisible unit. You can say "Socrates is mortal" and combine it with other statements, but you cannot look inside it. Predicate logic breaks statements open, analyzing their internal structure: subjects, predicates, and the quantifiers "all" and "some".

Consider the argument: "All humans are mortal. Socrates is human. Therefore, Socrates is mortal." Propositional logic cannot capture why this is valid. It would represent the three statements as $p$, $q$, and $r$, and the inference $p, q \models r$ is not valid in propositional logic. Predicate logic reveals the structure that makes it work.

A predicate is a property that objects can have. Write $H(x)$ for "$x$ is human" and $M(x)$ for "$x$ is mortal". The premises become "for all x, if x is human then x is mortal" and "Socrates is human". The conclusion "Socrates is mortal" follows by applying the general rule to the specific case.

The universal quantifier, written "for all", means "for every object in the domain". The existential quantifier, written "there exists", means "there is at least one object in the domain such that". These two quantifiers are the main new tools predicate logic adds to propositional logic. They let you express generalizations and existence claims that propositional logic cannot handle.

Think of the difference this way: propositional logic lets you say "this statement and that statement", while predicate logic lets you say "every object with this property also has that property" or "some object has this property". The quantifiers are the bridge from specific statements to general patterns.

## Visual [Beginner]

The structure of a quantified formula can be visualized as a tree. The formula "for all x, if H(x) then M(x)" has the universal quantifier at the root, binding the variable x throughout the subformula. Inside, the conditional connects two atomic predicates, each taking x as argument.

```
      FOR-ALL x
          |
        if-then
        /     \
     H(x)    M(x)
```

Another example: "there exists x such that P(x) and Q(x)" reads as "there is an object x that has both property P and property Q".

```
      THERE-EXISTS x
          |
         AND
        /   \
     P(x)   Q(x)
```

<!-- DIAGRAM: Euler diagram showing two overlapping circles labeled P and Q, with the intersection shaded to represent objects that have both properties -->

![Diagram showing an Euler diagram with two overlapping circles. Circle P represents objects with property P, circle Q represents objects with property Q. The intersection is shaded to show objects satisfying "there exists x such that P(x) and Q(x)".](/img/24.02.01-euler-overlap-placeholder.svg)

Consider the domain of all students in a classroom. Let S(x) mean "x studies logic" and P(x) mean "x passes the exam".

- "Every student who studies logic passes the exam": for all x, if S(x) then P(x).
- "Some student studies logic but does not pass": there exists x such that S(x) and not P(x).
- "No student studies logic and fails": there does not exist x such that S(x) and not P(x). This is equivalent to the first statement.

<!-- DIAGRAM: Euler diagram showing circle P inside circle S, with the region outside P but inside S shaded to represent students who study but fail -->

The relationship between the two quantifiers is one of duality. The statement "not everything has property P" is equivalent to "something does not have property P". And "there is nothing with property P" is equivalent to "everything lacks property P". This duality means you can always rewrite a statement with one quantifier using the other and negation.

## Worked example [Beginner]

**Example 1: Translating English to predicate logic.**

"All cats are mammals" becomes "for all x, if C(x) then M(x)", where C(x) means "x is a cat" and M(x) means "x is a mammal". Notice the conditional: we do not say "for all x, C(x) and M(x)", which would mean "everything is both a cat and a mammal". The conditional is the correct form for "all A are B" statements.

"Some cats are black" becomes "there exists x such that C(x) and B(x)". Here we use conjunction, not conditional. Using a conditional would be true if anything exists that is not a cat (since a false antecedent makes the conditional true), which is not what we mean.

**Example 2: Multiple quantifiers.**

"Every student has an advisor" translates as: for every x, if x is a student, then there exists y such that y is an advisor and y advises x. For each student, there is some advisor who advises them.

"Some professor advises every student" translates as: there exists y such that y is a professor, and for every x, if x is a student then y advises x. Here one professor advises all students.

**Example 3: Negating quantified statements.**

To negate "All birds can fly", push the negation inward. "Not all birds can fly" becomes "there exists a bird that cannot fly". In symbols: negating "for all x, if B(x) then F(x)" gives "there exists x such that B(x) and not F(x)".

**Example 4: A simple argument.**

Premise 1: All dogs are animals. For all x, if D(x) then A(x).
Premise 2: All animals need water. For all x, if A(x) then W(x).
Conclusion: All dogs need water. For all x, if D(x) then W(x).

This is valid. For any object x: if D(x), then A(x) by premise 1, then W(x) by premise 2. This chain of conditionals is the predicate-logic version of hypothetical syllogism.

## Formal definition [Intermediate+]

A first-order language $\mathcal{L}$ consists of:

1. **Logical symbols:** variables $v_1, v_2, \ldots$; connectives $\neg, \to, \land, \lor, \leftrightarrow$; quantifiers $\forall, \exists$; equality $=$; parentheses.
2. **Non-logical symbols:** a set of constant symbols $c_0, c_1, \ldots$; function symbols $f_0, f_1, \ldots$ each with an arity; predicate symbols $P_0, P_1, \ldots$ each with an arity.

A **term** is defined inductively: every variable is a term; every constant is a term; if $f$ is an $n$-ary function symbol and $t_1, \ldots, t_n$ are terms, then $f(t_1, \ldots, t_n)$ is a term. Terms name objects in the domain. For example, in the language of arithmetic with constant $0$ and unary function $S$ (successor), the terms $0$, $S(0)$, $S(S(0))$ name the numbers zero, one, and two. In a language with binary function $+$, the term $+(S(0), S(S(0)))$ names $1 + 2 = 3$.

A **formula** is defined inductively: if $P$ is an $n$-ary predicate and $t_1, \ldots, t_n$ are terms, then $P(t_1, \ldots, t_n)$ is an atomic formula; equality $t_1 = t_2$ is also an atomic formula; if $\varphi$ and $\psi$ are formulas, so are $\neg \varphi$, $(\varphi \land \psi)$, $(\varphi \lor \psi)$, $(\varphi \to \psi)$, $(\varphi \leftrightarrow \psi)$; if $\varphi$ is a formula and $x$ is a variable, then $\forall x \varphi$ and $\exists x \varphi$ are formulas. The set of well-formed formulas is the smallest set closed under these formation rules.

An occurrence of variable $x$ in formula $\varphi$ is **bound** if it falls within the scope of a quantifier $\forall x$ or $\exists x$. An occurrence that is not bound is **free**. The same variable can have both free and bound occurrences in a single formula: in $P(x) \land \forall x Q(x)$, the first occurrence of $x$ in $P(x)$ is free while the occurrence in $Q(x)$ is bound by $\forall x$. A formula with no free variables is a **sentence**. Sentences have determinate truth values relative to a structure; open formulas (with free variables) define sets within a structure.

A **structure** (or interpretation or model) $\mathfrak{A}$ for language $\mathcal{L}$ consists of:

1. A nonempty set $|\mathfrak{A}|$ called the **domain** or **universe**.
2. For each constant symbol $c$, an element $c^{\mathfrak{A}} \in |\mathfrak{A}|$.
3. For each $n$-ary function symbol $f$, a function $f^{\mathfrak{A}}: |\mathfrak{A}|^n \to |\mathfrak{A}|$.
4. For each $n$-ary predicate symbol $P$, a relation $P^{\mathfrak{A}} \subseteq |\mathfrak{A}|^n$.

A **variable assignment** in $\mathfrak{A}$ is a function $s: \text{Vars} \to |\mathfrak{A}|$. The **satisfaction relation** $\mathfrak{A}, s \models \varphi$ ("$\mathfrak{A}$ satisfies $\varphi$ under assignment $s$") is defined recursively:

- $\mathfrak{A}, s \models P(t_1, \ldots, t_n)$ iff $(t_1^{\mathfrak{A},s}, \ldots, t_n^{\mathfrak{A},s}) \in P^{\mathfrak{A}}$
- $\mathfrak{A}, s \models \neg \varphi$ iff $\mathfrak{A}, s \not\models \varphi$
- $\mathfrak{A}, s \models \varphi \land \psi$ iff $\mathfrak{A}, s \models \varphi$ and $\mathfrak{A}, s \models \psi$
- $\mathfrak{A}, s \models \varphi \lor \psi$ iff $\mathfrak{A}, s \models \varphi$ or $\mathfrak{A}, s \models \psi$
- $\mathfrak{A}, s \models \varphi \to \psi$ iff $\mathfrak{A}, s \not\models \varphi$ or $\mathfrak{A}, s \models \psi$
- $\mathfrak{A}, s \models \forall x \varphi$ iff for every $a \in |\mathfrak{A}|$, $\mathfrak{A}, s[x/a] \models \varphi$
- $\mathfrak{A}, s \models \exists x \varphi$ iff for some $a \in |\mathfrak{A}|$, $\mathfrak{A}, s[x/a] \models \varphi$

where $s[x/a]$ is the assignment identical to $s$ except that $s(x) = a$.

A sentence $\sigma$ is **true in $\mathfrak{A}$** (written $\mathfrak{A} \models \sigma$) if $\mathfrak{A}, s \models \sigma$ for every (equivalently, any) assignment $s$. A set of sentences $\Gamma$ **entails** $\varphi$ (written $\Gamma \models \varphi$) if every structure satisfying all of $\Gamma$ also satisfies $\varphi$.

### Key syntactic notions

**Substitution.** The notation $\varphi[t/x]$ denotes the result of substituting term $t$ for all free occurrences of $x$ in $\varphi$. A term $t$ is **free for $x$ in $\varphi$** if no free occurrence of $x$ in $\varphi$ falls within the scope of a quantifier that binds any variable of $t$. This condition ensures that substitution does not accidentally capture variables.

For example, in $\forall y (P(x,y) \to Q(y))$, substituting $y$ for $x$ yields $\forall y (P(y,y) \to Q(y))$, which changes the meaning: the formerly free $x$-position is now bound by $\forall y$. The term $y$ is not free for $x$ in this formula. Substituting $z$ instead gives $\forall y (P(z,y) \to Q(y))$, which preserves the intended meaning.

**Prenex normal form.** Every first-order formula is equivalent to one where all quantifiers appear at the front, in a block called the prefix, followed by a quantifier-free matrix. The equivalences needed are:

- $\neg \forall x \varphi \equiv \exists x \neg \varphi$ and $\neg \exists x \varphi \equiv \forall x \neg \varphi$
- $(\forall x \varphi) \land \psi \equiv \forall x (\varphi \land \psi)$ when $x$ is not free in $\psi$
- $(\exists x \varphi) \land \psi \equiv \exists x (\varphi \land \psi)$ when $x$ is not free in $\psi$
- $(\forall x \varphi) \lor \psi \equiv \forall x (\varphi \lor \psi)$ when $x$ is not free in $\psi$
- $(\forall x \varphi) \to \psi \equiv \exists x (\varphi \to \psi)$ when $x$ is not free in $\psi$
- $\psi \to (\forall x \varphi) \equiv \forall x (\psi \to \varphi)$ when $x$ is not free in $\psi$

When $x$ is free in $\psi$, renaming the bound variable resolves the conflict: $\forall x \varphi(x) \land \psi(x) \equiv \forall z (\varphi(z) \land \psi(x))$ for a fresh variable $z$.

### Semantic entailment and validity

A sentence $\sigma$ is **valid** (written $\models \sigma$) if $\mathfrak{A} \models \sigma$ for every structure $\mathfrak{A}$. A sentence is **satisfiable** if some structure makes it true. These are the fundamental semantic notions.

The relationship between validity and satisfiability mirrors propositional logic: $\sigma$ is valid if and only if $\neg \sigma$ is unsatisfiable. And $\Gamma \models \varphi$ if and only if $\Gamma \cup \{\neg \varphi\}$ is unsatisfiable. This reduction from entailment to satisfiability is the theoretical basis for automated theorem proving: to check whether $\varphi$ follows from $\Gamma$, check whether $\Gamma \cup \{\neg \varphi\}$ has a model.

**Examples of validity:**

- $\forall x P(x) \to \exists x P(x)$ is valid, because every structure has a nonempty domain.
- $\forall x (P(x) \to Q(x)) \to (\forall x P(x) \to \forall x Q(x))$ is valid. If everything with $P$ has $Q$, and everything has $P$, then everything has $Q$.
- $\exists x \forall y R(x,y) \to \forall y \exists x R(x,y)$ is valid. If one element relates to everything, then everything is related to by something. The converse is not valid.

**Examples of non-validity:**

- $\forall y \exists x R(x,y) \to \exists x \forall y R(x,y)$ is not valid. "Every number has a successor" does not entail "there is a number that is the successor of every number." A countermodel: let the domain be $\mathbb{N}$ and $R(x,y)$ mean $x > y$. Every number has a greater number, but no number is greater than every number.

- $\forall x (P(x) \lor Q(x)) \to (\forall x P(x) \lor \forall x Q(x))$ is not valid. "Everyone is a student or employed" does not entail "everyone is a student or everyone is employed." A countermodel: a domain of two people, one a student and not employed, the other employed and not a student.

### Theories and axiomatizations

A **theory** $T$ is a set of sentences closed under logical consequence: $T = \{\sigma : T \models \sigma\}$. A set of axioms $\Sigma$ **axiomatizes** $T$ if $T = \{\sigma : \Sigma \models \sigma\}$. The theory is **finitely axiomatizable** if it has a finite set of axioms.

**Examples:**

- The theory of dense linear orders without endpoints (DLO) is finitely axiomatized by: reflexivity, antisymmetry, transitivity, totality, density ($\forall x \forall y (x < y \to \exists z (x < z \land z < y))$), and no endpoints ($\forall x \exists y (y < x) \land \forall x \exists y (x < y)$). The unique countable model is $(\mathbb{Q}, <)$.

- Peano Arithmetic (PA) is axiomatized by finitely many axioms plus the induction schema: for each formula $\varphi(x)$, the axiom $\varphi(0) \land \forall x (\varphi(x) \to \varphi(S(x))) \to \forall x \varphi(x)$. The induction schema is infinite (one axiom per formula), so PA is not finitely axiomatizable.

- The theory of groups can be formulated in first-order logic with a binary function symbol $\cdot$ and a constant $e$: associativity, identity, and inverse. The theory of groups is finitely axiomatizable, but the theory of abelian groups requires an additional axiom ($\forall x \forall y (x \cdot y = y \cdot x)$).

### Definability

A set $X \subseteq |\mathfrak{A}|^n$ is **definable** in a structure $\mathfrak{A}$ if there exists a formula $\varphi(x_1, \ldots, x_n)$ such that $X = \{(a_1, \ldots, a_n) \in |\mathfrak{A}|^n : \mathfrak{A} \models \varphi(a_1, \ldots, a_n)\}$. A function is definable if its graph is a definable set.

Definability measures the expressive power of a language relative to a structure. In $(\mathbb{N}, +, \times)$, every computable set is definable (by the Matiyasevich-Davis-Putnam-Robinson theorem, every computably enumerable set is diophantine). But in $(\mathbb{N}, +)$ without multiplication, only the eventually periodic sets are definable. Multiplication vastly increases expressive power.

## Key theorem with proof [Intermediate+]

### The Soundness and Completeness of First-Order Logic (Godel's Completeness Theorem, 1929)

**Theorem (Godel).** For any set of first-order sentences $\Gamma$ and sentence $\varphi$: $\Gamma \models \varphi$ if and only if $\Gamma \vdash \varphi$ (where $\vdash$ denotes derivability in a standard proof system such as natural deduction or the Hilbert system).

The soundness direction ($\Gamma \vdash \varphi \implies \Gamma \models \varphi$) is straightforward: each inference rule preserves truth, so every theorem is a semantic consequence.

The completeness direction ($\Gamma \models \varphi \implies \Gamma \vdash \varphi$) is the deep result. The modern proof (Henkin's construction, 1949) proceeds in stages:

**Stage 1: Extending to a maximally consistent set.** If $\Gamma$ does not prove $\varphi$, then $\Gamma \cup \{\neg \varphi\}$ is consistent. By Lindenbaum's lemma, every consistent set extends to a maximally consistent set $\Gamma^*$: a consistent set such that adding any sentence not already in it produces an inconsistency.

**Stage 2: Adding witnessing constants (Henkin witnesses).** For each formula $\exists x \psi(x)$ in $\Gamma^*$, add a new constant $c_{\exists x \psi}$ and the witnessing axiom $\exists x \psi(x) \to \psi(c_{\exists x \psi})$. This ensures the expanded theory has names for everything it claims exists. Iterate through countably many extensions to handle all formulas.

**Stage 3: Building the term model.** The domain of the canonical model consists of equivalence classes of closed terms under the relation $t_1 \sim t_2$ iff $\Gamma^{**} \vdash t_1 = t_2$ (where $\Gamma^{**}$ is the final Henkin extension). Predicates are interpreted by: $P^{\mathfrak{A}}([t_1], \ldots, [t_n])$ iff $\Gamma^{**} \vdash P(t_1, \ldots, t_n)$.

**Stage 4: Truth lemma.** Prove by induction on formulas that for every sentence $\sigma$: $\mathfrak{A} \models \sigma$ if and only if $\sigma \in \Gamma^{**}$. The quantifier case uses the witnessing constants: $\mathfrak{A} \models \forall x \psi(x)$ iff $\mathfrak{A} \models \psi(c)$ for every constant $c$, which holds iff $\Gamma^{**} \vdash \psi(c)$ for every $c$, which by maximality holds iff $\forall x \psi(x) \in \Gamma^{**}$.

**Stage 5: Conclusion.** Since $\neg \varphi \in \Gamma^*$, the term model satisfies $\Gamma$ but not $\varphi$. So $\Gamma \not\models \varphi$, establishing the contrapositive.

This proof is constructive in the sense that it builds an explicit model from syntactic information. The method of Henkin witnesses is a powerful technique that has been adapted to many other completeness proofs in modal logic, infinitary logic, and type theory.

### Consequences of Completeness

**Compactness for first-order logic.** A set of first-order sentences $\Gamma$ is satisfiable if and only if every finite subset of $\Gamma$ is satisfiable. This follows from completeness: if $\Gamma$ is unsatisfiable, then $\Gamma \models \bot$, hence $\Gamma \vdash \bot$, and the proof uses only finitely many premises.

**The Lowenheim-Skolem theorem.** If a countable set of first-order sentences has a model, it has a countable model. If it has models of every infinite cardinality, it has models of no other cardinalities. First-order logic cannot control the cardinality of its models beyond the minimum, which is a fundamental limitation.

<aside class="exercise" data-difficulty="medium" data-type="symbolic" markdown="1">

**Exercise 3 (medium).** Negate $\forall x \exists y (P(x) \to Q(x,y))$ and push the negation inward as far as possible.

<details class="exercise__answer"><summary>Answer</summary>

$\neg \forall x \exists y (P(x) \to Q(x,y))$
$\equiv \exists x \neg \exists y (P(x) \to Q(x,y))$
$\equiv \exists x \forall y \neg (P(x) \to Q(x,y))$
$\equiv \exists x \forall y (P(x) \land \neg Q(x,y))$

"There exists an $x$ with property $P$ such that for every $y$, $Q(x,y)$ fails."

</details>

</aside>

<aside class="exercise" data-difficulty="medium" data-type="symbolic" markdown="1">

**Exercise 4 (medium).** Determine whether $\forall x P(x) \to \exists x P(x)$ is valid (true in every structure).

<details class="exercise__answer"><summary>Answer</summary>

Yes, it is valid. If every element of the domain has property $P$, and the domain is nonempty (a requirement of first-order structures), then at least one element has property $P$. Note: this relies on the convention that domains are nonempty.

</details>

</aside>

<aside class="exercise" data-difficulty="hard" data-type="symbolic" markdown="1">

**Exercise 5 (hard).** Convert $\neg(\forall x P(x) \lor \exists x Q(x))$ to prenex normal form.

<details class="exercise__answer"><summary>Answer</summary>

$\neg(\forall x P(x) \lor \exists x Q(x))$
$\equiv \neg \forall x P(x) \land \neg \exists x Q(x)$
$\equiv \exists x \neg P(x) \land \forall x \neg Q(x)$

Rename the second bound variable: $\exists x \neg P(x) \land \forall y \neg Q(y)$.

Prenex form: $\exists x \forall y (\neg P(x) \land \neg Q(y))$.

</details>

</aside>

<aside class="exercise" data-difficulty="hard" data-type="symbolic" markdown="1">

**Exercise 6 (hard).** Show that $\exists x \forall y R(x,y)$ does not entail $\forall y \exists x R(x,y)$ being false by finding a structure where the first is true but the second is false.

<details class="exercise__answer"><summary>Answer</summary>

In fact, the entailment goes the other direction. The correct exercise: show that $\forall y \exists x R(x,y)$ does not entail $\exists x \forall y R(x,y)$.

Let the domain be $\mathbb{N}$ and $R(x,y)$ mean $x > y$. For every $y$, there exists $x = y + 1$ with $x > y$, so $\forall y \exists x R(x,y)$ is true. But there is no single $x$ that is greater than every $y$ (no greatest natural number), so $\exists x \forall y R(x,y)$ is false. The order of quantifiers matters.

</details>

</aside>

<aside class="exercise" data-difficulty="hard" data-type="proof" markdown="1">

**Exercise 7 (hard).** Prove that if $\Gamma \models \varphi$ and $\Gamma$ is finite, then there exists a finite proof of $\varphi$ from $\Gamma$.

<details class="exercise__answer"><summary>Answer</summary>

By the completeness theorem, $\Gamma \vdash \varphi$. A proof is a finite sequence of formulas, each of which is either a logical axiom, a member of $\Gamma$, or follows from earlier formulas by an inference rule. Since proofs are finite by definition, $\varphi$ has a finite proof from $\Gamma$.

</details>

</aside>

<aside class="exercise" data-difficulty="medium" data-type="symbolic" markdown="1">

**Exercise 8 (medium).** Translate the following into first-order logic. Let $L(x,y)$ mean "x loves y" and let the domain be all people. (a) "Everybody loves somebody." (b) "Somebody loves everybody." (c) "There is somebody everybody loves." (d) "Nobody loves everybody."

<details class="exercise__answer"><summary>Answer</summary>

(a) $\forall x \exists y L(x,y)$
(b) $\exists y \forall x L(x,y)$
(c) $\exists y \forall x L(x,y)$ (same as b — "somebody" is the beloved)
(d) $\neg \exists x \forall y L(x,y)$, equivalently $\forall x \exists y \neg L(x,y)$

</details>

</aside>

<aside class="exercise" data-difficulty="hard" data-type="proof" markdown="1">

**Exercise 9 (hard).** Prove that the theory of equivalence relations (reflexive, symmetric, transitive) with infinitely many infinite equivalence classes is not finitely axiomatizable.

<details class="exercise__answer"><summary>Answer</summary>

The theory $T$ of equivalence relations with infinitely many classes, each infinite, is axiomatized by the three equivalence axioms plus infinitely many sentences: for each $n$, the sentence asserting "there are at least $n$ equivalence classes" and the sentence asserting "each class has at least $n$ elements." Suppose for contradiction that $T$ is finitely axiomatizable by $\sigma$. Then $\sigma$ can mention only finitely many of the "at least $n$" sentences. Let $N$ be the largest $n$ mentioned. A model with exactly $N$ equivalence classes would satisfy $\sigma$ but not $T$, contradiction.

</details>

</aside>

<aside class="exercise" data-difficulty="hard" data-type="symbolic" markdown="1">

**Exercise 10 (hard).** Let the language have one binary predicate $R$. Write a set of first-order sentences whose models are exactly the strict linear orders (irreflexive, transitive, total) with no greatest element.

<details class="exercise__answer"><summary>Answer</summary>

1. Irreflexivity: $\forall x \neg R(x,x)$
2. Transitivity: $\forall x \forall y \forall z ((R(x,y) \land R(y,z)) \to R(x,z))$
3. Totality: $\forall x \forall y (x \neq y \to (R(x,y) \lor R(y,x)))$
4. No greatest element: $\forall x \exists y R(x,y)$

The last sentence says every element has a strictly greater element, so there is no maximum.

</details>

## Advanced results [Master]

### The Compactness Theorem and Its Applications

The compactness theorem for first-order logic states that a (possibly infinite) set of sentences $\Gamma$ has a model if and only if every finite subset of $\Gamma$ has a model. This follows from the completeness theorem but can also be proved directly using ultraproducts.

**Application 1: Nonstandard models of arithmetic.** Let $\mathcal{L}_{\text{PA}}$ be the language of arithmetic $\{0, S, +, \times\}$. Extend the language with a new constant $c$. Let $\Gamma = \text{Th}(\mathbb{N}) \cup \{c \neq 0, c \neq S(0), c \neq S(S(0)), \ldots\}$. Every finite subset of $\Gamma$ is satisfiable (interpret $c$ as a sufficiently large natural number). By compactness, $\Gamma$ has a model, which must contain an element denoted by $c$ that is larger than every standard natural number. This nonstandard model of arithmetic satisfies all the same first-order sentences as $\mathbb{N}$ but has infinitely large elements.

**Application 2: The upward Lowenheim-Skolem theorem.** If a theory has an infinite model, it has models of every infinite cardinality $\kappa$. Add $\kappa$ many new constants and axioms stating they are pairwise distinct. Compactness gives a model, which must have size at least $\kappa$. Apply the downward Lowenheim-Skolem theorem to cut down to exactly $\kappa$.

**Application 3: The random graph.** The theory of the random graph is axiomatized by: for every $n, m \geq 1$, the sentence $\forall x_1 \ldots \forall x_n \forall y_1 \ldots \forall y_m (\bigwedge_{i \neq j} x_i \neq x_j \land \bigwedge_{i \neq j} y_i \neq y_j \land \bigwedge_{i,j} x_i \neq y_j \to \exists z (\bigwedge_i R(z, x_i) \land \bigwedge_j \neg R(z, y_j)))$. This says: for any finite disjoint sets of vertices, there is a vertex adjacent to everything in the first set and nothing in the second. Compactness shows this theory has a model, and it is the unique countable random graph (the Rado graph).

### The Lowenheim-Skolem Theorems

The **downward** Lowenheim-Skolem theorem states that if a countable first-order theory has a model, it has a countable model. More generally, if $\mathfrak{A}$ is a structure for a countable language and $X \subseteq |\mathfrak{A}|$, then there is an elementary submodel $\mathfrak{B} \preceq \mathfrak{A}$ with $X \subseteq |\mathfrak{B}|$ and $|\mathfrak{B}| \leq \max(|X|, \aleph_0)$.

The proof constructs $\mathfrak{B}$ by closing $X$ under Skolem functions: for each formula $\exists y \varphi(x_1, \ldots, x_n, y)$ true in $\mathfrak{A}$, choose a witnessing element $f_\varphi(a_1, \ldots, a_n)$ and close under all such functions. The result is a submodel that satisfies the same first-order sentences as $\mathfrak{A}$.

**Skolem's paradox.** Consider ZFC set theory (which is a countable first-order theory). If ZFC has a model, it has a countable model. But ZFC proves the existence of uncountable sets. How can a countable model contain uncountable sets? The resolution: the model is countable from the external (metatheoretic) perspective, but inside the model, no bijection exists between $\omega$ and the set that the model thinks is $\mathbb{R}$. Countability is not absolute: a set can be countable from outside but uncountable from inside.

### Decidability and Undecidability

First-order logic is **semidecidable**: there is an algorithm that, given $\Gamma \models \varphi$, will eventually confirm this (by searching for a proof). But there is no algorithm that always halts and correctly determines whether $\Gamma \models \varphi$. First-order validity is undecidable (Church's theorem, 1936).

The proof reduces the halting problem to first-order validity. Given a Turing machine $M$ and input $w$, construct a first-order sentence $\sigma_{M,w}$ such that $\sigma_{M,w}$ is valid if and only if $M$ halts on $w$. Since the halting problem is undecidable, so is first-order validity.

Despite this, many **fragments** of first-order logic are decidable:

- **Monadic first-order logic** (all predicates are unary) is decidable. This is the Lowenheim-Behmann theorem. The proof reduces satisfiability to checking finitely many "types" (complete descriptions of an element's properties).
- **The two-variable fragment** $\text{FO}^2$ (only variables $x$ and $y$) is decidable (Mortimer 1975, improved by Gradel, Kolaitis, and Vardi).
- **The guarded fragment** (quantifiers restricted by atomic formulas) is decidable (Gradel 1999). Many description logics used in knowledge representation fall in this fragment.
- **The Bernays-Schonfinkel class** ($\exists^*\forall^*$ sentences without function symbols) is decidable. Satisfiability reduces to checking models up to a finite bound.

### Quantifier Elimination

A theory $T$ admits **quantifier elimination** if for every formula $\varphi(\bar{x})$ there is a quantifier-free formula $\psi(\bar{x})$ such that $T \models \varphi \leftrightarrow \psi$. Theories with quantifier elimination are, in a precise sense, "simple" — every definable set is definable by a quantifier-free formula.

Important theories with quantifier elimination include:

- **Dense linear orders without endpoints** (DLO): the theory of $(\mathbb{Q}, <)$. Every first-order property of rationals expressible with $<$ can be stated without quantifiers.
- **Algebraically closed fields** (ACF): the theory of $\mathbb{C}$ as a field. The Chevalley theorem in algebraic geometry is the geometric content of quantifier elimination.
- **Real closed fields** (RCF): the theory of $\mathbb{R}$ as an ordered field. Tarski's theorem (1930) shows that the first-order theory of real numbers with addition and multiplication is decidable, giving a decision procedure for elementary geometry and algebra.

Quantifier elimination implies decidability (since quantifier-free formulas can be checked by computation) and model completeness (every embedding between models is elementary).

### The Craig Interpolation Theorem for First-Order Logic

Craig's interpolation theorem extends to first-order logic: if $\Gamma \models \Delta$ and both are sets of sentences, then there exists a sentence $\theta$ in the common language of $\Gamma$ and $\Delta$ such that $\Gamma \models \theta$ and $\theta \models \Delta$. If the common language is empty (no shared non-logical symbols), then either $\Gamma$ is unsatisfiable or $\Delta$ is valid.

The proof for first-order logic is more involved than for propositional logic. One approach uses the consistency property method: a consistency property is a set of finite sets of sentences satisfying certain closure conditions, and the existence of a consistency property containing $\Gamma \cup \{\neg \theta\}$ for every interpolant candidate $\theta$ leads to a contradiction.

**Applications of interpolation.** Craig interpolation has applications in modular verification: if two modules interact through a well-defined interface (the common language), any property of the combined system that depends on one module can be expressed in terms of the interface alone. This is the theoretical foundation of compositional reasoning in formal methods.

Interpolation also connects to definability through Beth's theorem: implicit and explicit definability coincide for first-order logic. If a theory uniquely determines the interpretation of a predicate symbol in every model, then the predicate is definable from the remaining symbols.

### Lindstrom's Theorem

Lindstrom's theorem (1969) characterizes first-order logic as the maximal logic satisfying both the compactness theorem and the Lowenheim-Skolem theorem. Any extension of first-order logic that satisfies both compactness and downward Lowenheim-Skolem is equivalent to first-order logic.

This means first-order logic occupies a unique position: it is the strongest logic with these two properties. Stronger logics (second-order logic, infinitary logics, logics with generalized quantifiers) must sacrifice at least one. Second-order logic fails compactness (it can characterize the natural numbers up to isomorphism, ruling out nonstandard models). The infinitary logic $\mathcal{L}_{\omega_1 \omega}$ (which allows countable conjunctions and disjunctions) fails compactness but retains a form of Lowenheim-Skolem.

Lindstrom's theorem explains why first-order logic is the "right" logic for model theory: it is the unique sweet spot between expressiveness and the meta-theoretic properties needed for a robust theory of models.

## Connections [Master]

This unit connects to several areas within the broader curriculum:

- **Propositional logic [49.01.01].** Predicate logic extends propositional logic with quantifiers and internal structure for atomic propositions. All propositional connectives and truth-functional semantics carry over. The completeness proof for predicate logic builds on the propositional case, adding the Henkin construction to handle quantifiers.

- **Set theory [01.XX].** First-order logic is the language of axiomatic set theory (ZFC). The quantifiers $\forall$ and $\exists$ range over sets, and the axioms are first-order sentences. The cumulative hierarchy $V_\alpha$ is defined by transfinite recursion, and the consistency of ZFC cannot be proved within ZFC itself (Godel's second incompleteness theorem).

- **Database theory [25.07.01].** Relational calculus (the theoretical foundation of SQL) is a variant of first-order logic. SQL queries are existential-quantifier-heavy: "SELECT name FROM students WHERE EXISTS (SELECT ...)" is $\exists x (\text{Student}(x) \land \ldots)$. The decidability of certain fragments of first-order logic ensures that database queries always terminate.

- **Artificial intelligence [25.09.01].** Knowledge representation uses first-order logic as a formal language for expressing facts, rules, and queries. Description logics, the formal foundation of ontologies (used in the Semantic Web), are decidable fragments of first-order logic. Prolog-style logic programming is based on the Horn clause fragment of first-order logic.

- **Programming language theory [25.04.01].** Type systems use quantifiers: parametric polymorphism corresponds to universal quantification over types ($\forall \alpha. \tau(\alpha)$), and existential types ($\exists \alpha. \sigma(\alpha)$) correspond to abstract data types. The Curry-Howard correspondence links proofs in predicate logic to programs in typed lambda calculi.

- **Formal verification [25.08.01].** Model checking and theorem proving use first-order logic to specify and verify system properties. SMT (Satisfiability Modulo Theories) solvers decide fragments of first-order logic extended with theories of arithmetic, arrays, and bit-vectors, enabling automated verification of hardware and software.

- **Computational complexity [25.03.01].** Descriptive complexity theory characterizes complexity classes in terms of the expressiveness of logical languages. Fagin's theorem (1974) shows that NP corresponds to existential second-order logic. The Immerman-Vardi theorem (1982) shows that P corresponds to first-order logic with least fixed points on ordered structures.

## Historical and philosophical context [Master]

### Frege's revolution

Gottlob Frege's *Begriffsschrift* (1879) introduced the first formal system for quantificational logic. Frege's key insight was that quantifiers are second-order concepts — they map first-level concepts (predicates) to truth values. The statement "all humans are mortal" does not predicate mortality of individual humans but rather asserts a relation between the concept "human" and the concept "mortal".

Frege's notation was two-dimensional and diagrammatic, making it nearly unreadable. Russell called it "cumbrous" and Peano's more linear notation eventually won out. But the conceptual content — the distinction between function and argument, the treatment of quantifiers as variable-binding operators, the strict separation of syntax from semantics — was entirely correct and remains the foundation of logic today.

Frege's system also introduced the distinction between sense (Sinn) and reference (Bedeutung). The morning star and the evening star have the same reference (the planet Venus) but different senses. This distinction, developed in "Uber Sinn und Bedeutung" (1892), became central to the philosophy of language and influenced Carnap, Church, and Montague.

### The Lowenheim-Skolem paradox

Leopold Lowenheim proved in 1915 that if a first-order sentence has a model, it has a countable model. Thoralf Skolem strengthened and generalized this result in 1920 and 1922. The consequence — that set theory, which postulates uncountable sets, has countable models — struck many as paradoxical.

Skolem himself used the result to argue against the absolute reality of set-theoretic concepts. In "Some Remarks on Axiomatized Set Theory" (1922), Skolem argued that all mathematical notions are relative to the axiomatic system in which they are formulated, and that no set-theoretic notion (including uncountability) has an absolute meaning. This position, known as Skolemism or set-theoretic relativism, remains a topic of philosophical debate.

Von Neumann responded that the countable model is "unnatural" and that the standard model should be preferred. Cohen (1963) showed with forcing that the size of the continuum (the set of real numbers) is not determined by ZFC, deepening the mystery. The independence of the continuum hypothesis from ZFC means that the question "how many real numbers are there?" has no answer within standard mathematics.

### Godel's completeness theorem and its context

Kurt Godel proved the completeness theorem in his 1929 doctoral dissertation at the University of Vienna, at the age of 23. The theorem shows that the syntactic notion of proof and the semantic notion of logical consequence coincide for first-order logic. This is a rare positive result in a field dominated by limitation theorems.

The proof method Godel used was syntactic, constructing proofs directly. Henkin's 1949 proof, which builds a model from the syntax of the theory, is now the standard approach because it generalizes more readily to other logics. Henkin's proof reveals that completeness is not really about the strength of the proof system but about the ability to build a syntactic model that mirrors semantic truth.

Godel proved the completeness theorem before his incompleteness theorems (1931). The completeness theorem says first-order logic is "complete" in the sense that every valid formula is provable. The incompleteness theorems say that certain theories (like arithmetic) are "incomplete" in the different sense that some true sentences are not provable. These are different notions of completeness applied to different objects.

### The foundational debates

The development of predicate logic was intertwined with the foundational crisis in mathematics (1900-1930). Three schools competed:

**Logicism** (Frege, Russell) held that all mathematical truths are logical truths. The logicist program required showing that mathematical concepts (numbers, functions, sets) are definable in purely logical terms, and that mathematical axioms are logical truths. Russell's type theory and the *Principia Mathematica* were the main technical vehicles. Godel's incompleteness theorems did not directly refute logicism, but the need for axioms like infinity and choice, which have a distinctly non-logical character, undermined the program.

**Formalism** (Hilbert) held that mathematics is a formal game played with symbols according to rules. The consistency of mathematical systems should be proved by finitary means — methods so elementary that no one could doubt them. Hilbert's program required a finitary proof of the consistency of arithmetic. Godel's second incompleteness theorem (1931) showed this is impossible: no consistent formal system containing arithmetic can prove its own consistency.

**Intuitionism** (Brouwer, later Heyting) held that mathematics is a construction of the human mind. The law of excluded middle ($\varphi \lor \neg \varphi$) was rejected for infinite domains, and existence proofs were required to be constructive (providing an explicit witness). Intuitionistic logic is a proper subsystem of classical logic, and the intuitionistic interpretation of the quantifiers differs: $\exists x \varphi(x)$ means "we can construct an $x$ satisfying $\varphi$", not merely "such an $x$ exists".

### Tarski's semantic theory of truth

Alfred Tarski's "The Concept of Truth in Formalized Languages" (1933, German translation 1936) provided the first rigorous definition of truth for formal systems. Tarski showed that truth can be defined for any first-order language in a metalanguage that is strictly stronger. The definition satisfies Convention T: for every sentence $\sigma$ of the object language, the metalanguage proves "$\sigma$ is true if and only if $\sigma^*$" where $\sigma^*$ is the translation of $\sigma$ into the metalanguage.

Tarski also proved that truth cannot be consistently defined in the object language itself (for languages containing arithmetic). This is the formal statement of the liar paradox: a language that contains its own truth predicate is inconsistent. This result has profound implications for the philosophy of language and for theories of semantic paradox.

### Model theory and the abstract approach

Model theory, the study of the relationship between formal languages and their interpretations, emerged as a distinct discipline in the 1950s through the work of Tarski, Robinson, and others. Abraham Robinson used model theory to develop nonstandard analysis (1961), providing rigorous foundations for reasoning with infinitesimals — a concept that Leibniz and Euler had used informally but that was thought to have been rendered obsolete by the epsilon-delta definitions of the 19th century.

Morley's categoricity theorem (1965) showed that if a countable first-order theory is categorical in one uncountable cardinality (all models of that size are isomorphic), then it is categorical in all uncountable cardinalities. This striking result connected logical properties of theories to structural properties of their models and launched stability theory, which classifies theories by the complexity of their models.

Shelah's classification theory (1970s-present) provides a taxonomy of first-order theories based on the number and complexity of their models. The dividing lines (stable, superstable, simple, NIP, NTP2, etc.) correspond to meaningful structural properties. This program shows that the universe of first-order theories is not chaotic but has a deep underlying order.

### Peano and the axiomatization of arithmetic

Giuseppe Peano published his axioms for the natural numbers in 1889 (in *Arithmetices principia, nova methodo exposita*). The original formulation used a simpler notation but the ideas were the same: the natural numbers are generated from zero by repeated application of the successor function, and induction holds for all properties. Peano's axioms demonstrated that arithmetic could be formalized in a precise logical language, separating the question of what arithmetic *is* from the question of what can be *proved* about it.

Peano used a logical notation influenced by Frege and Schroder but more readable than either. His notation for quantifiers and logical connectives was widely adopted in the early twentieth century. The *Formulario Mathematico* (1895-1908), a collaborative work edited by Peano, aimed to express all known mathematics in a formal logical notation. Though the project was never completed, it influenced Russell and Whitehead's *Principia Mathematica*.

### The role of quantifiers in mathematical practice

Quantifiers are so fundamental to modern mathematics that it is easy to forget they had to be invented. Before Frege, mathematicians used quantifier-like constructions informally. Euclid's "for every point P and line L not containing P, there is exactly one line through P parallel to L" is a quantified statement, but Euclid had no notation for it and no formal theory of how quantifiers interact.

The epsilon-delta definition of continuity ("for every $\epsilon > 0$, there exists $\delta > 0$ such that for all $x$, if $|x - a| < \delta$ then $|f(x) - f(a)| < \epsilon$") involves three alternating quantifiers: $\forall \epsilon \exists \delta \forall x$. The order matters: swapping the first two quantifiers gives a much weaker condition (uniform continuity only at a point), and swapping the last two gives a different condition altogether. Learning to navigate nested quantifiers is one of the main challenges in rigorous analysis.

In algebra, the definition of a group involves quantifiers: the identity axiom $\exists e \forall x (x \cdot e = x \land e \cdot x = x)$ and the inverse axiom $\forall x \exists y (x \cdot y = e \land y \cdot x = e)$. The alternation between $\forall$ and $\exists$ in the inverse axiom means the inverse depends on the element — this is essential to group theory.

### The development of proof theory

David Hilbert's program (1920s) aimed to prove the consistency of mathematical theories using finitary methods. The program required a precise formalization of the logic used in mathematical reasoning, which motivated the development of proof theory as a discipline.

Gerhard Gentzen's natural deduction system (1934) formalized the introduction and elimination rules for each logical connective and quantifier. The rules for $\forall$ are: $\forall$-introduction (from $\varphi(x)$ with $x$ arbitrary, infer $\forall x \varphi(x)$) and $\forall$-elimination (from $\forall x \varphi(x)$, infer $\varphi(t)$ for any term $t$). The rules for $\exists$ are: $\exists$-introduction (from $\varphi(t)$, infer $\exists x \varphi(x)$) and $\exists$-elimination (from $\exists x \varphi(x)$ and a derivation of $\psi$ from $\varphi(y)$ with $y$ fresh, infer $\psi$).

Gentzen's sequent calculus provides another proof system where the rules for quantifiers come in left and right variants. The cut-elimination theorem (Hauptsatz) shows that every proof in the sequent calculus can be transformed into a cut-free proof, where no intermediate formulas are introduced and then eliminated. Cut-free proofs have the subformula property: every formula in the proof is a subformula of the conclusion. This has profound consequences for proof search and automated theorem proving.

### Category-theoretic perspectives

Category theory provides an alternative foundation for quantifiers. In a topos (a category with sufficient structure to interpret logic), the universal quantifier $\forall x : A$ corresponds to a right adjoint to the pullback functor along the projection $A \times B \to B$, and the existential quantifier $\exists x : A$ corresponds to a left adjoint. This adjoint characterization of quantifiers reveals their fundamental algebraic nature.

The connection between adjoints and quantifiers was first observed by Lawvere in 1969. It shows that quantifiers are not arbitrary syntactic operations but arise naturally from the structure of substitution (pullback). This perspective has been influential in categorical logic and theoretical computer science, where it connects logical operations to computational ones through the Curry-Howard correspondence.

### Quantifiers in natural language

The study of quantifiers in natural language connects logic to linguistics. Richard Montague's PTQ (Proper Treatment of Quantification in English, 1973) showed that natural language quantifiers like "most", "few", "every other", and "more than three" can be given a precise semantic treatment using generalized quantifier theory.

Generalized quantifiers extend the standard $\forall$ and $\exists$ to arbitrary relations between sets. The quantifier "most" corresponds to the relation $\{(A,B) : |A \cap B| > |A \setminus B|\}$. Barwise and Cooper (1981) showed that the quantifiers found in natural languages have distinctive semantic properties (conservativity, extension, monotonicity) that constrain the set of logically possible quantifiers to a small and linguistically natural subset.

The study of generalized quantifiers bridges logic, linguistics, and cognitive science. Experimental work by psychologists including Lance Rips and Stephen Crain has investigated how children acquire the meaning of quantifiers, finding that some quantifier combinations (like "every...some...") are acquired much later than others (like "some...every..."), suggesting that the logical complexity of quantifier nesting is a genuine cognitive burden.

### Intuitionistic and constructive quantifiers

In intuitionistic logic, the quantifiers have a constructive interpretation. "For all x, P(x)" means we possess a method that, given any x, produces a proof of P(x). "There exists x such that P(x)" means we can exhibit a specific x and a proof that P(x) holds. The existential quantifier in intuitionistic logic requires a witness, not merely a proof that a contradiction would follow from there being no witness.

This constructive reading has direct computational content through the Curry-Howard correspondence (also known as the propositions-as-types interpretation). Under this correspondence, a proof of $\forall x : A. P(x)$ is a function that takes any $x : A$ and returns a proof of $P(x)$. A proof of $\exists x : A. P(x)$ is a pair $(a, p)$ where $a : A$ and $p : P(a)$. The constructive content of proofs in intuitionistic predicate logic is executable code.

Martin-Lof type theory (1970s-present) develops this correspondence fully, providing a foundational system where propositions are types and proofs are programs. Dependent types (types that depend on values) correspond precisely to the quantifiers: $\Pi_{x:A} B(x)$ (dependent product type) corresponds to $\forall x : A. B(x)$, and $\Sigma_{x:A} B(x)$ (dependent sum type) corresponds to $\exists x : A. B(x)$. This correspondence is the foundation of proof assistants like Coq, Agda, and Lean.
