## Intuition [Beginner]

When two people rank the same set of items differently, you want a number measuring "how far apart" their rankings are. Several natural distances exist: count how many items are in different positions, count how many pairs are in opposite order, or count the minimum number of swaps needed to turn one ranking into the other.

Each of these distances has a character-theoretic formula: it can be expressed as a sum involving the irreducible characters of the symmetric group. This is not a coincidence — the characters encode the deep symmetry structure of the group, and any function invariant under certain group operations can be expanded in characters.

Why does this concept exist? Because the most useful distances on permutations all have spectral expressions that connect them to representation theory, enabling character-based computations and statistical tests on ranking data.

## Worked example [Beginner]

Consider the permutations $\pi = (2,3,1,4)$ and $\sigma = (1,2,3,4)$ in one-line notation on four items. We compute three distances.

Step 1. **Cayley distance**: the minimum number of transpositions needed to transform $\pi$ into $\sigma$. The permutation $\sigma^{-1}\pi = (2,3,1,4)$ has cycle decomposition $(1\;2\;3)(4)$. A 3-cycle requires $3 - 1 = 2$ transpositions. So $d_C(\pi, \sigma) = 2$.

Step 2. **Hamming distance**: the number of positions where the two permutations differ. Comparing position by position: position 1 has 2 vs 1 (differ), position 2 has 3 vs 2 (differ), position 3 has 1 vs 3 (differ), position 4 has 4 vs 4 (same). So $d_H(\pi, \sigma) = 3$.

Step 3. **Kendall tau distance**: the number of pairs in opposite relative order. In $\pi = (2,3,1,4)$: the pair $(1,2)$ has $2 > 1$ but $2$ appears before $1$, which is the wrong order compared to $(1,2,3,4)$. Counting inversions: $(2,1), (3,1)$ = 2 inversions. So $d_K(\pi, \sigma) = 2$.

What this tells us: different metrics capture different aspects of the disagreement between two rankings. Cayley counts swaps; Hamming counts displaced items; Kendall counts pairwise disagreements.

## Formal definition [Intermediate+]

Let $S_n$ denote the symmetric group. We define four metrics on $S_n$.

**Cayley distance.** $d_C(\pi, \sigma) = n - c(\sigma^{-1}\pi)$ where $c(\tau)$ is the number of cycles in the cycle decomposition of $\tau$ (counting fixed points as cycles of length 1).

**Hamming distance.** $d_H(\pi, \sigma) = \#\{i : \pi(i) \neq \sigma(i)\}$. Equivalently, $d_H(\pi, \sigma) = n - \text{fix}(\sigma^{-1}\pi)$ where $\text{fix}(\tau)$ is the number of fixed points of $\tau$.

**Kendall tau distance.** $d_K(\pi, \sigma) = \#\{(i,j) : i < j,\; (\pi^{-1}\sigma(i) - \pi^{-1}\sigma(j))(i - j) < 0\}$. This counts the number of pairs whose relative order is reversed between $\pi$ and $\sigma$.

**Spearman rho (squared) distance.** $d_\rho(\pi, \sigma) = \sum_{i=1}^{n}(\pi(i) - \sigma(i))^2$.

Each metric is right-invariant: $d(\pi\tau, \sigma\tau) = d(\pi, \sigma)$ for all $\tau$. This means $d(\pi, \sigma) = d(e, \sigma^{-1}\pi)$, so every metric is determined by its values on pairs involving the identity.

**Definition (Character-theoretic expression).** A metric $d$ on $S_n$ has a character-theoretic expression if there exist coefficients $a_\lambda$ such that

$$
d(e, \pi) = \sum_{\lambda \vdash n} a_\lambda \chi_\lambda(\pi)
$$

for all $\pi \in S_n$, where $\chi_\lambda$ is the irreducible character indexed by $\lambda$.

### Counterexamples to common slips

- *Confusing right-invariance with left-invariance.* All four metrics are right-invariant but not all are left-invariant. The Cayley distance is bi-invariant (since it depends only on cycle structure), but the Kendall tau distance is not left-invariant.
- *Spearman rho as a metric.* The sum of squared position differences $d_\rho$ is not a metric in the strict sense (it does not satisfy the triangle inequality without taking square root). The square root $\sqrt{d_\rho}$ is a metric.
- *Kendall tau normalisation.* The raw inversion count ranges from $0$ to $n(n-1)/2$. The normalised version $\tau = 1 - 2d_K/(n(n-1)/2)$ is the Kendall tau correlation coefficient used in statistics.

## Key theorem with proof [Intermediate+]

**Theorem (Character-theoretic expressions for metrics on $S_n$ — Diaconis 1988).** Each of the four metrics has a character-theoretic expansion. In particular, for $\pi \in S_n$ with cycle type $\mu = (\mu_1, \mu_2, \ldots)$:

$$
d_C(e, \pi) = n - \ell(\mu) = n - \sum_{\lambda \vdash n} \frac{\chi_\lambda(\mu)}{d_\lambda} \cdot (n - \ell(\mu)),
$$

$$
d_H(e, \pi) = n - \text{fix}(\pi) = n - \frac{1}{(n-1)!}\sum_{\lambda \vdash n} d_\lambda \chi_\lambda(\pi)\chi_\lambda(\text{transposition}),
$$

and the Kendall tau distance satisfies

$$
d_K(e, \pi) = \binom{n}{2} - \frac{1}{n}\sum_{\lambda \vdash n} \frac{d_\lambda}{d_\lambda} \chi_\lambda(\pi) \binom{n}{2} = \frac{1}{2}\sum_{i=1}^{n-1}\frac{1}{m_i}|\chi^{(i)}(\pi)|^2
$$

where $\chi^{(i)}$ denotes the character of the representation carried by the $i$-th graded piece of the coinvariant algebra and $m_i$ is a combinatorial multiplicity.

*Proof.* We prove the Hamming distance character formula; the others follow similar strategies.

**Step 1 (Hamming as a class function).** The number of fixed points $\text{fix}(\pi) = \sum_{i=1}^n \mathbf{1}_{\pi(i)=i}$ can be written as $\text{fix}(\pi) = \sum_{i=1}^n \delta_{\pi(i),i}$. This is the character of the permutation representation on $\{1, \ldots, n\}$, which decomposes as $\mathbf{1} \oplus V^{(n-1,1)}$ where $\mathbf{1}$ is the identity representation. So $\text{fix}(\pi) = 1 + \chi_{(n-1,1)}(\pi)$.

**Step 2 (Inversion via Plancherel).** Express $\text{fix}(\pi)$ using the orthogonality relations. The function $\text{fix}$ is a class function (it depends only on the cycle type), so by character inversion:

$$
\text{fix}(\pi) = \sum_{\lambda \vdash n} \langle \text{fix}, \chi_\lambda \rangle \chi_\lambda(\pi), \quad \langle \text{fix}, \chi_\lambda \rangle = \frac{1}{n!}\sum_{\tau} \text{fix}(\tau)\overline{\chi_\lambda(\tau)}.
$$

The inner product $\langle \text{fix}, \chi_\lambda \rangle$ equals the multiplicity of $V^\lambda$ in the permutation representation. By the decomposition $\text{perm} = \mathbf{1} \oplus V^{(n-1,1)}$, only $\lambda = (n)$ and $\lambda = (n-1,1)$ contribute:

$$
\text{fix}(\pi) = 1 + \chi_{(n-1,1)}(\pi).
$$

Therefore $d_H(e, \pi) = n - 1 - \chi_{(n-1,1)}(\pi)$.

**Step 3 (Express via all characters).** Using the column orthogonality of the character table, the fixed point function can also be written as:

$$
\text{fix}(\pi) = \frac{1}{(n-1)!}\sum_{\lambda} d_\lambda \chi_\lambda(\pi) \chi_\lambda(\text{transposition})
$$

since the transposition class generates the $S_n$-invariant bilinear form that pairs with $\text{fix}$. This gives the stated formula for $d_H$. $\square$

**Bridge.** The character-theoretic expressions for metrics build toward the spectral analysis of permutation data in [07.05.11] where distances between distributions are measured by spectral norms, and appear again in [07.05.13] where metrics on cosets extend the framework to partial rankings. The foundational reason the character expansion works is that every right-invariant function on $S_n$ is a class function of $\sigma^{-1}\pi$, and the characters form an orthonormal basis for class functions. This is exactly the content that identifies the metric geometry of $S_n$ with the representation-theoretic Fourier analysis; the bridge is that computing a distance between two permutations is the same as evaluating a character sum, which is the same as computing a Fourier coefficient at each irreducible representation. Putting these together with the Diaconis-Graham inequalities, the four metrics are ordered by the first and second Diaconis-Graham bounds: $d_C \leq d_K \leq 2d_C$ and $d_H \leq 2d_K$.

**Exercise 8 (hard, symbolic).**

Prove the second Diaconis-Graham inequality: $d_K(\pi, \sigma) \leq 2 \cdot d_C(\pi, \sigma) \cdot (n - d_C(\pi, \sigma))$ for all $\pi, \sigma \in S_n$.

<details class="exercise__hint"><summary>Hint</summary>

A single transposition $(i\;j)$ introduces at most $2|j-i| - 1 \leq 2(n-1) - 1 = 2n - 3$ inversions. Use the bound $|j-i| \leq n-1$. Then refine using the fact that $k$ transpositions can introduce at most $2k(n-k)$ inversions by optimising the transposition sizes.

</details>

<details class="exercise__answer"><summary>Answer</summary>

Write $k = d_C(\pi, \sigma)$ and express $\sigma^{-1}\pi = \tau_1 \cdots \tau_k$ with each $\tau_i$ a transposition of elements at positions $a_i$ and $b_i$ with $a_i < b_i$. However, a tighter bound is obtained by noting that the total displacement $\sum_i (b_i - a_i)$ is at most $k(n - k)$ (since $k$ transpositions can move elements at most $k$ positions from their origin in each direction). This gives $d_K \leq 2 \cdot k \cdot (n - k)$, which is the stated inequality (actually a slight variant; the precise Diaconis-Graham bound is $d_K \leq 2d_C \cdot (n - d_C)$, proved by induction on $k$ using the fact that the inversion count after $k$ transpositions is maximised when the transpositions are chosen to maximise displacement at each step).

</details>

## Advanced results [Master]

**Theorem 1 (Diaconis-Graham inequalities 1977).** For all $\pi, \sigma \in S_n$:

$$
d_C(\pi, \sigma) \leq d_K(\pi, \sigma), \qquad d_K(\pi, \sigma) \leq 2\,d_C(\pi, \sigma)\,(n - d_C(\pi, \sigma)).
$$

These bounds are tight. The lower bound is achieved when $\sigma^{-1}\pi$ is a single transposition of adjacent elements. The upper bound is achieved when $\sigma^{-1}\pi$ is a product of disjoint transpositions that maximise the displacement.

**Theorem 2 (Cayley distance and characters).** The Cayley distance has the closed-form character expression:

$$
d_C(e, \pi) = n - \ell(\mu) = \sum_{\lambda \vdash n} \frac{n - \ell(\mu)}{\sum_\nu d_\nu^2} d_\lambda^2 \left(1 - \frac{\chi_\lambda(\mu)}{d_\lambda}\right),
$$

where $\mu$ is the cycle type of $\pi$ and $\ell(\mu)$ is the number of parts. The inner sum simplifies because only the identity and standard representations contribute substantively for small Cayley distance.

This expression was derived by Diaconis 1988 using the fact that the Cayley distance is a class function depending only on cycle type.

**Theorem 3 (Hamming distance and Fourier transform).** The Hamming distance decomposes spectrally as:

$$
d_H(e, \pi) = (n - 1)\left(1 - \frac{\chi_{(n-1,1)}(\pi)}{n-1}\right) = n - 1 - \chi_{(n-1,1)}(\pi).
$$

This is the simplest character formula among the four metrics because only the standard representation contributes.

**Theorem 4 (Spearman footrule and its spectral expansion).** The Spearman footrule $d_S(\pi, e) = \sum_i |\pi(i) - i|$ has a Fourier expansion involving the characters of the standard representation and its exterior powers. Diaconis and Graham 1977 showed that $d_S \leq 2d_K$ and $d_S \leq 2d_H(n - d_H)$.

**Theorem 5 (Metric properties and the Bruhat order).** The Kendall tau distance (equivalently, the Coxeter length) respects the Bruhat order on $S_n$: if $\pi \leq \sigma$ in Bruhat order, then $d_K(e, \pi) \leq d_K(e, \sigma)$. The Bruhat order is the partial order generated by the relation $\pi \leq \pi \tau$ whenever $\tau$ is a transposition that increases the inversion count.

This connects the metric geometry of $S_n$ to the combinatorics of the Bruhat order, which is fundamental in Schubert calculus.

**Theorem 6 (Expected distances under uniformity).** Under the uniform distribution on $S_n$:

$$
\mathbb{E}[d_K] = \frac{1}{2}\binom{n}{2}, \quad \mathbb{E}[d_H] = n\left(1 - \frac{1}{n}\right) = n - 1, \quad \mathbb{E}[d_C] = n - H_n,
$$

where $H_n = \sum_{k=1}^n 1/k$ is the $n$-th harmonic number. These expected values are computed from the character expressions by taking expectations of the characters, which evaluate to 0 for non-identity representations under the uniform distribution.

**Synthesis.** The character-theoretic expressions for metrics on $S_n$ provide the foundational reason that representation theory unifies the metric geometry of permutations. The central insight is that right-invariant metrics on $S_n$ are determined by their values at the identity, which are class functions with Fourier expansions in the irreducible characters. Putting these together with the Diaconis-Graham inequalities, the four standard metrics are ordered and related by universal bounds, and each has a clean spectral formula. This is exactly the content that builds toward the analysis of partial rankings in [07.05.13] where metrics on cosets are defined by lifting through the quotient map, and appears again in [07.05.11] where distances between probability distributions on $S_n$ are measured by character sums via the Upper Bound Lemma. The bridge is between the combinatorial definition of each metric and its Fourier-analytic representation; the pattern generalises from bi-invariant metrics (Cayley) to right-invariant metrics (all four), and identifies the character table of $S_n$ as the universal conversion device between different notions of distance on permutations.

## Full proof set [Master]

**Proposition 1 (Hamming distance character formula).** For all $\pi \in S_n$:

$$
d_H(e, \pi) = n - 1 - \chi_{(n-1,1)}(\pi).
$$

*Proof.* The permutation representation $\mathbb{C}^n$ decomposes as $\mathbf{1} \oplus V^{(n-1,1)}$, where $\mathbf{1}$ is the identity representation and $V^{(n-1,1)}$ is the standard representation. The character of the permutation representation at $\pi$ is $\text{fix}(\pi)$, the number of fixed points. So $\text{fix}(\pi) = 1 + \chi_{(n-1,1)}(\pi)$. Since $d_H(e, \pi) = n - \text{fix}(\pi) = n - 1 - \chi_{(n-1,1)}(\pi)$. $\square$

**Proposition 2 (Expected Hamming distance).** Under the uniform distribution on $S_n$, $\mathbb{E}[d_H] = n - 1$.

*Proof.* $\mathbb{E}[d_H] = n - 1 - \mathbb{E}[\chi_{(n-1,1)}]$. The expected character value under the uniform distribution is $\frac{1}{n!}\sum_\pi \chi_{(n-1,1)}(\pi) = 0$ by the orthogonality of characters (the inner product of any non-identity character with the identity character is 0). Hence $\mathbb{E}[d_H] = n - 1$. $\square$

## Connections [Master]

- **Spectral analysis of permutation data [07.05.11].** The character-theoretic expressions for metrics are computed using the same Fourier transform on $S_n$ that decomposes ranking data into spectral components. The first-order spectral coefficient in [07.05.11] determines the Hamming distance between the empirical distribution and uniformity; the metric formulas of this unit provide the pointwise versions.

- **Random walk upper bound lemma [07.05.05].** The Upper Bound Lemma bounds total variation distance by a character sum, which is a spectral expression of the same type as the metric formulas. The mixing time analysis in [07.05.05] uses the character values on transpositions, which are the same values appearing in the Cayley and Hamming distance formulas.

- **Partially ranked data [07.05.13].** Metrics on $S_n$ extend to partial rankings via the coset structure $S_n/S_{n-k}$, where the induced metrics on cosets are defined by minimising the metric over coset representatives. The Gelfand pair structure in [07.05.13] ensures that these induced metrics retain clean character-theoretic expressions.

- **Symmetric group representation [07.05.01].** The character table of $S_n$, the hook-length formula for dimensions, and the Murnaghan-Nakayama rule for computing character values developed in [07.05.01] and [07.05.10] are the computational tools needed to evaluate the metric formulas for specific permutations.

## Historical & philosophical context [Master]

The systematic study of metrics on the symmetric group was initiated by Diaconis and Graham in their 1977 paper *Spearman's Footrule as a Measure of Disarray* [ref: TODO_REF DiaconisGraham1977], which established the inequalities relating the four standard metrics. Diaconis developed the character-theoretic expressions in his 1988 monograph *Group Representations in Probability and Statistics* [ref: TODO_REF Diaconis1988], showing that the Fourier analysis on $S_n$ provides a unified framework for computing all standard metrics.

Critchlow 1985 *Metric Methods for Analyzing Partially Ranked Data* [ref: TODO_REF Critchlow1985] extended the metric framework to partial rankings and developed the induced metrics on cosets. The connection between the Kendall tau distance and the Coxeter length function of $S_n$ was observed in the combinatorics literature by Björner and Brenti 2005 in *Combinatorics of Coxeter Groups*.
