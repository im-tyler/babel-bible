## Intuition [Beginner]

Some quantities you care about are *ratios*. Think of fuel efficiency, miles per gallon: distance divided by fuel. Or a company's return on capital: profit divided by money invested. Or the cost per useful part produced. When you optimize a ratio, the thing you are minimizing or maximizing is not a simple bowl shape, so the convex-optimization toolkit does not apply directly. But these problems still have a friendly hidden structure, and this unit is about exploiting it.

The friendly structure is this: even though the ratio itself is not bowl-shaped, the question "can the ratio be brought below the number $t$?" is an easy, convex question for each fixed $t$. Asking whether efficiency can beat a target, or whether cost-per-part can be pushed under a threshold, turns into a standard convex check. So instead of attacking the ratio head-on, you guess a target value and ask the easy yes/no question.

Now comes the search idea. Suppose you ask "can the cost-per-part be below 10?" and the answer is yes. Then you ask "below 5?" If yes again, try lower; if no, the true best is between 5 and 10. Each yes-or-no answer cuts the range of possible best values in half. This halving search is called *bisection*, the same move you use to guess a number someone is thinking of between 1 and 100 by always splitting the gap.

Because each question halves the uncertainty, you close in fast: ten questions shrink the range by a factor of about a thousand, twenty questions by about a million. You never solve the hard ratio problem; you solve a short sequence of easy convex problems and let the halving do the work. That trade — replace one hard problem by a handful of easy ones — is the whole method, and ratios of linear quantities are its cleanest home, where the easy question is just a linear program.

## Visual [Beginner]

*Figure: a horizontal axis of candidate optimal values running from a lower guess l to an upper guess u. The true optimum p-star sits somewhere between. At each step a midpoint is tested with a convex feasibility check; a "feasible" answer pulls the upper end down to the midpoint, an "infeasible" answer pushes the lower end up to the midpoint. The bracket containing p-star halves every step until it is narrower than the tolerance.*

```
  step 0:   l |--------------------*--------------| u      test midpoint m0
                                    p*
  step 1:   l |----------*---------| u   (m0 feasible -> u = m0)
  step 2:        l |-----*----| u        (m1 infeasible -> l = m1)
  step 3:           l |--*--| u          bracket halves each step
                        p*
            ...                           stop when (u - l) < epsilon
            width after k steps = (u0 - l0) / 2^k
```

## Worked example [Beginner]

Maximize the ratio $\dfrac{x + 2}{x + 1}$ for $x$ between $0$ and $4$. We will find the best value by the halving search, treating "can the ratio be at least $t$?" as the easy question.

**Step 1. Turn the ratio question into a plain one.** The denominator $x + 1$ is positive on $[0,4]$, so "$\dfrac{x+2}{x+1} \ge t$" is the same as "$x + 2 \ge t(x+1)$", which rearranges to "$2 - t \ge (t - 1)x$". For each fixed $t$ this is a plain linear test on $x$.

**Step 2. Bracket the answer.** At $x = 0$ the ratio is $2/1 = 2$; at $x = 4$ it is $6/5 = 1.2$. The best value lies somewhere at or above $2$, so start with a lower guess $l = 1$ and an upper guess $u = 3$.

**Step 3. Test the midpoint $t = 2$.** The question becomes "$2 - 2 \ge (2-1)x$", i.e. "$0 \ge x$". The only $x$ in range that works is $x = 0$. So a value of $2$ is achievable: pull the upper guess down, $u = 2$.

**Step 4. Test the new midpoint $t = 1.5$.** Now "$2 - 1.5 \ge (1.5 - 1)x$", i.e. "$0.5 \ge 0.5x$", so $x \le 1$. Values up to $1$ work, so $1.5$ is achievable too; but $2$ was already achievable and is larger.

**What this tells us.** The ratio is largest at $x = 0$, where it equals $2$, and the halving search keeps confirming targets at or below $2$ while ruling out anything above. We never maximized the ratio directly — each step was a one-line linear check, and bisection steered us to the answer $2$.

## Formal definition [Intermediate+]

Throughout, the ambient space is $\mathbb{R}^n$ with the standard inner product $\langle a, x\rangle = \sum_{i=1}^n a_i x_i$. Quasiconvexity is recalled from [44.01.01]: a function $f : \mathbb{R}^n \to \mathbb{R}$ with convex domain is **quasiconvex** if every sublevel set $S_t = \{x \in \operatorname{dom} f : f(x) \le t\}$ is convex, equivalently $f(\theta x + (1-\theta)y) \le \max\{f(x), f(y)\}$ for $\theta \in [0,1]$. It is **quasiconcave** if $-f$ is quasiconvex, and **quasilinear** if both.

**Definition (quasiconvex program).** A **quasiconvex optimization problem** in standard form is

$$
p^\star = \inf_x\; f_0(x) \quad \text{subject to}\quad f_i(x) \le 0\ (i = 1,\dots,m),\quad Ax = b,
$$

where the objective $f_0$ is quasiconvex and the inequality functions $f_1,\dots,f_m$ are convex. The feasible set $C = \{x : f_i(x) \le 0,\ Ax = b\}$ is convex, but $f_0$ need not be, so the problem is not convex; what survives is that each sublevel set of $f_0$ is convex.

**Definition (sublevel representation by a convex family).** A function $f_0$ admits a **representation by a monotone family of convex inequalities** if there is a family of functions $\phi_t : \mathbb{R}^n \to \mathbb{R}$, indexed by $t \in \mathbb{R}$, each convex in $x$, such that

$$
\boxed{\; f_0(x) \le t \;\iff\; \phi_t(x) \le 0, \qquad \text{and}\qquad t \le u \implies \phi_t(x) \ge \phi_u(x)\ \text{for all } x. \;}
$$

The first condition says the $0$-sublevel set of $\phi_t$ is exactly the $t$-sublevel set $S_t$ of $f_0$; the second (monotone nonincreasing in $t$) makes the encoding consistent as $t$ varies. Every quasiconvex $f_0$ admits such a family — for instance $\phi_t(x) = 0$ when $f_0(x) \le t$ and $\phi_t(x) = +\infty$ otherwise, whose $0$-sublevel set is $S_t$ — but useful representations use finite, explicitly convex $\phi_t$ (often affine or convex-quadratic in $x$, affine in $t$) [ref: TODO_REF Boyd & Vandenberghe §4.2.5].

**Definition (convex feasibility subproblem).** For a fixed value $t$, the **feasibility problem** $\mathcal{F}(t)$ asks whether the convex set

$$
\{x : \phi_t(x) \le 0,\ f_i(x) \le 0\ (\forall i),\ Ax = b\}
$$

is nonempty. If it is, then $p^\star \le t$ (a feasible point achieves objective $\le t$); if it is empty, then $p^\star > t$ (with $\phi_t$ monotone, no feasible point reaches value $t$). $\mathcal{F}(t)$ is a convex feasibility problem because all the defining inequalities are convex.

**Definition (linear-fractional program).** A **linear-fractional program** (LFP) minimizes a ratio of affine functions over a polyhedron,

$$
\inf_x\; \frac{c^\top x + d}{e^\top x + f} \quad \text{subject to}\quad Gx \preceq h,\ Ax = b, \quad \text{on the domain } e^\top x + f > 0.
$$

The objective is quasiconvex (and quasiconcave, hence quasilinear) on the open halfspace $\{x : e^\top x + f > 0\}$, since each sublevel set $\{(c^\top x + d)/(e^\top x + f) \le t\} = \{c^\top x + d \le t(e^\top x + f)\}$ is a halfspace there. A **generalized linear-fractional program** replaces the single ratio by a worst-case maximum $\max_{i=1,\dots,K} (c_i^\top x + d_i)/(e_i^\top x + f_i)$ over a common positivity region $e_i^\top x + f_i > 0$, which is again quasiconvex as a pointwise maximum of quasiconvex functions [ref: TODO_REF Boyd & Vandenberghe §4.3.2].

## Key theorem with proof [Intermediate+]

The central result establishes that bisection solves a quasiconvex program to any prescribed accuracy in a number of convex feasibility solves logarithmic in the initial bracket width, and that quasiconvexity is exactly what makes the bisection logic sound (each feasibility answer is conclusive on one side).

**Theorem (bisection for quasiconvex optimization).** *Let $f_0$ be quasiconvex with a convex feasibility representation $\phi_t$ as above, let the feasible set $C$ be convex, and suppose $p^\star \in [l_0, u_0]$ with $l_0 < u_0$ initially bracketing the optimal value. Run bisection: at each step set $t = \tfrac12(l + u)$, solve $\mathcal{F}(t)$, set $u \leftarrow t$ if feasible and $l \leftarrow t$ if infeasible. Then after $k$ iterations $u - l = (u_0 - l_0)/2^k$, and $p^\star \in [l, u]$ throughout. Consequently, to guarantee $u - l \le \varepsilon$ it suffices to perform*

$$
k = \left\lceil \log_2 \frac{u_0 - l_0}{\varepsilon} \right\rceil
$$

*iterations, each a single convex feasibility solve.*

**Proof.** *Invariant.* We show $l \le p^\star \le u$ holds before and after every step. It holds at step $0$ by hypothesis. Suppose it holds with current bracket $[l, u]$ and let $t = \tfrac12(l+u)$.

If $\mathcal{F}(t)$ is feasible, some $x$ satisfies $\phi_t(x) \le 0$ together with the constraints, so by the representation $f_0(x) \le t$ and $x \in C$; hence $p^\star \le f_0(x) \le t$. Updating $u \leftarrow t$ preserves $p^\star \le u$, and $l$ is unchanged so $l \le p^\star$ still holds.

If $\mathcal{F}(t)$ is infeasible, no feasible point has $\phi_t(x) \le 0$, i.e. no $x \in C$ has $f_0(x) \le t$; therefore $p^\star > t$. Here quasiconvexity is what guarantees the answer is conclusive: $S_t \cap C$ is the intersection of the convex sublevel set $S_t$ with the convex set $C$, so it is convex, and a convex feasibility solver returns a definite empty/nonempty verdict rather than getting trapped in a nonconvex component. Updating $l \leftarrow t$ preserves $l \le p^\star$, and $u$ is unchanged so $p^\star \le u$ still holds. In both branches the invariant survives.

*Geometric width decay.* Each branch replaces one endpoint by the midpoint, so the new width is exactly half: $u' - l' = \tfrac12(u - l)$. By induction $u_k - l_k = (u_0 - l_0)/2^k$.

*Iteration count.* The width drops to $\le \varepsilon$ as soon as $(u_0 - l_0)/2^k \le \varepsilon$, i.e. $2^k \ge (u_0 - l_0)/\varepsilon$, i.e. $k \ge \log_2\!\big((u_0 - l_0)/\varepsilon\big)$. The least integer $k$ meeting this is $\lceil \log_2((u_0 - l_0)/\varepsilon)\rceil$. Since the invariant gives $p^\star \in [l, u]$ at termination, the midpoint $\tfrac12(l+u)$ estimates $p^\star$ within $\varepsilon/2$. $\square$

**Bridge.** This theorem **builds toward** the algorithmic core of every method that converts a nonconvex-looking objective into a sequence of convex solves, and **appears again in** the linear-fractional reductions below and in the parametric (Dinkelbach) view of fractional programming. **This is exactly** the device that **generalises** the elementary number-guessing game to optimization: the unknown is the scalar $p^\star$, each feasibility query is one comparison, and convexity of the sublevel sets is what makes the comparison return a side. **The foundational reason** bisection needs *quasi*convexity and not full convexity is that only the sublevel sets, not the epigraph, must be convex — a strictly weaker demand identified in [44.01.01] — which widens the reach of convex feasibility solvers to ratios and worst-case ratios. **Putting these together**, the convex feasibility subproblem $\mathcal{F}(t)$ **is dual to** the value question through the duality theory of [44.02.01]: an infeasibility certificate for $\mathcal{F}(t)$ is a dual proof that $p^\star > t$, so each bisection step is backed by a verifiable certificate, and the bridge is the sublevel-set family $\phi_t$ through which a quasiconvex program becomes $\lceil\log_2((u_0-l_0)/\varepsilon)\rceil$ convex programs.

## Exercises [Intermediate+]

<aside class="exercise" data-difficulty="easy" data-type="symbolic" markdown="1">

**Exercise 1 (easy, symbolic).**

Show that for the linear-fractional objective $f_0(x) = (c^\top x + d)/(e^\top x + f)$ on the domain $e^\top x + f > 0$, the sublevel set $\{x : f_0(x) \le t\}$ is a halfspace, and write the affine representative $\phi_t$.

<details class="exercise__hint"><summary>Hint</summary>

Multiply through by the positive denominator; the inequality direction is preserved.

</details>

<details class="exercise__answer"><summary>Answer</summary>

Since $e^\top x + f > 0$, the inequality $(c^\top x + d)/(e^\top x + f) \le t$ is equivalent to $c^\top x + d \le t(e^\top x + f)$, i.e. $(c - t e)^\top x + (d - t f) \le 0$. So $\phi_t(x) = (c - t e)^\top x + (d - t f)$ is affine in $x$ (and affine in $t$), and its $0$-sublevel set is the halfspace $\{x : (c - te)^\top x \le tf - d\}$. As $t$ increases, the coefficient $-te$ and intercept $-tf$ shift the halfspace, and one checks $\phi_t(x)$ is nonincreasing in $t$ on the domain because $\partial \phi_t/\partial t = -(e^\top x + f) < 0$. Each feasibility subproblem is therefore an LP.

</details>

</aside>

<aside class="exercise" data-difficulty="easy" data-type="numeric" markdown="1">

**Exercise 2 (easy, numeric).**

A quasiconvex program is bracketed in $[l_0, u_0] = [0, 100]$ and a tolerance $\varepsilon = 0.1$ is required. How many bisection (feasibility) solves does the iteration-count formula prescribe?

<details class="exercise__hint"><summary>Hint</summary>

Compute $\lceil \log_2(100/0.1)\rceil = \lceil \log_2 1000\rceil$.

</details>

<details class="exercise__answer"><summary>Answer</summary>

$\log_2 1000 = \log_2 10^3 = 3\log_2 10 \approx 3 \times 3.3219 = 9.966$, so $\lceil 9.966 \rceil = 10$. Ten convex feasibility solves bring the bracket below $0.1$. This is the practical message of the logarithmic bound: a thousand-fold accuracy improvement costs only ten convex solves.

</details>

</aside>

<aside class="exercise" data-difficulty="medium" data-type="symbolic" markdown="1">

**Exercise 3 (medium, symbolic).**

Carry out the **Charnes-Cooper transformation**: show that the LFP $\min (c^\top x + d)/(e^\top x + f)$ subject to $Gx \preceq h$, $e^\top x + f > 0$ becomes the LP $\min c^\top y + d z$ subject to $Gy \preceq h z$, $e^\top y + f z = 1$, $z \ge 0$, under the substitution $y = x/(e^\top x + f)$, $z = 1/(e^\top x + f)$.

<details class="exercise__hint"><summary>Hint</summary>

Scale every constraint by the positive quantity $z = 1/(e^\top x + f)$ and use $x = y/z$.

</details>

<details class="exercise__answer"><summary>Answer</summary>

Set $z = 1/(e^\top x + f) > 0$ and $y = z x$. Then $e^\top y + f z = z(e^\top x + f) = 1$, the normalization. The objective becomes $(c^\top x + d)/(e^\top x + f) = z(c^\top x + d) = c^\top(zx) + dz = c^\top y + dz$. The polyhedral constraint $Gx \preceq h$ multiplied by $z > 0$ gives $G(zx) \preceq hz$, i.e. $Gy \preceq hz$. Conversely, any $(y, z)$ feasible with $z > 0$ recovers $x = y/z$ feasible for the original with the same objective value. The case $z = 0$ corresponds to recession directions of the feasible polyhedron and is admitted as a limit (or excluded when the denominator is bounded away from $0$). Thus the LFP and LP have equal optimal values, and the ratio objective is solved by a single LP — no bisection needed in the linear-fractional case.

</details>

</aside>

<aside class="exercise" data-difficulty="medium" data-type="symbolic" markdown="1">

**Exercise 4 (medium, symbolic).**

Prove that the pointwise maximum $g(x) = \max_{i=1,\dots,K} f_i(x)$ of quasiconvex functions $f_i$ is quasiconvex, and deduce that a generalized linear-fractional objective is quasiconvex.

<details class="exercise__hint"><summary>Hint</summary>

A sublevel set of a maximum is an intersection of sublevel sets.

</details>

<details class="exercise__answer"><summary>Answer</summary>

For any $t$, $\{x : g(x) \le t\} = \{x : \max_i f_i(x) \le t\} = \bigcap_{i=1}^K \{x : f_i(x) \le t\}$, an intersection of the convex sublevel sets of the quasiconvex $f_i$, hence convex (intersection of convex sets is convex, [44.01.01]). Convex sublevel sets at every level give quasiconvexity of $g$. For the generalized LFP, each $f_i(x) = (c_i^\top x + d_i)/(e_i^\top x + f_i)$ is quasiconvex on the common region $\{e_i^\top x + f_i > 0\ \forall i\}$ (Exercise 1), so their maximum is quasiconvex; the feasibility subproblem at level $t$ is the LP $\{(c_i - t e_i)^\top x + (d_i - t f_i) \le 0\ \forall i\} \cap \{Gx \preceq h\}$, again solvable by bisection over LPs.

</details>

</aside>

<aside class="exercise" data-difficulty="medium" data-type="symbolic" markdown="1">

**Exercise 5 (medium, symbolic).**

Show that the sum of two ratios $\dfrac{c_1^\top x + d_1}{e_1^\top x + f_1} + \dfrac{c_2^\top x + d_2}{e_2^\top x + f_2}$ need not be quasiconvex, and explain why bisection does not directly apply to sum-of-ratios objectives.

<details class="exercise__hint"><summary>Hint</summary>

A sublevel set of a sum is not an intersection of sublevel sets; try a one-dimensional example whose sublevel set is disconnected.

</details>

<details class="exercise__answer"><summary>Answer</summary>

Take $h(x) = \dfrac{x}{1} + \dfrac{1}{x}$ on $x > 0$ (each term linear-fractional with positive denominator). Then $h(x) = x + 1/x$ has $h(1) = 2$, $h(2) = 2.5$, $h(1/2) = 2.5$, and $h \to \infty$ at both ends; its sublevel set $\{x > 0 : h(x) \le 2.4\}$ is two disjoint intervals around the two points where $h = 2.4$, hence not convex. So $h$ is not quasiconvex even though each summand is. Because the sublevel sets are nonconvex, the feasibility subproblem $\mathcal{F}(t)$ is no longer convex, and the bisection invariant (a conclusive empty/nonempty verdict) breaks; sum-of-ratios is in fact NP-hard in general and needs branch-and-bound or specialized global methods, not plain bisection.

</details>

</aside>

<details class="exercise__hint"><summary>Hint</summary>

$p(x)/q(x) \le t$ is equivalent to $p(x) - t q(x) \le 0$ when $q(x) > 0$. Relate the sign of $F(t)$ to whether $t$ over- or under-estimates $t^\star$.

</details>

<details class="exercise__answer"><summary>Answer</summary>

For each $x \in C$, $p(x) - tq(x)$ is affine and strictly decreasing in $t$ (slope $-q(x) < 0$); $F$ is a pointwise minimum of strictly decreasing affine functions, hence nonincreasing (in fact concave) in $t$. If $t < t^\star$ then for every $x$, $p(x)/q(x) \ge t^\star > t$, so $p(x) - tq(x) > 0$, giving $F(t) > 0$. If $t > t^\star$, the minimizer $x^\star$ of the ratio gives $p(x^\star)/q(x^\star) = t^\star < t$, so $p(x^\star) - t q(x^\star) < 0$ and $F(t) < 0$. At $t = t^\star$, $p(x) - t^\star q(x) \ge 0$ for all $x$ with equality at $x^\star$, so $F(t^\star) = 0$. Monotonicity makes the root unique, and $F(t^\star) = 0$ is the Dinkelbach equation; Newton's method on $F$ converges superlinearly, an alternative to bisection that uses the convex *value* rather than just feasibility.

</details>

</aside>

<aside class="exercise" data-difficulty="hard" data-type="symbolic" markdown="1">

**Exercise 7 (hard, symbolic).**

Show that an infeasibility certificate for the convex subproblem $\mathcal{F}(t)$ — produced by Lagrangian duality [44.02.01] — proves $p^\star > t$, and explain how this turns each bisection step into a verifiable certificate rather than a black-box answer.

<details class="exercise__hint"><summary>Hint</summary>

A separating hyperplane / Farkas-type multiplier exhibiting that $\{\phi_t(x) \le 0\} \cap C = \varnothing$ is a dual object; relate it to $p^\star$.

</details>

<details class="exercise__answer"><summary>Answer</summary>

$\mathcal{F}(t)$ is the convex system $\phi_t(x) \le 0$, $f_i(x) \le 0$, $Ax = b$. When it is infeasible, a theorem of the alternative (the convex Farkas / Lagrangian-duality result of [44.02.01]) supplies multipliers $\mu \ge 0$, $\lambda \succeq 0$, $\nu$, not all zero, with $\mu\,\phi_t(x) + \sum_i \lambda_i f_i(x) + \nu^\top(Ax - b) > 0$ for all $x$ in the domain — a single inequality certifying emptiness. Because $\phi_t(x) \le 0 \iff f_0(x) \le t$, emptiness of the system means no feasible $x$ attains $f_0(x) \le t$, i.e. $p^\star > t$. The multipliers are a short, independently checkable proof of that strict bound, so a verifier need not re-solve $\mathcal{F}(t)$; it only checks the certificate inequality. Each bisection step thus narrows $[l, u]$ with a dual proof attached, which is what makes bisection trustworthy as a building block in certified optimization.

</details>

</aside>

**Exercise 8 (hard, symbolic).**

Consider the worst-case fractional problem $\min_x \max_i \frac{c_i^\top x + d_i}{e_i^\top x + f_i}$ over $Gx \preceq h$ with all $e_i^\top x + f_i > 0$. Write the feasibility subproblem $\mathcal{F}(t)$ explicitly, confirm it is an LP, and state the resulting overall complexity in terms of LP solves.

<details class="exercise__hint"><summary>Hint</summary>

Use Exercise 1 on each ratio at level $t$; intersect the resulting halfspaces with the polyhedron.

</details>

<details class="exercise__answer"><summary>Answer</summary>

At level $t$, the objective is $\le t$ iff every ratio is $\le t$ iff $(c_i - t e_i)^\top x + (d_i - t f_i) \le 0$ for all $i$. So $\mathcal{F}(t)$ asks whether the polyhedron

$$
\{x : (c_i - t e_i)^\top x \le t f_i - d_i\ (i = 1,\dots,K),\ Gx \preceq h\}
$$

is nonempty — a linear feasibility problem (an LP with a dummy objective). Each step is one LP; bisection to tolerance $\varepsilon$ over an initial bracket $[l_0, u_0]$ takes $\lceil \log_2((u_0 - l_0)/\varepsilon)\rceil$ LP solves. Each LP has $K$ fractional-derived constraints plus the rows of $G$, and the positivity hypotheses $e_i^\top x + f_i > 0$ are maintained on the domain. The total cost is logarithmic in the accuracy and polynomial per step, so the worst-case fractional program is solved in polynomial time despite its nonconvex-looking objective.

## Advanced results [Master]

### The representation theorem and the geometry of quasiconvex sublevel families

A function is quasiconvex if and only if it admits a family $\{\phi_t\}$ of convex functions, nonincreasing in $t$, with $\{x : \phi_t(x) \le 0\} = S_t$ for every $t$. The forward direction packages the convex sublevel sets as the indicator-style family $\phi_t = \iota_{S_t}$ relaxed to a finite convex gauge; the converse reads convexity of every $S_t = \{\phi_t \le 0\}$ directly off convexity of each $\phi_t$, and quasiconvexity follows from convex sublevel sets at all levels. The content of the theorem is constructive: for the algorithmically interesting classes the $\phi_t$ are *jointly* structured — affine in $x$ for linear-fractional objectives, convex-quadratic for quadratic-over-affine objectives, linear matrix inequalities for the largest-generalized-eigenvalue objective $\lambda_{\max}(A(x), B(x))$ with $B(x) \succ 0$. The bisection method runs over whatever convex class the $\phi_t$ inhabit, so the *same* logarithmic outer loop solves a quasiconvex program over LPs, QPs, SOCPs, or SDPs depending only on the representation, which is why quasiconvex modeling sits one layer above the conic hierarchy of [44.05.01].

### First-order and differentiable characterizations of quasiconvexity

For differentiable $f_0$ on a convex open set, quasiconvexity is equivalent to the first-order condition $f_0(y) \le f_0(x) \implies \langle \nabla f_0(x), y - x\rangle \le 0$: along any direction in which the function does not increase the gradient cannot point uphill. This is strictly weaker than the convex supporting-line inequality of [44.01.01] and admits stationary points that are not global minimizers, so an unconstrained gradient zero is not by itself decisive — the reason bisection, which interrogates sublevel sets globally, is preferred to gradient descent for genuinely quasiconvex objectives. The twice-differentiable refinement involves the *bordered Hessian*: $f_0$ is quasiconvex on a convex set if, restricted to the subspace $\{v : \langle \nabla f_0(x), v\rangle = 0\}$, the Hessian is positive semidefinite, encoded by sign conditions on the leading principal minors of the matrix $\begin{psmallmatrix} 0 & \nabla f_0^\top \\ \nabla f_0 & \nabla^2 f_0 \end{psmallmatrix}$. These conditions are the generalized-concavity analogue of the PSD-Hessian test and are the working criteria in [ref: TODO_REF Avriel, Diewert, Schaible & Zang Ch. 3].

### Fractional programming: Charnes-Cooper, Dinkelbach, and the conic generalizations

Linear-fractional programs collapse entirely to LPs by the Charnes-Cooper change of variables [ref: TODO_REF Charnes & Cooper 1962], so they need no bisection; the homogenization $y = x/(e^\top x + f)$, $z = 1/(e^\top x + f)$ is the projective trick that linearizes a single ratio over a polyhedron. For nonlinear ratios $p(x)/q(x)$ with $p$ convex, $q$ concave, and both positive on a convex feasible set, the objective is quasiconvex, and two routes coexist: bisection on the optimal value, solving a convex feasibility problem per step, and the Dinkelbach parametric method [ref: TODO_REF Dinkelbach 1967], solving the convex *optimization* $F(t) = \min_x(p(x) - t q(x))$ and updating $t$ by Newton's method on the root equation $F(t) = 0$, which converges superlinearly versus bisection's linear (one-bit-per-step) rate. The generalized fractional program $\min_x \max_i p_i(x)/q_i(x)$ retains quasiconvexity and is handled by a Dinkelbach-type method on $\max_i(p_i - t q_i)$; its conic incarnations — the linear-fractional SOCP and the generalized-eigenvalue SDP $\min t$ s.t. $A(x) \preceq t B(x)$, $B(x) \succ 0$ — are exactly the bisection-over-conic-feasibility programs that modeling systems compile quasiconvex ratios into.

### Quasiconvexity that is not convexity: where the extra generality buys problems

The value of the quasiconvex layer is that it captures objects no convex reformulation reaches without lifting. The condition number $\lambda_{\max}(X)/\lambda_{\min}(X)$ of a positive-definite matrix depending affinely on $x$, the largest generalized eigenvalue, the maximum-of-ratios scheduling objectives, and the Von Neumann growth-rate model of an expanding economy are quasiconvex (or quasiconcave) without being convex; each becomes a sequence of SDP or LP feasibility checks under bisection. The price is that quasiconvexity does not compose the way convexity does: sums of quasiconvex functions are generally not quasiconvex (Exercise 5), so there is no quasiconvex analogue of the disciplined-convex-programming additive calculus of [44.01.01]. Quasiconvex modeling is therefore organized around a fixed list of recognized quasiconvex *atoms* combined only by the operations that preserve quasiconvexity — pointwise maximum, composition with a monotone map, and the ratio and perspective constructions — rather than by free addition.

**Synthesis.** The sublevel-family representation **is exactly** the device that **generalises** the epigraph encoding of [44.01.01] from convex to quasiconvex objectives: where a convex function is its convex epigraph, a quasiconvex function is its nested family of convex sublevel sets, and **the central insight** of this unit is that optimization over that family is bisection on a single scalar. **The foundational reason** the method works with only quasiconvexity is that bisection never needs the objective's epigraph to be convex — it needs only that each level question $\mathcal{F}(t)$ be a convex feasibility problem, which the monotone family $\phi_t$ guarantees, and this **is dual to** the duality theory of [44.02.01] in that each infeasible level supplies a certificate $p^\star > t$. **Putting these together**, the Charnes-Cooper transformation linearizes the single-ratio case to one LP, the Dinkelbach parametric root-finding trades bisection's linear convergence for Newton's superlinear convergence on the convex value function $F(t)$, and the generalized-eigenvalue and worst-case-ratio problems lift the whole construction onto the self-dual cones of [44.05.01]; **the bridge is** the monotone convex family $\phi_t$, the single object through which a nonconvex ratio objective becomes a logarithmic sequence of convex programs, and it **appears again in** the central-path homotopy of interior-point methods and in every parametric optimization scheme that solves a hard problem by sweeping a scalar through convex subproblems.

## Full proof set [Master]

**Proposition 1 (correctness and complexity of bisection).** *Under the hypotheses of the Key theorem, bisection terminates after $\lceil \log_2((u_0 - l_0)/\varepsilon)\rceil$ convex feasibility solves with a bracket $[l, u]$ of width $\le \varepsilon$ containing $p^\star$; the midpoint estimates $p^\star$ within $\varepsilon/2$.*

*Proof.* The bracket invariant $l \le p^\star \le u$ and the exact halving $u_k - l_k = (u_0 - l_0)/2^k$ are established in the Key theorem. The width meets $\le \varepsilon$ at the least $k$ with $2^k \ge (u_0 - l_0)/\varepsilon$, namely $k = \lceil \log_2((u_0 - l_0)/\varepsilon)\rceil$. With $p^\star \in [l, u]$ and $u - l \le \varepsilon$, the midpoint $m = \tfrac12(l + u)$ satisfies $|m - p^\star| \le \tfrac12(u - l) \le \varepsilon/2$. Each iteration solves exactly one $\mathcal{F}(t)$, a convex feasibility problem. $\square$

**Proposition 2 (quasiconvexity $\iff$ convex sublevel sets $\iff$ a monotone convex family).** *For $f_0 : \mathbb{R}^n \to \mathbb{R}$ with convex domain, the following are equivalent: (a) $f_0$ is quasiconvex; (b) every sublevel set $S_t$ is convex; (c) there is a family $\{\phi_t\}$, each convex in $x$ and nonincreasing in $t$, with $\{x : \phi_t(x) \le 0\} = S_t$ for all $t$.*

*Proof.* (a) $\Leftrightarrow$ (b) is the definition recalled from [44.01.01]: $f_0(\theta x + (1-\theta)y) \le \max\{f_0(x), f_0(y)\}$ holds for all $x, y, \theta$ if and only if each $S_t$ is convex. (b) $\Rightarrow$ (c): for each $t$ let $\phi_t = \iota_{S_t}$, the convex indicator that is $0$ on the convex set $S_t$ and $+\infty$ off it; its $0$-sublevel set is $S_t$, it is convex in $x$, and since $t \le u$ implies $S_t \subseteq S_u$ (sublevel sets nest upward) we have $\iota_{S_t} \ge \iota_{S_u}$, the required monotonicity. (c) $\Rightarrow$ (b): if $\phi_t$ is convex then $\{x : \phi_t(x) \le 0\}$ is a sublevel set of a convex function, hence convex; this set equals $S_t$, so each $S_t$ is convex. The indicator family proves existence; finite, explicitly convex $\phi_t$ exist for the structured classes and are what bisection actually consumes. $\square$

**Proposition 3 (Charnes-Cooper equivalence).** *Let the LFP be $\inf\{(c^\top x + d)/(e^\top x + f) : Gx \preceq h\}$ on $\{e^\top x + f > 0\}$, assumed nonempty with the denominator bounded away from $0$ on the feasible set. Then it has the same optimal value as the LP $\inf\{c^\top y + dz : Gy \preceq hz,\ e^\top y + fz = 1,\ z \ge 0\}$, and optimal solutions correspond via $y = zx$, $z = 1/(e^\top x + f)$.*

*Proof.* ($\le$) Given feasible $x$ for the LFP, set $z = 1/(e^\top x + f) > 0$ and $y = zx$. Then $e^\top y + fz = z(e^\top x + f) = 1$ and $Gy = zGx \preceq zh = hz$ (using $z > 0$), so $(y, z)$ is LP-feasible with objective $c^\top y + dz = z(c^\top x + d) = (c^\top x + d)/(e^\top x + f)$, the LFP objective at $x$. Thus the LP optimum is $\le$ the LFP optimum. ($\ge$) Given LP-feasible $(y, z)$ with $z > 0$, set $x = y/z$. Then $e^\top x + f = (e^\top y + fz)/z = 1/z > 0$ and $Gx = Gy/z \preceq h$, so $x$ is LFP-feasible with objective $(c^\top x + d)/(e^\top x + f) = z(c^\top(y/z) + d) = c^\top y + dz$, the LP objective. So the LFP optimum is $\le$ the LP optimum over the $z > 0$ region. The boundedness-away-from-$0$ hypothesis ensures the LP optimum is attained with $z > 0$ rather than only in the $z \to 0^+$ recession limit, giving equality and the stated correspondence. $\square$

**Proposition 4 (Dinkelbach root equation and convergence).** *Let $C$ be convex compact, $p$ convex, $q$ concave with $q > 0$ on $C$, and $t^\star = \min_{x \in C} p(x)/q(x)$. With $F(t) = \min_{x \in C}(p(x) - t q(x))$: $F$ is concave, strictly decreasing, $F(t^\star) = 0$, and the Newton-Dinkelbach iteration $t_{k+1} = p(x_k)/q(x_k)$ with $x_k \in \arg\min_x(p(x) - t_k q(x))$ converges monotonically to $t^\star$.*

*Proof.* For fixed $x$, $t \mapsto p(x) - tq(x)$ is affine with slope $-q(x) < 0$; $F$ is a pointwise minimum of these, hence concave and (since every slope is $\le -\min_C q < 0$) strictly decreasing. Sign analysis as in Exercise 6 gives $F(t) > 0$ for $t < t^\star$, $F(t) < 0$ for $t > t^\star$, and $F(t^\star) = 0$ with the minimizing $x^\star$ attaining the ratio $t^\star$; uniqueness of the root follows from strict monotonicity. For convergence, start with $t_0 \le t^\star$ feasible-side; given $t_k$, the minimizer $x_k$ of $p - t_k q$ has $F(t_k) = p(x_k) - t_k q(x_k) \ge 0$, and the update $t_{k+1} = p(x_k)/q(x_k)$ satisfies $p(x_k) - t_{k+1} q(x_k) = 0 \ge F(t_{k+1})$, so $t_{k+1} \ge t^\star$ is impossible to overshoot below and the sequence increases toward $t^\star$. Concavity of $F$ makes the update exactly a Newton step on $F(t) = 0$ using a supergradient $-q(x_k) \in \partial F(t_k)$, yielding superlinear convergence; compactness of $C$ secures attainment of each inner minimum. $\square$

## Connections [Master]

- The quasiconvexity definition, the convex sublevel sets, and the convexity-preserving operations consumed throughout — pointwise maximum, affine precomposition, perspective, and the intersection of convex sets — are exactly the apparatus of [44.01.01]; this unit specializes that apparatus to objectives that are quasiconvex but not convex, where the sublevel-set view rather than the epigraph view is the operative one, and the linear-fractional sublevel halfspaces are the simplest nontrivial instance of its operation calculus.

- Each bisection feasibility subproblem $\mathcal{F}(t)$ is a convex program whose solvability, and whose infeasibility certificate proving $p^\star > t$, rest on the Lagrangian-duality and Slater theory of [44.02.01]; the dual multipliers turn a black-box feasibility answer into a checkable proof, so the bisection outer loop inherits the certified-optimality guarantees of convex duality at every step.

- The conic and semidefinite programs of [44.05.01] are the feasibility classes that bisection runs over when the family $\phi_t$ consists of linear matrix inequalities: the generalized-eigenvalue problem $\min t$ s.t. $A(x) \preceq t B(x)$, $B(x) \succ 0$ is the canonical quasiconvex-over-SDP program, and the Charnes-Cooper-linearized LFP is its polyhedral shadow, tying the quasiconvex outer loop to the self-dual cone hierarchy.

- The central-path homotopy of interior-point methods sweeps a scalar barrier parameter through a family of convex subproblems exactly as bisection sweeps the value $t$ through the feasibility family $\mathcal{F}(t)$; both are parametric reductions of a hard problem to a one-parameter path of convex programs, and the Dinkelbach Newton iteration on $F(t) = 0$ is the value-based analogue that interior-point predictor-corrector schemes generalize.

## Historical & philosophical context [Master]

Quasiconvexity emerged from mathematical economics, where indifference curves and the diminishing-marginal-rate-of-substitution principle demanded a class of functions weaker than concave but still guaranteeing convex upper-contour sets. Bruno de Finetti's 1949 study of "stratified" functions and Werner Fenchel's 1953 Princeton lectures isolated quasiconvex and quasiconcave functions as the functions with convex sublevel (respectively superlevel) sets, and the systematic theory of generalized concavity was consolidated by Mordecai Avriel, W. E. Diewert, Siegfried Schaible, and Israel Zang [ref: TODO_REF Avriel, Diewert, Schaible & Zang Ch. 3] (Plenum, 1988), which catalogued the first-order, bordered-Hessian, and operation-preserving characterizations.

The fractional-programming thread runs in parallel. Abraham Charnes and William Cooper [ref: TODO_REF Charnes & Cooper 1962] published the projective change of variables in *Naval Research Logistics Quarterly* (volume 9, 1962, 181-186) that reduces a linear-fractional functional over a polyhedron to a linear program, and Werner Dinkelbach [ref: TODO_REF Dinkelbach 1967] gave the parametric method in *Management Science* (volume 13, 1967, 492-498), recasting $\max p/q$ as the root of $F(t) = \max(p - tq)$ and thereby connecting fractional optimization to root-finding on a concave value function. The bisection-on-the-optimal-value method and its $\log_2((u-l)/\varepsilon)$ complexity were given their modern algorithmic packaging for convex modeling by Stephen Boyd and Lieven Vandenberghe [ref: TODO_REF Boyd & Vandenberghe §4.2.5] (Cambridge, 2004), where quasiconvex optimization via a sequence of convex feasibility problems became a standard reduction in the disciplined-convex-programming toolkit.
